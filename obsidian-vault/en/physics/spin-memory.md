# Spin Memory

## Theoretical Foundation
Spin memory is a sub-leading gravitational memory effect. While the standard (displacement) memory causes a permanent change in the position of test masses, the spin memory effect causes a permanent change in the relative *timing* or *rotation* of masses in a circular arrangement. It is associated with the transfer of angular momentum to null infinity by gravitational waves and is linked to the sub-leading soft graviton theorem.

## Mathematical Formulation
Spin memory is related to the "magnetic" part of the gravitational radiation. It involves the integral of the angular momentum flux. The change in the rotation $\Delta \phi$ is related to the integral of the news tensor and the Bondi mass aspect:
$$ \Delta \theta \sim \int du \int d\Omega \sin\theta \dots \mathcal{J}(u, \Omega) $$
where $\mathcal{J}$ is the angular momentum aspect. Mathematically, it is the Ward identity for superrotations in the [[bms-group]].

## Physical Significance & Information Theory
Spin memory provides a more refined probe of the gravitational vacuum than standard memory. It demonstrates how angular momentum information is "remembered" by spacetime. In the context of the [[black-hole-soft-hair]], spin memory corresponds to the "superrotation hair," suggesting that black holes can store information about the spin and orbital dynamics of infalling matter in the form of soft excitations that affect the rotational state of the horizon.

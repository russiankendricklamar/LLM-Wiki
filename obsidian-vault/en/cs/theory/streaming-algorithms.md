# Streaming Algorithms

[[Streaming Algorithms]] process massive data with sublinear memory. Techniques like Count-Min Sketch estimate frequencies, and HyperLogLog estimates distinct elements cardinality using probabilistic hashing in $O(\log(\log N))$ space.

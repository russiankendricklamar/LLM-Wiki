# Byzantine Agreement (PBFT)

Practical Byzantine Fault Tolerance ([[PBFT]]) provides a replicated state machine protocol capable of tolerating up to $f$ malicious nodes given $N = 3f+1$ total nodes. It requires pre-prepare, prepare, and commit phases.

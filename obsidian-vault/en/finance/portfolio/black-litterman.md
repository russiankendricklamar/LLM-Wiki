# Black-Litterman Model

The [[Black-Litterman Model]] computes optimal portfolio weights by combining market equilibrium implied returns (prior) with investor views. Expected returns are $E[R] = [(\tau \Sigma)^{-1} + P^T \Omega^{-1} P]^{-1}[(\tau \Sigma)^{-1}\Pi + P^T \Omega^{-1} Q]$.

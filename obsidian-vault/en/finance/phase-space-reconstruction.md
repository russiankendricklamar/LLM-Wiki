# Phase Space Reconstruction

**Phase Space Reconstruction** is a technique from non-linear dynamics and chaos theory that allows for the recovery of a multi-dimensional portrait of a system using only a single observed time series (e.g., stock prices). In quantitative finance, it is used to uncover deterministic chaos and build non-linear predictive models.

## Prerequisites

[[spectral-analysis]], [[hurst-exponent]]

## Background and Motivation

Classical time series analysis assumes linearity (ARMA, GARCH). Non-linear dynamical systems — even purely deterministic ones — can produce extremely complex, apparently random output. The question of whether financial markets contain any genuine non-linear deterministic structure (as opposed to stochastic noise) has been debated since Mandelbrot's fractal market hypothesis in the 1960s.

Phase space reconstruction provides a rigorous framework for this question. If a low-dimensional attractor exists in financial data, the method can recover it, enabling non-linear short-term forecasting. The technique was made practically accessible by Takens' embedding theorem (1981), which proved that a scalar observable contains enough information to reconstruct the topology of the underlying dynamical system.

## Takens' Embedding Theorem

The foundation of this method is **Takens' Embedding Theorem** (1981). It states that if a time series $x(t)$ is generated by a dynamic system with a $d$-dimensional attractor, it is possible to reconstruct a space that is topologically equivalent to the original phase space using the "method of delays," provided the embedding dimension $m \geq 2d + 1$.

## The Method of Delays

A reconstructed state vector $\mathbf{y}_t$ at time $t$ is formed as:

$$
\mathbf{y}_t = [x(t),\; x(t-\tau),\; x(t-2\tau),\; \dots,\; x(t-(m-1)\tau)]
$$

Where:
- $x(t)$ is the value of the time series at time $t$.
- $\tau$ is the **time delay** (lag).
- $m$ is the **embedding dimension**.

The resulting matrix of state vectors $\{\mathbf{y}_t\}$ lives in $\mathbb{R}^m$ and — if Takens' conditions are met — is topologically equivalent to the original attractor.

## Parameter Selection

Choosing the correct parameters is critical for an accurate reconstruction:

### Time Delay $\tau$: Mutual Information Method

The optimal $\tau$ minimizes redundancy between $x(t)$ and $x(t-\tau)$ while preserving independence. The **Average Mutual Information** (AMI) function is:

$$
I(\tau) = \sum_{x,y} p(x, x_\tau) \log\frac{p(x, x_\tau)}{p(x)\,p(x_\tau)}
$$

The optimal $\tau$ is the first local minimum of $I(\tau)$ — at this lag, the components carry complementary rather than redundant information.

### Embedding Dimension $m$: False Nearest Neighbors

The **False Nearest Neighbor (FNN)** algorithm determines the minimum $m$ needed to unfold the attractor:
1. For each point $\mathbf{y}_t$ in $m$ dimensions, find its nearest neighbor $\mathbf{y}_{t^*}$.
2. Embed in $m+1$ dimensions and check if the nearest-neighbor distance increased dramatically.
3. A large increase indicates the neighbors were "false" — only close due to projection from higher dimensions.
4. The optimal $m$ is where the FNN fraction drops below a threshold (typically 1–5%).

## Lyapunov Exponents and Chaos Detection

Once the phase space is reconstructed, the **largest Lyapunov exponent** $\lambda_1$ quantifies the rate of divergence of nearby trajectories:

$$
d(t) \approx d_0\, e^{\lambda_1 t}
$$

- $\lambda_1 > 0$: Chaotic dynamics (exponential divergence, finite prediction horizon).
- $\lambda_1 = 0$: Borderline (quasi-periodic, periodic, or Brownian).
- $\lambda_1 < 0$: Stable fixed point or limit cycle.

For a pure random walk (Brownian motion), $\lambda_1 \approx +\infty$ — divergence is immediate. A positive but finite $\lambda_1$ in financial data suggests deterministic chaos with a finite prediction horizon of order $1/\lambda_1$.

## Correlation Dimension

The **correlation dimension** $D_2$ measures the fractal dimension of the attractor:

$$
C(r) = \lim_{N\to\infty} \frac{2}{N(N-1)} \sum_{i<j} \mathbf{1}[\|\mathbf{y}_i - \mathbf{y}_j\| < r]
$$

$$
D_2 = \lim_{r\to 0} \frac{\log C(r)}{\log r}
$$

A finite, non-integer $D_2$ is evidence of a fractal (chaotic) attractor. For pure white noise, $D_2 \to \infty$ as $m$ increases.

## Financial Applications

- **Attractor Identification**: Visualizing "strange attractors" in market data to distinguish complex non-linear structures from pure random noise.
- **Non-linear Forecasting**: Using local approximation methods (finding "nearest neighbors" in the phase space) to make short-term predictions for chaotic series.
- **Chaos vs. Noise**: Estimating the correlation dimension and Lyapunov exponents to determine if a market series is truly random or follows a low-dimensional deterministic process.
- **Embedding as features**: Using the reconstructed vectors $\mathbf{y}_t$ as input features for machine learning models (LSTM, reservoir computing), capturing non-linear temporal structure.

## Python: Phase Space Reconstruction and FNN

```python
import numpy as np
from scipy.spatial import cKDTree


def mutual_information(series, max_lag=50, n_bins=20):
    """
    Compute average mutual information for lags 1 to max_lag.
    Returns array of AMI values.
    """
    ami = []
    for lag in range(1, max_lag + 1):
        x = series[:-lag]
        y = series[lag:]
        # Bin both variables
        xy = np.column_stack([x, y])
        hist2d, _, _ = np.histogram2d(x, y, bins=n_bins)
        hist_x, _ = np.histogram(x, bins=n_bins)
        hist_y, _ = np.histogram(y, bins=n_bins)

        px = hist_x / hist_x.sum()
        py = hist_y / hist_y.sum()
        pxy = hist2d / hist2d.sum()

        # Compute mutual information
        mi = 0.0
        for i in range(n_bins):
            for j in range(n_bins):
                if pxy[i, j] > 0 and px[i] > 0 and py[j] > 0:
                    mi += pxy[i, j] * np.log(pxy[i, j] / (px[i] * py[j]))
        ami.append(mi)

    return np.array(ami)


def reconstruct_phase_space(series, tau, m):
    """
    Creates an embedding matrix from a 1D time series.
    Returns matrix of shape (n_vectors, m).
    """
    n = len(series)
    rows = n - (m - 1) * tau
    if rows <= 0:
        return np.array([])

    reconstructed = np.zeros((rows, m))
    for i in range(m):
        reconstructed[:, i] = series[i * tau: i * tau + rows]

    return reconstructed


def false_nearest_neighbors(series, tau, max_m=10, threshold=15.0):
    """
    Compute FNN fraction for embedding dimensions 1 to max_m.
    threshold: ratio of distance increase that defines a "false" neighbor.
    Returns array of FNN fractions.
    """
    fnn_fractions = []

    for m in range(1, max_m):
        embedded_m = reconstruct_phase_space(series, tau, m)
        embedded_m1 = reconstruct_phase_space(series, tau, m + 1)
        n = min(len(embedded_m), len(embedded_m1))
        embedded_m = embedded_m[:n]
        embedded_m1 = embedded_m1[:n]

        tree = cKDTree(embedded_m)
        distances, indices = tree.query(embedded_m, k=2)  # k=2: self + nearest
        nn_dist_m = distances[:, 1]
        nn_idx = indices[:, 1]

        # Distance in m+1 dimension
        nn_dist_m1 = np.linalg.norm(
            embedded_m1 - embedded_m1[nn_idx], axis=1
        )

        # FNN criterion
        with np.errstate(divide='ignore', invalid='ignore'):
            ratio = np.abs(nn_dist_m1 - nn_dist_m) / (nn_dist_m + 1e-10)

        fnn_fraction = np.mean(ratio > threshold)
        fnn_fractions.append(fnn_fraction)

    return np.array(fnn_fractions)


# Example: Lorenz system (known chaotic attractor)
def lorenz(n=5000, dt=0.01, sigma=10, rho=28, beta=8/3):
    x, y, z = 0.1, 0.0, 0.0
    xs = []
    for _ in range(n):
        dx = sigma * (y - x)
        dy = x * (rho - z) - y
        dz = x * y - beta * z
        x += dx * dt
        y += dy * dt
        z += dz * dt
        xs.append(x)
    return np.array(xs)

series = lorenz(n=3000)

ami_vals = mutual_information(series, max_lag=30)
optimal_tau = np.argmin(ami_vals[:15]) + 1
print(f"Optimal tau (first AMI minimum): {optimal_tau}")

fnn = false_nearest_neighbors(series, tau=optimal_tau, max_m=8)
optimal_m = np.argmax(fnn < 0.05) + 1
print(f"Optimal embedding dimension (FNN < 5%): {optimal_m}")

embedding = reconstruct_phase_space(series, tau=optimal_tau, m=optimal_m)
print(f"Reconstructed phase space shape: {embedding.shape}")
```

## Parameter Table

| Symbol | Name | Description | Selection Method |
|:---|:---|:---|:---|
| $\tau$ | Time delay | Lag between components | First AMI minimum |
| $m$ | Embedding dimension | State vector length | FNN algorithm |
| $\lambda_1$ | Largest Lyapunov exponent | Chaos measure | Wolf algorithm |
| $D_2$ | Correlation dimension | Fractal attractor dimension | Grassberger-Procaccia |
| $N$ | Series length | Data requirement grows with $m$ | $\gg 10^m$ |

## Key Limitations

- **Noise Sensitivity**: High-frequency market noise can easily obscure the underlying geometric structure of the attractor. Pre-filtering with wavelet denoising is often necessary.
- **Data Requirements**: High embedding dimensions require exponentially larger datasets to provide sufficient density in the phase space.
- **Stationarity**: Takens' theorem assumes a stationary attractor. Financial markets are non-stationary, so results should be interpreted carefully.
- **Skepticism in finance**: Most rigorous studies find that apparent low-dimensional chaos in financial data disappears after correcting for non-stationarity and small sample biases.

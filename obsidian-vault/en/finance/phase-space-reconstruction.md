# Phase Space Reconstruction

**Phase Space Reconstruction** is a technique from non-linear dynamics and chaos theory that allows for the recovery of a multi-dimensional portrait of a system using only a single observed time series (e.g., stock prices). In quantitative finance, it is used to uncover deterministic chaos and build non-linear predictive models.

## Takens' Embedding Theorem

The foundation of this method is **Takens' Embedding Theorem** (1981). It states that if a time series $x(t)$ is generated by a dynamic system, it is possible to reconstruct a space that is topologically equivalent to the original, unobserved phase space of the system using the "method of delays."

## The Method of Delays

A reconstructed state vector $\mathbf{y}_t$ at time $t$ is formed as follows:

$$
\mathbf{y}_t = [x(t), x(t-\tau), x(t-2\tau), \dots, x(t-(m-1)\tau)]
$$

Where:
- $x(t)$ is the value of the time series at time $t$.
- $\tau$ is the **time delay**.
- $m$ is the **embedding dimension**.

## Parameter Selection

Choosing the correct parameters is critical for an accurate reconstruction:
1.  **Time Delay ($\tau$)**: Often selected as the first local minimum of the **Mutual Information** function to ensure the components of the vector are sufficiently independent.
2.  **Embedding Dimension ($m$)**: Determined using the **False Nearest Neighbors** algorithm. It must be large enough to "unfold" the attractor and eliminate self-intersections of the trajectory.

## Financial Applications

- **Attractor Identification**: Visualizing "strange attractors" in market data to distinguish complex non-linear structures from pure random noise.
- **Non-linear Forecasting**: Using local approximation methods (finding "nearest neighbors" in the phase space) to make short-term predictions for chaotic series.
- **Chaos vs. Noise**: Estimating the correlation dimension to determine if a market series is truly random or follows a low-dimensional deterministic process.

## Python: Creating Embedding Vectors

```python
import numpy as np

def reconstruct_phase_space(series, tau, m):
    """
    Creates an embedding matrix from a 1D time series.
    """
    n = len(series)
    rows = n - (m - 1) * tau
    if rows <= 0:
        return np.array([])
    
    reconstructed = np.zeros((rows, m))
    for i in range(m):
        reconstructed[:, i] = series[i * tau : i * tau + rows]
    
    return reconstructed

# Example: Reconstructing a noisy oscillation
t = np.linspace(0, 10, 500)
prices = np.sin(t) + np.random.normal(0, 0.1, 500)
ps_matrix = reconstruct_phase_space(prices, tau=5, m=3)
```

## Key Limitations

- **Noise Sensitivity**: High-frequency market noise can easily obscure the underlying geometric structure of the attractor.
- **Data Requirements**: High embedding dimensions require exponentially larger datasets to provide sufficient density in the phase space.

# Dynamic Programming (HJB)

The [[Hamilton-Jacobi-Bellman Equation]] (HJB) is a nonlinear PDE for the value function in continuous-time optimal control: $-\frac{\partial V}{\partial t} = \max_u \left( L(x,u) + \nabla V \cdot f(x,u) \right)$. It provides a sufficient condition for optimality.

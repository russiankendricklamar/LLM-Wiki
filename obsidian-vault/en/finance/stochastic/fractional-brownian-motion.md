# Fractional Brownian Motion

[[Fractional Brownian Motion]] (fBm) generalizations standard Brownian motion with a Hurst parameter $H \in (0,1)$. If $H > 0.5$, increments exhibit long memory (persistence); if $H < 0.5$, they are anti-persistent. It is neither a Markov process nor a semimartingale.

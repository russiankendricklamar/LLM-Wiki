# SHA-256 (Secure Hash Algorithm)

**SHA-256** is a cryptographic hash function that produces a 256-bit (32-byte) signature for any given input. It is a member of the SHA-2 family, designed by the NSA. SHA-256 is characterized by its one-way nature, collision resistance, and deterministic output.

## Theoretical Foundation

A cryptographic hash function must satisfy three main properties:
1. **Pre-image Resistance**: Given a hash $h$, it should be computationally infeasible to find $x$ such that $H(x) = h$.
2. **Second Pre-image Resistance**: Given an input $x$, it should be infeasible to find another input $x'$ such that $H(x) = H(x')$.
3. **Collision Resistance**: It should be infeasible to find any two distinct inputs $x$ and $y$ such that $H(x) = H(y)$.

SHA-256 uses the **Merkle–Damgård construction** with a one-way compression function based on the Davies–Meyer structure.

## Mathematical Formulation

The algorithm operates on 512-bit message blocks and 256-bit intermediate hash values. The main loop consists of 64 rounds of operations, including:
- **Logical functions**: $Ch(E, F, G)$, $Maj(A, B, C)$, $\Sigma_0(A)$, $\Sigma_1(E)$.
- **Constant additions**: Using 64 fractional parts of the cube roots of the first 64 prime numbers ($K_t$).

One round transformation:
$$ T_1 = h + \Sigma_1(e) + Ch(e, f, g) + K_t + w_t $$
$$ T_2 = \Sigma_0(a) + Maj(a, b, c) $$
Where $a, \dots, h$ are the 32-bit working variables.

## Practical Relevance in Finance

SHA-256 is the bedrock of modern digital security and decentralized finance:
- **Bitcoin & Blockchain**: SHA-256 is used for Proof-of-Work (mining) and for generating addresses. It ensures the integrity of the ledger.
- **Digital Signatures**: Used in combination with ECDSA to sign transactions and verify identities.
- **Data Integrity**: Financial institutions use SHA-256 to create "fingerprints" of sensitive documents and transaction logs to prevent tampering.
- **Message Authentication Codes (HMAC)**: Securing API communication between high-frequency trading (HFT) systems and exchanges.

## Python: SHA-256 via hashlib

```python
import hashlib

def generate_hash(data: str) -> str:
    # Encode the string to bytes
    encoded_data = data.encode('utf-8')
    
    # Create SHA-256 hash object
    sha256_hash = hashlib.sha256(encoded_data)
    
    # Return hex representation
    return sha256_hash.hexdigest()

# Example usage
print(f"Hash of 'FinTech': {generate_hash('FinTech')}")
```

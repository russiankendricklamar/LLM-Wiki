# Hidden Markov Models (HMM)

A **Hidden Markov Model (HMM)** is a statistical Markov model in which the system being modeled is assumed to be a Markov process with unobserved (hidden) states. HMMs are powerful tools for temporal data analysis.

In finance, HMMs are extensively used for **Market Regime Detection**. For example, the returns and volatility of a stock may depend on the hidden state of the economy (e.g., Bull Market vs. Bear Market), which is not directly observable but can be inferred from market prices.

## Core Components

An HMM is defined by three sets of parameters ($\lambda = (A, B, \pi)$):
1.  **Transition Matrix (A)**: The probabilities of moving from one hidden state to another $P(z_t | z_{t-1})$.
2.  **Emission Matrix (B)**: The probability of observing a specific value given a hidden state $P(x_t | z_t)$.
3.  **Initial Distribution ($\pi$)**: The probability distribution of the starting state.

## Markov Property

The process is "memoryless": the future state depends only on the current state and not on the sequence of events that preceded it:

$$
P(z_{t+1} | z_t, z_{t-1}, \dots, z_1) = P(z_{t+1} | z_t)
$$

## The Three Fundamental Problems of HMMs

1.  **Evaluation**: What is the probability that the observed sequence was generated by a given model? (Forward-Backward Algorithm).
2.  **Decoding**: What is the most likely sequence of hidden states for a given sequence of observations? (Viterbi Algorithm).
3.  **Learning**: How should the model parameters $\lambda$ be adjusted to maximize the probability of the observations? (Baum-Welch Algorithm).

## Python: Detecting Market Regimes

Using the `hmmlearn` library to identify two regimes (Low Volatility vs. High Volatility) in stock returns:

```python
from hmmlearn import hmm
import numpy as np

# Sample returns (observations)
returns = np.random.normal(0.001, 0.02, 1000).reshape(-1, 1)

# Define and fit a Gaussian HMM with 2 components
model = hmm.GaussianHMM(n_components=2, covariance_type="full", n_iter=100)
model.fit(returns)

# Predict the hidden states
hidden_states = model.predict(returns)

print("Transition Matrix:")
print(model.transmat_)
# States might be: 0 - Low Vol, 1 - High Vol (or vice versa)
```

## Trading Applications

- **Adaptive Strategies**: Adjusting technical indicators or strategy parameters based on the identified market regime.
- **Risk Management**: De-risking portfolios when the model detects a transition into a high-volatility or high-risk regime.
- **Trend Following**: Estimating the probability that a current market trend will persist in the next time step.

## Overview
A vector norm is a function that assigns a strictly positive length or size to each vector in a vector space.

## L-p Norms
The $L_p$ norm for a vector $\mathbf{x} \in \mathbb{R}^n$ is:
$$ \|\mathbf{x}\|_p = \left( \sum_{i=1}^n |x_i|^p \right)^{1/p} $$
- **$L_1$ Norm:** Manhattan distance.
- **$L_2$ Norm:** Euclidean distance.
- **$L_\infty$ Norm:** Maximum absolute value of components.

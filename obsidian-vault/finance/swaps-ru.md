# Виды свопов

**Своп** (Swap) — это производный финансовый инструмент, представляющий собой соглашение между двумя сторонами об обмене денежными потоками в будущем. В отличие от фьючерсов, свопы обычно являются внебиржевыми (OTC) контрактами.

## 1. Процентный своп (Interest Rate Swap, IRS)

Самый распространенный вид. Стороны обмениваются процентными платежами: одна сторона платит по **фиксированной ставке**, а другая — по **плавающей** (например, SOFR, EURIBOR или ключевая ставка ЦБ).

### Зачем это нужно?
- Компании с кредитом под плавающую ставку могут «зафиксировать» свои расходы, если ожидают роста ставок.
- Банки управляют разрывом между активами и пассивами (ALM).

## 2. Валютный своп (Currency Swap)

Обмен номиналами и процентными платежами в разных валютах.
- **Cross-Currency Swap**: Обмен фиксированной ставки в одной валюте на плавающую в другой.
- **FX Swap**: Краткосрочная операция, сочетающая спот-покупку и форвардную продажу валюты.

## 3. Кредитно-дефолтный своп (Credit Default Swap, CDS)

Страховка от дефолта эмитента. Покупатель CDS платит регулярную премию, а продавец обязуется выплатить компенсацию (номинал облигации), если произойдет кредитное событие (дефолт).

## 4. Своп на совокупный доход (Total Return Swap, TRS)

Одна сторона (получатель дохода) получает все доходы от актива (дивиденды + рост цены), выплачивая другой стороне (собственнику) фиксированную или плавающую ставку. Это позволяет получить экспозицию на актив, не владея им физически.

## 5. Акционерный своп (Equity Swap)

Обмен дохода по акциям или индексу (например, S&P 500) на процентную ставку. Используется хедж-фондами для управления портфелями.

## Математическая оценка (IRS)

Стоимость процентного свопа для плательщика фиксированной ставки (Fixed Payer) в момент $t$ определяется как разница между приведенной стоимостью плавающей и фиксированной ноги:

$$
V_{swap} = PV_{float} - PV_{fixed}
$$

Где:
- $PV_{fixed} = \sum_{i=1}^{n} K \cdot N \cdot \Delta_i \cdot D(t_i)$
- $K$ — фиксированная ставка.
- $N$ — номинал.
- $D(t_i)$ — дисконт-фактор.

## Python: Расчет фиксированной ноги свопа

```python
def calculate_fixed_leg(notional, rate, tenors, discount_factors):
    """
    Приведенная стоимость фиксированной ноги процентного свопа.
    """
    leg_pv = sum(notional * rate * delta * df 
                 for delta, df in zip(tenors, discount_factors))
    return leg_pv

# Пример: 1 год, полугодовые выплаты
notional = 10_000_000
rate = 0.05
deltas = [0.5, 0.5]
dfs = [0.975, 0.951] # Дисконт-факторы

pv = calculate_fixed_leg(notional, rate, deltas, dfs)
print(f"PV фиксированной ноги: ${pv:,.2f}")
```

## Сравнение основных типов

| Тип свопа | Что обмениваем | Основной риск |
| :--- | :--- | :--- |
| **IRS** | Процентные ставки | Процентный риск |
| **Currency** | Валюты + Ставки | Валютный + Процентный |
| **CDS** | Премия за страховку | Кредитный (дефолт) |
| **TRS** | Доход актива vs Ставка | Рыночный риск актива |
| **Commodity** | Цена сырья vs Ставка | Риск цен на сырье |
| **Variance** | Реализованная vs Вмененная волатильность | Риск волатильности |

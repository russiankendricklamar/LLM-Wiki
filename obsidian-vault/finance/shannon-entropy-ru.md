# Энтропия Шеннона для обнаружения режимов

**Энтропия Шеннона** — это мера неопределенности или информационной насыщенности случайной величины. В количественных финансах она используется для анализа структуры временных рядов и обнаружения смены рыночных режимов.

## Математическое определение

Для дискретной случайной величины $X$ с возможными значениями $\{x_1, \dots, x_n\}$ и вероятностями их возникновения $p(x_i)$, энтропия $H(X)$ рассчитывается как:

$$
H(X) = -\sum_{i=1}^{n} p(x_i) \log_b p(x_i)
$$

Где $b$ — основание логарифма (обычно 2 для бит или $e$ для нат).

## Применение для обнаружения режимов

Основная идея заключается в том, что разные рыночные состояния имеют разную степень «упорядоченности»:
1.  **Высокая энтропия**: Рынок ведет себя как случайное блуждание (эффективный рынок). Потоки информации хаотичны, предсказуемость минимальна.
2.  **Низкая энтропия**: В данных появляются скрытые закономерности или сильные тренды. Это часто предшествует смене режима или крупным рыночным движениям (информационный «сгусток»).

## Энтропия как индикатор «черных лебедей»

Исследования показывают, что перед крупными обвалами энтропия часто начинает снижаться, так как участники рынка начинают действовать более синхронно (стадное поведение), что уменьшает «разнообразие» состояний системы.

## Python: Расчет энтропии доходностей

```python
import numpy as np
from scipy.stats import entropy

def calculate_market_entropy(returns, bins=10):
    # Гистограмма доходностей как аппроксимация вероятностей
    hist, _ = np.histogram(returns, bins=bins, density=True)
    probs = hist / np.sum(hist)
    
    # Расчет энтропии Шеннона
    return entropy(probs, base=2)

# Пример: сравнение стабильного и волатильного периодов
stable_returns = np.random.normal(0, 0.01, 1000)
chaotic_returns = np.random.laplace(0, 0.02, 1000)

print(f"Энтропия (нормальный рынок): {calculate_market_entropy(stable_returns):.4f}")
print(f"Энтропия (хвостатый/хаотичный рынок): {calculate_market_entropy(chaotic_returns):.4f}")
```

## Дополнительные виды энтропии в финансах

- **Approximate Entropy (ApEn)**: Устойчива к шуму, хорошо работает на коротких рядах.
- **Sample Entropy (SampEn)**: Улучшенная версия ApEn, не учитывающая самосовпадения.
- **Transfer Entropy**: Используется для измерения направленного влияния (причинности) одного актива на другой.

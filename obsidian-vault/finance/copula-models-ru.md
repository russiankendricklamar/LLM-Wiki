# Копула-модели

**Копула** (от лат. *copulare* — соединять) — это математическая функция, которая позволяет объединить одномерные маржинальные распределения в одно многомерное распределение. В количественных финансах копулы используются для моделирования структуры зависимости между активами, отделяя индивидуальное поведение активов от их взаимосвязи.

## Теорема Скляра

Фундаментальной основой является теорема Скляра. Она утверждает, что любое многомерное распределение $F(x_1, \dots, x_n)$ может быть представлено через копулу $C$ и маржинальные функции распределения $F_i(x_i)$:

$$
F(x_1, \dots, x_n) = C(F_1(x_1), \dots, F_n(x_n))
$$

Если маржинальные распределения непрерывны, то копула $C$ единственна.

## Зачем нужны копулы?

В отличие от стандартного коэффициента корреляции Пирсона, копулы позволяют:
1.  **Моделировать нелинейные зависимости**.
2.  **Учитывать зависимость в «хвостах» (Tail Dependence)**: активы могут иметь слабую корреляцию в обычное время, но сильно падать одновременно во время кризиса.
3.  **Объединять разные типы распределений**: например, один актив может быть распределен нормально, а другой иметь распределение Стьюдента.

## Типы копул

1.  **Гауссова (нормальная) копула**: Не учитывает зависимость в хвостах. Стала печально известна после кризиса 2008 года (модель Ли).
2.  **Копула Стьюдента (t-копула)**: Имеет симметричную зависимость в хвостах.
3.  **Архимедовы копулы**:
    - **Clayton**: Сильная зависимость в левом хвосте (одновременные обвалы).
    - **Gumbel**: Сильная зависимость в правом хвосте (одновременные взлеты).

## Python: Пример с Clayton Copula

Демонстрация того, как копула Клейтона моделирует зависимость при низких значениях:

```python
import numpy as np
import matplotlib.pyplot as plt

def clayton_copula(u, v, theta):
    return (u**(-theta) + v**(-theta) - 1)**(-1/theta)

# Генерация зависимых случайных величин
theta = 2.0 # Параметр зависимости
n = 1000
u = np.random.uniform(0, 1, n)
t = np.random.uniform(0, 1, n)
v = (t**(-theta/(theta+1)) * (u**(-theta) - 1) + 1)**(-1/theta)

plt.scatter(u, v, alpha=0.5)
plt.title(f"Clayton Copula (theta={theta})")
plt.xlabel("Asset A (CDF)")
plt.ylabel("Asset B (CDF)")
plt.show()
```

## Сравнение моделей

| Копула | Хвосты | Особенности |
| :--- | :---: | :--- |
| **Гауссова** | Нет | Легко считать, но опасно в кризис |
| **t-копула** | Оба | Подходит для волатильных рынков |
| **Clayton** | Левый | Идеальна для оценки рисков обвалов |
| **Gumbel** | Правый | Используется для моделирования пузырей |

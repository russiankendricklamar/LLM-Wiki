# Уравнение Фоккера-Планка

**Уравнение Фоккера-Планка (ФП)** — фундаментальное уравнение в описании эволюции плотности вероятности для стохастических систем, управляемых шумом. Оно связывает детерминированный дрейф и случайную диффузию через параболическое уравнение в частных производных, точно описывая переходную вероятность при переходе от [[stochastic-differential-equations|стохастических дифференциальных уравнений]] к вероятностным мерам.

## Вывод из расширения Крамерса-Мояла

Рассмотрим скалярный [[stochastic-differential-equations|СДУ]] в форме Ланжевена:

$$dX_t = a(X_t, t)\,dt + b(X_t, t)\,dW_t,$$

где $W_t$ — стандартный винеровский процесс. Для плотности вероятности $p(x,t) = \Pr(X_t = x)$ **расширение Крамерса-Мояла** дает точный ряд:

$$\frac{\partial p}{\partial t} = -\sum_{n=1}^{\infty} \frac{(-1)^n}{n!}\frac{\partial^n}{\partial x^n}\left[D_n(x,t) p(x,t)\right],$$

где $D_n(x,t) = \lim_{\Delta t \to 0} \mathbb{E}[\Delta X^n | X_t = x] / \Delta t$ — $n$-й момент приращения. Для непрерывного процесса все моменты $n \geq 3$ обращаются в нуль. Остаются только первые два:

$$D_1(x,t) = a(x,t), \quad D_2(x,t) = b(x,t)^2.$$

Это дает **уравнение Фоккера-Планка** (также называемое прямым уравнением Колмогорова):

$$\boxed{\frac{\partial p}{\partial t} = -\frac{\partial}{\partial x}[a(x,t) p(x,t)] + \frac{1}{2}\frac{\partial^2}{\partial x^2}[b(x,t)^2 p(x,t)]}.$$

## Многомерное обобщение

Для вектора $\mathbf{X}_t \in \mathbb{R}^d$ с $d\mathbf{X}_t = \mathbf{a}(\mathbf{x},t)\,dt + \mathbf{B}(\mathbf{x},t)\,d\mathbf{W}_t$:

$$\frac{\partial p}{\partial t} = -\sum_{i=1}^d \frac{\partial}{\partial x_i}[a_i(\mathbf{x},t) p] + \frac{1}{2}\sum_{i,j=1}^d \frac{\partial^2}{\partial x_i \partial x_j}[D_{ij}(\mathbf{x},t) p],$$

где $D_{ij} = (BB^T)_{ij}$ — матрица диффузии. В компактной форме:

$$\frac{\partial p}{\partial t} = -\nabla \cdot (\mathbf{a} p) + \frac{1}{2}\nabla^2 : (Dp) = -\nabla \cdot \mathbf{J},$$

где $\mathbf{J} = \mathbf{a} p - \frac{1}{2}\nabla \cdot (Dp)$ — **поток вероятности**.

## Стационарное распределение

Приравнивая $\partial p/\partial t = 0$ и требуя $\mathbf{J} \to 0$ на границах, получаем стационарное уравнение. Интегрируя один раз:

$$a(x) p_{\text{st}} = \frac{1}{2}\frac{d}{dx}[b(x)^2 p_{\text{st}}].$$

Это дает явное выражение:

$$p_{\text{st}}(x) \propto \frac{1}{b(x)^2} \exp\!\left(2\int \frac{a(x')}{b(x')^2}\,dx'\right).$$

## Процесс Орнштейна-Уленбека

Каноническим примером является [[ornstein-uhlenbeck-process|процесс Орнштейна-Уленбека]]: $dX_t = -\theta(X_t - \mu)\,dt + \sigma\,dW_t$. Здесь $a(x) = -\theta(x - \mu)$, $b(x) = \sigma$. Уравнение ФП:

$$\frac{\partial p}{\partial t} = \theta\frac{\partial}{\partial x}[(x - \mu) p] + \frac{\sigma^2}{2}\frac{\partial^2 p}{\partial x^2}.$$

Стационарное распределение:

$$p_{\text{st}}(x) \propto \exp\!\left(-\frac{2\theta(x - \mu)^2}{\sigma^2}\right) = \mathcal{N}\!\left(\mu, \frac{\sigma^2}{2\theta}\right).$$

## Уравнение теплоёмкости как частный случай

При $a = 0$ (отсутствие дрейфа) и $b = \sqrt{D}$ (постоянный коэффициент диффузии):

$$\frac{\partial p}{\partial t} = D \frac{\partial^2 p}{\partial x^2},$$

это **уравнение теплопроводности** — свободная диффузия. Решение с начальным условием $p(x, 0) = \delta(x - x_0)$:

$$p(x,t) = \frac{1}{\sqrt{4\pi D t}} \exp\!\left(-\frac{(x - x_0)^2}{4Dt}\right).$$

## Прямое и обратное уравнения Колмогорова

**Прямое уравнение (ФП)** описывает эволюцию плотности вероятности $p(x,t)$. **Обратное уравнение Колмогорова** эволюционирует переходную плотность по начальному условию:

$$\frac{\partial P(x,t|y,s)}{\partial s} = -a(y,s)\frac{\partial P}{\partial y} - \frac{1}{2}b(y,s)^2\frac{\partial^2 P}{\partial y^2}.$$

Здесь переходная вероятность $P(x,t|y,s) = \Pr(X_t = x | X_s = y)$ подчиняется дифференциальному уравнению по переменным $(y, s)$ с граничным условием $P(x,t|y,t) = \delta(x - y)$. Обратное уравнение фундаментально для вычисления цен опционов.

## Детальный баланс и статистическая механика

Для системы в контакте с тепловым резервуаром с температурой $T$, **условие детального баланса** требует:

$$p_{\text{st}}(x) P(y,t|x, 0) = p_{\text{st}}(y) P(x,t|y, 0).$$

При выполнении этого условия стационарное распределение совпадает с **распределением Больцмана**:

$$p_{\text{st}}(x) \propto \exp\!\left(-\frac{U(x)}{k_B T}\right),$$

где $U(x)$ — потенциальная энергия. Это основа связи уравнения ФП со статистической механикой.

## Применение к ценообразованию опционов

В стохастическом ценообразовании переходная плотность входит в формулу справедливой цены:

$$C(S, t) = \mathbb{E}^{\mathbb{Q}}[e^{-r(T-t)} \Phi(S_T) | S_t = S].$$

Для [[geometric-brownian-motion|геометрического броуновского движения]] $dS = rS\,dt + \sigma S\,dW$ уравнение ФП дает логнормальное распределение и приводит к формуле Блэка-Шоулза.

## Интегральное представление

[[path-integral|Интеграл по путям]] дает переходную плотность:

$$p(x, t | x_0, t_0) = \int \mathcal{D}[x(\cdot)] \, \exp\!\left(-\frac{1}{2}\int_{t_0}^t \frac{(dx - a \, dt)^2}{b^2} \right).$$

Это связывает стохастические процессы с дифференциальными уравнениями через функциональное интегрирование.

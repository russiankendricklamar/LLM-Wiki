# Интегралы по траекториям в КМ

Формализм [[интегралов по траекториям]] выражает квантовые амплитуды как сумму по всем возможным историям. Пропагатор равен $K = \int \mathcal{D}x \exp(iS[x]/\hbar)$, где $S$ — классическое действие.

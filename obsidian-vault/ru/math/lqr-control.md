# Линейно-квадратичный регулятор (LQR)

[[LQR]] минимизирует стоимость $J = \int (x^T Q x + u^T R u) dt$ для системы $\dot{x} = Ax + Bu$. Оптимальное управление $u = -Kx$, где $P$ решает алгебраическое уравнение Риккати.

# Полугруппы операторов

Теория $C_0$-полугрупп — это мощный язык для изучения **эволюционных уравнений** (систем, меняющихся во времени по фиксированному правилу). Она объединяет изучение PDE (например, уравнения теплопроводности) и марковских процессов.

## Основная идея

Семейство операторов $\{T(t)\}_{t \geq 0}$ называется полугруппой, если:
1.  $T(0) = I$ (состояние в момент 0 не меняется).
2.  $T(t+s) = T(t)T(s)$ (свойство детерминизма: результат через время $t+s$ — это результат через $s$, к которому применили эволюцию за время $t$).

## Инфинитезимальный производящий оператор ($A$)

«Скорость» полугруппы описывается её генератором $A$. Эволюционное уравнение в абстрактном виде записывается как:
$$\frac{du(t)}{dt} = Au(t)$$
Если $A$ — матрица, то решением будет матричная экспонента $e^{tA}$. Теория полугрупп обобщает это понятие на бесконечномерные пространства (функции).

## Примеры

- **Уравнение теплопроводности**: $A$ — это оператор Лапласа. Полугруппа описывает, как тепло распределяется по пространству.
- **Процессы Маркова**: полугруппа описывает эволюцию плотности вероятности во времени.

# Характеристические функции

## Обзор

Характеристическая функция — это **преобразование Фурье вероятностной меры**. Для случайной величины $X$ с распределением $\mu_X$:

$$\varphi_X(t) = \mathbb{E}[e^{itX}] = \int_{-\infty}^{\infty} e^{itx} \, d\mu_X(x).$$

Это одно из самых мощных средств теории вероятностей: характеристическая функция **полностью определяет распределение** (теорема единственности), упрощает доказательства предельных теорем (через теорему непрерывности Леви), и даёт прямой путь к моментам и кумулянтам.

§ 12 главы II [[shiryaev-probability|«Вероятности»]] Ширяева подробно её изучает. Применения: классическое доказательство [[central-limit-theorem|ЦПТ]], формула Леви-Хинчина для безгранично делимых распределений, устойчивые распределения, [[levy-esscher-continuous|процессы Леви]], характеристическая функция [[brownian-motion|броуновского движения]].

## Базовые свойства

1. **$\varphi_X(0) = 1$**. Очевидно из определения.
2. **$|\varphi_X(t)| \leq 1$** для всех $t$. Поскольку $|e^{itx}| = 1$.
3. **Равномерная непрерывность** на $\mathbb{R}$. Не просто непрерывность, а **равномерная** — следствие теоремы Лебега о доминируемой сходимости.
4. **$\varphi_X(-t) = \overline{\varphi_X(t)}$** — комплексное сопряжение.
5. **Для симметричных распределений** $\varphi_X$ вещественна.

## Связь с моментами

Если $X$ имеет конечные моменты до порядка $n$, то $\varphi_X$ дифференцируема до порядка $n$ в нуле, и

$$\varphi_X^{(k)}(0) = i^k \mathbb{E}[X^k], \quad k = 0, 1, \ldots, n.$$

То есть **разложение Тейлора** характеристической функции даёт моменты:

$$\varphi_X(t) = 1 + it\mathbb{E}[X] - \frac{t^2}{2}\mathbb{E}[X^2] - \frac{it^3}{6}\mathbb{E}[X^3] + \ldots$$

Это удобный путь к вычислению моментов, особенно для распределений с известной характеристической функцией (нормальное, Пуассон, бета).

## Сумма независимых

**Ключевое свойство.** Если $X$ и $Y$ независимы, то

$$\varphi_{X+Y}(t) = \varphi_X(t) \cdot \varphi_Y(t).$$

Распределение суммы соответствует **произведению** характеристических функций. Это многократно проще, чем свёртка плотностей. По этой причине характеристическая функция — главный инструмент для изучения сумм независимых величин и предельных теорем.

В частности, для $S_n = X_1 + \ldots + X_n$ из i.i.d. величин:

$$\varphi_{S_n}(t) = (\varphi_X(t))^n.$$

Это и есть отправная точка классических доказательств [[law-of-large-numbers|ЗБЧ]] и [[central-limit-theorem|ЦПТ]].

## Примеры

| Распределение | Характеристическая функция |
|---|---|
| Бернулли $\text{Ber}(p)$ | $1 - p + p e^{it}$ |
| Биномиальное $B(n, p)$ | $(1 - p + p e^{it})^n$ |
| Пуассона $\text{Pois}(\lambda)$ | $\exp(\lambda(e^{it} - 1))$ |
| Стандартное нормальное | $e^{-t^2/2}$ |
| Нормальное $\mathcal{N}(\mu, \sigma^2)$ | $e^{i\mu t - \sigma^2 t^2 / 2}$ |
| Экспоненциальное $\text{Exp}(\lambda)$ | $\lambda / (\lambda - it)$ |
| Коши | $e^{-|t|}$ |
| Равномерное $U(-a, a)$ | $\sin(at) / (at)$ |

Заметим, что характеристическая функция нормального распределения — снова гауссиана (с заменой переменной); это уникальное свойство, делающее нормальное распределение «фиксированной точкой» преобразования Фурье.

## Теорема единственности (Леви)

**Теорема.** Если $\varphi_X = \varphi_Y$, то $X$ и $Y$ имеют одинаковое распределение.

То есть характеристическая функция **однозначно определяет** распределение случайной величины. Это означает, что для проверки утверждения «$X$ имеет такое-то распределение» достаточно вычислить $\varphi_X$ и сравнить с известной характеристической функцией.

Доказательство — через формулу обращения Фурье: при условии интегрируемости $\varphi$ плотность восстанавливается как

$$f_X(x) = \frac{1}{2\pi} \int_{-\infty}^{\infty} e^{-itx} \varphi_X(t) \, dt.$$

В дискретном случае или при отсутствии плотности — формула Леви для $\mathbb{P}(a < X < b)$.

## Теорема непрерывности Леви

**Теорема.** Пусть $(X_n)$ — последовательность случайных величин с характеристическими функциями $(\varphi_n)$. Тогда:

$$X_n \xrightarrow{d} X \iff \varphi_n(t) \to \varphi_X(t) \text{ для каждого } t,$$

при условии что предельная функция $\varphi_X$ непрерывна в нуле (что эквивалентно тому, что предельное распределение — настоящая вероятностная мера, а не масса, утекающая в бесконечность).

Это **теорема непрерывности Леви** — мост между поточечной сходимостью характеристических функций и сходимостью по распределению. На ней построены **все классические доказательства** предельных теорем теории вероятностей, начиная с ЦПТ.

## Доказательство ЦПТ через характеристические функции

Стандартный путь:

1. Берём i.i.d. $X_1, X_2, \ldots$ с $\mathbb{E}[X] = 0$, $\text{Var}(X) = 1$, и образуем $S_n / \sqrt{n}$.
2. Характеристическая функция: $\varphi_{S_n/\sqrt{n}}(t) = (\varphi_X(t/\sqrt{n}))^n$.
3. Разлагаем $\varphi_X(t/\sqrt{n})$ в ряд Тейлора в нуле: $\varphi_X(t/\sqrt{n}) = 1 - t^2/(2n) + o(1/n)$.
4. Возводим в степень $n$: $(1 - t^2/(2n) + o(1/n))^n \to e^{-t^2/2}$.
5. Это характеристическая функция $\mathcal{N}(0, 1)$. По теореме непрерывности Леви $S_n/\sqrt{n} \xrightarrow{d} \mathcal{N}(0, 1)$.

Всё доказательство ЦПТ — на пол-страницы. Сравните с прямым подсчётом плотности свёртки $n$ копий — это адский путь.

## Безгранично делимые и формула Леви-Хинчина

Распределение называется **безгранично делимым**, если для каждого $n$ его можно представить как распределение суммы $n$ независимых одинаково распределённых величин. Все такие распределения — кандидаты в предельные для нормированных сумм.

**Формула Леви-Хинчина.** Безгранично делимые распределения имеют характеристическую функцию вида

$$\log \varphi(t) = i\beta t - \frac{1}{2}\sigma^2 t^2 + \int_{\mathbb{R}}\!\big( e^{itx} - 1 - itx \mathbf{1}_{|x|\leq 1} \big) \, \pi(dx),$$

где $(\beta, \sigma^2, \pi)$ — **триплет Леви-Хинчина**: снос, диффузионная компонента и мера Леви скачков.

Эта формула — **классификационная теорема** для безгранично делимых распределений и одновременно мост к [[levy-esscher-continuous|процессам Леви]] (которые есть стохастические аналоги безгранично делимых распределений во времени) и к [[stable-distributions|устойчивым распределениям]] (как частному случаю).

## Многомерные характеристические функции

Для случайного вектора $X = (X_1, \ldots, X_d)$:

$$\varphi_X(t) = \mathbb{E}[e^{i \langle t, X \rangle}] = \mathbb{E}[\exp(i (t_1 X_1 + \ldots + t_d X_d))], \quad t \in \mathbb{R}^d.$$

Все основные свойства переносятся: единственность, теорема непрерывности, разложение по моментам. В частности, **гауссовский вектор** $X \sim \mathcal{N}(\mu, \Sigma)$ имеет характеристическую функцию

$$\varphi_X(t) = \exp\!\big( i\langle\mu, t\rangle - \tfrac{1}{2}\langle t, \Sigma t\rangle \big).$$

Многомерные характеристические функции — стандартный инструмент для проверки **совместной нормальности**.

## Связь с производящими функциями

Для неотрицательных целочисленных случайных величин используется также **производящая функция** $G_X(z) = \mathbb{E}[z^X]$. Для общих случайных величин — **моментопроизводящая функция** $M_X(t) = \mathbb{E}[e^{tX}]$. Все они — варианты преобразования Фурье / Лапласа меры.

Преимущество $\varphi_X = \mathbb{E}[e^{itX}]$ перед $M_X$: характеристическая функция определена **всегда** (поскольку $|e^{itX}| = 1$), а $M_X$ может расходиться на бесконечности.

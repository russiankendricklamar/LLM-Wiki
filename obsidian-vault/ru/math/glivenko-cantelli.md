# Теорема Гливенко — Кантелли и эмпирические процессы

Теорему Гливенко — Кантелли называют **основной теоремой статистики**: она гарантирует, что эмпирическая функция распределения сходится к истинной равномерно и почти наверное. Это теоретический фундамент непараметрической статистики, критериев согласия и бутстрэпа. Теорема связывает теорию вероятностей ([[kolmogorov-probability-axioms|аксиомы Колмогорова]]) со статистической практикой самым прямым образом.

## Эмпирическая функция распределения

Пусть $X_1, \ldots, X_n$ — н.о.р. случайные величины с общей ФР $F$. **Эмпирическая ФР**:

$$F_n(x) = \frac{1}{n} \sum_{i=1}^{n} \mathbf{1}\{X_i \leq x\}.$$

Для каждого фиксированного $x$ индикатор $\mathbf{1}\{X_i \leq x\}$ — бернуллиевская величина с параметром $F(x)$, поэтому по [[law-of-large-numbers|усиленному ЗБЧ]] $F_n(x) \to F(x)$ п.н. Теорема Гливенко — Кантелли усиливает поточечную сходимость до **равномерной**.

## Теорема

**Теорема (Гливенко 1933, Кантелли 1933).** Для н.о.р. $X_1, X_2, \ldots$ с ФР $F$:

$$\sup_{x \in \mathbb{R}} |F_n(x) - F(x)| \xrightarrow{\text{п.н.}} 0 \quad \text{при } n \to \infty.$$

Супремум по **всем** $x$ стремится к нулю, а не только в отдельных точках.

## Схема доказательства

Доказательство комбинирует поточечную сходимость с монотонностью:

1. Фиксируем плотное счётное множество $\{r_k\}$ (например, рациональные числа). По усиленному ЗБЧ $F_n(r_k) \to F(r_k)$ п.н. для каждого $k$. По [[borel-cantelli-zero-one|лемме Бореля — Кантелли]] сходимость выполняется одновременно для всех рациональных на одном событии вероятности 1.
2. Для произвольного $x$ зажимаем $F_n(x)$ между значениями в ближайших рациональных точках, используя монотонность $F_n$ и $F$.
3. Правая непрерывность $F$ и плотность рациональных замыкают оценку: $\sup_x |F_n(x) - F(x)| \to 0$.

## Неравенство ДКВ

**Неравенство Дворецкого — Кифера — Вольфовица (1956)**, уточнённое Массаром (1990):

$$\mathbb{P}\!\left(\sup_{x} |F_n(x) - F(x)| > \varepsilon\right) \leq 2 e^{-2n\varepsilon^2}.$$

Замечательные свойства: оценка **свободна от распределения** (не зависит от $F$) и **неасимптотична** (верна при любом $n$). Из неё следует скорость сходимости $O(\sqrt{\log n / n})$ и точные доверительные полосы для $F$.

## Критерий Колмогорова — Смирнова

**Статистика Колмогорова — Смирнова (КС)**:

$$D_n = \sup_{x} |F_n(x) - F_0(x)|,$$

где $F_0$ — гипотетическая ФР. При $H_0: F = F_0$ Колмогоров (1933) доказал:

$$\sqrt{n}\, D_n \xrightarrow{d} \sup_{0 \leq t \leq 1} |B(t)|,$$

где $B$ — [[brownian-motion|броуновский мост]]. Предельное распределение — **распределение Колмогорова**, табулированное и используемое для точных p-значений. КС-критерий — канонический непараметрический критерий согласия: не требует группировки, работает для любой непрерывной $F_0$ и наследует гарантию состоятельности от Гливенко — Кантелли.

## VC-размерность и равномерная сходимость

Теорема Гливенко — Кантелли — утверждение о равномерной сходимости эмпирических средних по **классу множеств** $\mathcal{C} = \{(-\infty, x] : x \in \mathbb{R}\}$. Класс $\mathcal{C}$ называется **классом Гливенко — Кантелли**, если $\sup_{C \in \mathcal{C}} |P_n(C) - P(C)| \to 0$ п.н.

**Размерность Вапника — Червоненкиса (VC)** характеризует, какие классы являются классами Гливенко — Кантелли: если $\text{VC}(\mathcal{C}) < \infty$, то $\mathcal{C}$ — класс Гливенко — Кантелли для любой вероятностной меры $P$. Полупрямые имеют $\text{VC} = 1$; полуплоскости — $\text{VC} = 3$. Это связывает классическую статистику с **теорией обучения**: VC-размерность ограничивает выборочную сложность минимизации эмпирического риска.

## Эмпирические процессы

**Эмпирический процесс** — масштабированная флуктуация:

$$\mathbb{G}_n(x) = \sqrt{n}\,(F_n(x) - F(x)).$$

По теореме Донскера (функциональная [[central-limit-theorem|ЦПТ]]):

$$\mathbb{G}_n \xrightarrow{d} \mathbb{G} \quad \text{в } \ell^\infty(\mathbb{R}),$$

где $\mathbb{G}$ — $F$-броуновский мост. Теорема Донскера — процессное уточнение [[central-limit-theorem|ЦПТ]]: она даёт не только маргинальный гауссовский предел, но и предел по всей траектории.

Классы $\mathcal{F}$, для которых выполняется функциональная ЦПТ, называются **классами Донскера**. Каждый VC-класс с интегрируемой огибающей является донскеровским. Это основа полупараметрической теории эффективности.

## Приложения

**Непараметрическая статистика.** Эмпирическая ФР — **непараметрическая оценка максимального правдоподобия** для $F$. Гливенко — Кантелли гарантирует её равномерную состоятельность — отправная точка для ядерного оценивания плотности, ранговых критериев и оценки квантилей.

**Состоятельность бутстрэпа.** Бутстрэп Эфрона пересэмплирует из $F_n$. Теорема Гливенко — Кантелли обеспечивает $F_n \to F$ равномерно — ключевое условие состоятельности бутстрэпа.

**Критерии согласия.** Помимо Колмогорова — Смирнова: критерии Андерсона — Дарлинга, Крамера — фон Мизеса, Лиллиефорса — все используют функционалы от $F_n - F$. Их состоятельность следует из Гливенко — Кантелли.

**Машинное обучение.** Равномерная сходимость по классам функций (теория VC) обобщает Гливенко — Кантелли с множеств на классификаторы. Основная теорема статистического обучения: класс PAC-обучаем тогда и только тогда, когда его VC-размерность конечна.

# Дифференцирование мер

**Дифференцирование мер** — это классический раздел гармонического анализа и теории меры, устанавливающий связь между интегралом функции и её значением в почти каждой точке. Центральный результат — **теорема Лебега о дифференцировании** — показывает, что для локально интегрируемой функции среднее значение по шарам, сжимающимся к точке, сходится к значению функции в этой точке почти всюду.

## Теорема Лебега о дифференцировании

Для $f \in L^1_{loc}(\mathbb{R}^d)$ имеет место:
$$\lim_{r\to 0} \frac{1}{|B(x,r)|}\int_{B(x,r)} f(y)\,dy = f(x) \quad \text{п.в. } x \in \mathbb{R}^d$$

где $B(x,r)$ — открытый шар радиуса $r$ с центром в $x$, а $|B(x,r)|$ обозначает меру Лебега.

Эта теорема выражает глубокую идею: дифференцирование функции можно реализовать как локальное усреднение. Из неё следует, что для функции из $L^1_{loc}$, её интегральное определение через меру содержит полную информацию о её значениях почти всюду.

## Максимальная функция Харди–Литтлвуда

Технический инструмент для доказательства теоремы Лебега — **максимальная функция Харди–Литтлвуда**:
$$Mf(x) = \sup_{r>0} \frac{1}{|B(x,r)|}\int_{B(x,r)}|f(y)|\,dy$$

Эта функция измеряет максимальное среднее значение $|f|$ по всем шарам, содержащим точку $x$. Ключевое свойство:

**Слабая оценка типа $(1,1)$:** для любого $\lambda > 0$,
$$\left|\{Mf > \lambda\}\right| \leq \frac{C}{\lambda}\|f\|_1$$

где $C$ — абсолютная постоянная. Это означает, что уровни множества $\{Mf > \lambda\}$ убывают со скоростью $O(\lambda^{-1})$.

Для $p > 1$ верна сильная оценка в норме $L^p$:
$$\|Mf\|_p \leq C_p\|f\|_p$$

где $C_p$ зависит от $p$ и размерности. Эта граница следует из слабой оценки через **интерполяцию Маркинкевича**.

## Доказательство теоремы Лебега через максимальную функцию

Теорема Лебега выводится из свойств $Mf$ следующим образом. Для $f \in L^1_{loc}$, определим
$$L_*f(x) = \limsup_{r\to 0} \frac{1}{|B(x,r)|}\int_{B(x,r)} |f(y) - f(x)|\,dy$$

Тогда $L_*f(x) = 0$ п.в. на носителе функции $|f - f_0|$ для надлежащей функции $f_0$. По свойствам максимальной функции и регулярности ядер усреднения, для каждого $\epsilon > 0$ множество $\{L_*f > \epsilon\}$ имеет меру нуль, откуда следует результат.

## Лемма Безиковича о покрытиях

Для доказательства максимальных неравенств используется **лемма Безиковича** (Besicovitch covering lemma): если $\mathcal{F}$ — семейство шаров, покрывающее множество $E$, то существует подсемейство $\mathcal{F}'$ со следующими свойствами:

- $\mathcal{F}'$ попарно почти непересекающееся (пересечение каждой пары имеет меру нуль)
- $\mathcal{F}'$ по-прежнему покрывает $E$
- $\sum_{B \in \mathcal{F}'} |B| \leq C|E|$ для некоторой постоянной $C$, зависящей только от размерности

Эта лемма — комбинаторный фундамент для всех максимальных оценок в анализе.

## Разложение Кальдерона–Зигмунда

Разложение Кальдерона–Зигмунда служит мостом между локальными максимальными оценками и глобальными интегральными неравенствами. Для $f \in L^1(\mathbb{R}^d)$ и $\lambda > 0$ множество $\{\{f\} > \lambda\}$ можно покрыть шарами таким образом, что:
- вне объединения шаров $|f(x)| \leq \lambda$ п.в.
- внутри шаров локальное поведение контролируется

Этот метод лежит в основе доказательства слабой $(1,1)$ оценки для $M$.

## Дифференцирование общих мер

Идея дифференцирования распространяется на произвольные заряды и меры. Для знакопеременной меры $\nu$ (заряда) определяется **симметричная производная** относительно меры Лебега:
$$D\nu(x) = \lim_{r\to 0} \frac{\nu(B(x,r))}{|B(x,r)|}$$

когда предел существует. Обобщение теоремы Лебега утверждает: если $\nu$ абсолютно непрерывна относительно меры Лебега, то $D\nu$ существует п.в. и равна производной Радона–Никодима $\frac{d\nu}{d\lambda}$.

## Теорема Радона–Никодима как результат дифференцирования

Хотя теорема Радона–Никодима обычно доказывается посредством функционального анализа (в частности, через полярное разложение и проекционные свойства), её можно переформулировать в терминах дифференцирования: если $\nu$ абсолютно непрерывна относительно $\mu$, то существует локально интегрируемая функция $f$, такая что
$$\nu(E) = \int_E f\,d\mu$$
для всех измеримых множеств. Это свойство выводится из существования п.в. симметричной производной.

## Теорема Витали о покрытиях

Теорема Витали о покрытиях (Vitali covering theorem) предоставляет комбинаторный инструмент для выбора подпокрытия с хорошими метрическими свойствами:

**Теорема:** Если $\mathcal{V}$ — семейство замкнутых шаров, покрывающее множество $E$, то существует счётное подсемейство $\{B_n\}$ попарно непересекающихся шаров таких, что $|E \setminus \bigcup B_n| = 0$.

Это ключевой инструмент при работе с дифференцированием в классическом смысле и применяется в доказательстве теоремы Лебега.

## Применения и связи

Дифференцирование мер находит приложения в:
- Теории функций нескольких переменных (теория регулярности)
- Гармоническом анализе и сингулярных интегральных операторах
- Геометрической теории меры
- Стохастическом исчислении (локальные времена, квадратичная вариация)

Максимальная функция стала прообразом для множества операторов в анализе, включая операторы Данкела и Литтлвуда–Пэли.

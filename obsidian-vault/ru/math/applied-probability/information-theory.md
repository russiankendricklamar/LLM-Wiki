# Теория информации (Information Theory)

Теория информации, основанная Клодом Шенноном в 1948 году, формализует понятие информации как количественную меру неопределённости. Она устанавливает фундаментальные пределы сжатия данных и скорости их передачи по шумным каналам.

## Основные концепции

### 1. Энтропия Шеннона $H(X)$
Мера неопределённости случайной величины $X$:
$$H(X) = -\sum_{x \in \mathcal{X}} p(x) \log p(x)$$
- Максимальна на равномерном распределении.
- Измеряет минимальное среднее количество битов для кодирования исходов.

### 2. Взаимная информация $I(X;Y)$
Количество информации, которое $Y$ содержит о $X$:
$$I(X;Y) = H(X) - H(X|Y) = D_{\text{KL}}(P_{X,Y} \| P_X \otimes P_Y)$$

### 3. Дивергенция Кульбака-Лейблера (KL)
Мера «расстояния» (несходства) между распределениями $P$ и $Q$:
$$D_{\text{KL}}(P \| Q) = \sum_x p(x) \log \frac{p(x)}{q(x)}$$
- Всегда $D_{\text{KL}} \ge 0$.
- База для функции потерь в Cross-Entropy и Variational Inference.

## Фундаментальные теоремы

1. **Теорема о кодировании источника:** Энтропия $H(X)$ является неулучшаемым пределом сжатия данных без потерь.
2. **Теорема о пропускной способности канала:** Для любой скорости передачи $R < C$ (где $C = \max I(X;Y)$) существует код с вероятностью ошибки, стремящейся к нулю.
3. **Неравенство обработки данных:** $I(X;Z) \leq I(X;Y)$ для цепи Маркова $X \to Y \to Z$. Информация только теряется при обработке.

## Связь с другими областями

- **Физика:** Энтропия Шеннона эквивалентна термодинамической энтропии Больцмана ($S = k \ln W$).
- **Машинное обучение:** Обучение модели как минимизация KL-дивергенции между распределением данных и предсказаниями модели.
- **Статистика:** Информация Фишера определяет кривизну логарифмического правдоподобия и связана с дифференциальной энтропией.

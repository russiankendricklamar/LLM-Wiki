# Проверка гипотез II: Тесты отношения правдоподобия

Тесты отношения правдоподобия (Likelihood Ratio Tests, LRT) — это общий класс статистических тестов, используемых для сравнения качества подгонки двух моделей, одна из которых является вложенной версией другой.

## 1. Отношение правдоподобия

Рассмотрим проверку нулевой гипотезы $H_0: \theta \in \Theta_0$ против альтернативной $H_1: \theta \in \Theta$. Отношение правдоподобия $\Lambda$ определяется как:
$$\Lambda = \frac{\sup_{\theta \in \Theta_0} L(\theta)}{\sup_{\theta \in \Theta} L(\theta)}$$
Поскольку $\Theta_0 \subset \Theta$, отношение $\Lambda$ всегда находится в пределах от 0 до 1. Малое значение $\Lambda$ говорит о том, что данные гораздо более вероятны при альтернативной гипотезе, чем при нулевой.

## 2. Теорема Уилкса

Теорема Уилкса утверждает, что для больших выборок распределение величины $-2 \log \Lambda$ стремится к распределению хи-квадрат:
$$-2 \log \Lambda \xrightarrow{d} \chi^2_k$$
где $k$ — разность в количестве параметров между общей моделью и моделью при нулевой гипотезе (количество ограничений).

## 3. Мощность теста

**Мощность** теста — это вероятность правильного отклонения ложной нулевой гипотезы:
$$\text{Power} = P(\text{Reject } H_0 | H_1 \text{ is true}) = 1 - \beta$$
Согласно **лемме Неймана-Пирсона**, для простой нулевой гипотезы против простой альтернативной тест отношения правдоподобия является **наиболее мощным** тестом при заданном уровне значимости $\alpha$.

## 4. Связь с другими тестами

- **Тест Вальда**: Использует расстояние между MLE и значением при нулевой гипотезе.
- **Тест множителей Лагранжа (Score Test)**: Использует градиент логарифма правдоподобия в точке нулевой гипотезы.
Асимптотически LRT, тест Вальда и Score-тест эквивалентны.

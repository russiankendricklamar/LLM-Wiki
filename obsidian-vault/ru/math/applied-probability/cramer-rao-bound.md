# Граница Крамера-Рао

Неравенство Крамера-Рао устанавливает фундаментальный предел точности несмещенных оценок. Оно определяет минимально возможную дисперсию, которой может обладать несмещенная оценка.

## 1. Информация Фишера

**Функция счета (score function)** — это производная логарифма правдоподобия:
$$S(\theta) = \frac{\partial}{\partial \theta} \log f(X; \theta)$$
**Информация Фишера** $I(\theta)$ — это дисперсия функции счета:
$$I(\theta) = \mathbb{E}\left[ \left( \frac{\partial}{\partial \theta} \log f(X; \theta) \right)^2 \right] = -\mathbb{E}\left[ \frac{\partial^2}{\partial \theta^2} \log f(X; \theta) \right]$$

## 2. Неравенство

Для любой несмещенной оценки $\hat{\theta}$ параметра $\theta$:
$$\text{Var}(\hat{\theta}) \geq \frac{1}{n I(\theta)}$$
где $n$ — объем выборки.

### Эффективность
Несмещенная оценка, дисперсия которой совпадает с нижней границей Крамера-Рао, называется **эффективной оценкой**. Отношение границы Крамера-Рао к фактической дисперсии называется **статистической эффективностью**.

## 3. Общий вид (для смещенных оценок)
Если $\hat{\theta}$ — смещенная оценка с математическим ожиданием $\mathbb{E}[\hat{\theta}] = \psi(\theta)$, граница принимает вид:
$$\text{Var}(\hat{\theta}) \geq \frac{[\psi'(\theta)]^2}{n I(\theta)}$$

## 4. Значение в теории оценивания
- **MLE**: При определенных условиях регулярности [[mle-estimation|оценки максимального правдоподобия]] являются асимптотически эффективными, достигая границы Крамера-Рао при $n \to \infty$.
- **UMVUE**: Граница Крамера-Рао помогает в поиске равномерно несмещенной оценки с минимальной дисперсией (UMVUE).

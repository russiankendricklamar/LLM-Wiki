# Производящие функции моментов (MGFs) и характеристические функции

Производящие функции моментов (MGF) и характеристические функции — это методы на основе интегральных преобразований, используемые для однозначного описания вероятностных распределений. Они упрощают вычисление моментов и необходимы для доказательства предельных теорем.

## 1. Производящие функции моментов (MGF)

MGF случайной величины $X$ определяется как:
$$M_X(t) = \mathbb{E}[e^{tX}]$$
для $t$ в некоторой окрестности нуля.

### Свойства
1. **Генерация моментов**: $n$-й момент определяется $n$-й производной в нуле:
   $$\mathbb{E}[X^n] = M_X^{(n)}(0)$$
2. **Линейное преобразование**: $M_{aX+b}(t) = e^{bt}M_X(at)$.
3. **Суммы независимых величин**: Если $X, Y$ независимы, то $M_{X+Y}(t) = M_X(t)M_Y(t)$.

## 2. Характеристические функции

Характеристическая функция $\phi_X(t)$ определяется как:
$$\phi_X(t) = \mathbb{E}[e^{itX}] = \int_{-\infty}^{\infty} e^{itx} dF(x)$$
В отличие от MGF, характеристическая функция **существует всегда**, так как $|e^{itx}| = 1$.

### Свойства
1. **Теорема единственности**: Между распределениями и характеристическими функциями существует взаимно однозначное соответствие (теорема непрерывности Леви).
2. **Формула обращения**: Плотность $f(x)$ может быть восстановлена через:
   $$f(x) = \frac{1}{2\pi} \int_{-\infty}^{\infty} e^{-itx} \phi_X(t) dt$$
3. **Симметрия**: $X$ симметрична относительно 0 тогда и только тогда, когда $\phi_X(t)$ принимает только вещественные значения.

## 3. Приложения

- **Предельные теоремы**: Доказательство [[clt|Центральной предельной теоремы]] путем показа сходимости характеристических функций к $e^{-t^2/2}$.
- **Суммирование распределений**: Легкое нахождение распределения сумм (например, суммы независимых пуассоновских величин).

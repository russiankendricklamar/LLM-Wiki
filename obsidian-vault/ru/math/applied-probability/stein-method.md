# Метод Стейна

Метод Стейна — это мощный вероятностный инструмент для получения оценок сходимости распределений без явного вычисления характеристических функций. Разработанный Чарльзом Стейном в 1970-х годах, он дает количественные версии [[central-limit-theorem|центральной предельной теоремы]] и оценки для [[poisson-approximation|аппроксимации Пуассона]], находя приложения в комбинаторике, теории графов и статистической механике.

## Оператор Стейна и уравнение Стейна

Фундамент метода Стейна основан на замечательной характеризации: случайная величина $X$ имеет стандартное нормальное распределение $\mathcal{N}(0,1)$ если и только если

$$\mathbb{E}[f'(X) - Xf(X)] = 0$$

для всех абсолютно непрерывных функций $f$ с $\mathbb{E}[|f'(X)|] < \infty$. Оператор $\mathcal{A}f(x) = f'(x) - xf(x)$ называется **оператором Стейна** для нормального распределения.

Для измерения расстояния между распределениями фиксируют тестовую функцию $f$ и решают **уравнение Стейна**:

$$f'(x) - xf(x) = h(x)$$

где $h$ — заранее выбранная "гладкая" функция (обычно ограниченная). Если $W$ приближает $\mathcal{N}(0,1)$, то $\mathbb{E}[h(W)]$ должно быть близко к нулю. Стандартные оценки производных решений уравнения Стейна дают:

$$\|f'\|_\infty \leq 2\sqrt{\pi/2} \|h\|_\infty, \quad \|f''\|_\infty \leq 2\|h\|_\infty$$

Эти оценки разделяют анализ сходимости от свойств $h$, позволяя сравнивать распределения через [[concentration-inequalities|неравенства концентрации]] и оценки [[characteristic-functions|характеристических функций]].

## Обменяемые пары и дискрепанс Стейна

Практический способ применения метода Стейна — техника **обменяемых пар** (exchangeable pairs). Если $(W, W')$ — обменяемая пара случайных величин с $\mathbb{E}[W] = 0$ и $\text{Var}(W) = 1$, то при подходящих условиях регулярности

$$d_W(W, \mathcal{N}(0,1)) \lesssim \mathbb{E}[|W(W' - W)|] + \text{старшие поправки}$$

где $d_W$ — расстояние Вассерштейна. Величина $\mathbb{E}[|W(W' - W)|]$ измеряет, насколько в среднем величина $W$ "регрессирует" к нулю при связывании с "почти идентичной" копией. Это обеспечивает количественный результат, подобный локальной предельной теореме, без вычисления плотностей.

## Метод Чена–Стейна для аппроксимации Пуассона

Для распределения Пуассона оператор Стейна имеет вид

$$\mathcal{A}_\lambda f(k) = \lambda[f(k+1) - f(k)] - kf(k), \quad k \in \mathbb{Z}_{\geq 0}$$

где $\lambda > 0$ — параметр интенсивности. Дискретная случайная величина $K$ удовлетворяет $\mathbb{E}[\mathcal{A}_\lambda f(K)] = 0$ для всех ограниченных функций $f$ если и только если $K \sim \text{Пуассон}(\lambda)$.

Метод Чена–Стейна характеризует сходимость распределений решением

$$\lambda[f(k+1) - f(k)] - kf(k) = h(k) - \mathbb{E}[h(Y)]$$

где $Y \sim \text{Пуассон}(\lambda)$. Решения удовлетворяют равномерным оценкам:

$$\|f\|_\infty \leq \frac{2}{\lambda}\|h\|_\infty, \quad \|f(k+1) - f(k)\| \leq \frac{1}{\lambda}\|h\|_\infty$$

## Локальная зависимость и диссоциация

Реальные случайные величины редко бывают независимыми. Метод Стейна работает с зависимостью через концепцию **локальной зависимости**: если $W = \sum_{i=1}^n X_i$ и каждая $X_i$ существенно зависит только от ограниченного соседства индексов, то метод Стейна дает

$$d_{TV}(W, Y) \lesssim \sum_{i=1}^n \mathbb{E}[\mathbb{E}[X_i | \text{соседи}]] + \text{поправки к дисперсии}$$

где $d_{TV}$ — расстояние полной вариации. Условие диссоциации формализует, когда сумма зависимых случайных величин имеет "локально независимую" структуру, позволяя получить аппроксимации Пуассона или нормальные с явными скоростями сходимости.

## Оценки и метрики

Метод Стейна естественным образом дает оценки в нескольких метриках:

- **Расстояние Вассерштейна**: $d_W(\mathcal{L}(W), \mathcal{N}(0,1)) \sim O(\text{третий момент})$
- **Расстояние полной вариации**: $d_{TV}(\mathcal{L}(K), \text{Пуассон}(\lambda)) \sim O(\lambda^{-1})$ при диссоциации
- **Расстояние Колмогорова**: оценки на $\sup_x |P(W \leq x) - \Phi(x)|$ через [[local-limit-theorems|локальные предельные теоремы]]

Эти оценки часто совпадают с нижними оценками в литературе, демонстрируя, что метод Стейна точен.

## Конкретный пример: неподвижные точки случайной перестановки

Пусть $\sigma$ — равномерно случайная перестановка множества $\{1,2,\ldots,n\}$, и пусть $K_n = \sum_{i=1}^n \mathbb{1}_{\sigma(i)=i}$ подсчитывает количество неподвижных точек. Метод Стейна показывает:

$$d_{TV}(\mathcal{L}(K_n), \text{Пуассон}(1)) \leq \frac{2}{n}$$

Аргумент: разложим $K_n = \sum_{i=1}^n X_i$, где $X_i = \mathbb{1}_{\sigma(i)=i}$. Каждая $X_i$ имеет $\mathbb{P}(X_i = 1) = 1/n$, поэтому $\mathbb{E}[K_n] = 1$. Условие диссоциации выполнено, потому что $X_i$ и $X_j$ независимы при $|i-j| > 1$, с контролируемой зависимостью для соседних индексов. Метод Чена–Стейна затем дает указанную выше оценку без вычисления полного распределения $K_n$ — хотя при умеренных $n$ величина $K_n$ явно отклоняется от распределения Пуассон(1) из-за структуры зависимости.

## Приложения и расширения

Метод Стейна лежит в основе современного анализа:

- **Теория графов**: подсчет мотивов и подграфов в случайных графах через оценки локальной зависимости
- **Комбинаторика**: распределение вхождений паттернов в случайных последовательностях
- **Теория чисел**: аппроксимация подсчетов простых чисел и других числовых структур
- **Статистическая физика**: фазовые переходы и [[de-finetti-exchangeability|обменяемость]] в спин-системах

Недавние расширения включают многомерный метод Стейна (для многомерной нормальной и копула-аппроксимаций) и негладкие операторы Стейна для аппроксимации нестандартных предельных распределений.

## Ссылки и дальнейшее изучение

Метод Стейна продолжает развиваться, с современным подходом, подчеркивающим перспективу на основе генераторов и приложения к алгоритмам выборки. Фундаментальные работы остаются Stein (1986) и Chen (1998); современные обзоры акцентируют вычислительную эффективность и многомерные обобщения.

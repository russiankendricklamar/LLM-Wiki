# Эргодическая теория цепей Маркова

Эргодическая теория изучает долгосрочное статистическое поведение динамических систем. Для цепей Маркова она отвечает на фундаментальный вопрос: **придет ли система к устойчивому равновесию (стационарному распределению) независимо от того, где она началась?**

## Стационарное распределение ($\pi$)

Распределение $\pi$ называется стационарным для матрицы переходов $P$, если:
$$\pi P = \pi$$
Это означает, что если система уже находится в состоянии $\pi$, то через один шаг (и через любое количество шагов) её распределение не изменится.

## Эргодическая теорема

Для **неприводимой** (из любого состояния можно попасть в любое другое) и **апериодичной** (без жестких циклов) цепи Маркова верны следующие утверждения:

1.  **Единственность**: Существует единственное стационарное распределение $\pi$.
2.  **Сходимость**: Со временем распределение системы всегда стремится к $\pi$, «забывая» начальные условия.
3.  **Среднее по времени = Среднее по пространству**: Среднее значение любой функции вдоль одной длинной траектории совпадает со средним по всему распределению $\pi$.

Это свойство является математическим обоснованием алгоритмов **MCMC (Markov Chain Monte Carlo)**, которые используются повсеместно: от физики до байесовского вывода.

## Время перемешивания (Mixing Time)

Это количество шагов, необходимых для того, чтобы цепь оказалась «достаточно близко» к своему равновесному состоянию. Скорость сходимости определяется **спектральным зазором** матрицы переходов — разницей между самым большим (единичным) и вторым по величине собственным числом.

## Приложения

1.  **Алгоритм PageRank**: рейтинг страниц Google — это стационарное распределение случайного блуждания по графу интернета.
2.  **Байесовская статистика**: сэмплирование из сложных распределений с помощью алгоритма Метрополиса-Гастингса.
3.  **Физика**: описание перехода систем к термодинамическому равновесию.

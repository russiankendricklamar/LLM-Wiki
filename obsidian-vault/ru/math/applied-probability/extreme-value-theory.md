# Теория экстремальных значений (EVT)

Теория экстремальных значений (Extreme Value Theory, EVT) — это раздел теории вероятностей и статистики, изучающий поведение «хвостов» распределений и предельное поведение максимумов (или минимумов) последовательностей случайных величин. В отличие от центральной предельной теоремы, которая описывает поведение средних значений, EVT фокусируется на редких событиях — катастрофах, наводнениях, обвалах рынков и аномальных скоростях ветра.

## Центральная задача и максимумы блоков

Рассмотрим последовательность н.о.р. (независимых одинаково распределенных) случайных величин $X_1, X_2, \ldots, X_n$ с функцией распределения $F$. Максимум блока определяется как:

$$M_n = \max(X_1, X_2, \ldots, X_n)$$

Распределение $M_n$ легко выразить через $F$: $P(M_n \leq x) = F(x)^n$. Однако при $n \to \infty$ это значение стремится к нулю для любого фиксированного $x$ ниже верхней границы. Чтобы получить содержательный предел, мы нормируем максимум:

$$\frac{M_n - b_n}{a_n} \xrightarrow{d} G$$

где $a_n > 0$ и $b_n$ — нормирующие последовательности.

## Теорема Фишера — Типпетта — Гнеденко

Эта фундаментальная теорема утверждает, что если предел $G$ существует, то он обязан принадлежать к одному из трех типов:

1.  **Гумбель** (тип I): $G(x) = \exp(-e^{-x})$ — для распределений с экспоненциальными хвостами (нормальное, экспоненциальное).
2.  **Фреше** (тип II): $G(x) = \exp(-x^{-\alpha}), x > 0$ — для тяжелых, степенных хвостов (Парето, Коши).
3.  **Вейбулл** (тип III): $G(x) = \exp(-(-x)^\alpha), x < 0$ — для распределений с конечной верхней границей.

Эти три типа объединяются в **обобщенное распределение экстремальных значений (GEV)** с параметром формы $\xi$:

$$G_\xi(x) = \exp\left(-(1 + \xi x)^{-1/\xi}\right)$$

*   $\xi = 0$: Гумбель (тонкие хвосты)
*   $\xi > 0$: Фреше (тяжелые хвосты, $\alpha = 1/\xi$)
*   $\xi < 0$: Вейбулл (конечная граница)

## Метод превышения порога (POT) и GPD

Вместо максимумов блоков на практике часто используют метод превышения порога (**Peaks-Over-Threshold**). Он рассматривает все значения, превышающие достаточно высокий уровень $u$.

Согласно теореме Пикандса — Балкемы — де Хаана, распределение таких превышений при $u \to \infty$ стремится к **обобщенному распределению Парето (GPD)**:

$$W_{\xi, \beta}(y) = 1 - \left(1 + \xi \frac{y}{\beta}\right)^{-1/\xi}$$

Параметр $\xi$ здесь такой же, как в GEV, что позволяет оценивать «тяжесть хвоста» по всем экстремальным данным, а не только по одному максимуму за период.

## Применение в финансах: VaR и Expected Shortfall

EVT критически важна для управления рисками. В отличие от нормального распределения, которое катастрофически недооценивает риск «черных лебедей», EVT позволяет точно моделировать хвосты.

**Value-at-Risk (VaR)** на уровне $\alpha$ — это просто квантиль $F^{-1}(\alpha)$.
**Expected Shortfall (ES)** — это ожидаемый убыток при условии, что он превысил VaR:
$$\text{ES}_\alpha = \mathbb{E}[X \mid X > \text{VaR}_\alpha]$$

В рамках EVT-модели (GPD) ожидаемый убыток в хвосте вычисляется аналитически:
$$\text{ES}_\alpha = \frac{\text{VaR}_\alpha}{1-\xi} + \frac{\beta - \xi u}{1-\xi}$$
Это дает гораздо более адекватную оценку риска для активов с «толстыми хвостами» (криптовалюты, акции в кризис).

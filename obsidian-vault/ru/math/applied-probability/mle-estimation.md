# Точечное оценивание: Метод максимального правдоподобия (MLE)

Метод максимального правдоподобия (MLE) — это метод оценивания параметров вероятностного распределения путем максимизации функции правдоподобия. Согласно этому методу, в рамках принятой статистической модели наблюдаемые данные являются наиболее вероятными.

## 1. Формальный вывод

Пусть имеется выборка $X_1, ..., X_n$ из распределения $f(x; \theta)$. **Функция правдоподобия** определяется как:
$$L(\theta) = \prod_{i=1}^n f(x_i; \theta)$$
Для упрощения вычислений обычно максимизируют **логарифмическую функцию правдоподобия**:
$$\ell(\theta) = \sum_{i=1}^n \log f(x_i; \theta)$$
Оценка максимального правдоподобия $\hat{\theta}_{MLE}$ определяется как:
$$\hat{\theta}_{MLE} = \arg \max_{\theta} \ell(\theta)$$

## 2. Свойства MLE

Оценки MLE обладают рядом желательных асимптотических свойств:

1. **Состоятельность**: $\hat{\theta}_{MLE} \xrightarrow{P} \theta_0$ при $n \to \infty$.
2. **Асимптотическая нормальность**:
   $$\sqrt{n}(\hat{\theta}_{MLE} - \theta_0) \xrightarrow{d} \mathcal{N}(0, I(\theta_0)^{-1})$$
   где $I(\theta_0)$ — [[fisher-information|информация Фишера]].
3. **Эффективность**: MLE является асимптотически эффективной оценкой, достигая [[cramer-rao-bound|границы Крамера-Рао]].
4. **Инвариантность**: Если $\hat{\theta}$ — MLE для $\theta$, то $g(\hat{\theta})$ — MLE для $g(\theta)$.

## 3. Пример: Среднее нормального распределения
Для $X_i \sim \mathcal{N}(\mu, \sigma^2)$ логарифм правдоподобия равен:
$$\ell(\mu) = -\frac{n}{2}\log(2\pi\sigma^2) - \sum \frac{(x_i - \mu)^2}{2\sigma^2}$$
Дифференцируя по $\mu$ и приравнивая к нулю:
$$\frac{\partial \ell}{\partial \mu} = \frac{1}{\sigma^2} \sum (x_i - \mu) = 0 \implies \hat{\mu}_{MLE} = \bar{X}$$

# Теоремы Гёделя о неполноте

**Теоремы о неполноте** (Гёдель, 1931) — самые глубокие результаты математической логики XX века. Они показывают, что в любой достаточно богатой формальной системе существуют **истинные, но недоказуемые** утверждения. Программа Гильберта — формализовать всю математику — оказалась невозможной.

## Контекст: программа Гильберта

В 1900-х Гильберт хотел построить полную и непротиворечивую аксиоматическую систему для всей математики. Гёдель в 1931 году в 25 лет доказал, что это невозможно принципиально.

## Первая теорема о неполноте

> Любая непротиворечивая формальная система $T$, содержащая арифметику Пеано, **неполна**: существует предложение $G_T$ такое, что $T \nvdash G_T$ и $T \nvdash \neg G_T$.

Иными словами, $G_T$ истинно (в стандартной модели), но $T$ не может его доказать.

## Нумерация Гёделя

Ключевая идея — закодировать синтаксис формальной системы арифметикой:

$$
\ulcorner \varphi \urcorner \in \mathbb{N} \quad \text{— гёделев номер формулы } \varphi
$$

Любая формула $\varphi(x)$ с одной свободной переменной порождает **диагонализацию**:

$$
\text{diag}(\varphi) = \varphi(\ulcorner \varphi \urcorner)
$$

**Лемма о диагонализации**: для любой формулы $\psi(x)$ существует предложение $G$ такое, что $T \vdash G \leftrightarrow \psi(\ulcorner G \urcorner)$.

## Конструкция гёделевского предложения

Пусть $\text{Prov}_T(x)$ — формула, выражающая «$x$ — гёделев номер доказуемого в $T$ предложения».

По лемме о диагонализации существует $G$ такое, что:

$$
T \vdash G \leftrightarrow \neg\text{Prov}_T(\ulcorner G \urcorner)
$$

$G$ говорит: **«Я недоказуем в $T$»** (аналог парадокса лжеца в математике).

- Если $T \vdash G$ — то $T$ доказывает $\neg\text{Prov}_T(\ulcorner G \urcorner)$, что противоречит непротиворечивости.
- Если $T \vdash \neg G$ — то $T \vdash \text{Prov}_T(\ulcorner G \urcorner)$, что означает $T \vdash G$. Снова противоречие.

Следовательно, $G$ — истинное, но недоказуемое в $T$ предложение. $\blacksquare$

## Вторая теорема о неполноте

> Если $T$ непротиворечива и содержит арифметику, то $T$ **не может доказать свою собственную непротиворечивость**: $T \nvdash \text{Con}(T)$.

Где $\text{Con}(T) = \neg\text{Prov}_T(\ulcorner \bot \urcorner)$ — арифметизация утверждения «$T$ непротиворечива».

## Демонстрация нумерации Гёделя

```python
def godel_encode(expression: str) -> int:
    """
    Упрощённая нумерация Гёделя через простые числа.
    Каждый символ кодируется своим порядковым номером,
    число = p_1^{c_1} * p_2^{c_2} * ... * p_n^{c_n}
    """
    def nth_prime(n: int) -> int:
        primes = []
        candidate = 2
        while len(primes) < n:
            if all(candidate % p != 0 for p in primes):
                primes.append(candidate)
            candidate += 1
        return primes[-1]

    symbols = sorted(set(expression))
    encoding = {s: i + 1 for i, s in enumerate(symbols)}
    result = 1
    for i, char in enumerate(expression):
        prime = nth_prime(i + 1)
        result *= prime ** encoding[char]
    return result


def diagonal_lemma_demo():
    """
    Демонстрация диагонализации: функция, применяющая себя к своему коду.
    Аналог self-referential statements.
    """
    def psi(x: str) -> str:
        """Предикат: 'x — недоказуема'"""
        return f"'{x}' is unprovable"

    # Диагонализация: G = psi(G) → G говорит о себе
    G_template = "psi('psi(x) applied to itself')"
    G = psi(G_template)
    print(f"Гёделевское предложение G: {G}")
    print("G истинно, если 'psi(G_template) is unprovable'")
    print("→ G не может быть ни доказано, ни опровергнуто в T")


def fixed_point_example():
    """
    Неподвижная точка: f(x) = x применительно к высказываниям.
    Аналог леммы о диагонализации в Python.
    """
    quine_template = 'print(s % repr(s)); s = %r'
    # Квайн — программа, выводящая свой собственный код
    # Аналог предложения, описывающего само себя
    return quine_template % quine_template
```

## Последствия

| Следствие | Суть |
|-----------|------|
| Программа Гильберта невозможна | Нельзя финитно доказать непротиворечивость арифметики |
| Тьюринг (1936) | Проблема остановки неразрешима — следует из Гёделя |
| Чайтин | Большинство математических истин случайны и недоказуемы |
| Парадокс Рассела | Предвосхитил самореференцию до Гёделя |

## Связь с вычислимостью и ИИ

Диагонализация Гёделя — тот же приём, которым Тьюринг через пять лет доказал неразрешимость **проблемы остановки**, а Чайтин — определил случайное число $\Omega$ (вероятность того, что случайная программа остановится). **[[kolmogorov-complexity|Колмогоровская сложность]]** буквально гёдель-неполна: почти для каждой двоичной строки никакая формальная система не может доказать, какова её кратчайшая запись. Арифметический предел доказуемости и алгоритмический предел сжимаемости — две стороны одной стены.

Для языковых моделей это имеет конкретное значение. [[chain-of-thought|Chain-of-thought-рассуждения]] дают [[llm|LLM]] больше шагов на вывод, но не двигают стену: существуют арифметические утверждения, доказательства которых не может вывести никакая конечная цепочка. Жёсткий потолок того, что трансформер может *решить через рассуждение*, — это в точности гёделев потолок. Граница между [[mechanistic-interpretability|интерпретируемыми reasoning-схемами]] и недоступными им утверждениями — современное эмпирическое лицо первой теоремы о неполноте.

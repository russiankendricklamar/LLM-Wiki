# Вариационное исчисление

Вариационное исчисление — это математический аппарат для нахождения функций, экстремизирующих функционалы, т. е. отображения из пространств функций в вещественные числа. Оно составляет основу классической механики, оптимизации и дифференциальной геометрии.

## Основная задача

Центральный объект — это **функционал**:

$$J[y] = \int_a^b F(x, y(x), y'(x))\,dx$$

где $F$ — **лагранжиан**, $y(x)$ — искомая функция, $y' = dy/dx$. Требуется найти функцию $y$, экстремизирующую $J$ при фиксированных граничных условиях $y(a) = y_0$ и $y(b) = y_1$.

## Уравнение Эйлера–Лагранжа

Центральный результат — **уравнение Эйлера–Лагранжа**:

$$\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'} = 0$$

Это необходимое условие для того, чтобы $y$ была экстремалью функционала $J$. Раскрывая полную производную:

$$\frac{\partial F}{\partial y} - \frac{\partial^2 F}{\partial x \partial y'} - \frac{\partial^2 F}{\partial y \partial y'} y' - \frac{\partial^2 F}{\partial y'^2} y'' = 0$$

Уравнение обычно представляет собой дифференциальное уравнение второго порядка, общее решение которого содержит две произвольные постоянные, определяемые граничными условиями.

### Вывод через первую вариацию

Рассмотрим возмущение $y(x) + \epsilon \eta(x)$, где $\eta(a) = \eta(b) = 0$ (граничные вариации обращаются в нуль). **Первая вариация** функционала:

$$\delta J = \frac{d}{d\epsilon}\Big|_{\epsilon=0} J[y + \epsilon\eta] = \int_a^b \left(\frac{\partial F}{\partial y}\eta + \frac{\partial F}{\partial y'}\eta'\right)\,dx$$

Интегрируя второе слагаемое по частям и используя условие $\eta(a) = \eta(b) = 0$:

$$\delta J = \int_a^b \left(\frac{\partial F}{\partial y} - \frac{d}{dx}\frac{\partial F}{\partial y'}\right)\eta\,dx$$

Для того чтобы $\delta J = 0$ для всех допустимых $\eta$, подынтегральное выражение должно обращаться в нуль, откуда следует уравнение Эйлера–Лагранжа.

## Естественные граничные условия

Когда граничные значения не фиксированы — например, $y(b)$ свободно выбирается — граничные члены, возникающие при интегрировании по частям, не обращаются в нуль тождественно. Вместо этого получаются **естественные граничные условия**:

$$\frac{\partial F}{\partial y'}\bigg|_{x=b} = 0$$

Аналогично, если $y(a)$ свободно, то $\frac{\partial F}{\partial y'}\big|_{x=a} = 0$. Эти условия возникают естественным образом из вариационного принципа без явного предположения.

## Вторая вариация и условие Лежандра

Для различения минимумов, максимумов и седловых точек исследуется **вторая вариация**:

$$\delta^2 J = \frac{1}{2}\int_a^b \left(P(x)\eta^2 + Q(x)\eta'^2\right)\,dx$$

где $P = \frac{\partial^2 F}{\partial y^2} - \frac{d}{dx}\frac{\partial^2 F}{\partial y \partial y'}$ и $Q = \frac{\partial^2 F}{\partial y'^2}$.

Для минимума требуется $\delta^2 J \geq 0$ для всех $\eta$. **Достаточное условие** — **условие Лежандра**:

$$\frac{\partial^2 F}{\partial y'^2} > 0 \quad \text{(или } < 0 \text{ для максимума)}$$

Это гарантирует положительную определённость квадратичной формы (в усилённом смысле в сочетании с условиями на $P$).

## Тождество Бельтрами

Когда $F$ не зависит явно от $x$ — $F = F(y, y')$ — возникает полезный закон сохранения. **Тождество Бельтрами** утверждает:

$$F - y'\frac{\partial F}{\partial y'} = C \quad \text{(постоянна вдоль экстремалей)}$$

Это снижает порядок уравнения Эйлера–Лагранжа со второго на первый, что значительно упрощает вычисления.

## Классические задачи

### Брахистохрона

Кривая наискорейшего спуска под действием гравитации. Здесь $F = \sqrt{\frac{1+y'^2}{2gy}}$, и решением является **циклоида** — кривая, описываемая точкой на катящемся круге.

### Геодезические

Кратчайшие пути на римановом многообразии. В плоском пространстве $F = \sqrt{1 + y'^2}$ даёт $y'' = 0$ (прямые линии). На искривлённой поверхности уравнения Эйлера–Лагранжа определяют дифференциальные уравнения геодезических.

### Минимальные поверхности

Поверхности минимальной площади, натянутые на замкнутую кривую. Лагранжиан имеет вид $F = \sqrt{1 + u_x^2 + u_y^2}$ (в двумерном случае); решения удовлетворяют **уравнению минимальной поверхности**.

### Изопериметрическая задача

Максимизировать площадь, ограниченную кривой фиксированного периметра. Здесь $J[y]$ — площадь, $G[y]$ — длина дуги, и накладывается ограничение: $G[y] = L$. Используя **множители Лагранжа**, включаем ограничение в дополненный функционал:

$$\tilde{J}[y] = J[y] - \lambda G[y]$$

и решаем уравнения Эйлера–Лагранжа для дополненного функционала.

## Принцип Гамильтона и механика

В классической механике **действие** определяется как:

$$S[q] = \int_{t_1}^{t_2} L(t, q(t), \dot{q}(t))\,dt$$

где $L = T - V$ — лагранжиан (кинетическая минус потенциальная энергия). **Принцип Гамильтона** гласит, что истинная траектория экстремизирует $S$. Уравнения Эйлера–Лагранжа становятся **уравнениями Лагранжа движения**:

$$\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_i} - \frac{\partial L}{\partial q_i} = Q_i$$

где $Q_i$ — обобщённые силы от неконсервативных источников.

## Гамильтонова формулировка

Лагранжиан $L(q, \dot{q})$ может быть преобразован в гамильтониан посредством **преобразования Лежандра**. Определим **каноническую переменную импульса**:

$$p_i = \frac{\partial L}{\partial \dot{q}_i}$$

**Гамильтониан** определяется как:

$$H(q, p) = \sum_i p_i \dot{q}_i - L(q, \dot{q})$$

выраженный в переменных $q$ и $p$. Уравнения движения принимают вид **канонических уравнений**:

$$\dot{q}_i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q_i}$$

Этот формализм явно симплектичен и лежит в основе квантования и статистической механики.

## Теорема Нётер

**Теорема Нётер** связывает **симметрии с законами сохранения**. Если лагранжиан инвариантен относительно гладкого семейства преобразований (например, сдвига во времени, поворота, калибровочного преобразования), то соответствующая величина сохраняется.

Точнее: если $L(t, q(t), \dot{q}(t))$ инвариантен относительно $q \to q + \epsilon \phi(q)$, то:

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}} \cdot \phi(q)\right) = 0$$

Примеры:
- **Инвариантность относительно пространственного сдвига** → сохранение **импульса**
- **Инвариантность относительно поворота** → сохранение **углового момента**
- **Инвариантность относительно сдвига во времени** → сохранение **энергии**

Это фундаментальный принцип связывает геометрию и физику на глубочайшем уровне.

## Прямые методы и существование

Вместо явного решения уравнения Эйлера–Лагранжа **прямые методы** устанавливают существование минимизаторов путём минимизации $J$ по подходящему пространству функций (обычно по [[sobolev-spaces|пространству Соболева]]).

Ключевой результат: если $F$ **полунепрерывна снизу** и **коэрцитивна** (растёт достаточно быстро при $|y'| \to \infty$), то $J$ достигает своего минимума на слабо компактном подмножестве по **теореме Больцано–Вейерштрасса** в рефлексивных пространствах.

Этот подход распространяется на негладкие лагранжианы и охватывает случаи, когда классических экстремалей не существует.

## Связи с оптимальным управлением

**Принцип максимума Понтрягина** обобщает вариационное исчисление на оптимальное управление. Дано:

$$J[u] = \int_0^T L(t, x(t), u(t))\,dt$$

при ограничении $\dot{x} = f(t, x, u)$, оптимальное управление удовлетворяет необходимому условию с использованием **сопряжённой переменной** $\lambda(t)$:

$$H(t, x^*, u^*, \lambda^*) = \max_u H(t, x^*, u, \lambda^*)$$

где $H = L + \lambda \cdot f$ — гамильтониан. Присоединённые уравнения для $\lambda$ отражают канонические уравнения.

## Выпуклая двойственность

[[convex-analysis|Выпуклая двойственность]] предоставляет альтернативный взгляд: сопряжённый функционал (преобразование Лежандра–Фенхеля) связывает прямую задачу (минимизация $J$) с двойственной задачей. Результаты о слабой и сильной двойственности характеризуют случаи, когда прямая и двойственная цели совпадают, с приложениями к релаксации и аппроксимации.

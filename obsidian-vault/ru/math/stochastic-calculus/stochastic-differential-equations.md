# Стохастические дифференциальные уравнения

Стохастические дифференциальные уравнения (СДУ) — язык, на котором записывается непрерывная случайная динамика. Общее СДУ Ито:

$$dX_t = b(t, X_t)\,dt + \sigma(t, X_t)\,dW_t,$$

с дрейфом $b$ и коэффициентом диффузии $\sigma$, моделирует явления от цен финансовых активов до динамики популяций и сигнальных процессов в [[nonlinear-filtering|теории фильтрации]]. Липцер и Ширяев развивают теорию СДУ как фундамент частично наблюдаемых систем: сигнал $\theta_t$ и наблюдение $\xi_t$ — решения СДУ, и свойства этих решений (существование, единственность, регулярность) определяют возможности фильтрации.

Теория разделяется на два принципиально различных понятия решения: **сильное** (потраекторное) и **слабое** (по распределению). Их взаимосвязь, кристаллизованная в теореме Ямады-Ватанабэ, — один из элегантных структурных результатов стохастического анализа.

## Сильные решения

**Определение.** **Сильное решение** СДУ $dX_t = b(t,X_t)\,dt + \sigma(t,X_t)\,dW_t$ на фильтрованном вероятностном пространстве $(\Omega, \mathcal{F}, (\mathcal{F}_t), P)$, несущем броуновское движение $W$, — процесс $X$ такой, что:

1. $X$ согласован с фильтрацией, порождённой $W$ (и начальным условием $X_0$).
2. $\int_0^T |b(t,X_t)|\,dt + \int_0^T \sigma^2(t,X_t)\,dt < \infty$ п.н. для каждого $T$.
3. $X_t = X_0 + \int_0^t b(s,X_s)\,ds + \int_0^t \sigma(s,X_s)\,dW_s$ п.н. для всех $t$.

Ключевое требование — согласованность с $\mathcal{F}_t^W$: решение строится по заданному броуновскому движению, траектория за траекторией.

**Теорема (Ито).** Если $b$ и $\sigma$ глобально липшицевы по $x$ равномерно по $t$:

$$|b(t,x) - b(t,y)| + |\sigma(t,x) - \sigma(t,y)| \leq K|x - y|,$$

и удовлетворяют условию линейного роста $|b(t,x)| + |\sigma(t,x)| \leq K(1 + |x|)$, то для любого $\mathcal{F}_0$-измеримого $X_0$ с $\mathbb{E}[X_0^2] < \infty$ СДУ имеет единственное сильное решение на $[0, \infty)$.

Доказательство — итерации Пикара: $X^{(0)}_t = X_0$, $X^{(n+1)}_t = X_0 + \int_0^t b(s, X^{(n)}_s)\,ds + \int_0^t \sigma(s, X^{(n)}_s)\,dW_s$. Условие Липшица и изометрия Ито дают $\mathbb{E}[\sup_{s \leq t}|X^{(n+1)}_s - X^{(n)}_s|^2] \leq C^n t^n / n!$, обеспечивая сходимость.

## Слабые решения

**Определение.** **Слабое решение** — это вероятностное пространство $(\Omega, \mathcal{F}, P)$, фильтрация, броуновское движение $W$ и процесс $X$ на этом пространстве, удовлетворяющий СДУ. Пространство и броуновское движение — часть решения, а не задаются заранее.

Слабые решения — решения **по распределению**: два слабых решения считаются одинаковыми, если $X$ имеет одни и те же конечномерные распределения.

**Пример (Танака).** СДУ $dX_t = \mathrm{sgn}(X_t)\,dW_t$ имеет слабое решение (модуль броуновского движения $|W_t|$ — слабое решение по формуле Танаки), но не имеет сильного. Коэффициент диффузии $\sigma(x) = \mathrm{sgn}(x)$ не является липшицевым в нуле; потраекторное отображение $W \mapsto X$ не существует.

## Теорема Ямады-Ватанабэ

**Теорема (Ямада-Ватанабэ).** Для СДУ $dX_t = b(t,X_t)\,dt + \sigma(t,X_t)\,dW_t$:

$$\text{Слабое существование} + \text{Потраекторная единственность} \implies \text{Сильное существование (и единственность по распределению)}.$$

**Потраекторная единственность** означает: на любом вероятностном пространстве с любым броуновским движением $W$, если $X$ и $Y$ — два решения с $X_0 = Y_0$ п.н., то $X_t = Y_t$ для всех $t$ п.н.

Теорема разделяет вопрос существования (атакуемый аргументами компактности/плотности с помощью [[weak-convergence-prokhorov|теоремы Прохорова]] и [[skorokhod-topology|топологии Скорохода]]) от вопроса единственности (требующего аналитических оценок коэффициентов).

## Мартингальная задача

**Определение (Строк-Вараден).** **Мартингальная задача** для оператора $\mathcal{L} = b \cdot \nabla + \frac{1}{2}\sigma^2 \Delta$: найти вероятностную меру $P$ на пространстве траекторий $C([0,\infty); \mathbb{R}^d)$ такую, что для каждой $f \in C_c^2$:

$$f(X_t) - f(X_0) - \int_0^t \mathcal{L}f(X_s)\,ds$$

есть $P$-мартингал относительно канонической фильтрации.

**Теорема.** Слабое решение СДУ существует тогда и только тогда, когда мартингальная задача для $\mathcal{L}$ имеет решение. Единственность по распределению соответствует единственности мартингальной задачи.

Формулировка через мартингальную задачу не использует явного броуновского движения, что делает аргументы компактности естественными и связывает теорию СДУ с теорией марковских процессов и полугрупп.

## Одномерная теория

В одномерном случае доступны значительно более точные результаты.

**Теорема (Ямада-Ватанабэ, 1-d).** Потраекторная единственность выполнена для $dX_t = b(t,X_t)\,dt + \sigma(X_t)\,dW_t$, если $\sigma$ удовлетворяет:

$$|\sigma(x) - \sigma(y)| \leq \rho(|x - y|),$$

где $\int_0^{\varepsilon} \rho^{-2}(u)\,du = \infty$ при некотором $\varepsilon > 0$, а $b$ липшицев. Например, $\sigma(x) = |x|^\alpha$ при $\alpha \geq 1/2$ обеспечивает потраекторную единственность (классическое условие Липшица требует $\alpha \geq 1$).

**Теорема (Энгельберт-Шмидт).** Для $dX_t = \sigma(X_t)\,dW_t$ (без дрейфа) слабое решение существует и единственно по распределению тогда и только тогда, когда $\sigma^{-2}$ локально интегрируема ($\int_{x-\varepsilon}^{x+\varepsilon} \sigma^{-2}(y)\,dy < \infty$ для всех $x$).

## Теоремы сравнения

**Теорема.** Пусть $X$ и $Y$ решают $dX_t = b_1(t,X_t)\,dt + \sigma(t,X_t)\,dW_t$ и $dY_t = b_2(t,Y_t)\,dt + \sigma(t,Y_t)\,dW_t$ (одинаковый коэффициент диффузии, одно броуновское движение). Если $X_0 \leq Y_0$ п.н. и $b_1(t,x) \leq b_2(t,x)$ для всех $(t,x)$, то $X_t \leq Y_t$ для всех $t$ п.н.

Теоремы сравнения требуют совпадения коэффициента диффузии. Они необходимы в финансовой математике (оценки цен опционов) и при доказательстве свойств решений (неотрицательность, ограниченность).

## Тест Феллера на взрыв

Для одномерного СДУ $dX_t = b(X_t)\,dt + \sigma(X_t)\,dW_t$ на интервале $(l, r)$ тест Феллера определяет, достигаются ли границы $l$ и $r$ за конечное время. Масштабная функция $s(x) = \int^x \exp\bigl(-2\int^y b(z)/\sigma^2(z)\,dz\bigr)\,dy$ и мера скорости $m(dx) = [s'(x)\sigma^2(x)]^{-1}\,dx$ классифицируют границу $r$ как:

- **Естественную** (недостижимую), если $\int^r s'(x)\,dx = \infty$ или $\int^r m(dx)\,\int^x s'(y)\,dy = \infty$.
- **Входную** (можно стартовать, но не вернуться) или **выходную** (достигается и поглощает) — в зависимости от конечности соответствующих интегралов.

Эта классификация фундаментальна для понимания глобальности решений ($X_t$ существует для всех $t$) и взрыва к границе.

## Приложения

**Диффузионные модели в финансах.** [[geometric-brownian-motion|Геометрическое броуновское движение]] $dS_t = \mu S_t\,dt + \sigma S_t\,dW_t$ (липшицево на $(0,\infty)$, сильное решение существует) — основа теории Блэка-Шоулза. В [[heston-model|модели Хестона]] волатильность задаётся СДУ типа CIR $dV_t = \kappa(\bar{V} - V_t)\,dt + \xi\sqrt{V_t}\,dW_t$ — коэффициент $\sqrt{V}$ гёльдеров с показателем $1/2$, и теорема Ямады-Ватанабэ (1-d) гарантирует потраекторную единственность.

**Динамика популяций.** Диффузия Феллера $dX_t = \alpha X_t\,dt + \sigma\sqrt{X_t}\,dW_t$ моделирует размер популяции. Тест Феллера показывает, что $X = 0$ — поглощающая граница (вымирание) при определённых режимах параметров.

**Теория фильтрации.** В модели «сигнал-наблюдение» Липцера-Ширяева сигнал $\theta_t$ удовлетворяет СДУ. Регулярность решений (моментные оценки, марковское свойство) напрямую определяет корректность [[nonlinear-filtering|уравнений фильтрации]].

**Стохастическое управление.** Управляемые диффузии $dX_t = b(t, X_t, u_t)\,dt + \sigma(t, X_t)\,dW_t$ с допустимыми управлениями $u$ — основа динамического программирования и уравнения Гамильтона-Якоби-Беллмана.

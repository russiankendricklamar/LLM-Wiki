# Хаос Винера и кратные стохастические интегралы

Хаос Винера представляет собой фундаментальное разложение пространства квадратично интегрируемых случайных величин, порождённых броуновским движением. Эта теория, развитая Норбертом Винером и Киёси Ито, лежит в основе современного стохастического анализа и находит приложения в финансовой математике, квантовой теории поля и статистической механике.

## Полиномы Эрмита и ортогональность

Полиномы Эрмита образуют полный ортогональный базис в пространстве $L^2(\gamma)$, где $\gamma$ — стандартная гауссова мера на прямой. Они определяются формулой Родригеса:

$$H_n(x) = (-1)^n e^{x^2/2}\frac{d^n}{dx^n}e^{-x^2/2}$$

Явные примеры:
- $H_0(x) = 1$
- $H_1(x) = x$
- $H_2(x) = x^2 - 1$
- $H_3(x) = x^3 - 3x$
- $H_4(x) = x^4 - 6x^2 + 3$

Ортогональность в $L^2(\gamma)$:

$$\int_{-\infty}^{\infty} H_m(x) H_n(x) e^{-x^2/2} \frac{dx}{\sqrt{2\pi}} = \delta_{m,n}$$

Полиномы Эрмита удовлетворяют рекуррентному соотношению:

$$H_{n+1}(x) = x H_n(x) - n H_{n-1}(x)$$

и дифференциальному уравнению:

$$H_n''(x) - x H_n'(x) + n H_n(x) = 0$$

Функция Эрмита $h_n(x) = H_n(x)e^{-x^2/4}$ образует ортонормированный базис в $L^2(\mathbb{R})$ с нормой:

$$\int_{-\infty}^{\infty} |h_m(x)|^2 dx = \delta_{m,n}$$

## Первый хаос Винера

Первый хаос Винера $\mathcal{H}_1$ состоит из всех линейных функционалов от броуновского движения. Если $W_t$ — стандартное броуновское движение на $[0,T]$, то $\mathcal{H}_1$ изоморфен пространству $L^2([0,T])$:

$$\mathcal{H}_1 = \left\{ \int_0^T f(t) dW_t : f \in L^2([0,T]) \right\}$$

Случайная величина из $\mathcal{H}_1$ имеет гауссово распределение с нулевым средним. Если $X = \int_0^T f(t) dW_t$, то $X \sim N(0, \|f\|_{L^2}^2)$ и

$$E[X^2] = \|f\|_{L^2([0,T])}^2 = \int_0^T f(t)^2 dt$$

Элементы разных хаосов ортогональны: если $X \in \mathcal{H}_m$ и $Y \in \mathcal{H}_n$ с $m \neq n$, то $E[XY] = 0$.

## Кратные стохастические интегралы

Для симметричной функции $f \in L^2([0,T]^n)$ кратный стохастический интеграл Ито определяется индуктивно:

$$I_n(f) = \int_0^T I_{n-1}(f(\cdot, t)) dW_t$$

где $I_0(f) = f$ для константы $f$. В явном виде:

$$I_n(f) = \int_0^T \int_0^{t_1} \cdots \int_0^{t_{n-1}} f(t_1, \ldots, t_n) dW_{t_n} \cdots dW_{t_2} dW_{t_1}$$

Интеграл берётся по упорядоченной области $0 < t_n < t_{n-1} < \cdots < t_1 < T$.

**Эквивалентное определение** через симметризацию: если $f$ не обязательно симметрична, то

$$I_n(f) = \frac{1}{n!} \int_0^T \int_0^T \cdots \int_0^T \tilde{f}(t_1, \ldots, t_n) dW_{t_n} \cdots dW_{t_1}$$

где $\tilde{f}$ — симметризация $f$ по всем переменным.

## Разложение Винера-Ито

Центральный результат теории хаоса Винера — разложение произвольной квадратично интегрируемой случайной величины в ортогональный ряд:

$$L^2(\Omega, \mathcal{F}_W, P) = \bigoplus_{n=0}^\infty \mathcal{H}_n$$

где $\mathcal{H}_n$ — это пространство случайных величин вида $I_n(f)$ для всех $f \in L^2([0,T]^n)$, симметричных относительно перестановки переменных. $\mathcal{H}_0 = \mathbb{R}$ — константы.

Для любого $X \in L^2(\Omega, \mathcal{F}_W, P)$ существует единственное разложение:

$$X = \sum_{n=0}^\infty I_n(f_n)$$

где $f_n$ — симметричные ядра в $L^2([0,T]^n)$, и ряд сходится в $L^2(\Omega)$. Энергетическое соотношение:

$$E[X^2] = \sum_{n=0}^\infty n! \|f_n\|_{L^2([0,T]^n)}^2$$

## Изометрия Ито для кратных интегралов

Ключевое свойство кратных интегралов — это расширение классической изометрии Ито:

$$E[I_n(f)^2] = n! \|f\|_{L^2([0,T]^n)}^2$$

для симметричной $f \in L^2([0,T]^n)$. Это означает, что оператор симметризации и нормировки $f \mapsto I_n(f)/\sqrt{n!}$ изометричен из $L^2([0,T]^n)$ в $L^2(\Omega)$.

Для несимметричной $f$ имеем:

$$E\left[\left(\int_0^T \cdots \int_0^T f(t_1, \ldots, t_n) dW_{t_1} \cdots dW_{t_n}\right)^2\right] = n! \|f_{\text{sym}}\|_{L^2([0,T]^n)}^2$$

где $f_{\text{sym}}$ — симметризация $f$.

## Оператор Орнштейна-Уленбека

Оператор Орнштейна-Уленбека $L$ действует на хаосе Винера как оператор сдвига на каждом уровне:

$$L I_n(f) = -n I_n(f)$$

То есть $\mathcal{H}_n$ состоит из собственных функций $L$ с собственным значением $-n$. Оператор определяется через математическое ожидание условное на истории:

$$L F = -E[D_t D_t^* F]$$

где $D_t$ — оператор Маллявена (см. ниже).

**Число-операторный подход**: эквивалентно, $L = -N$, где $N I_n(f) = n I_n(f)$ — оператор числа частиц (в терминах квантовой теории поля).

## Гиперсжатие (Hypercontractivity)

Теорема Нельсона о гиперсжатии устанавливает, что элементы $n$-го хаоса удовлетворяют нетривиальным неравенствам между $L^p$-нормами при $p > q > 1$.

Для $F \in \mathcal{H}_n$:

$$\|F\|_{L^q(\Omega)} \leq \left(\frac{q-1}{p-1}\right)^{n/2} \|F\|_{L^p(\Omega)}$$

при $q > p$. В частности, для $q = 4$, $p = 2$:

$$\|F\|_{L^4}^4 \leq 3 \|F\|_{L^2}^4$$

для $F \in \mathcal{H}_1$. Граница точна и достигается на линейных функциях.

Гиперсжатие используется для доказательства [[characteristic-functions|предельных теорем]] и в анализе свойств конвергенции в хаосе.

## Производная Маллявена

Производная Маллявена $D$ — это оператор, обратный кратному интегралу в определённом смысле. Для $F = I_n(f) \in \mathcal{H}_n$:

$$D_t F = n I_{n-1}(f(\cdot, t))$$

То есть производная понижает порядок хаоса на единицу. Явно:

$$D_t I_n(f) = n \int_0^T \cdots \int_0^T f(t, t_1, \ldots, t_{n-1}) dW_{t_1} \cdots dW_{t_{n-1}}$$

**Правило произведения**:

$$D_t(FG) = F D_t G + G D_t F$$

**Интегрирование по частям** (формула Кларка-Окона):

$$F = E[F] + \int_0^T E[D_t F | \mathcal{F}_t] dW_t$$

Пространство Соболева-Маллявена $\mathbb{D}^{1,2}$ состоит из случайных величин $F$ с $E[\|DF\|_{L^2([0,T])}^2] < \infty$, и производная Маллявена — замкнутый оператор на этом пространстве.

## Теорема о четвёртом моменте (Нуалара-Печчи)

Одно из наиболее изящных приложений теории хаоса — теорема Нуалара-Печчи, связывающая четвёртый момент с центральной предельной теоремой.

**Теорема**: Пусть $\{F_n\}_{n \geq 1}$ — последовательность в пространстве Соболева-Маллявена с $E[F_n^4] \to 3 E[F_n^2]^2$. Тогда $\frac{F_n - E[F_n]}{\sqrt{\text{Var}(F_n)}} \xrightarrow{d} N(0,1)$.

Если $F_n = I_n(f_n) \in \mathcal{H}_n$ — чистый $n$-й хаос, то $E[F_n^4] = 3(n!)^2 \|f_n\|_{L^2}^4$ (при нормировке), так что условие выполняется автоматически, и гауссова сходимость следует из самой структуры хаоса.

Для смешанных хаосов теорема даёт точное условие в терминах множественных интегралов.

## Приложения в финансовой математике

В [[stochastic-differential-equations|стохастическом финансировании]] хаос Винера обеспечивает формальное разложение цены деривативов. Формула Дюпира выражает волатильность через частные производные цены:

$$\sigma(K, T) = \frac{\partial C / \partial T + r K (\partial C / \partial K)}{K^2 (\partial^2 C / \partial K^2)}$$

где $C(K,T)$ — цена call-опциона. Эта формула допускает разложение через хаос для анализа локальной волатильности и калибровки моделей.

Опциональные разложения и представления через хаос используются при наличии информационной асимметрии ([[information-theory|теория информации]]) и для построения хеджирующих стратегий.

## Связь с другими структурами

**Характеристические функции** ([[characteristic-functions|функции]]) элементов хаоса часто имеют явные формы. Для $F \in \mathcal{H}_n$:

$$E[e^{i\theta F}] = e^{-\frac{\theta^2}{2} E[F^2]} P_{n-1}\left(\frac{\theta^2}{E[F^2]}\right)$$

где $P_{n-1}$ — специальный полином, определяемый через полиномы Эрмита.

**Связь с [[malliavin-calculus|исчислением Маллявена]]**: весь формализм производной Маллявена строится на основе хаоса Винера, и изучение регулярности интегралов, формулы Кларка-Окона и вероятностных представлений полагается на структуру хаоса.

**[[stratonovich-integral|Интегралы Стратоновича]]** связаны с интегралами Ито соотношением симметризации, которое имеет естественное выражение через кратные интегралы разных хаосов.

**Квантовая теория поля**: фоковское пространство над $L^2([0,T])$ совпадает с хаосом Винера, и операторы рождения/уничтожения (creation/annihilation) соответствуют производным Маллявена и кратным интегралам.

## Заключение

Хаос Винера и кратные стохастические интегралы образуют мощный инструмент для анализа функционалов броуновского движения. Разложение Винера-Ито обеспечивает полную спектральную характеризацию пространства $L^2(\Omega)$ и позволяет применять гармонический анализ к вероятностным задачам. Теория имеет глубокие корни в гармоническом анализе, функциональном анализе и топологии, и её приложения охватывают финансовую математику, статистическую механику и квантовую механику.

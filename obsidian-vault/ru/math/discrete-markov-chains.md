# Дискретные марковские цепи

Марковская цепь — это случайный процесс, у которого **будущее зависит только от настоящего**, а не от полной истории. Это самая старая (А. А. Марков, 1906) и самая широко используемая модель «случайного процесса с памятью один шаг». Её достаточно, чтобы покрыть широкий класс реальных систем — от случайного блуждания до Google PageRank, от Метрополиса-Гастингса до моделей физических систем в равновесии.
[[shiryaev-probability|«Вероятности»]] Ширяева — самое полное введение в дискретные марковские цепи на русском.

## Определение

**Марковское свойство.** Случайная последовательность $(X_n)_{n \geq 0}$ называется **марковской цепью** со значениями в конечном или счётном пространстве состояний $S$, если для каждого $n$ и каждой последовательности $i_0, i_1, \ldots, i_{n+1} \in S$:

$$\mathbb{P}(X_{n+1} = i_{n+1} \mid X_0 = i_0, \ldots, X_n = i_n) = \mathbb{P}(X_{n+1} = i_{n+1} \mid X_n = i_n).$$

То есть «информация о прошлом не добавляет ничего к текущему состоянию» при предсказании следующего шага.

**Однородность по времени.** Если переходные вероятности $\mathbb{P}(X_{n+1} = j \mid X_n = i) = p_{ij}$ не зависят от $n$, цепь называется **однородной**. Все классические результаты — для однородных цепей.

**Матрица переходов.** Совокупность $P = (p_{ij})_{i,j \in S}$ — это **стохастическая матрица**: $p_{ij} \geq 0$ и $\sum_j p_{ij} = 1$ для каждого $i$. Каждая строка — распределение вероятностей следующего шага.

## Основные операции

**Распределение через $n$ шагов.** Если в момент 0 распределение состояний задано вектором $\mu_0$, то через $n$ шагов:

$$\mu_n = \mu_0 \cdot P^n.$$

То есть итерация цепи — это умножение распределения на $n$-ю степень матрицы переходов.

**Уравнения Чепмена-Колмогорова.** Для $n \leq m$:

$$p_{ij}^{(m)} = \sum_{k \in S} p_{ik}^{(n)} \cdot p_{kj}^{(m-n)},$$

где $p_{ij}^{(n)} = (P^n)_{ij}$. Это переписанное умножение матриц на языке вероятностей: чтобы попасть из $i$ в $j$ за $m$ шагов, нужно за $n$ шагов попасть в какое-то промежуточное $k$, а потом за $m-n$ шагов из $k$ в $j$.

## Классификация состояний

Состояния цепи делятся на классы по динамическому поведению.

**Сообщающиеся состояния.** $i$ и $j$ **сообщаются**, если из $i$ можно попасть в $j$ за конечное число шагов и обратно. Это отношение эквивалентности; цепь делится на **классы сообщения**.

**Возвратные / невозвратные.** Состояние $i$ называется **возвратным** (recurrent), если $\mathbb{P}(\text{вернуться в } i \text{ хотя бы один раз} \mid X_0 = i) = 1$. Иначе — **невозвратное** (transient).

- Для возвратного состояния число возвращений = $\infty$ почти наверное.
- Для невозвратного — конечно почти наверное.

**Положительно/нулево возвратные.** Возвратное состояние называется **положительно возвратным**, если ожидаемое время возврата $\mathbb{E}[\text{время возврата} \mid X_0 = i] < \infty$. Иначе — **нулево возвратным**.

Только положительно возвратные цепи имеют (нетривиальное) **стационарное распределение**.

## Стационарное распределение

**Определение.** Распределение $\pi$ на $S$ называется **стационарным** (или **инвариантным**), если

$$\pi \cdot P = \pi,$$

то есть $\pi$ — левый собственный вектор $P$ с собственным значением 1. В компонентах: $\pi_j = \sum_i \pi_i p_{ij}$.

Интерпретация: если в момент 0 распределение состояний цепи равно $\pi$, то и в каждый следующий момент оно равно $\pi$. Цепь «находится в равновесии».

**Существование и единственность.** Для конечной неразложимой (irreducible) цепи стационарное распределение существует и единственно. Для счётной — существует тогда и только тогда, когда цепь положительно возвратная.

## Эргодическая теорема для цепей

**Теорема.** Пусть $(X_n)$ — конечная неразложимая апериодическая марковская цепь с матрицей переходов $P$ и стационарным распределением $\pi$. Тогда:

1. **Сходимость к стационарному**: $p_{ij}^{(n)} \to \pi_j$ при $n \to \infty$ для всех $i, j$.
2. **Сходимость средних**: $\frac{1}{n} \sum_{k=1}^n f(X_k) \to \sum_j \pi_j f(j)$ почти наверное для любой функции $f$.

Это — частный случай [[ergodic-theorems-birkhoff|эргодической теоремы Биркгофа-Хинчина]] для конкретного случая марковского процесса.

**Скорость сходимости** определяется **спектральным зазором** $1 - |\lambda_2(P)|$, где $\lambda_2$ — второй по модулю собственный спектр $P$. Чем больше зазор, тем быстрее «забывается» начальное распределение.

## Апериодичность

Состояние $i$ называется **периодическим с периодом $d$**, если $\gcd\{n : p_{ii}^{(n)} > 0\} = d$. Цепь **апериодична**, если $d = 1$ для всех состояний.

Без апериодичности сходимость к стационарному распределению может нарушаться в чисто детерминистическом смысле: например, цепь, чередующая два состояния («туда-сюда»), не сходится к равновесию, хотя имеет стационарное распределение.

## Применения

**1. PageRank.** Алгоритм Google для ранжирования веб-страниц — это вычисление стационарного распределения марковской цепи на графе ссылок. Стационарное распределение — это «вес» каждой страницы в равновесии случайного блуждания по ссылкам.

**2. MCMC (Markov Chain Monte Carlo).** Алгоритмы Метрополиса-Гастингса, Гиббса, HMC — это построение марковской цепи с **заранее заданным** стационарным распределением и использование её для сэмплирования из этого распределения. Главный инструмент байесовской статистики.

**3. Скрытые марковские модели (HMM).** Модель сигнала, где наблюдаемая последовательность — это «шумная» функция от скрытой марковской цепи. Применения: распознавание речи, биоинформатика, NLP, технический анализ финансовых временных рядов. См. [[hidden-markov-models]].

**4. Финансовые модели режимов.** «Регимные» модели волатильности (например, Гамильтон 1989) предполагают, что рынок переключается между несколькими «режимами» (бычий, медвежий, кризис) по марковской цепи.

**5. Классическая физика и химия.** Модели реакций, диффузии, броуновского движения в дискретном времени — все это марковские цепи на конечных или счётных пространствах состояний.

**6. Reinforcement Learning.** Среда в RL обычно моделируется как марковский процесс принятия решений (MDP) — обобщение марковской цепи с действиями агента и наградами. Все классические алгоритмы (Q-learning, value iteration) работают именно с этой структурой.

## Цепи с непрерывным временем

Цепи с непрерывным временем — обобщение, в котором переходы происходят в моменты пуассоновского процесса с интенсивностями $q_{ij}$. Они описываются **уравнениями Колмогорова** для матрицы переходных вероятностей $P(t)$:

$$\frac{d}{dt} P(t) = Q \cdot P(t), \quad P(0) = I,$$

где $Q$ — генератор (матрица интенсивностей). Решение: $P(t) = e^{tQ}$.

Это раздел теории, который Ширяев в основной книге не покрывает (есть отдельная литература), но он непрерывный аналог дискретной теории.

## Связь с мартингалами

Многие функционалы марковской цепи являются **мартингалами**. Например, для функции $f$, удовлетворяющей $\sum_j p_{ij} f(j) = f(i)$ (так называемая **гармоническая** функция для $P$):

$$M_n = f(X_n) - n \cdot 0 = f(X_n)$$

— мартингал. Эти связи позволяют применять [[discrete-martingales|мартингальные методы]] к марковским задачам — теоремы остановки, неравенства Доба и т. д.

Сложные интегралы требуют продвинутых методов.

### Интегрирование по частям
Выводится из правила дифференцирования произведения:
$\int u \, dv = uv - \int v \, du$

### Дробно-рациональные функции
Используется для интегрирования рациональных дробей путем их разложения на простейшие.

### Тригонометрическая подстановка
Преобразует алгебраические выражения с радикалами (например, $\sqrt{a^2 - x^2}$) в тригонометрические.
Эти методы применяются для вычисления объемов с помощью [[coordinate-systems|Систем координат]].

# Фильтр Калмана

Фильтр Калмана — это оптимальный рекурсивный алгоритм для оценки внутреннего состояния линейной динамической системы на основе серии зашумленных измерений. Это аналог [[hmm|Скрытой марковской модели (HMM)]] для непрерывного пространства состояний, где все распределения являются [[multivariate-normal-distribution|многомерными нормальными]].

## Линейная модель пространства состояний
Система описывается двумя линейными уравнениями с аддитивным гауссовским шумом:
1. **Переход состояний (Динамика):**
   $$ x_t = F x_{t-1} + B u_t + w_t, \quad w_t \sim \mathcal{N}(0, Q) $$
   где $x_t$ — скрытое состояние, $F$ — матрица перехода, $u_t$ — вектор управления, $w_t$ — шум процесса с ковариацией $Q$.
   
2. **Наблюдение (Измерение):**
   $$ z_t = H x_t + v_t, \quad v_t \sim \mathcal{N}(0, R) $$
   где $z_t$ — измерение, $H$ — матрица наблюдения, $v_t$ — шум измерения с ковариацией $R$.

## Рекурсивное байесовское обновление
Фильтр поддерживает распределение вероятности состояния в виде гауссианы $\mathcal{N}(\hat{x}_t, P_t)$. Он чередует два шага:

### 1. Шаг предсказания (Predict)
Проекция текущей оценки состояния вперед во времени.
- **Предсказание состояния:** $\hat{x}_{t|t-1} = F \hat{x}_{t-1|t-1} + B u_t$
- **Предсказание ковариации:** $P_{t|t-1} = F P_{t-1|t-1} F^T + Q$

### 2. Шаг обновления (Update)
Корректировка с учетом нового измерения $z_t$.
- **Невязка (Инновация):** $y_t = z_t - H \hat{x}_{t|t-1}$
- **Ковариация невязки:** $S_t = H P_{t|t-1} H^T + R$
- **Оптимальный коэффициент Калмана:** $K_t = P_{t|t-1} H^T S_t^{-1}$
- **Обновленная оценка состояния:** $\hat{x}_{t|t} = \hat{x}_{t|t-1} + K_t y_t$
- **Обновленная ковариация:** $P_{t|t} = (I - K_t H) P_{t|t-1}$

## Оптимальность
Для линейных систем с гауссовским шумом фильтр Калмана дает точную байесовскую оптимальную оценку. Для нелинейных систем применяются модификации: **Расширенный фильтр Калмана (EKF)** (линеаризация через [[multivariable-taylor-series|ряд Тейлора]]) или **Unscented Kalman Filter (UKF)**.

# Методы ресэмплинга: Бутстрэп и Джекнайф

Методы ресэмплинга представляют собой класс статистических методов, основанных на многократном извлечении подвыборок из имеющихся данных для оценки характеристик распределения статистики (таких как дисперсия, смещение или доверительные интервалы). Эти методы незаменимы, когда теоретическое распределение статистики неизвестно или когда асимптотические аппроксимации (например, на основе [[central-limit-theorem|ЦПТ]]) неточны для малых выборок.

## 1. Джекнайф (Jackknife)

Метод джекнайфа («складного ножа»), предложенный Морисом Кенуем в 1949 году и развитый Джоном Тьюки, основан на систематическом исключении одного наблюдения из выборки.

Пусть $\hat{\theta}$ — оценка параметра $\theta$ по выборке $X_1, \dots, X_n$. Пусть $\hat{\theta}_{(-i)}$ — та же оценка, вычисленная по выборке без $i$-го элемента.

### Оценка смещения (Bias Reduction)
Средняя оценка джекнайфа:
$$\hat{\theta}_{(\cdot)} = \frac{1}{n} \sum_{i=1}^n \hat{\theta}_{(-i)}$$
Оценка смещения по джекнайфу:
$$\widehat{Bias}_{jack} = (n-1)(\hat{\theta}_{(\cdot)} - \hat{\theta})$$

### Оценка дисперсии
$$\widehat{Var}_{jack}(\hat{\theta}) = \frac{n-1}{n} \sum_{i=1}^n (\hat{\theta}_{(-i)} - \hat{\theta}_{(\cdot)})^2$$

**Ограничения:** Джекнайф плохо работает для негладких статистик (например, для медианы или других квантилей), так как исключение одного элемента может вызвать резкий скачок значения.

## 2. Бутстрэп (Bootstrap)

Брэдли Эфрон в 1979 году предложил метод бутстрэпа как более универсальное обобщение джекнайфа. Основная идея — **принцип подстановки (substitution principle)**: если выборка репрезентативна, то эмпирическая функция распределения $\hat{F}$ является наилучшей оценкой истинного распределения $F$.

### Алгоритм
1. Из исходной выборки $X = \{x_1, \dots, x_n\}$ извлекается $B$ бутстрэп-выборок $X^*_b$ размера $n$ **с возвращением**.
2. Для каждой выборки вычисляется статистика $\hat{\theta}^*_b$.
3. Распределение $\{\hat{\theta}^*_b\}_{b=1}^B$ используется для оценки свойств $\hat{\theta}$.

### Доверительные интервалы
- **Перцентильный интервал:** Квантили распределения $\{\hat{\theta}^*_b\}$.
- **BCA-интервал (Bias-Corrected and Accelerated):** Учитывает смещение и асимметрию, обеспечивая более высокую точность.

## 3. Теоретическое обоснование

Бутстрэп сходится, если статистика является гладкой функцией от эмпирического распределения. Формально, это требует существования производной Адамара для соответствующего статистического функционала. 

Для многих классических статистик (среднее, дисперсия, коэффициенты регрессии) бутстрэп дает **асимптотическую точность второго порядка**, что делает его точнее методов, основанных на простой нормальной аппроксимации.

## Применения
- Оценка стандартных ошибок в сложных эконометрических моделях.
- Валидация алгоритмов машинного обучения (Bagging — Bootstrap Aggregating).
- Построение доверительных интервалов для нелинейных параметров в финансах.

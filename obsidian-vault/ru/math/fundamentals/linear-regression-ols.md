# Линейная регрессия и МНК: Основа прогнозного моделирования

Линейная регрессия — это простейшая и наиболее интерпретируемая форма регрессии. Она предполагает линейную зависимость между входными переменными ($X$) и выходной переменной ($y$). **Метод наименьших квадратов (МНК)** — это стандартный способ оценки параметров, минимизирующий сумму квадратов отклонений между наблюдаемыми и предсказанными значениями.

## 1. Уравнение модели

Для набора из $n$ наблюдений модель выглядит так:
$$y = X\beta + \epsilon$$
- $y$: Вектор целевых значений ($n \times 1$).
- $X$: Матрица входных признаков ($n \times p$), включая столбец единиц для свободного члена (intercept).
- $\beta$: Вектор коэффициентов (параметров), которые нужно найти.
- $\epsilon$: Случайная ошибка (шум).

## 2. Метод наименьших квадратов (МНК)

Мы ищем такие $\beta$, которые минимизируют **остаточную сумму квадратов (RSS)**:
$$RSS(\beta) = \sum_{i=1}^n (y_i - x_i^T\beta)^2 = \|y - X\beta\|^2$$

### Нормальное уравнение
Приравняв производную RSS к нулю, мы получаем аналитическое решение:
$$\hat{\beta} = (X^T X)^{-1} X^T y$$

**Геометрическая интерпретация**: Предсказанные значения $\hat{y} = X\hat{\beta}$ являются **ортогональной проекцией** вектора целей $y$ на подпространство, образованное столбцами матрицы $X$. Остатки $e = y - \hat{y}$ ортогональны пространству признаков.

## 3. Условия Гаусса-Маркова

Чтобы МНК-оценка была лучшей линейной несмещенной оценкой (BLUE):
1. **Линейность**: Зависимость линейна по параметрам.
2. **Экзогенность**: $E[\epsilon | X] = 0$ (Нет корреляции между ошибкой и признаками).
3. **Гомоскедастичность**: Постоянная дисперсия ошибок.
4. **Отсутствие мультиколлинеарности**: Столбцы $X$ должны быть линейно независимы.

## 4. Метрики оценки

- **R-квадрат ($R^2$)**: Доля дисперсии $y$, объясненная моделью.
- **Среднеквадратичная ошибка (MSE)**: Среднее значение квадратов остатков.
- **t-статистика**: Проверка того, значимо ли отличается конкретный коэффициент $\beta_j$ от нуля.

## 5. Расширения: Регуляризация

Если матрица $X^T X$ необратима (из-за мультиколлинеарности) или для предотвращения переобучения:
- **Гребневая регрессия (Ridge)**: Добавляет $L_2$-штраф $\|\beta\|^2$.
- **Lasso-регрессия**: Добавляет $L_1$-штраф $|\beta|$, что приводит к разреженным решениям (отбор признаков).

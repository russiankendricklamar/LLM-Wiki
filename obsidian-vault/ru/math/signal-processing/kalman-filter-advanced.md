Стандартный фильтр Калмана оценивает состояние линейной системы с гауссовским шумом.

### Расширенный фильтр Калмана (EKF)
Для нелинейных систем EKF линеаризует модели вокруг текущей оценки с помощью Якобианов:
$F_k = \left. \frac{\partial f}{\partial x} \right|_{\hat{x}_{k-1}}, \quad H_k = \left. \frac{\partial h}{\partial x} \right|_{\hat{x}_{k|k-1}}$
Якобианы используются для обновления ковариации.

### Unscented Kalman Filter (UKF)
Линеаризация может давать большие ошибки. UKF использует **Unscented Transform**: детерминированно выбирает набор "сигма-точек", пропускает их через нелинейную функцию и восстанавливает среднее и ковариацию. Это дает точность ряда Тейлора 3-го порядка без вычисления Якобианов.

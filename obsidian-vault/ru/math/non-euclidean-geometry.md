# Неевклидова геометрия

**Неевклидова геометрия** возникла в XIX веке, когда математики независимо (Бойяи, Лобачевский, Риман) показали, что **пятый постулат Евклида** не является необходимым следствием остальных аксиом — и что существуют непротиворечивые геометрии, в которых он нарушается.

## Пятый постулат Евклида

> Через точку, не лежащую на прямой, можно провести **ровно одну** параллельную прямую.

2000 лет математики пытались доказать его из первых четырёх постулатов. Оказалось, это невозможно — он независим.

## Три геометрии

| | Евклидова | Гиперболическая | Эллиптическая |
|--|-----------|-----------------|---------------|
| Параллельные | Ровно одна | Бесконечно много | Ни одной |
| Кривизна $K$ | $K = 0$ | $K < 0$ | $K > 0$ |
| Сумма углов △ | $= \pi$ | $< \pi$ | $> \pi$ |
| Модель | Плоскость | Диск Пуанкаре | Сфера |

## Гауссова кривизна

Для поверхности в $\mathbb{R}^3$ кривизна в точке:

$$K = \kappa_1 \cdot \kappa_2$$

где $\kappa_1, \kappa_2$ — главные кривизны. **Теорема Гаусса Egregium**: $K$ — внутренняя характеристика, не зависящая от вложения.

## Гиперболическая геометрия

В **модели диска Пуанкаре** $\mathbb{D} = \{(x,y): x^2+y^2 < 1\}$:

- Прямые — дуги окружностей, ортогональных граничной окружности
- Метрика: $ds^2 = \frac{4(dx^2+dy^2)}{(1-x^2-y^2)^2}$
- Расстояние от центра до границы = $\infty$

$$d(0, r) = 2\tanh^{-1}(r) = \ln\frac{1+r}{1-r} \xrightarrow{r\to 1} \infty$$

```python
import numpy as np
import matplotlib.pyplot as plt

def poincare_distance(z1: complex, z2: complex) -> float:
    """Гиперболическое расстояние в диске Пуанкаре."""
    num = abs(z1 - z2)
    den = abs(1 - z1.conjugate() * z2)
    return 2 * np.arctanh(num / den)


def mobius(z: complex, a: complex) -> complex:
    """Изометрия диска Пуанкаре: Мёбиус-преобразование."""
    return (z - a) / (1 - a.conjugate() * z)


def triangle_angle_sum(a: complex, b: complex, c: complex) -> float:
    """
    Сумма углов гиперболического треугольника (< π).
    a, b, c — вершины в диске Пуанкаре.
    """
    def angle_at(v, u, w):
        # Касательные векторы к геодезическим из v
        du = mobius(u, v)
        dw = mobius(w, v)
        return abs(np.angle(dw / du))
    return angle_at(a, b, c) + angle_at(b, a, c) + angle_at(c, a, b)


def draw_poincare_disk(n_geodesics: int = 8):
    """Рисует диск Пуанкаре с геодезическими."""
    fig, ax = plt.subplots(1, 1, figsize=(6, 6))
    circle = plt.Circle((0, 0), 1, fill=False, color='black', lw=2)
    ax.add_patch(circle)

    for i in range(n_geodesics):
        angle = i * np.pi / n_geodesics
        # Геодезическая через центр = обычный диаметр
        ax.plot([-np.cos(angle), np.cos(angle)],
                [-np.sin(angle), np.sin(angle)], 'b-', alpha=0.5)

    ax.set_xlim(-1.1, 1.1); ax.set_ylim(-1.1, 1.1)
    ax.set_aspect('equal'); ax.set_title('Диск Пуанкаре')
    return fig
```

## Связь с физикой

Самое глубокое применение неевклидовой геометрии — **[[general-relativity|общая теория относительности]]** (Эйнштейн, 1915), в которой пространство-время — 4-мерное псевдориманово многообразие, кривизна которого определяется массой-энергией:

$$G_{\mu\nu} = \frac{8\pi G}{c^4} T_{\mu\nu}$$

Геодезические в искривлённом пространстве-времени — траектории свободно падающих тел. Риманова геометрия, построенная как раз для работы с внутренней кривизной из Theorema Egregium Гаусса, оказалась естественным языком гравитации. Без работ Римана у [[gravitational-waves-gr|гравитационных волн]] и [[asymptotic-spacetime|асимптотической структуры пространства-времени]] просто не было бы формализма.

Конформные компактификации, на которых строятся [[bms-group|группа BMS]] и [[celestial-holography|небесная голография]], тоже живут на неевклидовой территории: небесная сфера на световой бесконечности — риманова сфера со своей конформной геометрией.

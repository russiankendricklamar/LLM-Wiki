# Бутстрап

Бутстрап, введённый Эфроном в 1979 году, — это компьютерно-интенсивный метод переёшивания для оценки выборочного распределения статистики путём повторного переёшивания с возвращением из эмпирического распределения. Этот метод обходит ограничительные параметрические предположения и предоставляет обоснованные доверительные интервалы и тесты гипотез через эмпирическую аппроксимацию распределительных свойств.

## Эмпирический бутстрап

**Эмпирический бутстрап** или непараметрический бутстрап функционирует следующим образом: Дана выборка $X_1, \ldots, X_n$ из одного распределения с эмпирической функцией распределения $\hat{F}_n$. Мы конструируем бутстрап-реплики путём переёшивания $X_1^*, \ldots, X_n^* \sim \hat{F}_n$ с возвращением. Для статистики $T_n = g(X_1, \ldots, X_n)$ вычисляем бутстрап-реплики:

$$T_n^{*(b)} = g(X_1^{*(b)}, \ldots, X_n^{*(b)}), \quad b = 1, \ldots, B$$

**Бутстрап-распределение** приближает выборочное распределение $T_n$. При мягких условиях регулярности эмпирические квантили $\{T_n^{*(1)}, \ldots, T_n^{*(B)}\}$ сходятся к истинным квантилям $T_n$:

$$\mathbb{P}(T_n^* \leq t \mid \mathcal{D}_n) \xrightarrow{p} \mathbb{P}(T_n \leq t)$$

где $\mathcal{D}_n = (X_1, \ldots, X_n)$ и сходимость имеет место при $n, B \to \infty$.

## Теоретические основания

Состоятельность бутстрапа опирается на **функциональный дельта-метод** и теорию [[эмпирических-процессов]]. Определим бутстрап-эмпирическое распределение как $\mathbb{P}_{n,*}$. Ключевое наблюдение:

$$\sqrt{n}(\mathbb{P}_{n,*} - \hat{F}_n) \xrightarrow{d} G \quad \text{(бутстрап-эмпирический процесс)}$$

где $G$ — гауссов процесс с нулевым средним и ковариационной структурой, унаследованной от исходного эмпирического процесса. Для гладкого функционала $\Phi: \mathcal{D} \to \mathbb{R}$ (где $\mathcal{D}$ — пространство функций распределения),

$$\sqrt{n}(\Phi(\mathbb{P}_{n,*}) - \Phi(\hat{F}_n)) \approx \sqrt{n} D\Phi(\hat{F}_n)[G]$$

где $D\Phi$ — дифференциал Гато. Эта аппроксимация асимптотически распределения-свободна, когда $\Phi$ является Адамаром-дифференцируемой (Беран и Дюшарм).

## Условия состоятельности бутстрапа

Состоятельность бутстрапа требует **Адамаровой дифференцируемости** статистики в истинном распределении $F_0$:

$$\sup_{h \in \mathcal{C}([-1,1])} \left| \frac{\Phi((1-t)F_0 + t(F_0 + h)) - \Phi(F_0)}{t} - D\Phi(F_0)[h] \right| \to 0$$

при $t \to 0$. Это условие гарантирует, что:

$$\sqrt{n}(T_n - \theta_0) \xrightarrow{d} L, \quad \sqrt{n}(T_n^* - T_n \mid \mathcal{D}_n) \xrightarrow{d} L$$

где $L$ — предельное распределение и сходимость в бутстрап-мире отражает сходимость в реальности. Для статистик, которые лишь непрерывны (не Адамаром-дифференцируемы), бутстрап может не работать, как происходит с оценками максимального правдоподобия на границе пространства параметров или порядковыми статистиками.

## Конструирование доверительных интервалов

Существуют три основных метода построения доверительных интервалов бутстрапа:

**Метод процентилей:** Самый простой подход использует квантили бутстрапа непосредственно. Для доверительного интервала на уровне $100(1-\alpha)\%$:

$$[T_n^{*(\alpha/2)}, T_n^{*(1-\alpha/2)}]$$

где $T_n^{*(p)}$ обозначает $p$-й квантиль бутстрап-распределения. Этот метод является **ключевым** в смысле, что он зависит только от ранга $T_n$ внутри бутстрап-выборки, что делает его инвариантным к монотонным преобразованиям статистики.

**Базовый метод (процентиль-$t$):** Учитывает систематическое смещение и асимметрию, используя аппроксимацию на основе [[центральной-предельной-теоремы]]:

$$\left[ T_n - z_{1-\alpha/2} \cdot \widehat{\text{SE}}_{\text{boot}}, \quad T_n - z_{\alpha/2} \cdot \widehat{\text{SE}}_{\text{boot}} \right]$$

где $\widehat{\text{SE}}_{\text{boot}} = \sqrt{\text{Var}_*(T_n^*)}$ — стандартная ошибка бутстрапа. Этот интервал **инвариантен к сдвигам** и более точен для статистик с асимптотической нормальностью.

**Интервал BCa (с исправлением смещения и ускорением):** Учитывает как смещение, так и асимметрию в бутстрап-распределении через модифицированные квантили:

$$T_n^{*(p_1)} \leq \theta \leq T_n^{*(p_2)}$$

где $p_1, p_2$ — скорректированные квантили, включающие:
- **Коэффициент исправления смещения:** $\hat{z}_0 = \Phi^{-1}[\mathbb{P}_*(T_n^* < T_n)]$
- **Коэффициент ускорения:** $\hat{a} = \frac{\sum_i (T_n^{(-i)} - \bar{T}_n^{(-)})^3}{6[\sum_i (T_n^{(-i)} - \bar{T}_n^{(-)})^2]^{3/2}}$ (выведен из складного ножа [[influence-functions]])

Метод BCa достигает точности $O(n^{-1})$ в сравнении с $O(n^{-1/2})$ для методов процентилей и базового метода.

## Параметрический бутстрап

Когда предполагается параметрическая модель $\{f(x; \theta) : \theta \in \Theta\}$, **параметрический бутстрап** переёшивает из подогнанной модели, а не из эмпирического распределения. Дана оценка $\hat{\theta}$, выбираем $X_1^*, \ldots, X_n^* \sim f(\cdot; \hat{\theta})$ и вычисляем $T_n^*$ как прежде. Этот подход:

- Использует структуру модели для повышенной эффективности, когда модель правильна
- Может катастрофически не работать, если модель неправильна
- Предоставляет естественный фреймворк для проверки композитных гипотез и вычисления $p$-значений при нулевой гипотезе

Для проверки $H_0: \theta = \theta_0$ против альтернатив, параметрический бутстрап вычисляет:

$$p\text{-значение} = \frac{1}{B} \sum_{b=1}^B \mathbb{1}[|T_n^{*(b)}(\hat{\theta}_0)| \geq |T_n(\hat{\theta}_0)|]$$

где $\hat{\theta}_0$ — оценка максимального правдоподобия при нулевой гипотезе.

## Бутстрап для временных рядов: блочный и стационарный бутстрап

Стандартный бутстрап для независимых наблюдений не работает для зависимых данных. **Блочный бутстрап** (Кюнш 1989) разбивает временной ряд $X_1, \ldots, X_n$ на блоки длины $\ell$ и переёшивает блоки с возвращением. Это сохраняет временную зависимость до лага $\ell$:

$$\text{Блок } B_j = (X_{j\ell + 1}, \ldots, X_{j(\ell+1)})$$

Бутстрап-выборка: конкатенация $m = \lceil n/\ell \rceil$ случайно выбранных блоков. Выбор длины блока $\ell$ уравновешивает смещение (меньшее $\ell$ = менее полное сохранение зависимости) и дисперсию (большее $\ell$ = выше дисперсия из-за меньшего числа независимых блоков). Оптимальное $\ell \sim n^{1/3}$ при условиях на моменты.

**Стационарный бутстрап** (Политис–Романо): Решает проблемы краевых эффектов путём выбора длин блоков из геометрического распределения со средним $\bar{\ell}$. Каждый блок — случайное независимое рисование длины, производящее стационарную бутстрап-выборку:

$$\mathbb{P}(\text{длина блока} = k) = (1 - p)^{k-1} p, \quad p = 1/\bar{\ell}$$

Стационарный бутстрап более робастен к выбору $\bar{\ell}$ и сохраняет долгосрочную дисперсионную структуру слабо зависимых процессов эффективнее, чем методы с фиксированной длиной блока.

## Бутстрап для регрессии и обобщённых линейных моделей

Для регрессии $Y_i = m(X_i) + \epsilon_i$ с [[kernel-density-estimation]] или параметрическим сглаживанием существуют два подхода:

**Модельный (параметрический) бутстрап:** Подгонка модели регрессии, вычисление остатков $\hat{\epsilon}_i = Y_i - \hat{m}(X_i)$, переёшивание остатков с возвращением и конструирование бутстрап-наблюдений:

$$Y_i^* = \hat{m}(X_i) + \epsilon_i^*$$

Это предполагает, что ошибки одинаково распределены, что может не выполняться при гетероскедастичности.

**Бутстрап пар:** Переёшивание пар $(X_i, Y_i)$ непосредственно с возвращением. Это не требует параметрических предположений, но теряет информацию об упорядочивании и может быть менее эффективным для упорядоченных категорических предикторов. Бутстрап пар **робастен** к любой форме условной гетероскедастичности.

Для обобщённых линейных моделей с нелинейной связью $g(m(X_i)) = \eta(X_i)$, оба подхода естественно расширяются, с модельным бутстрапом использующим предсказанные $\mu_i^*$ и пуассоновским или бинарным переёшиванием.

## M-из-n бутстрап для нерегулярных статистик

Состоятельность бутстрапа нарушается для **нерегулярных статистик** типа максимума или размаха, где предельное распределение ненормально и параметр может находиться на границе пространства параметров. **M-из-n бутстрап**, предложенный Бикелем, Гётцем и ван Цветом, решает это проблему переёшиванием $m < n$ наблюдений:

$$T_m^{*(b)} = g(X_1^{*(b)}, \ldots, X_m^{*(b)}) \quad \text{где} \quad m = m(n) \to \infty, \quad m/n \to 0$$

Выбор $m \sim n^{2/3}$ оптимален для многих нерегулярных статистик, дающий состоятельность бутстрапа по распределению. M-из-n бутстрап особенно полезен для экстремальных статистик, порядковых статистик и оценок максимального правдоподобия на граничных точках.

## Связи со складным ножом и функциями влияния

Бутстрап тесно связан с **складным ножом** Кеноля и Тьюки. Складной нож вычисляет:

$$\hat{\theta}_{\text{jack}} = n \cdot T_n - (n-1) \cdot \frac{1}{n}\sum_{i=1}^n T_n^{(-i)}$$

где $T_n^{(-i)}$ исключает $i$-е наблюдение. Для гладких функционалов оценка дисперсии складного ножа:

$$\hat{\sigma}^2_{\text{jack}} = \frac{n-1}{n} \sum_{i=1}^n (T_n^{(-i)} - \bar{T}_n^{(-)})^2$$

приближает дисперсию бутстрапа до второго порядка. Оба метода захватывают влияние отдельных наблюдений через переёшивание, но бутстрап непосредственно приближает выборочное распределение, в то время как складной нож — приближение рядом Тейлора.

## Режимы отказа бутстрапа

Несмотря на его универсальность, бутстрап имеет несколько режимов отказа:

1. **Нерегулярные статистики:** Максимум, размах и оценки экстремальных значений могут дать несостоятельное бутстрап-распределение, когда истинный параметр находится на границе.

2. **Слабая асимптотика:** Для статистик с медленно сходящимися распределениями (например, отношения нормалей), бутстрап может быть неточен на конечных выборках даже если асимптотически состоятелен.

3. **Зависимые данные с долгой памятью:** Блочный бутстрап может быть неэффективным или несостоятельным для сильно зависимых процессов с гиперболически затухающей автокорреляцией ($\rho(k) \sim k^{-\alpha}$ с $0 < \alpha < 1$). [[concentration-inequalities]] плохо контролируют дисперсию бутстрапа в этом режиме.

4. **Высокоразмерные данные:** В высоких измерениях эмпирическое распределение становится всё более разреженным, и бутстрап-распределение может расходиться от истинного распределения даже при большом $n$.

5. **[[cramer-rao-bound|Информационно-теоретические границы]]:** Для статистик, которые суперэффективны (превосходят границу Крамера–Рао на множестве положительной меры), бутстрап не захватывает правильную асимптотическую дисперсию.

## Продвинутые темы и недавние разработки

**Сглаженный бутстрап:** Для непрерывных статистик сглаживание эмпирического распределения перед переёшиванием (например, через оценку ядерной плотности) может улучшить точность и ослабить краевые эффекты.

**Двойной бутстрап:** Конструирование доверительных интервалов для доверительных интервалов бутстрапа через вложенную процедуру бутстрапа для уточненных поправок высших порядков.

**Подвыборка и подсерийные методы:** Обобщения m-из-n, которые применяются к более общим зависимым структурам и слабым условиям сходимости, избегая необходимости асимптотической нормальности.

**Множительный бутстрап:** Взвешивание наблюдений независимыми случайными множителями $w_i \sim \text{Exp}(1)$ или Рэдемахер(1/2) вместо переёшивания, снижая вычислительную стоимость при сохранении асимптотической теории.

## Ключевые результаты — краткое резюме

1. **Состоятельность бутстрапа (Адамаров-дифференцируемый случай):** $\sqrt{n}(T_n - \theta_0) \stackrel{d}{=} \sqrt{n}(T_n^* - T_n \mid \mathcal{D}_n)$ по распределению
2. **Точность BCa:** Скорость ошибки $O(n^{-1})$ для доверительных интервалов против $O(n^{-1/2})$ для базовых методов
3. **Блочный бутстрап для смешивания:** Сохраняет долгосрочную дисперсию для слабо зависимых процессов с $\ell \sim n^{1/3}$
4. **M-из-n для нерегулярных:** Состоятельность восстановлена с $m = o(n)$ для максимума и граничных статистик
5. **Параметрический бутстрап:** Асимптотически верен при правильной спецификации модели; может отказать при ошибочной спецификации

Бутстрап — краеугольный камень современного статистического вывода, предоставляющий унифицированный фреймворк, который связывает теорию [[эмпирических-процессов]], [[u-statistics]], и [[локальной-асимптотической-нормальности]] через эмпирическую аппроксимацию.

## Литература

Эфрон (1979) ввёл бутстрап для общей оценки и проверки гипотез. Беран и Дюшарм (1991) установили условия [[функционального-дельта-метода]] для валидности бутстрапа. Кюнш (1989) разработал блочный бутстрап для временных рядов, а Политис и Романо (1994) ввели стационарный бутстрап. Бикель, Гётц и ван Цвет (1997) анализировали состоятельность m-из-n бутстрапа для нерегулярных статистик. Холл (1992) предоставляет всеобъемлющее лечение доверительных интервалов бутстрапа и [[concentration-inequalities]] о режимах отказа бутстрапа.

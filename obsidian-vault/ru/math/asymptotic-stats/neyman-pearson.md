# Теория Неймана–Пирсона

**Теория Неймана–Пирсона** — математическое основание оптимального статистического тестирования, разработанное Ежи Нейманом и Эгоном Пирсоном в 1930-х годах. Она отвечает на фундаментальный вопрос: при заданном уровне значимости $\alpha$, какой тест имеет максимальную мощность для различения двух гипотез? Ответ — выраженный в **лемме Неймана–Пирсона** — состоит в том, что оптимальный тест использует отношение правдоподобия. Этот результат заложил основы частотистского статистического вывода.

## Простые и сложные гипотезы

**Гипотеза** — это вероятностное предположение о механизме генерирования данных. Формально пространство параметров $\Theta$ разбивается на два непересекающихся множества: нулевая гипотеза $H_0: \theta \in \Theta_0$ и альтернатива $H_1: \theta \in \Theta_1$.

- **Простые гипотезы**: $H_0: \theta = \theta_0$ и $H_1: \theta = \theta_1$. Обе гипотезы задают одно единственное значение параметра; отношение правдоподобия вычисляется ровно.
- **Сложные гипотезы**: $\Theta_0$ или $\Theta_1$ содержат несколько значений параметра. Сложные нулевые гипотезы стандартны на практике (например, $H_0: \mu \leq \mu_0$ при тестировании среднего нормального распределения), но отношение правдоподобия становится функцией неизвестных параметров.

## Размер и мощность теста

Тест $\phi: \mathcal{X} \to [0, 1]$ (где $\phi(x) \in [0,1]$ — вероятность отклонения $H_0$ при наблюдении $x$) имеет два критических свойства:

- **Ошибка первого рода (размер)**: $\alpha = \Pr_{\theta_0}(\text{отклонить } H_0) = \mathbb{E}_{\theta_0}[\phi(X)]$. Её контролируют на предзаданном уровне, например $\alpha = 0{,}05$.
- **Ошибка второго рода (мощность)**: $\beta = \Pr_{\theta_1}(\text{принять } H_0) = 1 - \mathbb{E}_{\theta_1}[\phi(X)]$. **Мощность теста** — это $1 - \beta = \mathbb{E}_{\theta_1}[\phi(X)]$, вероятность корректного обнаружения альтернативы.

Рамка Неймана–Пирсона фиксирует размер на уровне $\alpha$ и максимизирует мощность: найти тест, который отклоняет $H_0$ как можно чаще под альтернативой $H_1$, сохраняя редкость ложных отклонений.

## Лемма Неймана–Пирсона

**Теорема**: Для тестирования $H_0: \theta = \theta_0$ против $H_1: \theta = \theta_1$ с размером $\alpha$ наиболее мощный тест — это **тест отношения правдоподобия**:

$$\phi^*(x) = \begin{cases} 1 & \text{если } \frac{L(\theta_1; x)}{L(\theta_0; x)} > c \\ \gamma & \text{если } \frac{L(\theta_1; x)}{L(\theta_0; x)} = c \\ 0 & \text{если } \frac{L(\theta_1; x)}{L(\theta_0; x)} < c \end{cases}$$

где $c$ и $\gamma \in [0, 1]$ выбираются так, чтобы $\Pr_{\theta_0}(\phi^*(X) = 1) + \gamma \Pr_{\theta_0}(\phi^*(X) = \gamma) = \alpha$.

**Набросок доказательства**: Предположим, другой тест $\psi$ достигает размера $\alpha$ и мощности $\mathbb{E}_{\theta_1}[\psi(X)] > \mathbb{E}_{\theta_1}[\phi^*(X)]$. Определим множество $A = \{x : \phi^*(x) = 1\}$ (область отклонения). По конструкции $\phi^*$,
$$\mathbb{E}[\psi(X) - \phi^*(X)] \leq c \mathbb{E}[\phi^*(X) - \psi(X)]$$
под $\theta_0$. Но оба теста имеют размер $\alpha$, поэтому это неравенство становится равенством, противоречие с $\mathbb{E}_{\theta_1}[\psi(X)] > \mathbb{E}_{\theta_1}[\phi^*(X)]$, если только $\psi \neq \phi^*$ почти всюду.

## Сложные гипотезы: равномерно наиболее мощные тесты

Когда гипотезы сложные, ищут **равномерно наиболее мощный (РНМ) тест**: область отклонения, которая максимизирует мощность одновременно для всех параметров под альтернативой $H_1$.

**Монотонное отношение правдоподобия**: Семейство плотностей $\{f_\theta\}$ обладает монотонным отношением правдоподобия в достаточной статистике $T(X)$, если
$$\frac{f_{\theta_1}(x)}{f_{\theta_0}(x)} \text{ монотонно растёт как функция } T(x)$$
для любых $\theta_1 > \theta_0$.

**Теорема Карлина–Рубина**: Если $\{f_\theta\}$ имеет монотонное отношение правдоподобия в $T(X)$, то для тестирования $H_0: \theta \leq \theta_0$ против $H_1: \theta > \theta_0$ тест
$$\phi(x) = \begin{cases} 1 & \text{если } T(x) > c \\ 0 & \text{если } T(x) \leq c \end{cases}$$
является РНМ на уровне $\alpha$ (где $c$ выбирается из условия $\Pr_{\theta_0}(T(X) > c) = \alpha$).

## РНМ несмещённые тесты и экспоненциальные семейства

Для [[exponential-families|экспоненциальных семейств]] рамка Неймана–Пирсона расширяется естественно. Экспоненциальное семейство
$$f_\theta(x) = h(x) \exp\left\{\eta(\theta) T(x) - A(\theta)\right\}$$
допускает РНМ несмещённые тесты для двусторонних гипотез на $\eta(\theta)$. Область отклонения обычно строится путём инверсии отношения правдоподобия, включающего как достаточную статистику $T(X)$, так и мешающий параметр, что приводит к точным тестам (например, точный тест Фишера для таблиц $2 \times 2$).

## Рандомизированные тесты и дискретные распределения

Для дискретных распределений граничный случай, когда отношение правдоподобия равно пороговому значению $c$, может иметь положительную вероятность. Лемма Неймана–Пирсона это допускает посредством **рандомизированных тестов**: с вероятностью $\gamma$ мы случайным образом отклоняем или принимаем гипотезу, когда отношение правдоподобия равно $c$. Хотя рандомизация кажется искусственной, она необходима для достижения ровно размера $\alpha$ — нерандомизированный тест может достичь только размер $\leq \alpha$.

## P-значения как тестовые статистики

Ключное понимание в современной практике: любая тестовая статистика $S(X)$ порождает **p-значение**
$$p(X) = \inf\{\alpha : \text{уровневый-}\alpha\text{ тест отклоняет } H_0\}$$
или эквивалентно, для теста отношения правдоподобия, $p(X) = \Pr_{\theta_0}(S(X') \geq S(X))$, где неравенство в направлении, благоприятном для $H_1$. Само p-значение становится основой для вывода: если $p(X) < \alpha$, отклоняем; если $p(X) \geq \alpha$, не отклоняем. Это превращает рецепт Неймана–Пирсона в правило решения, параметризованное одной наблюдаемой величиной.

## Эффективность тестов и асимптотика мощности

В пределе при размере выборки $n \to \infty$ свойства мощности тестов управляются информационно-теоретическими величинами.

**Эффективность по Бахадуру** измеряет скорость расхождения отношения правдоподобия под альтернативой:
$$\limsup_{n \to \infty} \frac{\log p_n}{\log p^*_n}$$
где $p_n$ — p-значение для данного теста, а $p^*_n$ — p-значение для наилучшего возможного теста. Дивергенция Кульбака–Лейблера $D_{\text{KL}}(\theta_1 || \theta_0)$ управляет этой скоростью.

**Эффективность по Питмену** сравнивает требуемые размеры выборок для достижения фиксированной мощности. При [[local-asymptotic-normality|локальной асимптотической нормальности]] она зависит от информации Фишера $I(\theta)$ и часто вычисляется через предельную мощность теста под локальной альтернативой $\theta = \theta_0 + c/\sqrt{n}$.

## Связь с information-geometry

Геометрия статистического тестирования возникает из рамки Неймана–Пирсона: метрика информации Фишера кодирует «расстояние» между значениями параметра таким образом, который определяет оптимальное тестирование. Скорости сжатия и асимптотическая мощность интимно связаны с геометрией модели. Тесты, достигающие наиболее быстрых скоростей сжатия относительно расстояния Кульбака–Лейблера до истины, оптимальны в сильном смысле.

## Связи с [[sequential-hypothesis-testing|последовательным тестированием]] и [[cramer-rao-bound|границей Крамера–Рао]]

[[cramer-rao-bound|Граница Крамера–Рао]] устанавливает нижнюю границу дисперсии любой несмещённой оценки; двойственно, она задаёт нижнюю границу на информационное содержание, необходимое для различения гипотез. Последовательное тестирование обобщает теорию Неймана–Пирсона на моменты остановки, позволяя самому размеру выборки быть случайным, сохраняя при этом гарантии размера и мощности — тест Вальда SPRT является каноническим примером.

## Ограничения и расширения

Классическая теория Неймана–Пирсона предполагает:
- Точное знание нулевого распределения.
- Чёткую спецификацию альтернативы.
- Независимость наблюдений.

Современные расширения адресуют:
- **Сложные нулевые гипотезы** с мешающими параметрами посредством условного правдоподобия или методов профильного правдоподобия.
- **Высокомерные режимы**, когда $p \gg n$, где центральную роль играют [[empirical-processes|эмпирические процессы]] и [[local-asymptotic-normality|локальная асимптотическая нормальность]].
- **Нерегулярные модели** (например, обнаружение точки перелома, модели смеси), где классическая асимптотика нарушается.

Несмотря на свой возраст, рамка Неймана–Пирсона остаётся фундаментальной: каждый статистический тест оценивается относительно идеала максимизации мощности при фиксированном размере, и принцип отношения правдоподобия, который она утверждает, универсален в научном выводе.

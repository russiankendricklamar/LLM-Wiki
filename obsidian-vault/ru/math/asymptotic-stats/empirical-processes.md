## Определение и основной объект

Пусть $X_1, \ldots, X_n$ — независимые одинаково распределённые наблюдения с неизвестным распределением $P$, и $\mathcal{F}$ — класс функций $f: \mathcal{X} \to \mathbb{R}$. **Эмпирический процесс** определяется как:

$$G_n(f) = \frac{1}{\sqrt{n}} \sum_{i=1}^n \left( f(X_i) - \mathbb{E}[f(X_i)] \right), \quad f \in \mathcal{F}$$

Это случайный элемент пространства $\ell^\infty(\mathcal{F})$ — пространства ограниченных функций на $\mathcal{F}$ с нормой супремума. Ключевая идея: вместо изучения асимптотики отдельных статистик мы изучаем поведение $G_n$ как целого процесса, что позволяет получать **равномерные** результаты сразу для всего класса $\mathcal{F}$.

## Теорема Гливенко—Кантелли

Для класса $\mathcal{F}$ справедлив **закон больших чисел в равномерной норме** если, обозначив эмпирическое распределение как $F_n$:

$$\sup_{f \in \mathcal{F}} \left| \frac{1}{n} \sum_{i=1}^n f(X_i) - \mathbb{E}[f(X_i)] \right| \xrightarrow{a.s.} 0$$

**Класс $\mathcal{F}$ называется Гливенко—Кантельи (GC-класс)**, если это свойство выполняется. Классический пример: $\mathcal{F} = \{1_{(-\infty, t]}: t \in \mathbb{R}\}$, где сходимость сводится к равномерной сходимости эмпирической функции распределения $F_n(t)$ к теоретической $F(t)$:

$$\sup_{t \in \mathbb{R}} |F_n(t) - F(t)| \xrightarrow{a.s.} 0$$

Условие Гливенко—Кантелли существенно слабее, чем требуемое для функциональной центральной предельной теоремы.

## Теорема Донскера и функциональный ЦПТ

**Теорема Донскера** — функциональная версия центральной предельной теоремы для эмпирических процессов. Если класс $\mathcal{F}$ — **класс Донскера**, то:

$$G_n \Rightarrow G_P \quad \text{в } \ell^\infty(\mathcal{F})$$

где $G_P$ — **гауссовский процесс** с ковариационной функцией:

$$\mathrm{Cov}(G_P(f), G_P(g)) = \mathbb{E}[f(X)g(X)] - \mathbb{E}[f(X)]\mathbb{E}[g(X)] = \mathrm{Cov}(f(X), g(X))$$

Для функций индикаторов на $\mathbb{R}$, то есть $\mathcal{F} = \{1_{(-\infty, t]}: t \in \mathbb{R}\}$, гауссовский процесс $G_P$ — **броуновский мост** $B(t) = W(F(t)) - F(t)W(1)$, где $W$ — стандартный винеровский процесс.

## Энтропия и скобочные числа

Поведение эмпирического процесса определяется **сложностью** класса $\mathcal{F}$, измеряемой через **числа покрытия** и **скобочные числа**.

**Число покрытия** $N(\mathcal{F}, L^2(P), \varepsilon)$ — минимальное число $L^2$-шаров радиуса $\varepsilon$ (относительно нормы $\|f\|_{L^2} = \sqrt{\mathbb{E}[f^2]}$), необходимых для покрытия класса $\mathcal{F}$.

**Скобочное число** $N_{[\,]}(\varepsilon, \mathcal{F}, L^2(P))$ — минимальное число пар $(l_j, u_j)$ скобок $l_j \le f \le u_j$ с $\mathbb{E}[u_j - l_j] \le \varepsilon$, покрывающих весь класс.

## Условие Донскера через энтропийный интеграл

Достаточное условие для принадлежности классом Донскера:

$$\int_0^\infty \sqrt{\log N_{[\,]}(\varepsilon, \mathcal{F}, L^2(P))} \, d\varepsilon < \infty$$

Это условие охватывает практически все классы, встречающиеся в приложениях:
- Гладкие параметрические семейства (конечномерные)
- Монотонные функции
- Функции с ограниченной вариацией
- **VC-классы** (обсуждаются ниже)

## VC-классы и размерность Вапника—Червоненкиса

Для класса **множеств** $\mathcal{C} = \{C: \mathcal{X} \to \{0,1\}\}$, **размерность VC** определяется как максимальный размер множества, которое может быть **расколото** (shattered) — то есть существует набор из $d$ точек такой, что для каждого из $2^d$ подмножеств найдётся множество $C \in \mathcal{C}$ с $C \cap \{\text{points}\}$ равным данному подмножеству.

Если $\text{VC}(\mathcal{C}) = d$, то:

$$\log N(\mathcal{C}, L^2(P), \varepsilon) \le d \log\left(\frac{2}{\varepsilon}\right)$$

**Фундаментальный результат**: VC-классы — это Донскеровы классы. Это означает, что для любого VC-класса справедлив функциональный ЦПТ:

$$\sqrt{n}(P_n - P) \Rightarrow G_P$$

где сходимость равномерна по всем множествам в классе. Это обеспечивает **равномерный контроль** над эмпирическим риском.

## Радемахерова сложность и обобщение

**Радемахерова сложность** класса $\mathcal{F}$:

$$\mathcal{R}_n(\mathcal{F}) = \mathbb{E}_\sigma\left[\sup_{f \in \mathcal{F}} \left|\frac{1}{n}\sum_{i=1}^n \sigma_i f(X_i)\right|\right]$$

где $\sigma_1, \ldots, \sigma_n$ — независимые случайные переменные Радемахера (равномерно принимающие значения $\pm 1$).

Через симметризацию:

$$\mathbb{E}\left[\sup_{f \in \mathcal{F}} G_n(f)\right] \le 2\sqrt{n} \cdot \mathcal{R}_n(\mathcal{F})$$

Это позволяет получить **гарантии обобщения** в машинном обучении:

$$\mathbb{E}_S[\sup_{f \in \mathcal{F}} (R(f) - \hat{R}_S(f))] \le 2\mathcal{R}_n(\mathcal{F})$$

где $R(f)$ — истинный риск, а $\hat{R}_S(f)$ — эмпирический риск на выборке $S$.

## Приложение к машинному обучению

Теория эмпирических процессов служит фундаментом статистического обучения. Для гипотезного класса $\mathcal{H}$ с VC-размерностью $d$:

$$\text{Bound}_{\text{gen}} = O\left(\sqrt{\frac{d \log n}{n}}\right) + \text{union bound correction}$$

гарантирует, что если эмпирический риск мал, то истинный риск также мал с высокой вероятностью. Это объясняет, почему классы с малой сложностью (малая размерность VC, малая радемахерова сложность) лучше обобщаются.

## Бутстреп для эмпирического процесса

Эмпирический процесс можно оценивать данными с помощью **бутстрепа**. Если $X_1^*, \ldots, X_n^*$ — выборка с возвращением из $X_1, \ldots, X_n$, и $P_n^*$ — соответствующая эмпирическая мера, то:

$$G_n^* = \sqrt{n}(P_n^* - P_n) \Rightarrow G_P$$

в том же смысле, что и $G_n$. Это позволяет оценивать распределение процесса без знания истинного $P$, что критично для построения доверительных интервалов в высокомерных регрессионных задачах.

## Связанные концепции

Теория эмпирических процессов обобщается на [[weak-convergence-prokhorov|слабую сходимость в пространствах Прохорова]], что необходимо для изучения процессов с зависимостями. [[local-asymptotic-normality|Локальная асимптотическая нормальность (LAN)]] уточняет поведение отношения правдоподобия через эмпирические процессы. В задачах [[pac-learning|PAC-обучения]] радемахерова сложность заменяет VC-размерность для более точных гарантий.

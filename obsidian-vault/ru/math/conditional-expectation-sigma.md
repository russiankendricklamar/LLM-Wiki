# Условное математическое ожидание относительно σ-алгебры

Условное матожидание $\mathbb{E}[X \mid \mathcal{G}]$ — это **самый полезный объект** в теории вероятностей после самой вероятности. Это «лучшее предсказание $X$, использующее только информацию из σ-алгебры $\mathcal{G}$». В отличие от элементарного определения «при условии события $B$» (которое работает только при $\mathbb{P}(B) > 0$), колмогоровская версия позволяет обуславливать **на любую σ-алгебру** — даже на ту, которая порождена непрерывной случайной величиной (для которой все индивидуальные события имеют нулевую вероятность).

Этому посвящён § 7 главы II [[shiryaev-probability|«Вероятности»]] Ширяева. Без условного матожидания нельзя сформулировать определение **мартингала**, теорему Фубини в её полном виде, теорему Радона-Никодима или [[girsanov-semimartingales|теорему Гирсанова]]. Это технический фундамент всей продвинутой теории вероятностей.

## Мотивация: проблема обусловливания на нулевое событие

Элементарное определение:

$$\mathbb{P}(A \mid B) = \frac{\mathbb{P}(A \cap B)}{\mathbb{P}(B)}, \quad \text{при } \mathbb{P}(B) > 0.$$

Это работает, пока $\mathbb{P}(B) > 0$. Но что делать в задаче «найти распределение $X$ при условии $Y = y$», если $Y$ — непрерывная случайная величина? Тогда $\mathbb{P}(Y = y) = 0$ для каждого $y$, и формула не определена.

Колмогоров в 1933 году обошёл эту проблему **глобально**, не пытаясь определять «$X$ при условии $Y = y$» для каждого $y$ отдельно, а сразу определяя случайную величину $\mathbb{E}[X \mid \sigma(Y)]$, которая «несёт всю условную информацию одновременно».

## Определение

**Определение.** Пусть $X$ — интегрируемая случайная величина ($\mathbb{E}|X| < \infty$) на $(\Omega, \mathcal{F}, \mathbb{P})$, $\mathcal{G} \subset \mathcal{F}$ — под-σ-алгебра. **Условное математическое ожидание** $\mathbb{E}[X \mid \mathcal{G}]$ — это любая случайная величина $Y$ такая, что:

1. $Y$ **измерима относительно $\mathcal{G}$**.
2. $\int_A Y \, d\mathbb{P} = \int_A X \, d\mathbb{P}$ для каждого $A \in \mathcal{G}$.

**Утверждение (Радон-Никодим).** Такая случайная величина $Y$ существует и единственна почти наверное.

Существование вытекает из теоремы Радона-Никодима (мера $A \mapsto \int_A X \, d\mathbb{P}$ абсолютно непрерывна относительно $\mathbb{P}$, ограниченной на $\mathcal{G}$). Единственность — из того, что две функции, имеющие одинаковые интегралы по всем $A \in \mathcal{G}$, равны почти наверное на $\mathcal{G}$.

## Интуиция

Условие 1 ($\mathcal{G}$-измеримость) означает: значение $Y$ полностью определяется информацией из $\mathcal{G}$. Если бы вы знали только $\mathcal{G}$, вы могли бы вычислить $Y$.

Условие 2 («среднее как у $X$ на каждом $\mathcal{G}$-событии») означает: на любом подмножестве, которое $\mathcal{G}$ способна различить, $Y$ совпадает с $X$ **в среднем**. Внутри каждого «класса неразличимости» $\mathcal{G}$ величина $Y$ — это **средняя** $X$ по этому классу.

Это в точности «лучшее усреднение $X$, использующее только информацию из $\mathcal{G}$».

## L²-проекция

Если $X \in L^2(\Omega, \mathcal{F}, \mathbb{P})$, то $\mathbb{E}[X \mid \mathcal{G}]$ — это **ортогональная проекция** $X$ на подпространство $L^2(\Omega, \mathcal{G}, \mathbb{P})$ ($\mathcal{G}$-измеримых функций). То есть:

$$\mathbb{E}[X \mid \mathcal{G}] = \arg\min_{Y \in L^2(\mathcal{G})} \mathbb{E}\big[(X - Y)^2\big].$$

Это **наилучшее предсказание $X$ в смысле среднеквадратичного отклонения** среди всех функций, использующих только информацию из $\mathcal{G}$.

В машинном обучении: если $\mathcal{G} = \sigma(Y)$ — σ-алгебра, порождённая признаками $Y$, то $\mathbb{E}[X \mid \mathcal{G}]$ — это идеальный (но не вычислимый напрямую) предиктор $X$ по $Y$. Любая регрессия — попытка приблизить эту функцию.

## Свойства

Условное матожидание удовлетворяет всем свойствам обычного матожидания плюс несколько новых:

1. **Линейность**: $\mathbb{E}[aX + bY \mid \mathcal{G}] = a \mathbb{E}[X \mid \mathcal{G}] + b \mathbb{E}[Y \mid \mathcal{G}]$.
2. **Сохранение полного среднего**: $\mathbb{E}[\mathbb{E}[X \mid \mathcal{G}]] = \mathbb{E}[X]$ — закон полного матожидания.
3. **«Вынесение известного»**: если $Y$ $\mathcal{G}$-измерима, то $\mathbb{E}[XY \mid \mathcal{G}] = Y \cdot \mathbb{E}[X \mid \mathcal{G}]$.
4. **Башенное свойство**: если $\mathcal{H} \subset \mathcal{G}$, то $\mathbb{E}[\mathbb{E}[X \mid \mathcal{G}] \mid \mathcal{H}] = \mathbb{E}[X \mid \mathcal{H}]$ — «меньшая информация выигрывает».
5. **Независимость**: если $X$ независима от $\mathcal{G}$, то $\mathbb{E}[X \mid \mathcal{G}] = \mathbb{E}[X]$ — константа.
6. **Неравенство Йенсена**: для выпуклой $\varphi$, $\varphi(\mathbb{E}[X \mid \mathcal{G}]) \leq \mathbb{E}[\varphi(X) \mid \mathcal{G}]$.
7. **Контрактивность**: $\|\mathbb{E}[X \mid \mathcal{G}]\|_p \leq \|X\|_p$ для $p \geq 1$.

Башенное свойство (4) — самое часто используемое в теории мартингалов: оно позволяет «съезжать» с маленькой σ-алгебры на большую и обратно.

## Связь с мартингалами

Случайный процесс $(X_n)_{n \geq 0}$ — **мартингал** относительно фильтрации $(\mathcal{F}_n)$, если:

1. $X_n$ адаптирован: $X_n$ $\mathcal{F}_n$-измерим.
2. $X_n$ интегрируем: $\mathbb{E}|X_n| < \infty$.
3. **Свойство мартингала**: $\mathbb{E}[X_{n+1} \mid \mathcal{F}_n] = X_n$.

То есть «лучшее предсказание следующего значения по всей доступной информации совпадает с текущим». Это математически точное выражение «честной игры»: ожидаемая прибыль в следующий раунд равна нулю.

Без условного матожидания относительно σ-алгебры это определение **не имело бы смысла** для общих фильтраций. Колмогоровская конструкция — то, что делает мартингалы настоящим математическим объектом.

См. [[discrete-martingales]] для подробного развития теории.

## Регулярная условная вероятность

При условии $\mathcal{G}$ можно говорить о **условном распределении** случайной величины $X$. Существует функция $\mu_X(\cdot \mid \mathcal{G})(\omega)$ — для почти всех $\omega$ — которая является вероятностной мерой на $\mathbb{R}$ и для которой

$$\mathbb{E}[f(X) \mid \mathcal{G}](\omega) = \int_\mathbb{R} f(x) \, \mu_X(dx \mid \mathcal{G})(\omega).$$

Это **регулярная условная вероятность**. Она существует для широкого класса пространств (в частности, для борелевских) — и именно она позволяет говорить о «распределении $X$ при условии $Y = y$» в строгом смысле, минуя нулевые знаменатели.

## В финансах

Условное матожидание — главный технический инструмент финансовой математики:

- Цена опциона: $C_t = \mathbb{E}^\mathbb{Q}[H \mid \mathcal{F}_t] / B_t$ — условное матожидание выплаты под мартингальной мерой.
- Греки (дельта, гамма): производные условного матожидания по параметрам.
- Hedging: воспроизводящий портфель — это разложение мартингала $\mathbb{E}^\mathbb{Q}[H \mid \mathcal{F}_t]$ через стохастический интеграл.
- Стохастическая фильтрация (Калман): $\hat{X}_t = \mathbb{E}[X_t \mid \mathcal{F}_t^{\text{obs}}]$ — оптимальная оценка состояния по наблюдениям.

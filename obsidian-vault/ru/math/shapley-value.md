# Вектор Шепли

[[Вектор Шепли]] обеспечивает справедливое распределение излишка в кооперативных играх: $\phi_i(v) = \sum \frac{|S|!(|N|-|S|-1)!}{|N|!} (v(S \cup \{i\}) - v(S))$. Широко используется в машинном обучении (SHAP).

# Дискретные мартингалы

Мартингал — это математическая модель **«честной игры»**: процесс, у которого ожидаемое следующее значение, при условии всей доступной информации, совпадает с текущим. Понятие восходит к идее «системы Мартингал» в азартных играх XVIII века, но его строгая теория была построена Жозефом Дубом (Joseph L. Doob) в 1940-х.
[[shiryaev-probability|«Вероятности»]] Ширяева — самое подробное доступное изложение дискретной мартингальной теории на русском языке. Без этой техники невозможны ни современные предельные теоремы, ни оптимальная остановка, ни [[ftap-first-fundamental-theorem|FTAP]] и финансовая математика, ни машинное обучение с подкреплением.

## Определение

Пусть $(\Omega, \mathcal{F}, \mathbb{P})$ — вероятностное пространство, $(\mathcal{F}_n)_{n \geq 0}$ — фильтрация (см. [[sigma-algebra-measurability]]).

**Определение.** Случайный процесс $(X_n)_{n \geq 0}$ называется **мартингалом** (относительно фильтрации $(\mathcal{F}_n)$), если:

1. **Адаптированность**: $X_n$ измерим относительно $\mathcal{F}_n$ для каждого $n$.
2. **Интегрируемость**: $\mathbb{E}|X_n| < \infty$ для каждого $n$.
3. **Свойство мартингала**: $\mathbb{E}[X_{n+1} \mid \mathcal{F}_n] = X_n$ почти наверное.

Свойство (3) — это «честность»: ожидаемое значение в следующий момент, при всей информации, накопленной к текущему, **в точности равно** текущему значению. Никаких «ожидаемых выигрышей» от правильной игры.

**Связанные понятия:**

- **Супермартингал**: $\mathbb{E}[X_{n+1} \mid \mathcal{F}_n] \leq X_n$. Игра «убыточная для игрока в среднем». Капитал монотонно убывает в среднем.
- **Субмартингал**: $\mathbb{E}[X_{n+1} \mid \mathcal{F}_n] \geq X_n$. Игра «прибыльная в среднем».

В финансах: цена актива под исторической мерой — субмартингал (есть положительная риск-премия), под мартингальной мерой — мартингал (нет арбитража).

## Примеры

**1. Симметричное случайное блуждание.** $X_n = X_{n-1} + \xi_n$, где $\xi_n \in \{-1, +1\}$ независимо с вероятностью $1/2$. Это мартингал.

**2. Условное матожидание.** Если $Y$ — интегрируемая случайная величина, то $X_n = \mathbb{E}[Y \mid \mathcal{F}_n]$ — мартингал. Это так называемый **мартингал, замкнутый в $Y$**, и любой равномерно интегрируемый мартингал имеет такой вид.

**3. Произведение независимых единичных случайных величин.** Если $\xi_1, \xi_2, \ldots$ независимы с $\mathbb{E}[\xi_n] = 1$, то $M_n = \prod_{k=1}^n \xi_k$ — мартингал.

**4. Экспоненциальный мартингал.** В дискретном виде: $M_n = \exp(\sum_k \theta_k Z_k - \sum_k \kappa_k(\theta_k))$, где $\kappa$ — кумулянтная функция. Это **дискретный аналог [[girsanov-semimartingales|плотности Гирсанова]]**.

## Неравенства Доба

**Максимальное неравенство Доба.** Для неотрицательного субмартингала $(X_n)$:

$$\mathbb{P}\!\left(\max_{k \leq n} X_k \geq \lambda \right) \leq \frac{\mathbb{E}[X_n]}{\lambda}.$$

Это обобщение неравенства Маркова на максимум по траектории, а не одно значение. Принципиально важно, потому что многие теоремы (типа [[law-iterated-logarithm|закона повторного логарифма]]) требуют контроля именно максимума.

**$L^p$-неравенство Доба.** Для неотрицательного субмартингала с $X_n \in L^p$, $p > 1$:

$$\mathbb{E}\!\left[ \max_{k \leq n} X_k^p \right] \leq \left(\frac{p}{p-1} \right)^p \mathbb{E}[X_n^p].$$

То есть моменты максимума контролируются моментами одного значения. Это «магия» мартингальной теории: добавление дополнительного «функционала максимума» обходится только постоянной мультипликативной поправкой.

## Теоремы о сходимости мартингалов

**Теорема Доба о сходимости.** Если $(X_n)$ — субмартингал с $\sup_n \mathbb{E}[X_n^+] < \infty$, то существует случайная величина $X_\infty$ такая, что

$$X_n \to X_\infty \quad \text{п.н.}$$

То есть **ограниченные в $L^1$ субмартингалы почти наверное сходятся**. Это нетривиальное обобщение «монотонная ограниченная последовательность сходится» на случайные процессы.

**Теорема о сходимости в $L^p$.** Если дополнительно $\sup_n \mathbb{E}|X_n|^p < \infty$ для $p > 1$, то $X_n \to X_\infty$ ещё и в $L^p$.

Эта теорема — фундамент:
- **Доказательства сильного [[law-of-large-numbers|ЗБЧ]]** через мартингальный подход.
- **Существования предельных распределений** для статистических процедур.
- **Теории лицензий и кредитного риска** (модели потерь как сходящиеся мартингалы).

## Optional Stopping (теорема о моменте остановки)

**Момент остановки** $\tau$ — случайная величина $\tau : \Omega \to \mathbb{N} \cup \{\infty\}$ такая, что $\{\tau \leq n\} \in \mathcal{F}_n$ для каждого $n$. Интуитивно: «решение остановиться принимается на основе доступной информации, без подсматривания в будущее».

**Теорема Доба об остановке (optional stopping).** Если $(X_n)$ — мартингал и $\tau$ — ограниченный момент остановки ($\tau \leq T$ для константы $T$), то

$$\mathbb{E}[X_\tau] = \mathbb{E}[X_0].$$

То есть «нельзя выиграть в среднем правильным выбором момента остановки» — это математическая формулировка невозможности «победить честную игру».

**Обобщения:** теорема верна и для неограниченных $\tau$ при подходящих условиях интегрируемости. Контрпример: блуждание $S_n = \sum \xi_n$, $\tau$ = момент достижения уровня $a$. Тогда $\mathbb{E}[S_\tau] = a \neq 0 = \mathbb{E}[S_0]$, но $\tau$ не является ограниченным.

В финансах optional stopping — фундамент **american option pricing** (см. [[optimal-stopping-american]]): цена американского опциона = $\sup_\tau \mathbb{E}^\mathbb{Q}[H(\tau, S_\tau)]$, где супремум по моментам остановки.

## Разложение Доба

**Теорема (разложение Доба).** Каждый дискретный субмартингал $(X_n)$ единственным образом представим в виде

$$X_n = X_0 + M_n + A_n,$$

где $M$ — мартингал ($M_0 = 0$) и $A$ — **предсказуемый** возрастающий процесс ($A_0 = 0$, $A_n$ измерим относительно $\mathcal{F}_{n-1}$).

Это **дискретный аналог разложения Доба-Мейера** для непрерывных супермартингалов и **базис всей теории мартингалов**: позволяет «отделить» мартингальную часть от предсказуемой («снос» — тренд процесса).

## Дискретная формула Ито

В Ширяев приводит **дискретный аналог формулы Ито**: для функции $f$ и мартингала $M$:

$$f(M_n) = f(M_0) + \sum_{k=1}^n f'(M_{k-1}) \Delta M_k + \frac{1}{2} \sum_{k=1}^n f''(\xi_k) (\Delta M_k)^2,$$

где $\xi_k$ — некоторая точка между $M_{k-1}$ и $M_k$. В пределе при «непрерывном времени» это превращается в классическую формулу Ито с членом $\frac{1}{2} f''(M_t) \, d\langle M\rangle_t$.

Дискретный вариант особенно удобен для понимания **происхождения** «лишнего» члена с второй производной, который кажется странным в непрерывном случае.

## Применения

**1. Финансовая математика.** [[ftap-first-fundamental-theorem|FTAP]]: рынок безарбитражен ⟺ существует EMM, под которой дисконтированные цены — мартингал. Все ценообразование опционов — это мартингальная теория.

**2. Теория алгоритмов.** Анализ рандомизированных алгоритмов часто сводится к построению подходящего мартингала и применению теорем сходимости.

**3. Машинное обучение.** В Reinforcement Learning value-функции при оптимальной стратегии — мартингалы относительно правильных мер. TD-learning — это итеративная оценка мартингальных свойств.

**4. Эконометрика.** Большинство «статистик» в эконометрике временных рядов — мартингалы относительно подходящей фильтрации, что позволяет применять теоремы сходимости и ЦПТ для мартингальных разностей.

**5. Страховая математика.** Мартингальный метод в задачах разорения: процесс капитала страховой компании — мартингал под подходящей мерой.

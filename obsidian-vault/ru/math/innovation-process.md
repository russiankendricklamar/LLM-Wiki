# Инновационный процесс

Инновационный процесс — фундаментальная конструкция теории стохастической фильтрации. По процессу наблюдений $\xi_t$ и оптимальному фильтру $\pi_t(h)$ определяется инновация $\nu_t = \xi_t - \int_0^t \pi_s(h)\,ds$ — часть наблюдений, не предсказуемая по прошлому. Замечательный факт, установленный Липцером и Ширяевым, состоит в том, что $\nu$ всегда является винеровским процессом относительно фильтрации наблюдений, независимо от распределения скрытого сигнала. Это означает, что «новая информация», поступающая через наблюдения, всегда гауссова.

Инновационный процесс преобразует задачу нелинейной фильтрации: вместо работы с «сырыми» наблюдениями $\xi_t$ (несущими сложную зависимость от сигнала) все уравнения фильтрации можно переписать через инновацию $\nu_t$, обладающую чистой вероятностной структурой.

## Определение и основные свойства

Рассмотрим стандартную модель фильтрации: сигнал $\theta_t$ и наблюдения:

$$\xi_t = \int_0^t h(s, \theta_s)\,ds + W_t,$$

где $W$ — винеровский процесс, независимый от $\theta$. **Инновационный процесс** определяется как:

$$\nu_t = \xi_t - \int_0^t \pi_s(h)\,ds = \int_0^t \bigl[h(s, \theta_s) - \pi_s(h)\bigr]\,ds + W_t,$$

где $\pi_s(h) = \mathbb{E}[h(s,\theta_s) \mid \mathcal{F}_s^\xi]$ — оптимальная фильтровая оценка дрейфа наблюдений.

**Теорема (Липцер-Ширяев).** Инновационный процесс $\nu_t$ является винеровским процессом относительно фильтрации наблюдений $(\mathcal{F}_t^\xi)$ и исходной вероятностной меры $P$.

Доказательство использует характеризацию Леви броуновского движения: показывают, что $\nu$ — непрерывный $\mathcal{F}_t^\xi$-локальный мартингал с квадратической вариацией $\langle \nu \rangle_t = t$. Мартингальное свойство следует из проекции: $\mathbb{E}[d\xi_t - \pi_t(h)\,dt \mid \mathcal{F}_t^\xi] = 0$ по определению [[conditional-expectation-sigma|условного математического ожидания]].

## Теорема о представлении через инновацию

**Теорема.** Фильтрация наблюдений и фильтрация инноваций совпадают:

$$\mathcal{F}_t^\xi = \mathcal{F}_t^\nu \quad \text{для всех } t \geq 0.$$

Это глубокий результат. Включение $\mathcal{F}_t^\nu \subseteq \mathcal{F}_t^\xi$ очевидно, поскольку $\nu$ строится из $\xi$ и $\pi_s(h)$ ($\mathcal{F}_s^\xi$-измеримого). Обратное включение $\mathcal{F}_t^\xi \subseteq \mathcal{F}_t^\nu$ требует доказательства того, что $\xi_t$ восстанавливается из $\{\nu_s : 0 \leq s \leq t\}$: поскольку $\pi_s(h)$ является $\mathcal{F}_s^\xi = \mathcal{F}_s^\nu$-измеримым, $\xi_t = \nu_t + \int_0^t \pi_s(h)\,ds$ восстанавливается из инновации (интеграл согласован с $\mathcal{F}^\nu$).

**Следствие.** Каждый $\mathcal{F}_t^\xi$-мартингал $M_t$ имеет **интегральное представление** по инновации:

$$M_t = M_0 + \int_0^t \varphi_s\,d\nu_s$$

для некоторого $\mathcal{F}_t^\nu$-предсказуемого процесса $\varphi$. Это аналог теоремы о мартингальном представлении для [[brownian-motion|броуновского движения]], но в фильтрации наблюдений.

## Связь с теоремой Гирсанова

Инновационный процесс имеет естественную интерпретацию через замену меры. При **эталонной мере** $\tilde{P}$ (построенной через [[girsanov-semimartingales|теорему Гирсанова]]) наблюдение $\xi$ — винеровский процесс. При исходной мере $P$ наблюдение имеет дрейф $h(\theta_t)$. Инновация $\nu_t$ получается вычитанием наилучшей оценки этого дрейфа:

$$\nu_t = \xi_t - \int_0^t \pi_s(h)\,ds.$$

При $\tilde{P}$ инновация $\nu$ отличается от $\xi$ на предсказуемый дрейф и остаётся семимартингалом. Но при $P$ инновация — винеровский процесс, объект более чистый, чем сам $\xi$. Это происходит именно потому, что фильтр $\pi_t(h)$ удаляет всю предсказуемую структуру из наблюдений.

## Инновационный подход к фильтрации

Уравнения фильтрации из [[nonlinear-filtering|нелинейной фильтрации]] переписываются целиком через инновацию. **Уравнение Кушнера-Стратоновича** принимает вид:

$$d\pi_t(f) = \pi_t(\mathcal{L}f)\,dt + \bigl[\pi_t(fh) - \pi_t(f)\pi_t(h)\bigr]\,d\nu_t.$$

Структура прозрачна: фильтр эволюционирует через детерминированный дрейф $\pi_t(\mathcal{L}f)\,dt$ (динамика сигнала, усреднённая по текущему условному распределению) плюс поправку, управляемую инновацией (новой информацией). При большом $d\nu_t$ — когда наблюдение отклоняется от прогноза — фильтр обновляется существенно. При малом $d\nu_t$ фильтр опирается на модель сигнала.

## Каузальная и некаузальная оценка

Инновационная формулировка чётко разделяет три задачи оценивания:

- **Фильтрация** (каузальная): оценка $\theta_t$ по $\{\xi_s : 0 \leq s \leq t\}$. Достаточно инноваций $\nu_s$ при $s \leq t$.
- **Прогнозирование**: оценка $\theta_{t+\delta}$ по $\{\xi_s : 0 \leq s \leq t\}$. Фильтр продвигается вперёд по динамике сигнала без новой инновации.
- **Сглаживание** (некаузальная оценка): оценка $\theta_s$ при $s < t$ по $\{\xi_u : 0 \leq u \leq t\}$. Требуются «будущие» инновации $\nu_u$ при $u > s$, что приводит к обратным стохастическим уравнениям.

Теорема о представлении гарантирует, что фильтрация наблюдений несёт ту же информацию, что и фильтрация инноваций, — при переходе к $\nu$ информация не теряется.

## Обеление наблюдений

В терминологии обработки сигналов инновационный процесс **обеляет** наблюдения. «Сырое» наблюдение $\xi_t$ имеет сложную корреляционную структуру (из-за сигнала $\theta$). Инновация $\nu_t$ имеет независимые приращения — это белый шум. Отображение $\xi \mapsto \nu$ является **обратимым каузальным преобразованием** (по теореме о представлении), поэтому обеление не теряет информации.

Практическое следствие: любой каузальный оценщик, оптимальный для белошумового входа, остаётся оптимальным при применении к инновации, извлечённой из окрашенных наблюдений. [[kalman-filter|Фильтр Калмана]] целиком выводится из этого принципа.

## Пример: скалярная линейная модель

Для линейной системы $d\theta_t = a\theta_t\,dt + b\,dV_t$, $d\xi_t = h\theta_t\,dt + dW_t$ инновация:

$$d\nu_t = d\xi_t - h\,m_t\,dt,$$

где $m_t = \mathbb{E}[\theta_t \mid \mathcal{F}_t^\xi]$ — условное среднее. Фильтр Калмана-Бьюси в инновационной форме:

$$dm_t = a\,m_t\,dt + \gamma_t h\,d\nu_t,$$

где $\gamma_t$ удовлетворяет уравнению Риккати $\dot{\gamma} = 2a\gamma + b^2 - \gamma^2 h^2$. Инновационный коэффициент усиления $\gamma_t h$ определяет, насколько сильно фильтр реагирует на новую информацию. Обобщения см. в [[conditionally-gaussian-processes|условно-гауссовских процессах]].

## Приложения

**Рекурсивное оценивание.** Любой каузальный алгоритм оценивания переформулируется как функционал от инновационного процесса — основа современной адаптивной фильтрации.

**Вычисление правдоподобия.** Логарифм правдоподобия наблюдений выражается через инновацию, что позволяет проводить оценку максимального правдоподобия для частично наблюдаемых систем.

**Теория управления.** В стохастическом оптимальном управлении с неполными наблюдениями принцип разделения утверждает, что оптимальное управление — функция фильтра $\pi_t$, а обновление информации контроллера определяется инновационным процессом.

В ориентированном графе сильно связная компонента (SCC) — это максимальный подграф, где есть путь между любыми двумя вершинами.

### Алгоритм Тарьяна
Алгоритм находит все SCC за время $O(V + E)$ с помощью одного обхода в глубину (DFS). Он использует стек и вычисляет значение `low-link` для каждого узла $v$ — минимальный индекс, достижимый по ребрам дерева DFS и не более чем одному обратному ребру.
Если $v.low == v.index$, $v$ является корнем SCC, и компонента извлекается из стека.

### Алгоритм Косарайю
Альтернативный метод Косарайю делает два прохода DFS: один по $G$ для фиксации времени завершения, второй — по транспонированному $G^T$ в порядке убывания времени.

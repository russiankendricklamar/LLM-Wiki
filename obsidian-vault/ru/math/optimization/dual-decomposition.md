Двойственная декомпозиция разбивает большие задачи с глобальными ограничениями на независимые подзадачи.

### Лагранжева релаксация
Для $\min f_1(x_1) + f_2(x_2)$ при условии $A x_1 + B x_2 = c$, формируется лагранжиан:
$L(x_1, x_2, \lambda) = f_1(x_1) + f_2(x_2) + \lambda^T(A x_1 + B x_2 - c)$

### Итеративное обновление
Лагранжиан разделяется на независимые задачи для фиксированного $\lambda$.
1. **Прямое обновление (параллельно):** Минимизация $L$ отдельно по $x_1$ и $x_2$.
2. **Двойственное обновление:** Градиентный подъем по $\lambda$:
$\lambda^{(k+1)} = \lambda^{(k)} + \alpha (A x_1^{(k+1)} + B x_2^{(k+1)} - c)$
Хотя метод параллелизуется, он сходится медленно и требует строгой выпуклости.

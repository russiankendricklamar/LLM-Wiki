# Проксимальные алгоритмы и ADMM

## Что это такое

Проксимальные алгоритмы — это класс методов оптимизации для решения негладких, ограниченных или крупномасштабных задач. Они опираются на **проксимальный оператор**, который обобщает понятие проекции на множество. **Переменный метод множителей Лагранжа (ADMM)** — это мощный алгоритм, сочетающий преимущества дуальной декомпозиции и методов дополненного Лагранжиана, что позволяет параллельно решать задачи с разделяемыми целевыми функциями.

## Проксимальный оператор

Для замкнутой собственной выпуклой функции $f$ проксимальный оператор $\text{prox}_{\lambda f}: \mathbb{R}^n \to \mathbb{R}^n$ определяется как:
$$\text{prox}_{\lambda f}(v) = \arg\min_x \left( f(x) + \frac{1}{2\lambda} \|x - v\|_2^2 \right)$$
где $\lambda > 0$ — параметр.
- Если $f$ — индикаторная функция множества $C$, то проксимальный оператор — это проекция $\Pi_C(v)$.
- Если $f(x) = \|x\|_1$, то проксимальный оператор — это оператор **мягкого порога** (soft-thresholding): $S_{\lambda}(v)_i = \text{sign}(v_i)\max(|v_i|-\lambda, 0)$.

## ADMM: Alternating Direction Method of Multipliers

ADMM решает задачи вида:
$$\min f(x) + g(z) \quad \text{при условии } Ax + Bz = c$$
где $f$ и $g$ выпуклы.

**Дополненный Лагранжиан** (Augmented Lagrangian) имеет вид:
$$L_\rho(x, z, y) = f(x) + g(z) + y^T(Ax + Bz - c) + \frac{\rho}{2}\|Ax + Bz - c\|_2^2$$
где $y$ — дуальная переменная (множитель Лагранжа), а $\rho > 0$ — параметр штрафа.

Шаги обновления ADMM:
1. $x^{k+1} = \arg\min_x L_\rho(x, z^k, y^k)$
2. $z^{k+1} = \arg\min_z L_\rho(x^{k+1}, z, y^k)$
3. $y^{k+1} = y^k + \rho(Ax^{k+1} + Bz^{k+1} - c)$

«Переменный» характер алгоритма позволяет обновлять $x$ и $z$ последовательно, что зачастую намного проще, чем совместная оптимизация.

## Проксимальный градиентный метод

Для задач вида $\min f(x) + g(x)$, где $f$ дифференцируема, а $g$ негладкая (но имеет известный проксимальный оператор), шаг обновления выглядит так:
$$x^{k+1} = \text{prox}_{\lambda g}(x^k - \lambda \nabla f(x^k))$$
Это основа таких алгоритмов, как ISTA (Iterative Soft-Thresholding Algorithm) для Lasso.

## Сходимость

ADMM гарантированно сходится для выпуклых функций $f$ и $g$ при наличии линейного ограничения. Скорость сходимости к оптимальному значению составляет $O(1/k)$, хотя на практике она может быть значительно выше. Алгоритм особенно эффективен для задач с огромными объемами данных и недифференцируемыми регуляризаторами.

## ВЫВОД: Проксимальный оператор нормы $L_1$

Пусть $f(x) = \|x\|_1$. Проксимальный оператор:
$$\text{prox}_{\lambda \|\cdot\|_1}(v) = \arg\min_x \left( \sum_i |x_i| + \frac{1}{2\lambda} \sum_i (x_i - v_i)^2 \right)$$
Целевая функция разделяема по компонентам, поэтому минимизируем для каждого $x_i$:
$h(x_i) = |x_i| + \frac{1}{2\lambda} (x_i - v_i)^2$.
Условие субградиента: $0 \in \partial |x_i| + \frac{1}{\lambda}(x_i - v_i)$.
1. Если $x_i > 0$: $1 + \frac{1}{\lambda}(x_i - v_i) = 0 \implies x_i = v_i - \lambda$ (требуется $v_i > \lambda$).
2. Если $x_i < 0$: $-1 + \frac{1}{\lambda}(x_i - v_i) = 0 \implies x_i = v_i + \lambda$ (требуется $v_i < -\lambda$).
3. Если $x_i = 0$: $|v_i| \le \lambda$.
Это дает оператор мягкого порога $S_\lambda(v)$.

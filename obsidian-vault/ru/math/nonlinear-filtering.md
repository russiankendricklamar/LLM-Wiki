# Нелинейная фильтрация

Нелинейная фильтрация — центральная задача, исследуемая Липцером и Ширяевым: по частично наблюдаемой стохастической системе вычислить условное распределение скрытого сигнала по имеющимся наблюдениям. В отличие от [[kalman-filter|фильтра Калмана]], работающего лишь в линейно-гауссовском случае, общая нелинейная теория вынуждена отслеживать бесконечномерный объект — полное условное распределение, — что делает эту задачу одной из глубочайших в стохастическом анализе.

Математический аппарат объединяет мерно-теоретическую вероятность, стохастическое исчисление и теорию стохастических уравнений в частных производных. Два основных уравнения — Кушнера-Стратоновича и Закаи — дают дополняющие описания эволюции условного распределения; их изучение определило развитие современной теории фильтрации, стохастического управления и обработки сигналов.

## Частично наблюдаемая система

Стандартная модель «сигнал-наблюдение» включает две компоненты. **Сигнал** $\theta_t$ — марковская диффузия:

$$d\theta_t = a(t,\theta_t)\,dt + b(t,\theta_t)\,dV_t,$$

где $V$ — винеровский процесс. Сигнал не наблюдается непосредственно. Вместо этого доступен **процесс наблюдений**:

$$\xi_t = \int_0^t h(s,\theta_s)\,ds + W_t,$$

где $W$ — винеровский процесс, независимый от $V$, а $h$ — функция датчика. Цель — вычисление **оптимального фильтра**:

$$\pi_t(f) = \mathbb{E}[f(\theta_t) \mid \mathcal{F}_t^\xi],$$

для пробных функций $f$, где $\mathcal{F}_t^\xi = \sigma(\xi_s : 0 \leq s \leq t)$ — фильтрация наблюдений. Условное распределение $\pi_t$ — случайный мерозначный процесс.

## Уравнение Кушнера-Стратоновича

Условное распределение $\pi_t$ удовлетворяет **уравнению Кушнера-Стратоновича (КС)**:

$$d\pi_t(f) = \pi_t(\mathcal{L}f)\,dt + \bigl[\pi_t(fh) - \pi_t(f)\pi_t(h)\bigr]\bigl(d\xi_t - \pi_t(h)\,dt\bigr),$$

где $\mathcal{L}$ — генератор диффузии сигнала:

$$\mathcal{L}f = a \cdot \nabla f + \tfrac{1}{2}\,\mathrm{tr}(bb^\top \nabla^2 f).$$

Член $d\xi_t - \pi_t(h)\,dt$ есть **инновация** $d\nu_t$ — «новая информация» в наблюдениях (см. [[innovation-process|инновационный процесс]]). Уравнение КС нелинейно: коэффициенты зависят от самого $\pi_t$.

## Уравнение Закаи

Дункан, Мортенсен и Закаи ввели **ненормированную** условную меру $\sigma_t$, удовлетворяющую **уравнению Закаи**:

$$d\sigma_t(f) = \sigma_t(\mathcal{L}f)\,dt + \sigma_t(fh)\,d\xi_t.$$

Уравнение Закаи **линейно** по $\sigma_t$, что является главным техническим преимуществом. Нормированный фильтр восстанавливается формулой $\pi_t(f) = \sigma_t(f)/\sigma_t(1)$. Ненормированная мера связана с условным распределением через замену меры типа Гирсанова: при эталонной мере, относительно которой $\xi$ — винеровский процесс, $\sigma_t$ описывает условное математическое ожидание, взвешенное производной Радона-Никодима.

Существование и единственность решений уравнения Закаи устанавливаются при достаточно широких условиях на $a$, $b$ и $h$ средствами теории мерозначных стохастических эволюционных уравнений.

## Почему нелинейная фильтрация трудна

Условное распределение $\pi_t$ живёт в бесконечномерном пространстве (пространстве вероятностных мер). В отличие от линейно-гауссовского случая, где $\pi_t$ гауссово и характеризуется конечным числом параметров (среднее и ковариация), в общей ситуации конечномерной достаточной статистики не существует.

**Теорема (Бенеш-Хазевинкель-Маркус).** Единственные диффузионные модели, для которых нелинейный фильтр допускает конечномерное рекурсивное представление, — это линейно-гауссовский случай ([[kalman-filter|фильтр Калмана]]), фильтр Бенеша (дрейф удовлетворяет специальному условию типа Риккати) и некоторые экспоненциальные семейства.

Это **проклятие размерности** мотивирует как изучение [[conditionally-gaussian-processes|условно-гауссовских]] приближений, так и численные методы — [[particle-filter|метод частиц]].

## Архитектура доказательства

Вывод уравнений фильтрации проходит несколько этапов:

1. **Метод эталонной меры.** Строится вероятностная мера $\tilde{P}$, при которой $\xi$ — стандартный винеровский процесс, независимый от $\theta$. По теореме Гирсанова ([[girsanov-semimartingales|замена меры]]) производная Радона-Никодима есть [[doleans-dade-exponential|экспонента Долеан-Дад]] $\mathcal{E}_t = \exp\bigl(\int_0^t h(\theta_s)\,d\xi_s - \frac{1}{2}\int_0^t h^2(\theta_s)\,ds\bigr)$.
2. **Формула Каллианпура-Стрибеля.** При $\tilde{P}$: $\pi_t(f) = \tilde{\mathbb{E}}[f(\theta_t)\mathcal{E}_t \mid \mathcal{F}_t^\xi] / \tilde{\mathbb{E}}[\mathcal{E}_t \mid \mathcal{F}_t^\xi]$.
3. **Вывод уравнения Закаи** применением формулы Ито к числителю $\sigma_t(f) = \tilde{\mathbb{E}}[f(\theta_t)\mathcal{E}_t \mid \mathcal{F}_t^\xi]$.
4. **Вывод КС** из Закаи формулой Ито для отношения $\sigma_t(f)/\sigma_t(1)$.

## Примеры и приложения

**Линейно-гауссовский случай.** При $a(\theta) = A\theta$, $b = \text{const}$, $h(\theta) = H\theta$ условное распределение гауссово. Уравнение КС сводится к фильтру Калмана-Бьюси: $dm_t = Am_t\,dt + \gamma_t H^\top(d\xi_t - Hm_t\,dt)$ с матричным уравнением Риккати для $\gamma_t$. См. [[kalman-filter|фильтр Калмана]].

**Фильтр Бенеша.** При постоянном $h$ и дрейфе $a$, удовлетворяющем $a' + a^2/2 = \text{квадратичная}$, условная плотность остаётся в известном экспоненциальном семействе — точно решаемый нелинейный фильтр.

**Стохастическая волатильность.** В [[heston-model|модели Хестона]] волатильность $V_t$ играет роль $\theta_t$, а доходности актива — наблюдений. Фильтрация $V_t$ — задача нелинейной фильтрации, решаемая [[particle-filter|методом частиц]] или дискретизациями [[hidden-markov-models|скрытых марковских моделей]].

**Слежение за целями и навигация.** Радары и сонары измеряют дальность и пеленг — сильно нелинейные функции состояния — каноническое приложение теории нелинейной фильтрации.

## Численные методы

Поскольку нелинейный фильтр бесконечномерен, практическая реализация требует аппроксимации. Основные подходы:

- **Расширенный фильтр Калмана (EKF).** Линеаризация $h$ вокруг текущей оценки и применение [[kalman-filter|калмановского обновления]]. Прост, но может расходиться при сильной нелинейности.
- **Метод частиц.** Представление $\pi_t$ взвешенной эмпирической мерой из $N$ частиц, продвигаемых последовательным методом Монте-Карло. См. [[particle-filter|метод частиц]]. Сходимость при $N \to \infty$ следует из закона больших чисел для систем взаимодействующих частиц.
- **Проекционные фильтры.** Проекция уравнения Кушнера-Стратоновича на конечномерное многообразие плотностей (например, экспоненциальные семейства), дающая конечное СДУ для параметров.
- **Спектральные методы.** Разложение условной плотности по собственным функциям $\mathcal{L}$ с усечением, переводящее СДУЧП Закаи в конечную систему СДУ.

Выбор метода зависит от размерности $\theta$, степени нелинейности $h$ и требований реального времени.

# Стохастические рекуррентные последовательности

Стохастическая рекуррентная последовательность (СРП) определяется соотношением

$$X_{n+1} = f(X_n, \xi_{n+1}),$$

где $(\xi_n)$ — н.о.р. случайные «входы», а $f$ — измеримая функция. Эта простая конструкция охватывает огромный круг моделей: системы массового обслуживания, авторегрессионные процессы, модели финансовой волатильности и итерированные функциональные системы, порождающие фракталы. Центральный вопрос — **устойчивость**: сходится ли $X_n$ к единственному стационарному распределению независимо от начального условия? Подход Боровкова к этой задаче через сцепление и обновляющие события даёт элегантную и мощную методологию.

## Схема Лойнса

Наиболее классический пример возникает в теории массового обслуживания. **Рекурсия Линдли** для времени ожидания в очереди GI/GI/1 имеет вид

$$W_{n+1} = (W_n + X_n)^+,$$

где $X_n = \text{обслуживание}_n - \text{интервал}_n$ и $(x)^+ = \max(x, 0)$.

**Теорема (Лойнс, 1962).** Если $f$ **монотонна** (т.е. $x \leq y$ влечёт $f(x, \xi) \leq f(y, \xi)$) и $\mathbb{E}[X_n] < 0$ (условие устойчивости: интенсивность обслуживания превышает интенсивность поступления), то, стартуя с $W_0 = +\infty$ («максимального» начального условия), $W_n$ убывает к пределу $W_\infty$ — единственному стационарному решению.

Конструкция Лойнса работает в обратном времени: $W_\infty = \sup_{n \geq 0} (X_0 + X_{-1} + \cdots + X_{-n+1})$. По [[law-of-large-numbers|закону больших чисел]] этот супремум конечен п.н. при $\mathbb{E}[X] < 0$, что даёт собственное стационарное распределение.

## Сжимаемость и ключевое условие

Для общих (немонотонных) рекурсий фундаментальный критерий устойчивости — **сжимаемость в среднем**.

**Определение.** Рекурсия $X_{n+1} = f(X_n, \xi_{n+1})$ **сжимающая**, если

$$\mathbb{E}\!\left[\log \operatorname{Lip}(f(\cdot, \xi))\right] < 0,$$

где $\operatorname{Lip}(f(\cdot, \xi)) = \sup_{x \neq y} |f(x, \xi) - f(y, \xi)| / |x - y|$ — константа Липшица.

Интуитивно это означает, что в среднем отображение $x \mapsto f(x, \xi)$ сжимает расстояния. Даже если некоторые реализации $\xi$ растягивают, логарифмическое среднее должно быть отрицательным.

**Теорема.** При условии сжимаемости и мягком моментном условии ($\mathbb{E}[\log^+ |f(x_0, \xi)|] < \infty$ для некоторого $x_0$) рекурсия имеет единственное стационарное распределение $\pi$, и для любого начального $X_0$:

$$X_n \xrightarrow{d} \pi \quad \text{при } n \to \infty.$$

Более того, сходимость экспоненциально быстра в подходящей метрике.

## Линейный случай: произведения случайных матриц

Линейная СРП $X_{n+1} = A_n X_n + B_n$ (с н.о.р. $(A_n, B_n)$) — наиболее изученный случай.

**Теорема (Кестен, 1973; Верваат, 1979).** Линейная рекурсия $X_{n+1} = A_n X_n + B_n$ имеет единственное стационарное решение тогда и только тогда, когда

$$\mathbb{E}[\log |A|] < 0 \quad \text{и} \quad \mathbb{E}[\log^+ |B|] < \infty.$$

Стационарное решение задаётся п.н. сходящимся рядом

$$X_\infty = B_0 + A_0 B_{-1} + A_0 A_{-1} B_{-2} + A_0 A_{-1} A_{-2} B_{-3} + \cdots$$

Условие $\mathbb{E}[\log |A|] < 0$ означает отрицательность **старшего показателя Ляпунова** — связь с мультипликативной эргодической теоремой Фюрстенберга-Кестена для произведений случайных матриц.

## Связь с мультипликативной эргодической теоремой

**Теорема (Фюрстенберг-Кестен, 1960).** Для н.о.р. случайных $d \times d$ матриц $A_1, A_2, \ldots$ с $\mathbb{E}[\log^+ \|A_1\|] < \infty$:

$$\frac{1}{n} \log \|A_n A_{n-1} \cdots A_1\| \to \gamma \quad \text{п.н.},$$

где $\gamma$ — **старший показатель Ляпунова**. При $\gamma < 0$ произведение $A_n \cdots A_1 \to 0$ экспоненциально быстро, что есть в точности условие устойчивости для линейной СРП в $\mathbb{R}^d$.

Это мультипликативный аналог [[law-of-large-numbers|закона больших чисел]]: вместо сходимости сумм к среднему произведения матриц имеют определённую экспоненциальную скорость роста.

## Подход Боровкова через обновляющие события

Боровков вводит **обновляющие события** (renovation events, они же события сцепления): случайные моменты $T$, в которые рекурсия «забывает» начальное условие. Формально обновляющее событие происходит в момент $n$, если $f(x, \xi_n) = f(y, \xi_n)$ для всех $x, y$ — отображение становится константой независимо от входа.

Если обновляющие события происходят с положительной вероятностью и имеют конечный средний промежуток, то:

1. Рекурсия эргодична (существует единственное стационарное распределение).
2. Сходимость к стационарности геометрическая.
3. [[ergodic-theorems-birkhoff|Эргодическая теорема]] применима к временным средним вдоль траекторий.

Этот подход гибче сжимаемости — он охватывает рекурсии, где $f$ не липшицева или где сжимаемость нарушается глобально, но сцепление происходит через редкие «обнуляющие» события.

## Связь с GARCH

Модель GARCH(1,1) для финансовой волатильности (см. [[garch-models]]) определяет

$$\sigma_{n+1}^2 = \omega + \alpha \varepsilon_n^2 + \beta \sigma_n^2 = \omega + (\alpha Z_n^2 + \beta) \sigma_n^2,$$

где $Z_n = \varepsilon_n / \sigma_n$ — н.о.р. Это линейная СРП $X_{n+1} = A_n X_n + B_n$ с $A_n = \alpha Z_n^2 + \beta$ и $B_n = \omega$.

Условие стационарности:

$$\mathbb{E}[\log(\alpha Z^2 + \beta)] < 0,$$

что есть в точности условие на показатель Ляпунова. При его выполнении волатильность GARCH сходится к единственному стационарному распределению с **тяжёлыми (паретовскими) хвостами** — явление, открытое Кестеном (1973) и переоткрытое в контексте GARCH Нельсоном (1990). Индекс хвоста удовлетворяет неявному уравнению $\mathbb{E}[(\alpha Z^2 + \beta)^\kappa] = 1$.

## Приложения

**Теория массового обслуживания.** Рекурсия Линдли $W_{n+1} = (W_n + X_n)^+$ и её многомерные обобщения управляют временами ожидания практически во всех одноканальных и сетевых моделях обслуживания.

**Авторегрессионные процессы.** Модель AR(1) $X_{n+1} = \phi X_n + \varepsilon_{n+1}$ — линейная СРП; стационарность требует $|\phi| < 1$, т.е. $\log|\phi| < 0$. Модели более высоких порядков сводятся к векторным СРП.

**Итерированные функциональные системы.** Выбираем сжатие $f_i$ с вероятностью $p_i$ и итерируем: $X_{n+1} = f_{I_{n+1}}(X_n)$. Аттрактор — фрактал (например, треугольник Серпинского, папоротник Барнсли). Сжимаемость гарантирует сходимость к единственной инвариантной мере на фрактале.

**Финансы за пределами GARCH.** Модели стохастической волатильности, модели с переключением режимов ([[hidden-markov-models]]) и многие модели риска в страховании — стохастические рекурсии. Их эргодичность определяет осмысленность долгосрочного статистического оценивания.

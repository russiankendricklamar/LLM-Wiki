# Ветвящиеся процессы

Ветвящиеся процессы моделируют популяции, в которых каждый индивид независимо порождает случайное число потомков. Процесс Гальтона-Ватсона, введённый в 1874 году для изучения вымирания фамилий, остаётся прототипом: единственный предок порождает детей, каждый из которых независимо размножается по тому же закону. Несмотря на простоту, модель обнаруживает резкий фазовый переход — вымирание против экспоненциального роста — управляемый единственным параметром: средним числом потомков. Боровков развивает теорию через производящие функции и мартингальные аргументы, связывая её с [[law-of-large-numbers|законом больших чисел]], [[central-limit-theorem|центральной предельной теоремой]] и [[characteristic-functions|характеристическими функциями]].

## Процесс Гальтона-Ватсона

Пусть $X$ — неотрицательная целочисленная случайная величина (распределение потомства) с производящей функцией (п.ф.)

$$f(s) = \mathbb{E}[s^X] = \sum_{k=0}^{\infty} p_k s^k, \quad |s| \leq 1.$$

Определим процесс рекурсивно: $Z_0 = 1$ и

$$Z_{n+1} = \sum_{i=1}^{Z_n} X_i^{(n)},$$

где $X_i^{(n)}$ — н.о.р. копии $X$. Ключевой параметр — **среднее число потомков** $m = \mathbb{E}[X] = f'(1)$.

П.ф. числа $Z_n$ удовлетворяет **итерационному соотношению** $f_n(s) = f(f_{n-1}(s))$, где $f_1 = f$. Это сводит анализ $n$-го поколения к композиции одной функции — подход через производящие функции, делающий теорию столь элегантной.

## Вероятность вымирания

Пусть $q = \mathbb{P}(Z_n = 0 \text{ для некоторого } n \geq 1)$ — вероятность вымирания.

**Теорема.** Вероятность вымирания $q$ есть наименьший неотрицательный корень уравнения $f(s) = s$. При этом:

- **Докритический случай** ($m < 1$): $q = 1$. Вымирание достоверно.
- **Критический случай** ($m = 1$, $\sigma^2 > 0$): $q = 1$. Вымирание всё равно достоверно, но медленнее.
- **Надкритический случай** ($m > 1$): $q < 1$. Существует положительная вероятность бесконечного выживания.

Доказательство — аргумент неподвижной точки: поскольку $f$ выпукла на $[0,1]$ и $f(1) = 1$, уравнение $f(s) = s$ имеет не более одного корня в $[0,1)$. В надкритическом случае выпуклость гарантирует существование такого корня строго ниже 1.

**Набросок доказательства.** Вероятности вымирания $q_n = \mathbb{P}(Z_n = 0)$ удовлетворяют $q_{n+1} = f(q_n)$ при $q_0 = 0$. Поскольку $f$ неубывает и $q_1 = f(0) = p_0 \geq 0$, последовательность $(q_n)$ неубывает и ограничена 1, поэтому $q_n \uparrow q$, где $q = f(q)$. Выпуклость $f$ обеспечивает, что наименьшая неподвижная точка в $[0,1]$ равна 1 при $f'(1) = m \leq 1$ и строго меньше 1 при $m > 1$.

## Асимптотическое поведение

**Критический случай (Колмогоров, 1938).** Если $m = 1$ и $\sigma^2 = \text{Var}(X) \in (0, \infty)$, то

$$\mathbb{P}(Z_n > 0) \sim \frac{2}{\sigma^2 n}, \quad n \to \infty.$$

При условии выживания $Z_n / n$ сходится по распределению к экспоненциальной случайной величине. Популяция выживает лишь за счёт редких больших флуктуаций.

**Надкритический случай (теорема Кестена-Стигума).** Если $m > 1$ и $\mathbb{E}[X \log X] < \infty$, то

$$W_n = \frac{Z_n}{m^n} \to W \quad \text{п.н.},$$

где $W$ — неотрицательная случайная величина с $\mathbb{P}(W > 0) = 1 - q$. Предел $W$ удовлетворяет **функциональному уравнению** $\mathbb{E}[e^{-tW}] = f(\mathbb{E}[e^{-tW/m}])$ и имеет непрерывную плотность на $(0, \infty)$.

**Докритический случай.** Если $m < 1$, то $\mathbb{P}(Z_n > 0) \sim C \cdot m^n$ для некоторой константы $C$, и при условии невымирания размер популяции имеет предельное распределение на $\{1, 2, \ldots\}$.

## Мартингал $W_n$

Нормированный процесс $W_n = Z_n / m^n$ — неотрицательный [[discrete-martingales|мартингал]]:

$$\mathbb{E}[W_{n+1} \mid \mathcal{F}_n] = \frac{\mathbb{E}[Z_{n+1} \mid Z_n]}{m^{n+1}} = \frac{m Z_n}{m^{n+1}} = W_n.$$

По теореме о сходимости мартингалов $W_n \to W$ п.н. Условие Кестена-Стигума $\mathbb{E}[X \log X] < \infty$ необходимо и достаточно для $\mathbb{E}[W] = 1$ (т.е. сходимости в $L^1$ и невырожденности на событии выживания). Без этого условия $W = 0$ п.н. даже в надкритическом случае — нормировка $m^n$ оказывается слишком грубой.

## Обобщения

**Непрерывное ветвление (процесс Беллмана-Харриса).** Каждый индивид живёт случайное время $L$ с распределением $G$, затем порождает потомков. При условиях регулярности $\mathbb{E}[Z(t)] \sim C e^{\alpha t}$, где $\alpha$ — мальтузианский параметр, единственный корень $m \int_0^\infty e^{-\alpha t}\, dG(t) = 1$.

**Многотипное ветвление.** Индивиды $d$ типов размножаются по типозависимым законам потомства. Матрица средних потомков $M$ заменяет скаляр $m$; критичность определяется собственным числом Перрона-Фробениуса $\rho(M)$, а теорема Кестена-Стигума обобщается с ведущим собственным вектором, определяющим асимптотический типовой состав.

**Ветвление в случайной среде.** Распределение потомства меняется случайно от поколения к поколению. Классификация на до/над/критический случаи зависит от $\mathbb{E}[\log m_n]$, а не от $\mathbb{E}[m_n]$, что порождает явления, отсутствующие в классической теории (например, выживание при $\mathbb{E}[m_n] < 1$).

## Примеры

**1. Бинарное деление.** Каждый индивид порождает ровно 2 потомка ($p_2 = 1$). Тогда $m = 2$, $Z_n = 2^n$ детерминированно, и $W = 1$ п.н. Это вырожденный надкритический случай без случайности.

**2. Геометрическое потомство.** Пусть $p_k = (1-p)p^k$ для $k = 0, 1, 2, \ldots$ Тогда $f(s) = (1-p)/(1-ps)$, $m = p/(1-p)$. Процесс надкритичен тогда и только тогда, когда $p > 1/2$. Вероятность вымирания: $q = (1-p)/p$ при $p > 1/2$.

**3. Пуассоновское потомство.** Пусть $X \sim \text{Poisson}(\lambda)$. Тогда $f(s) = e^{\lambda(s-1)}$, $m = \lambda$, и $q$ — единственный корень $e^{\lambda(s-1)} = s$ в $[0,1)$ при $\lambda > 1$. Замкнутой формулы нет, но $q \approx 1 - 2(\lambda - 1)/\lambda^2$ при $\lambda$ чуть больше 1.

## Приложения

**1. Популяционная генетика.** Модель Райта-Фишера с мутациями можно рассматривать через ветвление: полезный аллель с селективным преимуществом $s$ закрепляется с вероятностью $\approx 2s$, выводимой из надкритического приближения Гальтона-Ватсона.

**2. Ядерные цепные реакции.** Первоначальная физическая мотивация: каждый акт деления порождает случайное число нейтронов, каждый из которых может вызвать дальнейшее деление. Критичность ($m = 1$) — расчётная точка управляемого реактора.

**3. Моделирование эпидемий.** Базовое репродуктивное число $R_0$ в эпидемиологии — среднее число потомков. Эпидемия затухает п.н. тогда и только тогда, когда $R_0 \leq 1$ — теорема о вымирании ветвящегося процесса на языке эпидемиологии.

**4. Вирусное распространение контента.** Каждый репост генерирует случайное число повторных репостов. Вероятность «стать вирусным» (невымирание) равна $1 - q$, где $q$ — решение $f(s) = s$ для распределения репостов.

# Стохастическая экспонента Долеан-Дада

В обычном анализе решение уравнения $dZ = Z \, dX$ есть $Z = e^X$. В стохастическом исчислении наличие квадратической вариации в формуле Ито приводит к тому, что наивная экспонента $e^X$ **не** является решением стохастического дифференциального уравнения $dZ = Z_{-} \, dX$. Правильное решение — **стохастическая экспонента** (экспонента Долеан-Дада) $\mathcal{E}(X)$, играющая центральную роль в книге Жакода и Ширяева «Предельные теоремы для случайных процессов» как универсальный процесс плотности для замен меры.

Каждая [[girsanov-semimartingales|плотность Гирсанова]] — стохастическая экспонента. Каждое [[geometric-brownian-motion|геометрическое броуновское движение]] — стохастическая экспонента. Стохастическая экспонента для теории семимартингалов — то же, что обычная экспонента для линейной алгебры: отображение из аддитивной группы в мультипликативную.

## Определение и уравнение

**Определение.** Пусть $X$ — [[semimartingale-models|семимартингал]] с $X_0 = 0$. **Стохастическая экспонента** $\mathcal{E}(X)$ — единственное решение СДУ:

$$Z_t = 1 + \int_0^t Z_{s-} \, dX_s.$$

Эквивалентно, $\mathcal{E}(X)$ — единственный семимартингал $Z$, удовлетворяющий $dZ = Z_{-} \, dX$ с $Z_0 = 1$.

Ключевой момент: подынтегральное выражение $Z_{s-}$ (левый предел), а не $Z_s$ — подынтегральное выражение **предсказуемо**, что обеспечивает корректность стохастического интеграла.

## Явная формула

**Теорема.** Стохастическая экспонента имеет явный вид:

$$\mathcal{E}(X)_t = \exp\!\left(X_t - \frac{1}{2}\langle X^c \rangle_t\right) \prod_{0 < s \leq t} (1 + \Delta X_s) \, e^{-\Delta X_s},$$

где $X^c$ — непрерывная мартингальная часть $X$, $\langle X^c \rangle$ — её квадратическая вариация, и произведение берётся по всем моментам скачков $X$ до момента $t$.

**Набросок доказательства.** Применим формулу Ито к $f(X_t, \langle X^c \rangle_t) = \exp(X_t - \frac{1}{2}\langle X^c \rangle_t)$. Для непрерывной части поправка квадратической вариации в формуле Ито производит в точности член $-\frac{1}{2}\langle X^c \rangle$. Для каждого скачка $\Delta X_s$ экспонента $e^{\Delta X_s}$ должна быть исправлена на $(1 + \Delta X_s)$, поскольку СДУ использует левый предел $Z_{s-}$: необходимо $Z_s = Z_{s-}(1 + \Delta X_s)$, но $e^{\Delta X_s} \neq 1 + \Delta X_s$ вообще говоря. Компенсирующий множитель $e^{-\Delta X_s}$ в произведении исправляет это.

## Частные случаи

**Непрерывный семимартингал.** Если $X$ не имеет скачков, формула упрощается:

$$\mathcal{E}(X)_t = \exp\!\left(X_t - \frac{1}{2}\langle X \rangle_t\right).$$

Это форма, появляющаяся в классической теореме Гирсанова для броуновского движения: если $X_t = \int_0^t \theta_s \, dW_s$, то $\mathcal{E}(X)$ — плотность Радона-Никодима.

**Чисто скачкообразный процесс.** Если $X$ чисто разрывный с $X^c = 0$:

$$\mathcal{E}(X)_t = \prod_{0 < s \leq t} (1 + \Delta X_s).$$

Это произведение мультипликативных приращений — естественный аналог геометрического случайного блуждания.

**Броуновское движение со сносом.** Если $X_t = \theta W_t + \mu t$, то $\langle X^c \rangle_t = \theta^2 t$ и:

$$\mathcal{E}(X)_t = \exp\!\left(\theta W_t + \mu t - \frac{\theta^2}{2} t\right).$$

При $\mu = 0$ это классическая экспоненциальная мартингала, которая является истинной мартингалой при ограниченном $\theta$.

## Положительность и строгая положительность

$\mathcal{E}(X)_t \geq 0$ для всех $t$ тогда и только тогда, когда $\Delta X_t > -1$ для всех $t$ (никакой скачок не уменьшает $X$ более чем на 1). Если дополнительно $\Delta X_t > -1$ строго, то $\mathcal{E}(X) > 0$ всюду.

Это условие существенно для теоремы Гирсанова: процесс плотности должен быть строго положительным, чтобы новая мера была эквивалентна (а не просто абсолютно непрерывна).

## Формула Йора

**Теорема (Йор).** Для двух семимартингалов $X, Y$:

$$\mathcal{E}(X) \cdot \mathcal{E}(Y) = \mathcal{E}(X + Y + [X, Y]),$$

где $[X, Y]$ — квадратическая ковариация.

**Идея доказательства.** Применим формулу произведения Ито к $Z = \mathcal{E}(X)\mathcal{E}(Y)$:

$$dZ = \mathcal{E}(X)_{-} \mathcal{E}(Y)_{-} \, dX + \mathcal{E}(X)_{-} \mathcal{E}(Y)_{-} \, dY + \mathcal{E}(X)_{-} \mathcal{E}(Y)_{-} \, d[X,Y]$$
$$= Z_{-} \, d(X + Y + [X,Y]).$$

Значит, $Z$ решает СДУ для $\mathcal{E}(X + Y + [X,Y])$, и единственность даёт результат.

**Следствие.** Обратный элемент к $\mathcal{E}(X)$ есть $\mathcal{E}(\tilde{X})$, где $\tilde{X}$ — единственный семимартингал, удовлетворяющий $X + \tilde{X} + [X, \tilde{X}] = 0$. Для непрерывного $X$ это даёт $\tilde{X}_t = -X_t + \langle X \rangle_t$.

## Связь с теоремой Гирсанова

[[girsanov-semimartingales|Теорема Гирсанова]] утверждает: если $L = \mathcal{E}(M)$ для локальной мартингалы $M$ с $\Delta M > -1$, и если $L$ — истинная мартингала, то можно определить новую вероятностную меру $Q$ формулой $dQ/dP|_{\mathcal{F}_t} = L_t$. При мере $Q$ предсказуемые характеристики каждого семимартингала изменяются по явным формулам, включающим $M$.

Стохастическая экспонента — **мост** между аддитивными объектами (локальными мартингалами $M$, параметризующими замену меры) и мультипликативными объектами (процессами плотности $L$, определяющими новую меру).

## Приложения

**Геометрические модели Леви.** Ценовой процесс $S_t = S_0 \mathcal{E}(X)_t$ для [[levy-processes|процесса Леви]] $X$ — стандартная экспоненциальная модель Леви. Явная формула показывает, что $S$ остаётся положительным при $\Delta X > -1$, то есть когда скачки не могут сделать цену отрицательной.

**Экспоненциальные мартингалы.** В теории больших уклонений (см. [[large-deviations-cramer]]) изучают экспоненциальную мартингалу $\mathcal{E}(\lambda M)$, где $M$ — мартингала и $\lambda$ — параметр. Поведение этого семейства при варьировании $\lambda$ даёт информацию о хвостовых вероятностях.

**Отношения правдоподобия в статистике.** Логарифм отношения правдоподобия для наблюдений точечного процесса — стохастический интеграл логарифма отношения интенсивностей по компенсированному точечному процессу. Само отношение правдоподобия — стохастическая экспонента этого интеграла.

**Условие Новикова.** Достаточное условие того, что $\mathcal{E}(M)$ — истинная (равномерно интегрируемая) мартингала, когда $M_t = \int_0^t \theta_s \, dW_s$:

$$\mathbb{E}\!\left[\exp\!\left(\frac{1}{2}\int_0^T \theta_s^2 \, ds\right)\right] < \infty.$$

Это стандартный инструмент проверки корректности замены меры Гирсанова.

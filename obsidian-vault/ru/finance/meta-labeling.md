# Фреймворк мета-лейблинга (Marcos Lopez de Prado)

**Мета-лейблинг (Meta-Labeling)** — это метод машинного обучения, предложенный Маркосом Лопесом де Прадо для повышения эффективности финансовых стратегий. Суть метода заключается в разделении принятия решения о **стороне сделки** (покупка/продажа) и о **размере позиции** (входить/не входить).

## Механизм работы

1. **Первичная модель**: Стандартная модель (например, пересечение скользящих средних) определяет направление сделки ($y \in \{-1, 1\}$).
2. **Мета-разметка**: Создается вторичная метка $z \in \{0, 1\}$:
   - $z=1$, если прогноз первичной модели оказался верным (прибыль).
   - $z=0$, если прогноз первичной модели оказался ошибочным (убыток).
3. **Вторичная модель**: Обучается классификатор для предсказания $z$. Если вторичная модель предсказывает $z=0$, сделка игнорируется, даже если первичная модель дает сигнал.

## Преимущества

- **Борьба с переобучением**: Вторичная модель учится распознавать рыночные режимы, в которых первичная модель работает плохо.
- **Повышение Precision**: За счет отсеивания ложноположительных сигналов значительно улучшается качество стратегии.
- **Оптимизация сайзинга**: Вероятность, выдаваемая вторичной моделью, может напрямую использоваться для определения веса позиции.

## Применение в финансах

1. **Трендовые стратегии**: Отсеивание ложных сигналов в боковом тренде.
2. **Арбитраж**: Выполнение сделок только тогда, когда рыночные условия (волатильность, ликвидность) благоприятствуют сближению цен.
3. **Управление капиталом**: Использование вероятности мета-модели для динамического изменения кредитного плеча.

## Python: Логика мета-лейблинга

```python
import pandas as pd
from sklearn.ensemble import RandomForestClassifier

def get_meta_signals(X, primary_signals, true_returns):
    # z=1 если знак сигнала совпал со знаком доходности
    z = (primary_signals == np.sign(true_returns)).astype(int)
    
    model = RandomForestClassifier()
    model.fit(X, z)
    
    # Вероятность того, что первичный сигнал верен
    meta_probs = model.predict_proba(X)[:, 1]
    return meta_probs

# Если meta_probs > 0.5 — подтверждаем первичный сигнал
```

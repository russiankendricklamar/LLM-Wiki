# Сверточная кластеризация разрешений волатильности (CVRC)

**Сверточная кластеризация разрешений волатильности (CVRC)** — метод глубокого обучения, рассматривающий частотно-временное представление волатильности как двумерное «изображение» и применяющий CNN для извлечения иерархических признаков с последующей кластеризацией режимов без учителя. Метод позволяет выявлять скрытые рыночные режимы, невидимые для традиционных скользящих показателей.

## Необходимые знания

[[spectral-analysis]], [[wavelet-transform]], [[garch-models]]

## Обзор и предпосылки

Традиционные модели волатильности (GARCH, реализованная волатильность) используют одномерные временные ряды. Однако волатильность имеет иерархическую структуру: внутридневные микро-шоки, дневные события, недельные макро-циклы. Представление вейвлет-преобразования волатильности как изображения позволяет CNN извлекать многомасштабные признаки одновременно.

Метод DEC (Deep Embedded Clustering, Xie et al., 2016) и VAE (Variational Autoencoder, Kingma & Welling, 2013) обеспечивают обучение признаков без разметки — критически важно, поскольку рыночные режимы не наблюдаются непосредственно.

## Математическая база

**Сверточная операция** на карте волатильности $V \in \mathbb{R}^{T \times F}$:

$$
y_{i,j} = \sigma\!\left(\sum_m\sum_n V_{i+m,\,j+n} \cdot K_{m,n} + b\right)
$$

где $K$ — обучаемый фильтр, $\sigma$ — активация ReLU.

**Вариационный нижний предел (ELBO)** для VAE:

$$
\mathcal{L}_{\text{VAE}} = \mathbb{E}_{q_\phi(z|x)}\!\left[\ln p_\theta(x|z)\right] - D_{\text{KL}}\!\left(q_\phi(z|x)\,\|\,p(z)\right)
$$

где $q_\phi(z|x) = \mathcal{N}(\mu_\phi(x), \text{diag}(\sigma^2_\phi(x)))$ — приближённое апостериорное распределение.

**Трюк репараметризации**: $z = \mu_\phi(x) + \sigma_\phi(x) \odot \epsilon$, $\epsilon \sim \mathcal{N}(0, I)$.

**Мягкое назначение кластеров (DEC)** через распределение Стьюдента с $\nu$ степенями свободы:

$$
q_{ij} = \frac{(1 + \|z_i - \mu_j\|^2 / \nu)^{-(\nu+1)/2}}{\sum_{j'} (1 + \|z_i - \mu_{j'}\|^2 / \nu)^{-(\nu+1)/2}}
$$

**Вспомогательное распределение** для заострения назначений:

$$
p_{ij} = \frac{q_{ij}^2 / \sum_i q_{ij}}{\sum_{j'} q_{ij'}^2 / \sum_i q_{ij'}}
$$

**Потери кластеризации** — минимизация KL-дивергенции между $Q$ и $P$:

$$
\mathcal{L}_{\text{cluster}} = D_{\text{KL}}(P \| Q) = \sum_i\sum_j p_{ij}\ln\frac{p_{ij}}{q_{ij}}
$$

## Ключевые свойства

- Многомасштабные ядра (3×3, 5×5, 7×7) захватывают структуры разных временных горизонтов
- VAE обеспечивает непрерывное латентное пространство — плавные переходы между режимами
- DEC итеративно уточняет границы кластеров без ручной разметки
- Три интерпретируемых режима: равновесие (низкая волатильность), переходная турбулентность, системный кризис

## Реализация на Python

```python
import torch
import torch.nn as nn
import torch.nn.functional as F
import numpy as np

class VolatilityEncoder(nn.Module):
    """CNN-кодировщик для поверхностей волатильности."""
    def __init__(self, latent_dim: int = 16):
        super().__init__()
        self.conv = nn.Sequential(
            nn.Conv2d(1, 16, kernel_size=3, padding=1), nn.ReLU(), nn.MaxPool2d(2),
            nn.Conv2d(16, 32, kernel_size=3, padding=1), nn.ReLU(), nn.MaxPool2d(2),
            nn.Conv2d(32, 64, kernel_size=3, padding=1), nn.ReLU(),
        )
        self.fc_mu  = nn.Linear(64 * 8 * 8, latent_dim)
        self.fc_log = nn.Linear(64 * 8 * 8, latent_dim)

    def forward(self, x):
        h = self.conv(x).flatten(1)
        return self.fc_mu(h), self.fc_log(h)

class VolatilityDecoder(nn.Module):
    """CNN-декодировщик для реконструкции поверхности волатильности."""
    def __init__(self, latent_dim: int = 16):
        super().__init__()
        self.fc = nn.Linear(latent_dim, 64 * 8 * 8)
        self.deconv = nn.Sequential(
            nn.ConvTranspose2d(64, 32, kernel_size=2, stride=2), nn.ReLU(),
            nn.ConvTranspose2d(32, 16, kernel_size=2, stride=2), nn.ReLU(),
            nn.Conv2d(16, 1, kernel_size=3, padding=1),
        )

    def forward(self, z):
        h = self.fc(z).view(-1, 64, 8, 8)
        return self.deconv(h)

def vae_loss(x_recon, x, mu, log_var, beta: float = 1.0):
    """ELBO: реконструкция + KL-дивергенция."""
    recon = F.mse_loss(x_recon, x, reduction='sum')
    kl    = -0.5 * torch.sum(1 + log_var - mu.pow(2) - log_var.exp())
    return recon + beta * kl

def cluster_soft_assignment(z: torch.Tensor, centroids: torch.Tensor,
                             nu: float = 1.0) -> torch.Tensor:
    """Мягкое назначение кластеров по распределению Стьюдента."""
    dist = torch.cdist(z, centroids)**2
    q = (1 + dist / nu).pow(-(nu + 1) / 2)
    return q / q.sum(dim=1, keepdim=True)

# Симуляция: 100 поверхностей волатильности 32×32
np.random.seed(42)
vol_surfaces = torch.randn(100, 1, 32, 32).abs()
encoder = VolatilityEncoder(latent_dim=16)
decoder = VolatilityDecoder(latent_dim=16)
mu_z, log_var_z = encoder(vol_surfaces)
z = mu_z + torch.exp(0.5 * log_var_z) * torch.randn_like(mu_z)
x_recon = decoder(z)
loss = vae_loss(x_recon, vol_surfaces, mu_z, log_var_z)
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $K_{m,n}$ | Сверточный фильтр | Обучаемые веса CNN | Инициализация Xavier |
| $d_z$ | Размерность латентного пространства | Размер скрытого представления | $8$ – $64$ |
| $\beta$ | Вес KL в ELBO | Компромисс реконструкция vs диверсификация | $0.1$ – $4.0$ |
| $\nu$ | Степени свободы DEC | Параметр Стьюдента для мягкого назначения | $1$ (Коши) – $\infty$ (гауссово) |
| $k$ | Число кластеров | Количество режимов волатильности | $3$ – $8$ |

## Ограничения и расширения

**Ограничения:**
- Требует значительного объёма данных для обучения CNN и VAE; при малых выборках возможно переобучение.
- Интерпретируемость кластеров не гарантирована — присвоение экономических меток требует экспертизы.
- Стационарность предполагается внутри каждого кластера, что нарушается при структурных изломах рынка.

**Расширения:**
- **[[wavelet-transform]]**: вейвлет-спектрограмма как входное представление для CVRC.
- **[[gan]]**: GAN-аугментация для искусственной генерации дефицитных кризисных режимов.
- **Временная когерентность**: рекуррентный слой (LSTM) поверх латентных кодов для плавных переходов между режимами.

## Практическое применение

1. **Адаптивный риск-менеджмент**: масштабирование лимитов VaR по текущему режиму кластера.
2. **Стресс-тестирование**: генерация сценариев из кластера «системный кризис» для экстремальных проверок.
3. **Адаптивные торговые стратегии**: переключение между агрессивными и консервативными параметрами MM в зависимости от режима.
4. **Ценообразование опционов**: поверхность подразумеваемой волатильности классифицируется по режимам для выбора модели.
5. **Мониторинг системного риска**: ранее предупреждение перехода в кластер «заражение» по паттернам латентного пространства.

# Показатель Херста (H)

**Показатель Херста (H)** — статистическая мера «долгосрочной памяти» или персистентности временного ряда. Назван в честь гидролога Гарольда Херста, разработавшего его при изучении паводков на Ниле (1951). В финансы метод адаптировали Мандельброт и ван Несс (1968) в контексте дробного броуновского движения. Показатель Херста — ключевой инструмент для классификации рыночной динамики: трендовая, возвращающаяся к среднему или чисто случайная.

## Предпосылки

Херст заметил, что нормированный размах накопленных отклонений от среднего растёт по степенному закону во времени. На эффективных рынках (броуновское движение) показатель должен быть ровно 0.5. Отклонения от 0.5 указывают либо на долгую память (персистентность), либо на антиперсистентность — оба случая дают сигнал для выбора торговой стратегии.

## Интерпретация значений

Показатель принимает значения от 0 до 1, выделяя три режима:

1. **$H = 0.5$**: **Случайное блуждание (броуновское движение)**. Будущие изменения независимы от прошлых. Соответствует гипотезе эффективного рынка.
2. **$0.5 < H < 1$**: **Персистентный (трендовый) ряд**. Прошлый рост предсказывает будущий рост. Чем $H$ ближе к 1, тем сильнее и продолжительнее тренд.
3. **$0 < H < 0.5$**: **Антиперсистентный ряд (возврат к среднему)**. Прошлый рост предсказывает будущее снижение. Чем $H$ ближе к 0, тем быстрее возврат.

Фрактальная размерность траектории связана с $H$: $D = 2 - H$.

## Математическая основа: R/S-анализ

Исходный метод Херста — **нормированный размах (R/S)**. Для временного ряда $\{X_t\}$ длины $T$:

1. Вычисляем среднее: $\bar{X}_T = \frac{1}{T}\sum_{t=1}^T X_t$
2. Накопленные отклонения: $Y_t = \sum_{s=1}^t (X_s - \bar{X}_T)$
3. Размах: $R(T) = \max_{1\leq t \leq T} Y_t - \min_{1\leq t \leq T} Y_t$
4. Стандартное отклонение: $S(T) = \sqrt{\frac{1}{T}\sum_{t=1}^T (X_t - \bar{X}_T)^2}$

Нормированный размах удовлетворяет:

$$
\mathbb{E}\!\left[\frac{R(T)}{S(T)}\right] = C \cdot T^H
$$

Логарифмирование: $\ln(R/S) = \ln C + H \ln T$, поэтому $H$ — наклон регрессии в log-log координатах.

## Анализ вычитания тренда (DFA)

DFA (Peng et al. 1994) более устойчив к нестационарности, чем R/S. Для ряда накопленной суммы $Y(k) = \sum_{i=1}^k (X_i - \bar{X})$:

1. Разбиваем на непересекающиеся окна размера $n$.
2. В каждом окне подгоняем локальный полиномиальный тренд и вычисляем остаточную дисперсию $F^2(n, \nu)$.
3. Усредняем: $F(n) = \sqrt{\frac{1}{N_n}\sum_\nu F^2(n,\nu)}$.
4. Показатель DFA $\alpha$ (эквивалент $H$) — наклон $\log F(n)$ относительно $\log n$.

DFA устраняет полиномиальную нестационарность до порядка $m$ (DFA $m$-го порядка), поэтому предпочтителен для рядов финансовых доходностей.

## Тест отношения дисперсий

Дополнительный статистический тест для проверки $H = 0.5$:

$$
\text{VR}(q) = \frac{\text{Var}(r_t + r_{t-1} + \dots + r_{t-q+1})/q}{\text{Var}(r_t)}
$$

При нулевой гипотезе случайного блуждания $\text{VR}(q) = 1$ для любого $q$. $\text{VR}(q) > 1$ указывает на положительную автокорреляцию (тренд, $H > 0.5$); $\text{VR}(q) < 1$ — на возврат к среднему.

## Python: Несколько оценщиков показателя Херста

```python
import numpy as np
from typing import Tuple

def hurst_rs(series: np.ndarray) -> float:
    """
    Оценка показателя Херста методом R/S-анализа (log-log регрессия).
    """
    lags = np.unique(np.logspace(1, np.log10(len(series) // 2), 30).astype(int))
    rs_values = []
    for lag in lags:
        chunks = [series[i:i+lag] for i in range(0, len(series) - lag, lag)]
        rs_chunk = []
        for chunk in chunks:
            mean_c = chunk.mean()
            cumdev = np.cumsum(chunk - mean_c)
            R = cumdev.max() - cumdev.min()
            S = chunk.std(ddof=1)
            if S > 0:
                rs_chunk.append(R / S)
        if rs_chunk:
            rs_values.append(np.mean(rs_chunk))
    slope, _ = np.polyfit(np.log(lags[:len(rs_values)]), np.log(rs_values), 1)
    return slope

def hurst_dfa(series: np.ndarray, order: int = 1) -> float:
    """
    Оценка показателя Херста методом DFA (анализ вычитания тренда).
    """
    cumsum = np.cumsum(series - series.mean())
    lags = np.unique(np.logspace(1, np.log10(len(series) // 4), 20).astype(int))
    fluctuations = []
    for lag in lags:
        segments = len(series) // lag
        f2 = []
        for s in range(segments):
            seg = cumsum[s*lag:(s+1)*lag]
            t   = np.arange(lag)
            coeffs = np.polyfit(t, seg, order)
            trend  = np.polyval(coeffs, t)
            f2.append(np.mean((seg - trend)**2))
        fluctuations.append(np.sqrt(np.mean(f2)))
    slope, _ = np.polyfit(np.log(lags[:len(fluctuations)]), np.log(fluctuations), 1)
    return slope

def hurst_variance_ratio(series: np.ndarray, max_q: int = 20) -> np.ndarray:
    """Тест отношения дисперсий для горизонтов 2..max_q."""
    var1 = np.var(np.diff(series), ddof=1)
    vr = []
    for q in range(2, max_q + 1):
        multi_returns = series[q:] - series[:-q]
        vr.append(np.var(multi_returns, ddof=1) / (q * var1))
    return np.array(vr)

# --- Пример ---
np.random.seed(42)
# Трендовый ряд (H > 0.5)
trend_prices  = np.cumsum(np.random.normal(0.05, 1, 2000))
# Случайное блуждание (H = 0.5)
random_prices = np.cumsum(np.random.normal(0, 1, 2000))

h_rs_trend  = hurst_rs(np.diff(trend_prices))
h_dfa_trend = hurst_dfa(np.diff(trend_prices))
h_rs_rw     = hurst_rs(np.diff(random_prices))

print(f"Трендовый R/S:         {h_rs_trend:.3f}  (ожидается >0.5)")
print(f"Трендовый DFA:         {h_dfa_trend:.3f}  (ожидается >0.5)")
print(f"Случайное блуждание R/S: {h_rs_rw:.3f}    (ожидается ≈0.5)")
```

## Таблица параметров

| Символ | Название | Описание | Примечание |
|--------|----------|----------|------------|
| $H$ | Показатель Херста | Мера долгосрочной памяти | 0–1; 0.5 = броуновский |
| $R(T)$ | Размах | Max – min накопленного отклонения | Растёт как $T^H$ |
| $S(T)$ | Ст. отклонение | Нормировочный множитель | Выборочное ст. откл. ряда |
| $n$ | Окно DFA | Размер блока локального вычитания | Лог-равномерные, 10–$T/4$ |
| $\alpha$ | Показатель DFA | Наклон DFA ≡ $H$ | Та же интерпретация |
| $\text{VR}(q)$ | Отношение дисперсий | Тест автокорреляции | =1 при случайном блуждании |

## Ограничения и расширения

- **Смещение оценки**: R/S завышает $H$ для коротких рядов и коротко-памятных ARMA-процессов; DFA менее смещён, но требует $T > 500$ для надёжной оценки.
- **Нестационарность**: оба метода предполагают стационарность приращений; структурные сдвиги (смена режимов) дают ложное $H \neq 0.5$.
- **Мультифрактальные рынки**: единственного $H$ может быть недостаточно; **Мультифрактальная модель доходностей (MMAR)** и **Мультифрактальный DFA (MFDFA)** характеризуют полный спектр показателей масштабирования $h(q)$.
- **Нестационарный H**: скользящая оценка $H$ на окнах 250 дней показывает значительные колебания; разные классы активов демонстрируют разную персистентность в разных режимах.
- **Ложная долгая память**: кластеризация волатильности типа GARCH может порождать видимую долгую память в модулях доходностей даже при отсутствии истинной долгой памяти в процессе.

## Применение в финансах

- **Выбор стратегии**: $H > 0.55$ на окне оценки обосновывает трендовые стратегии и моментум. $H < 0.45$ указывает на прибыльность стратегий возврата к среднему (парный трейдинг, стат-арбитраж).
- **Проверка эффективности рынка**: мониторинг $H$ во времени позволяет обнаруживать периоды временной предсказуемости рынка.
- **Фрактальный анализ**: $H$ определяет фрактальную размерность ценовой траектории и подходящее масштабирование времени в моделях риска.
- **Классификация режимов**: в сочетании со [[hidden-markov-models]] или кластеризацией, скользящие оценки $H$ помогают количественно маркировать рыночные режимы.

# Преобразование Гильберта-Хуанга (HHT)

**Преобразование Гильберта-Хуанга (HHT)** — высокоточный метод обработки сигналов, специально разработанный для анализа нелинейных и нестационарных временных рядов. Создано Норденом Хуангом и коллегами в NASA в 1998 году. В отличие от Фурье и вейвлетов, HHT — адаптивный, управляемый данными метод, который обеспечивает точное описание мгновенной частоты и амплитуды сигнала без допущений о линейности или стационарности.

## Предпосылки и мотивация

Финансовые временные ряды не являются ни стационарными, ни линейными. Режимы волатильности меняются, циклы варьируются по частоте, взаимосвязи нелинейны. Классический анализ Фурье предполагает стационарность и разлагает сигнал по фиксированным синусоидальным базисным функциям — плохо подходит для эволюционирующей рыночной динамики. Вейвлеты улучшают это, но всё равно используют фиксированный базис. HHT не использует фиксированного базиса: он извлекает собственные моды из самих данных, делая метод полностью адаптивным.

## Структура метода

### Этап 1: Эмпирическая модовая декомпозиция (EMD)

EMD разлагает сигнал $x(t)$ на конечный набор **собственных модовых функций (IMF)** и остаток через процедуру «просеивания»:

1. Находим все локальные максимумы и минимумы $x(t)$.
2. Строим верхнюю огибающую $e_{\max}(t)$ через кубический сплайн по максимумам, аналогично — нижнюю $e_{\min}(t)$.
3. Вычисляем среднюю огибающую: $m(t) = \frac{1}{2}[e_{\max}(t) + e_{\min}(t)]$.
4. Кандидат на IMF: $h(t) = x(t) - m(t)$.
5. Повторяем шаги 1–4 для $h(t)$ до выполнения критерия остановки (стандартное отклонение последовательных $h$ ниже 0.2).
6. Найденная IMF $c_1(t)$ вычитается из $x(t)$; процедура повторяется на остатке до монотонности остатка.

Исходный сигнал восстанавливается точно:

$$
x(t) = \sum_{k=1}^n c_k(t) + r_n(t)
$$

где $r_n(t)$ — монотонный остаток (долгосрочный тренд).

### Этап 2: Спектральный анализ Гильберта

Для каждой IMF $c_k(t)$ вычисляем преобразование Гильберта:

$$
\hat{c}_k(t) = \frac{1}{\pi} \mathcal{P}\int_{-\infty}^{\infty} \frac{c_k(\tau)}{t - \tau}\, d\tau
$$

Строим аналитический сигнал:

$$
z_k(t) = c_k(t) + i\hat{c}_k(t) = a_k(t)\, e^{i\theta_k(t)}
$$

Мгновенная амплитуда и фаза:

$$
a_k(t) = \sqrt{c_k^2(t) + \hat{c}_k^2(t)}, \qquad \theta_k(t) = \arctan\!\frac{\hat{c}_k(t)}{c_k(t)}
$$

**Мгновенная частота** моды $k$:

$$
\omega_k(t) = \frac{d\theta_k(t)}{dt}
$$

Полное временно-частотное представление энергии — **спектр Гильберта-Хуанга** $H(\omega, t) = \sum_k a_k^2(t) \delta(\omega - \omega_k(t))$ — описывает распределение энергии по частотам, которые сами меняются во времени.

## HHT vs. Фурье vs. Вейвлеты

| Метод | Базисные функции | Применимость | Разрешение |
|:------|:----------------:|:------------:|:----------:|
| **Фурье** | Синусы/косинусы (фикс.) | Линейные, стационарные | Только глобальное |
| **Вейвлеты** | Материнский вейвлет (фикс.) | Нестационарные | Зависит от масштаба |
| **HHT** | Адаптивные (IMF) | **Нелинейные, нестационарные** | **Мгновенное (локальное)** |

## Python: Рабочий процесс EMD

```python
import numpy as np
from PyEMD import EMD
from scipy.signal import hilbert

def hht_analysis(signal: np.ndarray, fs: float = 1.0) -> dict:
    """
    HHT: декомпозиция через EMD, затем мгновенная частота и амплитуда для каждой IMF.

    Args:
        signal: одномерный временной ряд (цены или доходности)
        fs:     частота дискретизации (например, 252 для дневных данных)

    Returns:
        dict с IMF, мгновенными частотами, амплитудами и остатком
    """
    emd = EMD()
    imfs = emd(signal)
    residual = signal - imfs.sum(axis=0)

    inst_freqs = []
    inst_amps  = []
    for imf in imfs:
        analytic = hilbert(imf)
        amp   = np.abs(analytic)
        phase = np.unwrap(np.angle(analytic))
        freq  = np.diff(phase) / (2 * np.pi) * fs
        inst_freqs.append(np.append(freq, freq[-1]))
        inst_amps.append(amp)

    return {
        "imfs":       imfs,
        "residual":   residual,
        "inst_freqs": np.array(inst_freqs),
        "inst_amps":  np.array(inst_amps),
    }

# Пример: нестационарный сигнал
np.random.seed(42)
T = 500
t = np.arange(T)
signal = (np.sin(2 * np.pi * t / 50)
          + np.sin(2 * np.pi * t / 10 + 0.5 * t / 100)
          + 0.01 * t
          + 0.3 * np.random.randn(T))

result = hht_analysis(signal, fs=252)
print(f"Извлечено IMF: {len(result['imfs'])}")
for k, (freq, amp) in enumerate(zip(result['inst_freqs'], result['inst_amps'])):
    print(f"  IMF {k+1}: ср. частота={freq.mean():.4f}, ср. амплитуда={amp.mean():.4f}")
```

## Таблица параметров

| Параметр | Описание | Типичное значение |
|----------|----------|-------------------|
| Критерий остановки | Порог СО для цикла просеивания | 0.1–0.3 |
| Интерполяция | Порядок сплайна для огибающей | Кубический (по умолчанию) |

| `fs` | Частота дискретизации | 252 (дневная), 1 (тик) |

## Ключевые ограничения

- **Смешение мод (mode mixing)**: разные циклы могут попадать в одну IMF. **Ensemble EMD (EEMD)** и **CEEMDAN** снижают этот эффект, добавляя белый шум и усредняя по ансамблю.
- **Краевые эффекты**: интерполяция сплайном нестабильна у границ ряда; зеркальное дополнение снижает артефакты.
- **Вычислительная стоимость**: итеративное просеивание — $O(N \log N)$ с большой константой, значительно медленнее FFT.
- **Неоднозначность**: EMD не гарантирует уникальность декомпозиции; результаты могут варьироваться в зависимости от реализации.

## Применение в финансах

- **Извлечение рыночных циклов**: разложение ценового ряда на доминирующие циклы (краткосрочный возврат к среднему, сезонный, деловой цикл) без артефактов фильтров с фиксированным окном.
- **Адаптивное денойзинг**: удаление высокочастотных IMF (рыночный шум) с сохранением нелинейного ценового тренда.
- **Анализ волатильности**: мгновенная амплитуда доминирующей IMF — динамическая мера интенсивности рыночных колебаний.
- **Торговля на основе циклов**: определение текущего доминирующего периода цикла из спектра Гильберта для динамической настройки периода осцилляторов.

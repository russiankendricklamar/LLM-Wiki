# Построение собственных портфелей через RMT

**Собственные портфели (Eigenportfolios)** — это набор некоррелированных портфелей, веса которых определяются собственными векторами корреляционной или ковариационной матрицы. В сочетании с **Теорией случайных матриц (RMT)** этот метод позволяет выделять устойчивые инвестиционные факторы, очищенные от рыночного шума. Впервые применён в финансах Laloux et al. (1999) и Plerou et al. (1999).

## Предпосылки и мотивация

Выборочная ковариационная матрица, оцененная по $T$ наблюдениям $N$ активов, содержит значительный шум, когда $T$ и $N$ сопоставимы по величине. Отношение $q = T/N$ определяет долю спектральной информации, являющейся чистой ошибкой оценки. RMT предоставляет нулевую гипотезу: при отсутствии истинной кросс-корреляции спектр собственных значений выборочной матрицы описывается **распределением Марченко-Пастура**. Любое собственное значение, попадающее в этот «шумовой» диапазон, следует отбросить или скорректировать перед использованием в оптимизации.

## Математическое обоснование

Для корреляционной матрицы $C$ активов размерности $N$ спектральное разложение выглядит следующим образом:

$$
C = V \Lambda V^T
$$

Где:
- $\Lambda = \text{diag}(\lambda_1, \lambda_2, \dots, \lambda_N)$ — собственные значения в порядке возрастания.
- $V = [v_1, v_2, \dots, v_N]$ — соответствующие ортонормированные собственные векторы.

**Собственный портфель** $E_i$ имеет веса $w_i$, пропорциональные компонентам $i$-го собственного вектора $v_i$. Нормировка обеспечивает инвестируемость портфеля:

$$
w_{i,j} = \frac{v_{i,j}}{\sum_k |v_{i,k}|}
$$

Дисперсия собственного портфеля $i$ равна его собственному значению $\lambda_i$, а любые два собственных портфеля $i \neq j$ ортогональны: $w_i^T C w_j = 0$.

## Граница Марченко-Пастура

При нулевой гипотезе i.i.d. гауссовских доходностей и соотношении $q = T/N$ теоретическая верхняя граница шумовых собственных значений:

$$
\lambda_+ = \sigma^2 \left(1 + \frac{1}{\sqrt{q}}\right)^2
$$

где $\sigma^2$ — дисперсия внедиагональных корреляций. Собственные значения $\lambda_i > \lambda_+$ несут истинный кросс-секционный сигнал:

- **Первый собственный вектор** ($v_1$) — **рыночный фактор**: все веса имеют один знак.
- **Последующие значимые векторы** — секторные факторы и факторы стиля (Value, Momentum, Quality).
- **Шумовой кластер** ($\lambda_i \leq \lambda_+$) — отбрасывается или заменяется оценкой сжатия.

## Очищенная корреляционная матрица

После выделения сигнальных собственных векторов строится **очищенная** корреляционная матрица:

$$
\tilde{C} = \sum_{i:\, \lambda_i > \lambda_+} \lambda_i v_i v_i^T + \hat{\lambda}_{\text{шум}} \sum_{i:\, \lambda_i \leq \lambda_+} v_i v_i^T
$$

где $\hat{\lambda}_{\text{шум}}$ выбирается так, чтобы $\text{tr}(\tilde{C}) = N$. Эта очищенная матрица используется в оптимизации Марковица, обеспечивая существенно более низкий оборот и устойчивые веса.

## Python: Извлечение сигнальных собственных портфелей

```python
import numpy as np

def get_marchenko_pastur_upper(q: float, sigma_sq: float = 1.0) -> float:
    """Верхняя граница шумовых собственных значений (Марченко-Пастур)."""
    return sigma_sq * (1 + 1 / np.sqrt(q)) ** 2

def get_signal_eigenportfolios(corr_matrix: np.ndarray, T: int):
    """
    Возвращает сигнальные собственные векторы и очищенную матрицу корреляций.
    corr_matrix: (N, N) матрица корреляций
    T: число наблюдений в выборке
    """
    N = corr_matrix.shape[0]
    q = T / N
    lambda_plus = get_marchenko_pastur_upper(q)

    eigenvalues, eigenvectors = np.linalg.eigh(corr_matrix)  # порядок возрастания

    signal_mask = eigenvalues > lambda_plus
    signal_eigenvalues = eigenvalues[signal_mask]
    signal_eigenvectors = eigenvectors[:, signal_mask]

    # Очищенная матрица
    noise_eigenvalue_mean = eigenvalues[~signal_mask].mean()
    denoised = (
        signal_eigenvectors @ np.diag(signal_eigenvalues) @ signal_eigenvectors.T
        + noise_eigenvalue_mean
        * eigenvectors[:, ~signal_mask] @ eigenvectors[:, ~signal_mask].T
    )
    diag = np.diag(denoised)
    scale = np.sqrt(np.outer(diag, diag))
    denoised_corr = denoised / scale

    # Веса собственных портфелей (нормировка по L1)
    weights = {}
    for i, (lam, vec) in enumerate(
        zip(signal_eigenvalues[::-1], signal_eigenvectors[:, ::-1].T)
    ):
        w = vec / np.sum(np.abs(vec))
        weights[f"EP_{i+1}_lambda={lam:.2f}"] = w

    return weights, denoised_corr

# Пример: матрица 3x3, T=500 наблюдений
C = np.array([[1.0, 0.7, 0.5], [0.7, 1.0, 0.6], [0.5, 0.6, 1.0]])
ep_weights, C_clean = get_signal_eigenportfolios(C, T=500)
for name, w in ep_weights.items():
    print(f"{name}: {np.round(w, 3)}")
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|--------|----------|----------|-------------------|
| $N$ | Число активов | Размерность матрицы корреляций | 50–3000 |
| $T$ | Число наблюдений | Длина окна оценки | 250–2000 дней |
| $q = T/N$ | Соотношение выборки | Управляет уровнем шума | >2 желательно |
| $\lambda_+$ | Верхняя граница MP | Порог шумовых значений | 1.0–3.0 |
| $\lambda_1$ | Рыночное собств. значение | Наибольшее значение | $0.25N$–$0.5N$ |

## Ограничения и расширения

- **Нестационарность**: корреляции меняются во времени; скользящие или экспоненциально взвешенные оценки предпочтительнее.
- **Негауссовые доходности**: закон MP предполагает гауссовость; для доходностей с тяжёлыми хвостами граница шума смещается, нужен робастный оценщик ковариации.
- **Неоднозначность поворота**: собственные векторы в вырожденных подпространствах не уникальны; вращения PCA (varimax) улучшают интерпретируемость.

## Применение в финансах

1. **Статистический арбитраж**: торговля отклонениями доходности актива от его проекции на сигнальные собственные портфели.
2. **Декомпозиция риска**: выделение рыночного, секторного и идиосинкратического вкладов в дисперсию портфеля.
3. **Очищенная оптимизация**: замена сырой ковариации RMT-очищенной снижает концентрацию и нестабильность весов.
4. **Построение альфа-сигналов**: идиосинкратическая компонента имеет меньшую эффективную размерность и лучше поддаётся предсказанию.

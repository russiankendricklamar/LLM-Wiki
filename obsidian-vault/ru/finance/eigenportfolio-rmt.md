# Построение собственных портфелей через RMT

**Собственные портфели (Eigenportfolios)** — это набор некоррелированных портфелей, веса которых определяются собственными векторами корреляционной или ковариационной матрицы. В сочетании с **Теорией случайных матриц (RMT)** этот метод позволяет выделять устойчивые инвестиционные факторы, очищенные от рыночного шума.

## Математическое обоснование

Для корреляционной матрицы $C$ активов размерности $N$ спектральное разложение выглядит так:
$$
C = V \Lambda V^T
$$
Где:
- $\Lambda = \text{diag}(\lambda_1, \lambda_2, \dots, \lambda_N)$ — собственные значения.
- $V = [v_1, v_2, \dots, v_N]$ — соответствующие собственные векторы.

**Собственный портфель** $E_i$ имеет веса $w_i$, пропорциональные компонентам $i$-го собственного вектора $v_i$. Нормировка весов обычно выполняется следующим образом:
$$
w_{i,j} = \frac{v_{i,j}}{\sum_k |v_{i,k}|}
$$

## Фильтрация через RMT

Согласно [[random-matrix-theory]], мы учитываем только те портфели, чьи собственные значения $\lambda_i > \lambda_+$. 
- **Первый собственный вектор** ($v_1$) обычно представляет **Рыночный фактор** (все веса имеют один знак).
- Следующие значимые векторы представляют **Секторные факторы** или факторы стиля (Value, Momentum).
- Векторы из «шумового кластера» ($\lambda_i < \lambda_+$) игнорируются, так как они не несут предсказательной силы.

## Применение в финансах

1. **Статистический арбитраж**: Торговля отклонениями доходности актива от его проекции на основные собственные портфели.
2. **Декомпозиция риска**: Определение того, какие «скрытые факторы» доминируют в волатильности портфеля.
3. **Очищенная оптимизация**: Использование только значимых собственных портфелей для построения портфеля Марковица, что снижает оборачиваемость.

## Python: Расчет рыночного собственного портфеля

```python
import numpy as np

def get_market_eigenportfolio(corr_matrix):
    # Спектральное разложение
    eigenvalues, eigenvectors = np.linalg.eigh(corr_matrix)
    
    # Самое большое собственное значение — последнее в массиве
    market_vector = eigenvectors[:, -1]
    
    # Нормировка весов (сумма модулей = 1)
    weights = market_vector / np.sum(np.abs(market_vector))
    return weights

# Пример корреляционной матрицы 3x3
C = np.array([[1.0, 0.7, 0.5], [0.7, 1.0, 0.6], [0.5, 0.6, 1.0]])
w = get_market_eigenportfolio(C)
print(f"Веса рыночного фактора: {w}")
```

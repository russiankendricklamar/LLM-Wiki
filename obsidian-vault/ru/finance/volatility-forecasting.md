# Прогнозирование волатильности (GARCH, HAR)

**Прогнозирование волатильности** — это процесс предсказания будущей амплитуды колебаний цен. В отличие от направления цены, волатильность часто демонстрирует кластеризацию (периоды высокой волатильности сменяются аналогичными) и возврат к среднему, что делает её более предсказуемой. Это критически важно для оценки опционов и управления рисками.

## Практическое руководство по внедрению

1.  **GARCH (Generalized Autoregressive Conditional Heteroskedasticity)**:
    *   Улавливает изменяющуюся во времени дисперсию.
    *   Моделирует текущую дисперсию $\sigma_t^2$ как функцию прошлых квадратов доходностей и прошлых дисперсий.
2.  **модель HAR (Heterogeneous Autoregressive)**:
    *   Использует высокочастотные данные (реализованная волатильность).
    *   Основана на «гипотезе гетерогенного рынка»: волатильность определяется разными группами трейдеров (дневные, недельные, месячные).
3.  **Реализованная волатильность (Realized Volatility)**: Расчет волатильности на основе внутридневных (например, 5-минутных) доходностей для большей точности.

## Математическая формулировка

Стандартная модель GARCH(1,1):
$$
\sigma_t^2 = \omega + \alpha \epsilon_{t-1}^2 + \beta \sigma_{t-1}^2
$$
где $\epsilon_{t-1}$ — доходность в момент $t-1$.

Модель HAR-RV:
$$
RV_{t+1d} = \beta_0 + \beta_d RV_{t} + \beta_w RV_{t,w} + \beta_m RV_{t,m} + \epsilon_{t+1}
$$
где $RV_{t,w}$ — средняя реализованная волатильность за последнюю неделю. Для оценки параметров стохастических процессов часто применяются методы [[mcmc]] для более точной работы с негауссовыми распределениями.

## Ключевые метрики
- **Среднеквадратичная ошибка (MSE)**: Разница между предсказанной и реализованной волатильностью.
- **QLIKE Loss**: Робастная функция потерь, специально разработанная для волатильности.
- **Персистентность ($\alpha + \beta$)**: Измеряет, как долго длится шок волатильности.
- **Sharpe Ratio портфеля с Vol-Targeting**: Насколько улучшение прогноза повышает доходность на единицу риска.

## Python: GARCH(1,1) с библиотекой `arch`

```python
import numpy as np
import pandas as pd
from arch import arch_model

# Генерация тестовых доходностей
np.random.seed(42)
returns = np.random.normal(0, 1, 1000)

# Обучение GARCH(1,1)
model = arch_model(returns, vol='Garch', p=1, q=1)
model_fit = model.fit(disp='off')

# Прогноз на следующие 5 дней
forecasts = model_fit.forecast(horizon=5)
print(forecasts.variance.dropna().tail())
```

## Финансовый контекст

Прогнозирование волатильности — это «двигатель» стратегий **Volatility Targeting**, где экспозиция портфеля снижается в периоды высокой турбулентности. В управлении рисками это основной входной параметр для расчета [[value-at-risk]] (VaR). Продвинутые модели используют [[gan]] для симуляции реалистичных поверхностей волатильности при стресс-тестировании.

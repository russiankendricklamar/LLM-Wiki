# Процесс Хоукса (Hawkes Process)

**Процесс Хоукса** — это класс самовозбуждающихся точечных процессов, в которых наступление события увеличивает вероятность появления будущих событий в краткосрочной перспективе. В отличие от стандартного процесса Пуассона, где события независимы, процессы Хоукса описывают эффект «кластеризации», часто наблюдаемый на финансовых рынках.

## Математическое определение

Функция интенсивности $\lambda(t)$ одномерного процесса Хоукса задается формулой:

$$
\lambda(t) = \mu + \int_{-\infty}^t \phi(t-s) dN(s) = \mu + \sum_{T_i < t} \phi(t - T_i)
$$

Где:
- $\mu$ — базовая интенсивность (экзогенные события).
- $\phi(\tau)$ — ядро процесса, обычно экспоненциально затухающее: $\phi(\tau) = \alpha e^{-\beta \tau}$.
- $T_i$ — временные метки предыдущих событий.

Для стационарности процесса коэффициент ветвления $n = \int_0^\infty \phi(\tau) d\tau = \alpha / \beta$ должен быть меньше 1.

## Применение в финансах

Процессы Хоукса широко используются в высокочастотной торговле (HFT) и анализе микроструктуры рынка:
1. **Динамика книги ордеров**: Моделирование потока лимитных и рыночных ордеров.
2. **Flash Crashes (мгновенные обвалы)**: Понимание того, как одна крупная сделка может спровоцировать каскад ликвидаций.
3. **Кластеризация волатильности**: Поскольку события порождают события, это естественно объясняет периоды затяжной высокой волатильности.

## Python: Симуляция процесса Хоукса

```python
import numpy as np

def simulate_hawkes(mu, alpha, beta, T):
    t = 0
    events = []
    
    while t < T:
        # Алгоритм прореживания (Thinning algorithm)
        M = mu + alpha * 10 # Грубая верхняя граница
        t += np.random.exponential(1/M)
        
        if t > T: break
        
        # Текущая интенсивность
        lambda_t = mu + sum(alpha * np.exp(-beta * (t - np.array(events))))
        
        if np.random.rand() < lambda_t / M:
            events.append(t)
            
    return np.array(events)

# Параметры: mu=0.1, alpha=0.5, beta=0.8, Длительность=100
events = simulate_hawkes(0.1, 0.5, 0.8, 100)
print(f"Сгенерировано событий: {len(events)}")
```

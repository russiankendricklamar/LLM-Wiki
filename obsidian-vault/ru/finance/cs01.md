# CS01 (Credit Spread 01)

**CS01** (Credit Spread 01, также DVCS или Credit DV01) — мера чувствительности стоимости кредитного инструмента к изменению кредитного спреда на 1 базисный пункт ($0.0001$). CS01 является стандартным показателем кредитного риска для портфелей корпоративных облигаций, кредитно-дефолтных свопов (CDS) и структурированных продуктов.

## Обзор и предпосылки

Кредитный риск — риск того, что эмитент не выполнит обязательства — реализуется в ценах через кредитный спред: разницу между доходностью инструмента и безрисковой ставкой. CS01 отвечает на вопрос: сколько стоит портфель потеряет при расширении всех спредов на 1 б.п.? Агрегация CS01 по эмитентам, секторам и срокам даёт трейдерам и риск-менеджерам полное понимание кредитной экспозиции.

## Математическая база

**Определение CS01** через конечную разность:

$$
\text{CS01} \approx -\frac{V(s + \Delta s) - V(s)}{\Delta s}, \quad \Delta s = 0.0001
$$

**Аналитический CS01 для корпоративной облигации** через спред-дюрацию $D_S$:

$$
\text{CS01} = MV \cdot D_S \cdot 0.0001
$$

где $D_S \approx D_{\text{mod}} \cdot \frac{1}{1+s}$ при небольших спредах, $MV$ — рыночная стоимость.

**RPV01 кредитно-дефолтного свопа** (Risky PV01) — стоимость 1 б.п. годовой купонной выплаты:

$$
\text{RPV01} = \sum_{i=1}^n \delta_i\,B(0,T_i)\,Q(T_i)
$$

где $\delta_i$ — дробный год периода $i$, $B(0,T_i)$ — дисконт-фактор, $Q(T_i) = e^{-\int_0^{T_i}\lambda_u\,du}$ — вероятность выживания.

**CS01 CDS** — производная RPV01 по базовой интенсивности:

$$
\text{CS01}_{\text{CDS}} \approx \text{RPV01} \cdot 0.0001
$$

**Бакетированный (поключевой) CS01** — чувствительность к сдвигу $k$-й точки кривой спреда:

$$
\text{KR-CS01}_k = -\frac{V(s_1,\dots,s_k + \Delta s,\dots,s_n) - V(\mathbf{s})}{\Delta s}
$$

## Ключевые свойства

- CS01 > 0 для длинной позиции в облигациях: рост спреда снижает цену
- CS01 < 0 для покупателя защиты CDS: рост спреда увеличивает стоимость защиты
- Для нулевой суммарной кредитной экспозиции: сумма CS01 по всем позициям = 0
- CS01 нелинеен для дальних сроков из-за дисконтирования и кривизны

## Реализация на Python

```python
import numpy as np
import pandas as pd

def cs01_by_bump(cashflows: np.ndarray, times: np.ndarray, discount_rates: np.ndarray,
                 spread: float, bump: float = 0.0001) -> float:
    """
    CS01 методом полного перерасчёта (bump-and-reprice).
    cashflows   : денежные потоки инструмента
    times       : времена платежей в годах
    discount_rates: безрисковые дисконтные ставки по каждому сроку
    spread      : текущий кредитный спред
    """
    def price(s):
        df = np.exp(-(discount_rates + s) * times)
        return float(np.sum(cashflows * df))
    return -(price(spread + bump) - price(spread)) / bump

def bucketed_cs01(cashflows: np.ndarray, times: np.ndarray, discount_rates: np.ndarray,
                  spreads: np.ndarray, buckets: np.ndarray,
                  bump: float = 0.0001) -> dict:
    """
    Бакетированный CS01 по ключевым точкам кривой спреда.
    buckets: тенора ключевых точек (например, [1, 2, 3, 5, 7, 10])
    """
    result = {}
    for k, bucket in enumerate(buckets):
        spreads_up = spreads.copy()
        mask = np.abs(times - bucket) < 0.5  # ±0.5 года от бакета
        spreads_up[mask] += bump
        def price_base():
            df = np.exp(-(discount_rates + spreads) * times)
            return float(np.sum(cashflows * df))
        def price_up():
            df = np.exp(-(discount_rates + spreads_up) * times)
            return float(np.sum(cashflows * df))
        result[f'{bucket}Y'] = -(price_up() - price_base()) / bump
    return result

def cds_cs01(notional: float, spread: float, recovery: float,
             maturities: np.ndarray, survival_probs: np.ndarray,
             risk_free_rates: np.ndarray) -> float:
    """CS01 CDS через RPV01."""
    rpv01 = 0.0
    prev_t = 0.0
    for i, T in enumerate(maturities):
        dt = T - prev_t
        disc = np.exp(-risk_free_rates[i] * T)
        surv = survival_probs[i]
        rpv01 += dt * disc * surv
        prev_t = T
    return notional * rpv01 * 0.0001

# Пример: 5-летняя корпоративная облигация
times = np.array([0.5, 1.0, 1.5, 2.0, 2.5, 3.0, 3.5, 4.0, 4.5, 5.0])
coupons = np.full(len(times), 25000.0)  # купон $25k
coupons[-1] += 1000000.0               # + номинал
discount_rates = np.full(len(times), 0.04)
spread = 0.0120  # 120 б.п.
cs01_total = cs01_by_bump(coupons, times, discount_rates, spread)
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $s$ | Кредитный спред | Текущий спред над безрисковой ставкой | $20$ – $2000$ б.п. |
| $D_S$ | Спред-дюрация | Чувствительность цены к спреду | $0.5$ – $9$ лет |
| $\text{RPV01}$ | Рисковая аннуитетная стоимость | Цена 1 б.п. купона CDS с учётом дефолта | $0$ – $5$ |
| $Q(T)$ | Вероятность выживания | Безусловная вероятность дожить до $T$ | $(0, 1)$ |
| $\delta_i$ | Дробный год | Длина купонного периода | $0.25$ (квартал) – $1.0$ (год) |

## Ограничения и расширения

**Ограничения:**
- CS01 — линейная аппроксимация; для больших сдвигов спреда нелинейность (convexity) существенна.
- При параллельном сдвиге кривой суммарный CS01 корректен, но не отражает риск изменения наклона кривой.
- Для экзотических кредитных инструментов (TRS, CLN) расчёт CS01 требует полной модели дисконтирования.

**Расширения:**
- **[[dv01]]**: аналогичная мера для процентного риска; CS01 и DV01 агрегируются раздельно.
- **[[cox-process]]**: интенсивностная модель дефолта для аналитического CS01 CDS.
- **XVA-интеграция**: CS01 CVA зависит от и кредитного спреда контрагента, и рыночного риска портфеля.

## Практическое применение

1. **Хеджирование кредитного риска**: покупка/продажа CDS для нейтрализации суммарного CS01 портфеля.
2. **Лимиты кредитного риска**: установка максимального CS01 на эмитента, сектор и регион.
3. **P&L-атрибуция**: ежедневный P&L = $\sum_i \text{CS01}_i \times \Delta s_i$ по каждому эмитенту.
4. **Структурированные продукты**: CS01 транша CDO — нелинейная функция коррелированных спредов пула.
5. **Торговля кривой**: разница CS01 по коротким и длинным тенорам — позиция на наклон кривой спредов.

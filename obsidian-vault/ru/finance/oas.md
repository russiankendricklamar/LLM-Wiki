# Option-Adjusted Spread (OAS)

**Option-Adjusted Spread (OAS)** — это разница в доходности между облигацией со встроенным опционом (например, правом отзыва или правом погашения) и безрисковой кривой доходности, скорректированная на стоимость этого опциона.

OAS позволяет инвесторам сравнивать облигации с разными типами встроенных опционов на равных условиях, выделяя чистую премию за кредитный риск и риск ликвидности.

## Зачем нужна корректировка?

Обычный Z-спред (Zero-volatility spread) просто показывает разницу в доходности, не учитывая, что встроенный опцион может изменить денежные потоки облигации при изменении процентных ставок. 

- **Для Callable-облигаций (с правом отзыва)**: Инвестор несет риск того, что эмитент вернет долг раньше при падении ставок. Этот риск требует дополнительной доходности. Поэтому для таких облигаций **OAS < Z-спред**.
- **Для Putable-облигаций (с правом погашения)**: Инвестор имеет право вернуть облигацию эмитенту при росте ставок. Это право стоит денег. Поэтому для таких облигаций **OAS > Z-спред**.

## Математическая концепция

OAS рассчитывается с использованием стохастических моделей процентных ставок (например, модели Халла-Уайта или Black-Derman-Toy). Суть метода заключается в нахождении такого постоянного спреда $s$, который при добавлении к каждой ветке дерева процентных ставок уравнивает теоретическую цену модели с рыночной ценой облигации:

$$
P_{market} = \frac{1}{N} \sum_{path=1}^{N} \text{PV}(path, r + s)
$$

Где $s$ — это и есть OAS.

## OAS в ипотечных бумагах (MBS)

Для MBS показатель OAS является основным, так как заемщики имеют право на досрочное погашение ипотеки (prepayment risk) при снижении ставок. OAS помогает понять, насколько доходность MBS превышает доходность казначейских облигаций после учета риска досрочного погашения.

## Python: Схематичный расчет

Полный расчет OAS требует построения дерева процентных ставок, но концептуально это выглядит так:

```python
def estimate_oas(z_spread, option_cost):
    """
    Упрощенная связь между Z-спредом, OAS и стоимостью опциона.
    """
    # Для Callable облигации:
    oas = z_spread - option_cost
    return oas

z_spread = 150 # б.п.
option_val = 40 # стоимость риска отзыва в б.п.
oas_result = estimate_oas(z_spread, option_val)

print(f"Чистый кредитный спред (OAS): {oas_result} б.п.")
```

## Сравнение спредов

| Тип спреда | Учитывает кривую | Учитывает опционы | Основное применение |
| :--- | :---: | :---: | :--- |
| **G-Spread** | Нет | Нет | Простые корпоративные облигации |
| **Z-Spread** | Да | Нет | Облигации без опционов |
| **OAS** | Да | Да | MBS, Callable/Putable Bonds |

## Резюме

OAS — это «очищенный» спред. Если OAS облигации кажется слишком высоким по сравнению с аналогами того же кредитного рейтинга, это может сигнализировать о недооцененности бумаги рынком.

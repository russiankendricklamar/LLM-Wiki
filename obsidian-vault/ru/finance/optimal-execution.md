# Кросс-импакт и оптимальное исполнение (Almgren-Chriss)

**Оптимальное исполнение (Optimal Execution)** — это процесс выполнения крупного торгового поручения с минимальными суммарными издержками, которые включают постоянное и временное воздействие на рынок (market impact), а также риск изменения цены. Модель **Алмгрена-Крисса** является отраслевым стандартом для определения оптимальной траектории торговли.

## Фреймворк Алмгрена-Крисса

Модель минимизирует ожидаемые издержки $E[X]$ и их волатильность $V[X]$ (риск). Целевая функция: $U = E[X] + \lambda V[X]$.

### Ключевые формулы

1. **Динамика цены**:
   $P_k = P_{k-1} + \sigma \sqrt{\tau} \epsilon_k - \tau g(n_k / \tau)$ (Постоянный импакт)
   $\tilde{P}_k = P_k - h(n_k / \tau)$ (Временный импакт)

2. **Оптимальная траектория ликвидации**:
   Для риск-нейтрального трейдера ($\lambda=0$) оптимальной стратегией является равномерная продажа (TWAP).
   Для трейдера, избегающего риска ($\lambda > 0$), количество оставшихся акций $x_j$ в момент $t_j$:
   
   $$
   x_j = \frac{\sinh(\kappa (T - t_j))}{\sinh(\kappa T)} X
   $$
   
   Где $\kappa \approx \sqrt{\lambda \sigma^2 / \eta}$ — параметр «срочности» (urgency).

## Кросс-импакт (Cross-Impact)

В мультиактивных портфелях торговля одним активом влияет на цены других. Матрица кросс-импакта $\Omega$ описывает эти зависимости:
$$
\Delta P = \Sigma \cdot \text{Sign}(\text{Trades}) \cdot |\text{Trades}|^\delta
$$

## Применение в финансах

1. **VWAP/TWAP алгоритмы**: Определение того, как разбить ордер в 1 млн акций на части в течение дня.
2. **Ребалансировка портфеля**: Минимизация издержек при одновременном изменении весов в сотнях инструментов.
3. **Управление рисками**: Оценка «ликвидационной стоимости» фонда в условиях рыночного стресса.

## Python: Расчет оптимальной траектории

```python
import numpy as np

def almgren_chriss_trajectory(X, T, N, kappa):
    """
    X: Общий объем к продаже
    T: Общее время
    N: Количество шагов
    kappa: Параметр срочности
    """
    times = np.linspace(0, T, N)
    holdings = X * np.sinh(kappa * (T - times)) / np.sinh(kappa * T)
    return holdings

# Пример: Продажа 1 млн акций, 10 шагов, kappa=0.5
X_total = 1_000_000
traj = almgren_chriss_trajectory(X_total, 1.0, 10, 0.5)

print("Остаток позиции на каждом шаге:")
print(traj.astype(int))
```

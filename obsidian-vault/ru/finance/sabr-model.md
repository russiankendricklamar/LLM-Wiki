# Модель SABR

**Модель SABR** (Stochastic Alpha, Beta, Rho) — это модель стохастической волатильности, широко используемая на рынках производных финансовых инструментов на процентные ставки. Она была разработана Патриком Хаганом и другими для описания «улыбки волатильности» (volatility smile) или «скоса» (skew), наблюдаемых в рыночных ценах.

## Динамика (УСДУ)

Модель описывает эволюцию форвардной цены $F_t$ и её мгновенной волатильности $\alpha_t$:

$$
\begin{aligned}
dF_t &= \alpha_t F_t^\beta dW_t^1 \\
d\alpha_t &= \nu \alpha_t dW_t^2
\end{aligned}
$$

Где два винеровских процесса коррелированы: $d[W^1, W^2]_t = \rho dt$.

### Параметры:
- $\alpha$: Начальный уровень волатильности («Альфа»).
- $\beta$: Экспонента для форвардной цены («Бета»). $\beta=1$ соответствует логнормальной модели, $\beta=0$ — нормальной.
- $\rho$: Корреляция между ценой и волатильностью («Ро»). Влияет на скос улыбки.
- $\nu$: Волатильность волатильности («Vol-of-vol»). Влияет на выпуклость улыбки.

## Формула Хагана для вмененной волатильности

Одним из главных преимуществ модели SABR является наличие точной асимптотической формулы для вмененной волатильности Блэка-Шоулза $\sigma_{impl}$:

$$
\sigma_{impl}(K, F) \approx \frac{\alpha}{(FK)^{(1-\beta)/2} [1 + \frac{(1-\beta)^2}{24} \ln^2(F/K) + \dots]} \cdot \frac{z}{x(z)} \cdot [1 + (\dots)T]
$$

(Примечание: Полная формула сложна, но позволяет проводить экстремально калибровку модели под рыночные данные.)

## Python: Расчет вмененной волатильности

```python
import numpy as np

def sabr_vol(K, F, T, alpha, beta, rho, nu):
    if F == K: # Около денег (At-the-money)
        return alpha / (F**(1-beta)) * (1 + ((1-beta)**2/24 * alpha**2/(F**(2-2*beta)) + 0.25*rho*beta*nu*alpha/(F**(1-beta)) + (2-3*rho**2)/24 * nu**2)*T)
    
    # Упрощенная версия для визуализации
    logFK = np.log(F/K)
    z = (nu/alpha) * (F*K)**((1-beta)/2) * logFK
    xz = np.log((np.sqrt(1 - 2*rho*z + z**2) + z - rho) / (1 - rho))
    
    vol = (alpha / ((F*K)**((1-beta)/2) * (1 + (1-beta)**2/24 * logFK**2 + (1-beta)**4/1920 * logFK**4))) * (z/xz)
    return vol
```

## Чувствительность параметров

- **Увеличение $\rho$**: Усиливает скос (наклоняет улыбку).
- **Увеличение $\nu$**: Усиливает эффект «улыбки» (кривизну).
- **Изменение $\beta$**: Управляет «скелетом» модели (тем, как улыбка движется при изменении форвардной цены).

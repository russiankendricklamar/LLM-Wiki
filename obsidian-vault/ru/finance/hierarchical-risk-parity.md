# Иерархический паритет риска (HRP)

**Иерархический паритет риска (Hierarchical Risk Parity, HRP)** — это современный метод аллокации активов, разработанный Маркосом Лопесом де Прадо. HRP решает главную проблему классической оптимизации Марковица (Mean-Variance): высокую чувствительность к ошибкам оценки ковариации и нестабильность весов при наличии сильно коррелированных активов.

## Почему это работает?

В отличие от традиционных методов, HRP использует **теорию графов** и **машинное обучение** (иерархическую кластеризацию) для построения диверсифицированного портфеля без необходимости обращать матрицу ковариации.

## Алгоритм HRP состоит из трех этапов:

1.  **Иерархическая кластеризация**: Группировка активов на основе их корреляций. Похожие активы объединяются в кластеры, образуя древовидную структуру (дендрограмму).
2.  **Квази-диагонализация**: Перестановка активов в ковариационной матрице таким образом, чтобы сильно коррелированные активы находились рядом. Это делает матрицу «почти диагональной».
3.  **Рекурсивное деление (Recursive Bisection)**: Распределение весов сверху вниз по дереву. На каждом уровне кластеры делятся на две группы, и капитал распределяется между ними обратно пропорционально их совокупному риску.

## Математическое преимущество

HRP не требует обращения матрицы ковариации ($\Sigma^{-1}$). Если два актива почти идеально коррелированы, метод Марковица может выдать огромные веса с разным знаком (лонг/шорт), тогда как HRP просто объединит их в один кластер и распределит вес поровну, обеспечивая стабильность.

## Python: Концептуальный пример

Использование библиотеки `scipy` для кластеризации и расчета весов:

```python
import numpy as np
import scipy.cluster.hierarchy as sch

def get_hrp_weights(cov, corr):
    # 1. Кластеризация
    dist = np.sqrt(0.5 * (1 - corr)) # Метрика расстояния
    link = sch.linkage(dist, 'single')
    
    # 2. Квази-диагонализация (упрощенно)
    sort_ix = sch.leaves_list(link)
    
    # 3. Рекурсивное распределение весов
    # (В реальности это рекурсивная функция по дендрограмме)
    weights = np.ones(len(sort_ix)) / len(sort_ix)
    return weights[sort_ix]

# Пример данных
cov_matrix = np.array([[0.04, 0.02], [0.02, 0.09]])
corr_matrix = np.array([[1.0, 0.5], [0.5, 1.0]])
w = get_hrp_weights(cov_matrix, corr_matrix)
print(f"Веса HRP: {w}")
```

## Сравнение методов аллокации

| Характеристика | Markowitz (MV) | Risk Parity (RP) | HRP |
| :--- | :---: | :---: | :---: |
| **Обращение матрицы** | Да | Нет | **Нет** |
| **Чувствительность к шуму** | Высокая | Средняя | **Низкая** |
| **Учет иерархии активов** | Нет | Нет | **Да** |
| **Стабильность весов** | Низкая | Высокая | **Очень высокая** |
| **Нужен прогноз доходности**| Да | Нет | **Нет** |

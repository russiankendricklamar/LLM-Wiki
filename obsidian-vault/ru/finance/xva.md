# XVA (Корректировки стоимости)

**XVA** — это общее название для различных поправок к безрисковой цене производного финансового инструмента. Эти корректировки необходимы для учета кредитного риска, стоимости фондирования и регуляторных требований к капиталу в посткризисном финансовом мире.

## Основные компоненты XVA

1.  **CVA (Credit Valuation Adjustment)**: Корректировка на кредитный риск контрагента. Отражает рыночную стоимость риска того, что контрагент объявит дефолт, когда контракт имеет для нас положительную стоимость.
2.  **DVA (Debit Valuation Adjustment)**: Корректировка на наш собственный кредитный риск. Отражает выгоду для нас (снижение обязательств), если мы сами объявим дефолт, когда контракт имеет для нас отрицательную стоимость.
3.  **FVA (Funding Valuation Adjustment)**: Корректировка на стоимость финансирования. Учитывает разницу между ставками по займам и ставками по обеспечению (colateral).
4.  **KVA (Capital Valuation Adjustment)**: Отражает стоимость удержания регуляторного капитала (Basel III) в течение всего срока жизни сделки.
5.  **MVA (Margin Valuation Adjustment)**: Учитывает стоимость финансирования начальной маржи (Initial Margin).

## Математическая концепция (CVA)

CVA рассчитывается как ожидаемый убыток по сделке в случае дефолта контрагента:

$$
\text{CVA} \approx (1-R) \sum_{i=1}^{n} \text{EE}(t_i) \cdot PD(t_{i-1}, t_i) \cdot D(t_i)
$$

Где:
- $R$ — коэффициент возмещения (Recovery Rate).
- $\text{EE}(t_i)$ — ожидаемая экспозиция (Expected Exposure) в момент времени $t_i$.
- $PD$ — вероятность дефолта (Probability of Default) на интервале.
- $D(t_i)$ — дисконт-фактор.

## Почему это важно?

До кризиса 2008 года деривативы часто оценивались по безрисковым кривым. Сегодня XVA может составлять значительную часть P&L банка. Неправильный расчет XVA может привести к тому, что сделка, кажущаяся прибыльной, на самом деле будет убыточной после учета стоимости капитала и риска.

## Python: Простейшая оценка CVA

```python
import numpy as np

def calculate_simple_cva(expected_exposures, pd_profile, recovery_rate=0.4):
    """
    Упрощенный расчет CVA на основе профиля экспозиций и вероятностей дефолта.
    """
    lgd = 1 - recovery_rate # Loss Given Default
    cva = lgd * np.sum(expected_exposures * pd_profile)
    return cva

# Прогноз экспозиций на 5 лет
ee = np.array([100, 120, 150, 130, 90])
# Вероятности дефолта по годам
pd = np.array([0.01, 0.012, 0.015, 0.018, 0.02])

result = calculate_simple_cva(ee, pd)
print(f"Оценочный CVA: ${result:.2f}")
```

| :--- | :--- | :---: |
| **CVA** | Дефолт контрагента | Снижает цену |
| **DVA** | Наш собственный дефолт | Повышает цену |
| **FVA** | Стоимость ликвидности | Зависит от базиса |

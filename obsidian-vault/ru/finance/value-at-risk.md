# Value at Risk (VaR)

**Value at Risk (VaR)** — это стандартная мера рыночного риска, которая оценивает максимально возможные убытки портфеля активов за определенный период времени с заданным уровнем доверия (confidence level).

Например, если дневной VaR портфеля составляет $1 млн с уровнем доверия 95%, это означает, что в 95% случаев убытки за день не превысят $1 млн, и лишь в 5% случаев убытки могут быть больше.

## Методы расчета

Существует три основных подхода к расчету VaR:

1.  **Параметрический метод (Вариация-Ковариация)**: Предполагает, что доходности активов распределены нормально.
2.  **Историческое моделирование**: Использует реальные исторические данные о доходностях для оценки распределения будущих убытков.
3.  **Метод Монте-Карло**: Генерирует тысячи случайных сценариев развития рынка на основе заданных параметров.

## Математическая формула (Параметрический VaR)

Для одного актива параметрический VaR рассчитывается по формуле:

$$
\text{VaR}_{\alpha} = S \cdot \sigma \cdot \sqrt{t} \cdot Z_{\alpha}
$$

Где:
- $S$ — стоимость портфеля.
- $\sigma$ — волатильность доходности актива.
- $t$ — временной горизонт (в днях).
- $Z_{\alpha}$ — квантиль стандартного нормального распределения для заданного уровня доверия $\alpha$.

## Python: Расчет VaR

Пример расчета параметрического и исторического VaR на Python:

```python
import numpy as np
from scipy.stats import norm

def calculate_var(returns, confidence_level=0.95, portfolio_value=1000000):
    # Параметрический VaR
    mu = np.mean(returns)
    sigma = np.std(returns)
    z_score = norm.ppf(confidence_level)
    parametric_var = portfolio_value * (z_score * sigma - mu)
    
    # Исторический VaR
    historical_var = portfolio_value * -np.percentile(returns, (1 - confidence_level) * 100)
    
    return parametric_var, historical_var

# Пример данных (дневные доходности)
np.random.seed(42)
returns = np.random.normal(0.001, 0.02, 1000)
p_var, h_var = calculate_var(returns)

print(f"Параметрический VaR (95%): ${p_var:,.2f}")
print(f"Исторический VaR (95%): ${h_var:,.2f}")
```

## Преимущества и недостатки

| Характеристика | Плюсы | Минусы |
| :--- | :--- | :--- |
| **Простота** | Легко интерпретировать в денежном выражении | Не учитывает размер убытков в "хвосте" (проблема Expected Shortfall) |
| **Стандартизация** | Позволяет сравнивать риски разных портфелей | Сильно зависит от предположения о нормальности (для парам. метода) |
| **Регуляция** | Используется в Базельских соглашениях | Плохо работает во время рыночных кризисов |

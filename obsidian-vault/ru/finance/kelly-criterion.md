# Критерий Келли

**Критерий Келли** — это математическая формула, используемая для определения оптимального размера ставки в серии пари с целью максимизации логарифма капитала. В количественных финансах этот метод является ключевым инструментом для распределения капитала и управления рисками.

## Максимизация логарифма капитала

Основная идея критерия Келли заключается в максимизации математического ожидания логарифма капитала, что эквивалентно максимизации геометрического темпа роста. Для серии ставок с результатами $W_i$ цель состоит в том, чтобы найти долю $f$ от текущего баланса, которая максимизирует:

$$
E[\ln(W_n)] = E\left[\sum_{i=1}^n \ln(1 + f X_i)\right]
$$

где $X_i$ — доходность $i$-й ставки.

## Формула Келли

Для простого пари с двумя исходами (выигрыш или проигрыш):
- Вероятность выигрыша $p$, коэффициент $b$ к 1 (чистая прибыль $b$).
- Вероятность проигрыша $q = 1 - p$ (потеря всей ставки).

Оптимальная доля $f^*$ для ставки определяется как:

$$
f^* = \frac{bp - q}{b} = \frac{p(b+1) - 1}{b}
$$

В непрерывном случае (например, доходность акций по модели GBM), доля Келли составляет:

$$
f^* = \frac{\mu - r}{\sigma^2}
$$

где $\mu$ — ожидаемая доходность, $r$ — безрисковая ставка, а $\sigma$ — волатильность.

## Применение и ограничения

### Беттинг и формирование портфеля
Критерий Келли широко используется в азартных играх (спортивные ставки, блэкджек) и финансах (определение размера позиции в алгоритмической торговле). Он устанавливает теоретический предел кредитного плеча.

### Дробный Келли (Fractional Kelly)
На практике использование полной доли Келли ($1.0 \times f^*$) может привести к экстремальной волатильности и риску глубокой просадки, если параметры $(\mu, \sigma)$ оценены неточно. Многие практики используют **дробный Келли** (например, «Половина Келли», $0.5 \times f^*$), чтобы снизить волатильность, сохраняя значительную часть темпа роста.

## Ожидаемая полезность

Критерий Келли соответствует логарифмической функции полезности $U(W) = \ln(W)$, что подразумевает коэффициент относительного неприятия риска (CRRA), равный 1.

$$
E[U(W)] = p \ln(1 + fb) + q \ln(1 - f)
$$

Максимизация этого выражения по $f$ дает классическую формулу Келли.

## Реализация на Python: Симуляция роста

Следующая симуляция сравнивает управление капиталом по Келли с фиксированной долей.

```python
import numpy as np

def simulate_kelly(p, b, initial_wealth=100, n_steps=100, fraction=1.0):
    f_star = (b * p - (1 - p)) / b
    bet_size = f_star * fraction
    
    wealth = [initial_wealth]
    outcomes = np.random.choice([b, -1], size=n_steps, p=[p, 1-p])
    
    for outcome in outcomes:
        current_wealth = wealth[-1]
        wager = current_wealth * bet_size
        wealth.append(current_wealth + wager * outcome)
        
    return np.array(wealth)

# Параметры
p = 0.55  # 55% вероятность выигрыша
b = 1     # Коэффициент 1:1
n_steps = 500

kelly_wealth = simulate_kelly(p, b, n_steps=n_steps, fraction=1.0)
half_kelly_wealth = simulate_kelly(p, b, n_steps=n_steps, fraction=0.5)

print(f"Финальный капитал (Келли): {kelly_wealth[-1]:.2f}")
print(f"Финальный капитал (Половина Келли): {half_kelly_wealth[-1]:.2f}")
```

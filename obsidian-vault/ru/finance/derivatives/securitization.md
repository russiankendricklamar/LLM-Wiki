# Секьюритизация

**Секьюритизация** — процесс объединения неликвидных активов (ипотек, автокредитов, дебиторской задолженности по кредитным картам, студенческих кредитов) в **специальную проектную компанию (SPV)**, выпускающую торгуемые ценные бумаги (транши), обеспеченные денежными потоками пула. Преобразуя балансовые активы в инструменты рынка капитала, секьюритизация обеспечивает оригинаторам фондирование и передачу риска, а инвесторам — доступ к диверсифицированным кредитным экспозициям с настраиваемым профилем «риск — доходность».

## Механика

### Структура SPV

Оригинатор (банк, финансовая компания) продаёт пул активов в банкротоизолированный SPV. SPV финансирует покупку выпуском траншей ценных бумаг, обеспеченных активами (ABS). Денежные потоки (платежи заёмщиков по основному долгу и процентам) распределяются по траншам согласно **каскаду (waterfall)** — контрактному приоритету платежей.

### Каскадная структура

Типичный последовательный каскад:

1. **Старшие комиссии**: доверительный управляющий, сервисер, административные расходы
2. **Процент старшего транша**: класс A получает проценты первым
3. **Процент мезанинного транша**: класс B получает проценты следующим
4. **Основной долг старшего транша**: класс A получает плановое и досрочное погашение
5. **Основной долг мезанинного транша**: класс B после полного погашения A
6. **Equity/остаток**: остаток поступает equity-траншу (позиция первого убытка)

**Тест избыточного обеспечения (OC test)** перенаправляет денежные потоки от младших к старшим траншам при падении покрытия ниже порога.

## Ипотечные ценные бумаги (MBS)

### Агентские vs. неагентские

**Агентские MBS** (Ginnie Mae, Fannie Mae, Freddie Mac): государственная гарантия основного долга и процентов; основной риск — досрочное погашение; огромный рынок (~$9 трлн).

**Неагентские MBS**: без государственной гарантии; кредитный риск существен; структурируются в транши с кредитным усилением. Субпрайм и Alt-A неагентские MBS стали центром кризиса 2007-08.

### Риск досрочного погашения

Заёмщики могут рефинансировать или переехать, создавая **риск досрочного погашения**. Модель **PSA (Public Securities Association)**:

$$
\text{CPR}(t) = \min(0.06, \; 0.002 \cdot t) \quad \text{(100% PSA)}
$$

где CPR — условная ставка досрочного погашения, $t$ — возраст кредита в месяцах. Ежемесячная смертность:

$$
\text{SMM}(t) = 1 - (1 - \text{CPR}(t))^{1/12}
$$

### OAS — спред с учётом опциона

Поскольку MBS содержат встроенный опцион досрочного погашения, правильная мера спреда — **OAS**: постоянный спред над безрисковой кривой, уравнивающий модельную цену (среднее по траекториям ставок и зависимым от них досрочным погашениям) с рыночной:

$$
P_{\text{рынок}} = \frac{1}{N}\sum_{k=1}^{N} \sum_{t=1}^{T} \frac{CF_t^{(k)}}{(1 + r_t^{(k)} + \text{OAS})^t}
$$

где $CF_t^{(k)}$ — денежные потоки на траектории $k$, $r_t^{(k)}$ — смоделированные короткие ставки. См. [[oas]] для полного вывода.

## Кредитное усиление

Механизмы защиты старших траншей от убытков:

| Метод | Описание |
|:---|:---|
| **Субординация** | Младшие транши поглощают убытки раньше старших |
| **Избыточное обеспечение** | Номинал пула превышает выпущенные ценные бумаги |
| **Избыточный спред** | Купон пула превышает средневзвешенный купон траншей |
| **Резервный счёт** | Денежные средства из начального выпуска |
| **Страхование монолайнером** | Гарантия третьей стороны (до кризиса) |

Уровень кредитного усиления определяет рейтинг: 30% субординации может обеспечить AAA для старшего транша.

## ABS (ценные бумаги, обеспеченные активами)

Помимо ипотек, секьюритизация охватывает разнообразное обеспечение:

- **ABS на автокредиты**: короткая дюрация (3-5 лет), низкое досрочное погашение, предсказуемые убытки
- **ABS на кредитные карты**: возобновляемая структура с управляемой амортизацией; мастер-траст выпускает несколько серий
- **ABS на студенческие кредиты**: длинная дюрация; неопределённость из-за программ погашения, привязанных к доходу
- **CLO (обеспеченные кредитные обязательства)**: пулы левериджированных кредитов с активным управлением; крупнейший рынок структурированного кредита после кризиса

## CDO-квадрат

**CDO-квадрат** (CDO$^2$) — CDO, обеспечением которого служат транши других CDO. Повторное транширование создаёт экстремальный леверидж: небольшой рост дефолтности базовых активов каскадирует через два уровня субординации. Эти инструменты усилили убытки в 2007-08 и сейчас практически не выпускаются.

## Синтетическая секьюритизация

Вместо физической передачи активов в SPV оригинатор использует **CDS-контракты** (см. [[credit-derivatives-cds]]) для синтетической передачи кредитного риска. SPV продаёт защиту на референсный портфель и выпускает кредитные ноты. Преимущества: не требуется юридическая передача кредитов; ускоренное исполнение; снижение регуляторного капитала по Basel III/IV.

## Кризис 2007-08

Кризис обнажил системные сбои по всей цепочке секьюритизации:

- **Originate-to-distribute**: оригинаторы не имели стимулов к проверке заёмщиков — кредиты NINJA (No Income, No Job, No Assets) распространились массово
- **Провал рейтинговых агентств**: модели использовали гауссовы копулы с низкими предположениями о корреляции (см. [[cdo-tranches]]), выдавая завышенные рейтинги AAA субпрайм-траншам
- **Спирали переоценки**: падение цен MBS вынуждало левериджированных держателей продавать, ещё более снижая цены — классическая экстерналия пожарной распродажи
- **Коллапс корреляций**: дефолты субпрайм-ипотек оказались гораздо более коррелированными, чем предполагали исторические данные
- **Испарение ликвидности**: SIV и ABCP-кондуиты, финансировавшие долгосрочные MBS краткосрочными бумагами, столкнулись с кризисом overnight-рефинансирования

## Посткризисное регулирование

**Dodd-Frank (2010)**: правило удержания риска — секьюритизатор обязан сохранить не менее 5% кредитного риска («skin in the game»). Квалифицированные жилищные ипотеки (QRM) освобождены.

**EU Securitisation Regulation (2019)**: рамка STS (Simple, Transparent, Standardised) для льготного капитального режима. Обязательная проверка должной осмотрительности, шаблоны прозрачности, запрет ре-секьюритизации.

**Basel III/IV**: повышенные веса риска для секьюритизационных экспозиций; иерархия подходов (SEC-IRBA, SEC-ERBA, SEC-SA) со сквозным анализом качества базового пула.

## Реализация на Python

```python
import numpy as np

def psa_prepayment(age_months: int, psa_speed: float = 1.0) -> float:
    """Условная ставка досрочного погашения по модели PSA."""
    cpr = min(0.06, 0.002 * age_months) * psa_speed
    smm = 1 - (1 - cpr) ** (1 / 12)
    return smm

def mbs_cash_flows(
    balance: float, wac: float, wam: int,
    psa_speed: float = 1.0
) -> np.ndarray:
    """Проекция денежных потоков MBS с досрочным погашением PSA."""
    monthly_rate = wac / 12
    cash_flows = np.zeros(wam)
    remaining = balance

    for t in range(1, wam + 1):
        if remaining <= 0:
            break
        interest = remaining * monthly_rate
        scheduled_principal = (remaining * monthly_rate /
            (1 - (1 + monthly_rate) ** -(wam - t + 1))) - interest
        smm = psa_prepayment(t, psa_speed)
        prepayment = (remaining - scheduled_principal) * smm
        total_principal = scheduled_principal + prepayment
        cash_flows[t - 1] = interest + total_principal
        remaining -= total_principal

    return cash_flows

# Пример: пул $100M, WAC 5.5%, 360 мес., 150% PSA
cfs = mbs_cash_flows(100_000_000, 0.055, 360, psa_speed=1.5)
print(f"Совокупные денежные потоки: ${cfs.sum():,.0f}")
print(f"WAL: {sum((t+1)/12 * cfs[t] for t in range(len(cfs))) / cfs.sum():.1f} лет")
```

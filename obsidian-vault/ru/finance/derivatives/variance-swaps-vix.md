# Свопы на дисперсию и индекс VIX

**Своп на дисперсию** выплачивает разницу между реализованной и страйковой дисперсией по истечении срока. Справедливый страйк определяется модельно-независимо из континуума цен опционов — один из глубочайших результатов теории деривативов. **Индекс VIX** операционализирует эту репликацию по опционам SPX, предоставляя индикатор 30-дневной подразумеваемой волатильности в реальном времени.

## Механика свопа на дисперсию

### Выплата

Своп на дисперсию выплачивает номинал $N_{\text{var}}$, умноженный на разницу между годовой реализованной дисперсией и согласованным страйком:

$$
\text{Выплата} = N_{\text{var}} \cdot (\sigma^2_{\text{realized}} - K_{\text{var}})
$$

Реализованная дисперсия при дискретном наблюдении с $n$ точками:

$$
\sigma^2_{\text{realized}} = \frac{252}{n} \sum_{i=1}^{n} \left(\ln \frac{S_{i}}{S_{i-1}}\right)^2
$$

Без вычитания среднего — свопы на дисперсию используют «безсредневую» конвенцию, согласованную с аргументом репликации.

### Модельно-независимая репликация

Фундаментальный результат (Carr, Madan, 1998; Demeterfi, Derman, Kamal, Zou, 1999): справедливый страйк равен удвоенной цене лог-контракта:

$$
K_{\text{var}} = \frac{2}{T}\, e^{rT} \left[\int_0^{F} \frac{P(K)}{K^2}\,dK + \int_{F}^{\infty} \frac{C(K)}{K^2}\,dK\right]
$$

где $F$ — форвардная цена, $P(K)$ и $C(K)$ — цены OTM путов и коллов. Это **модельно-независимый** результат — никаких предположений о динамике базового актива кроме непрерывности траекторий.

## Индекс VIX

CBOE VIX реализует репликацию свопа на дисперсию для 30-дневного горизонта по опционам SPX:

$$
\text{VIX}^2 = \frac{2}{T} \sum_i \frac{\Delta K_i}{K_i^2}\, e^{rT}\, Q(K_i) - \frac{1}{T}\left(\frac{F}{K_0} - 1\right)^2
$$

где $Q(K_i)$ — срединные цены OTM-опционов, $K_0$ — первый страйк ниже $F$, $T = 30/365$. VIX котируется в годовых пунктах волатильности: $\text{VIX} = 100 \cdot \sqrt{K_{\text{var}}}$.

Ключевые свойства:
- VIX ~ 15 означает дневные движения SPX ~1%; VIX ~ 30 — ~1.9%
- VIX отражает риск-нейтральное ожидание, а не прогноз реализованной волатильности
- **Временная структура VIX** обычно находится в контанго

## Своп на дисперсию vs своп на волатильность

Своп на волатильность выплачивает $\sigma_{\text{realized}} - K_{\text{vol}}$ — линейно по волатильности. По неравенству Йенсена:

$$
\mathbb{E}[\sigma] \leq \sqrt{\mathbb{E}[\sigma^2]}
$$

Справедливый страйк свопа на волатильность строго ниже $\sqrt{K_{\text{var}}}$. Поправка на выпуклость:

$$
K_{\text{vol}} \approx \sqrt{K_{\text{var}}} - \frac{\text{Var}(\sigma)}{8\, K_{\text{var}}^{3/2}}
$$

Свопы на дисперсию реплицируемы; свопы на волатильность — нет — требуется модель волатильности волатильности.

## P&L дельта-хеджированного опциона

P&L непрерывно дельта-хеджированного опциона раскладывается в ставку на дисперсию:

$$
\text{P\&L} = \int_0^T \frac{1}{2}\,\Gamma_t\, S_t^2 \left(\sigma^2_{\text{realized},t} - \sigma^2_{\text{implied}}\right) dt
$$

Результат Карра-Мадана: дельта-хеджированный опцион — это позиция в реализованной vs подразумеваемой дисперсии, взвешенная долларовой гаммой $\frac{1}{2}\Gamma S^2$. Своп на дисперсию устраняет зависимость от траектории гамма-взвешивания.

## Деривативы на VIX

**Фьючерсы VIX** торгуются на CBOE Futures Exchange. Цена фьючерса $F_{\text{VIX}}(t,T)$ не равна текущему VIX — она отражает риск-нейтральное ожидание VIX на экспирацию. **Базис VIX**:

$$
\text{Basis} = F_{\text{VIX}}(t,T) - \text{VIX}(t)
$$

обычно положителен (контанго) из-за отрицательной корреляции доходности SPX и волатильности, создавая устойчивый ролл-доход для стратегий коротких по волатильности.

**Опционы на VIX** — европейские, расчёт по специальной котировке открытия (SOQ). Подразумеваемая волатильность опционов на VIX — «vol-of-vol» — обычно превышает 80%.

## Премия за риск волатильности

**Премия за риск дисперсии** (VRP) — систематическая разница между подразумеваемой и реализованной дисперсией:

$$
\text{VRP} = K_{\text{var}} - \mathbb{E}^{\mathbb{P}}[\sigma^2_{\text{realized}}]
$$

Эмпирически VRP устойчиво положительна: подразумеваемая дисперсия превышает последующую реализованную на 2-4 пункта волатильности в среднем. Эта премия компенсирует продавцам дисперсии поглощение крашевого риска.

## Реализация на Python

```python
import numpy as np

def vix_calculation(
    strikes: np.ndarray, otm_prices: np.ndarray,
    F: float, K0: float, T: float, r: float
) -> float:
    """Расчёт подразумеваемой дисперсии в стиле VIX по OTM-опционам."""
    dK = np.diff(strikes)
    dK = np.append(dK, dK[-1])
    sigma_sq = (2 / T) * np.exp(r * T) * np.sum(
        dK / strikes**2 * otm_prices
    ) - (1 / T) * (F / K0 - 1)**2
    return np.sqrt(max(sigma_sq, 0)) * 100

def realized_variance(prices: np.ndarray, ann_factor: int = 252) -> float:
    """Годовая реализованная дисперсия по ценовому ряду."""
    log_returns = np.log(prices[1:] / prices[:-1])
    return float(ann_factor * np.mean(log_returns**2))

def variance_swap_pnl(
    K_var: float, realized_var: float, notional_vega: float
) -> float:
    """P&L длинного свопа на дисперсию (конвенция вега-номинала)."""
    N_var = notional_vega / (2 * np.sqrt(K_var))
    return N_var * (realized_var - K_var)

def convexity_adjustment(K_var: float, vol_of_vol: float) -> float:
    """Справедливый страйк свопа на волатильность с поправкой на выпуклость."""
    return np.sqrt(K_var) - vol_of_vol**2 / (8 * K_var**1.5)
```

## Применения

1. **Торговля волатильностью**: направленные позиции по дисперсии независимо от направления акций
2. **Хеджирование хвостовых рисков**: выпуклость длинного свопа на дисперсию непропорционально выплачивает при крахах
3. **Дисперсионный трейдинг**: продажа индексной дисперсии, покупка одноименной — монетизация премии за корреляционный риск
4. **Керри-стратегии**: систематическая короткая позиция по фьючерсам VIX извлекает ролл-доход контанго (со значительным хвостовым риском)
5. **Бюджетирование рисков**: VIX как вход в динамическую аллокацию активов и целевой левередж

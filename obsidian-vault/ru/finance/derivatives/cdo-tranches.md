# Транши CDO

**Обеспеченное долговое обязательство (CDO)** — структурированный кредитный продукт, объединяющий портфель кредитных экспозиций (облигации, кредиты или CDS) и перераспределяющий совокупный кредитный риск путём выпуска траншей различной старшинства. Каждый транш поглощает убытки в строгом порядке: сначала equity, затем мezzanine, затем senior. Это «нарезание» риска превращает однородный пул в ценные бумаги с принципиально разными профилями риска — от equity-траншей с доходностью 15-30% до super-senior траншей с рейтингом AAA.

## Структура траншей

CDO с номиналом $N$ делится на транши, определённые точками **присоединения** $a$ и **отсоединения** $d$. Убыток транша:

$$
L_{\text{транш}}(a, d) = \frac{\min(L, d \cdot N) - \min(L, a \cdot N)}{(d - a) \cdot N}
$$

где $L$ — совокупный убыток портфеля. Стандартные точки индексных траншей:

| Транш | Присоединение | Отсоединение | Характер |
|:---|:---:|:---:|:---|
| **Equity** | 0% | 3% | Первый убыток, максимальный спред |
| **Mezzanine** | 3% | 7% | Левериджированная кредитная экспозиция |
| **Senior** | 7% | 15% | Инвестиционный грейд |
| **Super-senior** | 15% | 30% | Почти безрисковый (до кризиса) |
| **Остаток** | 30% | 100% | Минимальный риск |

## Модель гауссовой копулы

### Подход Ли (2000)

Модель Дэвида Ли сопоставляет каждому заёмщику $i$ латентную переменную:

$$
X_i = \sqrt{\rho}\,M + \sqrt{1-\rho}\,Z_i
$$

где $M \sim \mathcal{N}(0,1)$ — общий рыночный фактор, $Z_i \sim \mathcal{N}(0,1)$ — идиосинкратические шоки, $\rho$ — попарная корреляция активов. Дефолт наступает при:

$$
X_i < \Phi^{-1}(F_i(T))
$$

где $F_i(T) = 1 - e^{-\lambda_i T}$ — маржинальная вероятность дефолта, $\Phi^{-1}$ — квантиль стандартного нормального распределения.

### Условная независимость

При условии $M = m$ дефолты независимы. Условная вероятность дефолта:

$$
p_i(m) = \Phi\!\left(\frac{\Phi^{-1}(F_i(T)) - \sqrt{\rho}\,m}{\sqrt{1-\rho}}\right)
$$

Распределение убытков портфеля вычисляется интегрированием по $m$:

$$
\mathbb{P}(L \leq x) = \int_{-\infty}^{\infty} \mathbb{P}(L \leq x \mid M = m)\,\phi(m)\,dm
$$

## Корреляция и ценообразование

### Подразумеваемая корреляция

**Compound (подразумеваемая) корреляция** — единственное значение $\rho$, которое в модели гауссовой копулы воспроизводит рыночный спред транша. Аналог подразумеваемой волатильности Блэка-Шоулза.

### Базовая корреляция

**Базовая корреляция** сопоставляет каждой точке отсоединения $d$ корреляцию $\rho(d)$, правильно оценивающую «базовый транш» $[0, d]$:

$$
\text{PV}_{\text{транш}}[a, d] = \text{PV}_{\text{баз}}[0, d] - \text{PV}_{\text{баз}}[0, a]
$$

Базовая корреляция монотонно возрастает с точкой отсоединения — **улыбка корреляции** — отражая, что старшим траншам для соответствия рыночным ценам требуется более высокая корреляция.

## Кризис 2007-08

Фатальный недостаток гауссовой копулы: **нулевая хвостовая зависимость** ($\lambda_L = \lambda_U = 0$ при $\rho < 1$). На практике:

- Пулы субпрайм-ипотек демонстрировали коррелированные дефолты, многократно превышающие гауссовы прогнозы
- Super-senior транши с рейтингом AAA понесли катастрофические убытки
- Модель на порядки занижала вероятность 10+ одновременных дефолтов в портфеле из 125 имён
- Рейтинговые агентства опирались на выходы гауссовой копулы, присваивая вводящие в заблуждение рейтинги AAA

См. [[copula-models]] для математического объяснения провала хвостовой зависимости.

## Альтернативные модели

**Копула Стьюдента**: заменяет $\Phi$ на $t_\nu$, вводя симметричную хвостовую зависимость $\lambda > 0$ при конечном $\nu$. Лучше описывает совместные экстремальные дефолты.

**Случайная факторная нагрузка**: $\rho$ стохастична — $\rho(M)$ растёт в стрессе, порождая более толстые хвосты распределения убытков.

**Копула Маршалла-Олкина**: моделирует одновременные дефолты через общие шоки, создавая массу в точках совместного дефолта вместо непрерывной корреляции.

**Стохастическое возмещение**: ставка возмещения $R$ отрицательно коррелирована с частотой дефолтов — при массовых дефолтах возмещение падает, усиливая убытки старших траншей.

## Наличные CDO vs. синтетические CDO

| Характеристика | Наличный CDO | Синтетический CDO |
|:---|:---|:---|
| **Обеспечение** | Физические облигации/кредиты в SPV | CDS-контракты на референсные имена |
| **Фондирование** | Фондированный (инвесторы платят номинал) | Нефондированный (своповая экспозиция) |
| **Риск досрочного погашения** | Да (для CLO, MBS) | Нет |
| **Ликвидность** | Ниже (bespoke) | Выше (индексные транши) |

**Bespoke-транши** — заказные синтетические CDO с портфелем, выбранным клиентом, — оцениваются через интерполяцию базовых корреляций из ликвидных индексных траншей.

## Реализация на Python

```python
import numpy as np
from scipy.stats import norm

def gaussian_copula_loss_distribution(
    default_probs: np.ndarray, rho: float,
    n_simulations: int = 50000, seed: int = 42
) -> np.ndarray:
    """Симуляция числа дефолтов через однофакторную гауссову копулу."""
    rng = np.random.default_rng(seed)
    n_names = len(default_probs)
    thresholds = norm.ppf(default_probs)

    M = rng.standard_normal(n_simulations)
    Z = rng.standard_normal((n_simulations, n_names))
    X = np.sqrt(rho) * M[:, None] + np.sqrt(1 - rho) * Z

    defaults = (X < thresholds).astype(int)
    return defaults.sum(axis=1)

def tranche_expected_loss(
    loss_counts: np.ndarray, n_names: int,
    attach: float, detach: float
) -> float:
    """Ожидаемый убыток транша как доля от номинала транша."""
    portfolio_loss_frac = loss_counts / n_names
    tranche_loss = np.clip(portfolio_loss_frac - attach, 0, detach - attach)
    return float(tranche_loss.mean() / (detach - attach))

n_names = 125
dp = np.full(n_names, 0.02)
losses = gaussian_copula_loss_distribution(dp, rho=0.30)

for a, d, name in [(0, 0.03, "Equity"), (0.03, 0.07, "Mezz"),
                    (0.07, 0.15, "Senior"), (0.15, 0.30, "Super-Sr")]:
    el = tranche_expected_loss(losses, n_names, a, d)
    print(f"{name:>10} [{a:.0%}-{d:.0%}]: EL = {el:.4%}")
```

## Уроки кризиса

- **Модельный риск экзистенциален**: один параметр корреляции не способен охватить полную структуру зависимости убытков 125+ заёмщиков
- **Хвостовая зависимость критична**: гауссова копула применима только для неэкстремальных сценариев; кризисное ценообразование требует толстохвостых альтернатив
- **Опора на рейтинги**: механическое использование модельных выходов для рейтингов без стресс-тестирования создало системную хрупкость
- **Прозрачность**: посткризисное регулирование (Dodd-Frank, EU Securitisation Regulation) требует удержания риска и раскрытия информации — см. [[securitization]]

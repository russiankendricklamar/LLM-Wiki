# Стоимость удержания (Cost of Carry)

Модель **стоимости удержания** (cost of carry) определяет теоретическую форвардную цену актива, учитывая все затраты и выгоды от владения физическим активом по сравнению с владением форвардным контрактом. Это фундаментальное соотношение, связывающее спотовые и форвардные рынки, а его нарушения определяют само понятие **арбитража** на товарных и финансовых рынках.

## Общая формула

Форвардная цена $F$ с поставкой в момент $T$:

$$
F = S\,e^{(r - q + u - y)\,T}
$$

| Символ | Компонент | Описание |
|:-------|:----------|:---------|
| $S$ | Спот-цена | Текущая рыночная цена |
| $r$ | Безрисковая ставка | Стоимость финансирования |
| $q$ | Дивидендная доходность | Доход от владения активом |
| $u$ | Стоимость хранения | Хранение, страхование (товары) |
| $y$ | Convenience yield | Выгода от владения физическим товаром |

Чистая стоимость удержания: $b = r - q + u - y$. При $b > 0$ — контанго; при $b < 0$ — бэквордация.

## Финансовые активы

Для финансовых активов (без хранения и convenience yield) формула упрощается.

**Акция без дивидендов**: $F = Se^{rT}$

**Акция с непрерывной дивидендной доходностью $q$**: $F = Se^{(r-q)T}$

**Акция с дискретными дивидендами** $D_i$ в моменты $t_i$:

$$
F = \left(S - \sum_i D_i\,e^{-rt_i}\right)e^{rT}
$$

**Валюта (покрытый паритет процентных ставок)**: $F = Se^{(r_d - r_f)T}$, где $r_d$ — домашняя ставка, $r_f$ — иностранная. Основа ценообразования FX-форвардов и [[swaps]].

**Фьючерсы на индексы**: $F = Se^{(r-q)T}$, где $q$ — агрегированная дивидендная доходность. Разница $F - Se^{rT}$ — **справедливая стоимость**, за которой следят индексные арбитражёры.

## Товарные рынки

Товары вводят стоимость хранения $u$ и convenience yield $y$:

$$
F = Se^{(r + u - y)T}
$$

**Convenience yield** $y$ — неденежная выгода от владения физическими запасами: возможность удовлетворить неожиданный спрос, избежать остановки производства, использовать временный дефицит. Не наблюдается напрямую — выводится из спот-форвардного соотношения.

### Контанго и бэквордация

**Контанго** ($F > S$): затраты на хранение и финансирование превышают convenience yield. Форвардная кривая направлена вверх. Типично: золото, цветные металлы в профиците, нефть при накоплении запасов.

**Бэквордация** ($F < S$): высокий convenience yield при дефиците. Форвардная кривая наклонена вниз. Типично: нефть при перебоях поставок, сельхозтовары вблизи неурожайных сезонов.

**Базис**:

$$
\text{Basis} = S - F
$$

Сходится к нулю при экспирации ($F(T,T) = S_T$). Базисный риск возникает при хеджировании фьючерсами из-за непредсказуемых изменений базиса.

## Cash-and-Carry арбитраж

Если $F > Se^{(r-q+u-y)T}$ (форвард переоценён):

1. Занять $S$ под ставку $r$.
2. Купить спот-актив.
3. Оплатить хранение $u$, получить дивиденды $q$.
4. Продать форвард по цене $F$.
5. В момент $T$: поставить актив, получить $F$, вернуть $Se^{(r-q+u)T}$.

**Обратный cash-and-carry** при $F < Se^{(r-q+u-y)T}$: короткая продажа актива, покупка форварда. Работает только при возможности шорта — ключевое ограничение для физических товаров.

## Индексный арбитраж

Справедливая стоимость фьючерса на индекс: $F^* = Se^{(r-q)T}$. При $F > F^*$ (фьючерс «дорогой»): покупка корзины индекса, продажа фьючерса. При $F < F^*$: обратная операция. Безарбитражная полоса: $F^* \pm \text{транзакционные издержки}$.

## Фьючерсы vs форварды: конвекционная поправка

Фьючерсы переоцениваются ежедневно (mark-to-market); форварды — только при погашении. При стохастических ставках, коррелированных с ценой актива, возникает **конвекционное смещение**:

$$
F_{\text{futures}} \approx F_{\text{forward}}\,e^{\rho_{S,r}\,\sigma_S\,\sigma_r\,T^2 / 2}
$$

Для процентных фьючерсов эта поправка существенна при построении кривых доходности. См. [[swaps]], [[caps-floors-swaptions]].

## Когда модель не работает

- **Ограничения на короткие продажи**: для физических товаров шорт невозможен, обратный арбитраж не исполним.
- **Нескладируемые активы**: электроэнергия, скоропортящиеся продукты — связь спот-форвард ослабевает.
- **Контрагентский риск**: форварды несут кредитный риск — см. [[xva]].
- **Неликвидность**: расширяет арбитражную полосу.

## Форвардные кривые и временная структура

Форвардная кривая $F(T)$ кодирует рыночные ожидания баланса спроса/предложения:

$$
F(T) = S\,e^{(r(T) - q(T) + u(T) - y(T))\,T}
$$

Все параметры могут зависеть от срока. Товарные трейдеры анализируют форму кривой для оценки экономики хранения и сезонных паттернов.

## Реализация на Python

```python
import numpy as np

def forward_price(S: float, r: float, T: float, q: float = 0.0,
                  u: float = 0.0, y: float = 0.0) -> float:
    """Форвардная цена по модели cost of carry."""
    return S * np.exp((r - q + u - y) * T)

def implied_convenience_yield(S: float, F: float, r: float, T: float,
                               q: float = 0.0, u: float = 0.0) -> float:
    """Подразумеваемый convenience yield из наблюдаемых спота и форварда."""
    return r - q + u - np.log(F / S) / T

# Примеры
F_index = forward_price(S=100, r=0.05, T=0.5, q=0.02)
F_gold  = forward_price(S=1900, r=0.05, T=1.0, u=0.005)
F_oil   = forward_price(S=80, r=0.05, T=0.25, u=0.02, y=0.08)

y_impl = implied_convenience_yield(S=80, F=78, r=0.05, T=0.25, u=0.02)
```

## Применения

1. **Справедливая стоимость индексных фьючерсов**: $F^* = Se^{(r-q)T}$ — ориентир для программного трейдинга.
2. **Товарные форвардные кривые**: модель cost of carry структурирует всю временную структуру товарных цен.
3. **FX-форварды**: покрытый паритет $F = Se^{(r_d - r_f)T}$ — основа валютного хеджирования и [[swaps]].
4. **Опционное ценообразование**: форвардная цена $F$ входит в [[blacks-model]] напрямую.
5. **Финансирование акций**: стоимость удержания определяет справедливый уровень equity total return swaps.

# Броуновское движение

**Броуновское движение**, также известное как винеровский процесс, — это стохастический процесс с непрерывным временем, являющийся пределом как случайных блужданий, так и различных других стохастических процессов.

## Свойства

Стохастический процесс $W_t$ является винеровским процессом, если:
1. $W_0 = 0$.
2. Он имеет независимые приращения: $W_{t+u} - W_t$ не зависит от $W_s$ для $s \le t$.
3. Он имеет гауссовские приращения: $W_{t+u} - W_t \sim N(0, u)$.
4. Он имеет почти наверное непрерывные траектории.

## Стохастические дифференциальные уравнения (СДУ)
Дифференциальная форма броуновского движения — $dW_t$, представляющая бесконечно малое приращение, где $dW_t^2 = dt$ в смысле Ито.

## Реализация на Python

```python
import numpy as np

def simulate_brownian_motion(T, n_steps):
    dt = T / n_steps
    steps = np.random.normal(0, np.sqrt(dt), n_steps)
    path = np.concatenate([[0], np.cumsum(steps)])
    return path
```

# Броуновское движение

**Броуновское движение (винеровский процесс)**, названное в честь ботаника Роберта Броуна (1827) и формализованное Норбертом Винером (1923), — фундаментальный стохастический процесс непрерывного времени. В финансах оно служит строительным блоком для всех моделей непрерывного времени: от формулы Блэка-Шоулза до стохастических процентных ставок.

## Обзор и предпосылки

Луи Башелье (1900) первым применил броуновское движение к ценам акций в своей диссертации — за 5 лет до Эйнштейна. Современные финансовые модели: геометрическое броуновское движение (логнормальные цены), процессы Орнштейна-Уленбека (возврат к среднему), модели Хестона/CIR — все являются преобразованиями базового броуновского движения через стохастическое исчисление Ито.

## Математическая база

**Аксиоматическое определение**. Стохастический процесс $W = (W_t)_{t \geq 0}$ является стандартным броуновским движением (СБД), если:

1. $W_0 = 0$ почти наверное
2. **Независимые приращения**: $W_{t} - W_s \perp \mathcal{F}_s$ при $t > s$
3. **Гауссовские приращения**: $W_t - W_s \sim \mathcal{N}(0, t-s)$
4. **Непрерывность траекторий**: $t \mapsto W_t$ непрерывна почти наверное

**Квадратическая вариация** — ключевое отличие от гладких функций:

$$
[W, W]_t = \lim_{\|\Pi\| \to 0} \sum_{i} (W_{t_{i+1}} - W_{t_i})^2 = t \quad \text{п.н.}
$$

**Лемма Ито** для $f \in \mathcal{C}^{2,1}$:

$$
df(W_t, t) = \frac{\partial f}{\partial t}dt + \frac{\partial f}{\partial w}dW_t + \frac{1}{2}\frac{\partial^2 f}{\partial w^2}dt
$$

**Принцип отражения**: для максимума $M_t = \max_{s \leq t} W_s$:

$$
\mathbb{P}(M_t \geq a) = 2\mathbb{P}(W_t \geq a) = 2\left(1 - \Phi\!\left(\frac{a}{\sqrt{t}}\right)\right)
$$

**Время первого прохождения** уровня $a > 0$: $\tau_a = \inf\{t \geq 0 : W_t = a\}$ имеет обратное гауссово распределение:

$$
f_{\tau_a}(t) = \frac{a}{\sqrt{2\pi t^3}}\exp\!\left(-\frac{a^2}{2t}\right)
$$

**Многомерное броуновское движение**. Вектор $\mathbf{W}_t \in \mathbb{R}^n$ с корреляционной матрицей $\Sigma = LL^\top$ (разложение Холецкого):

$$
\mathbf{W}_t = L\,\mathbf{Z}_t, \quad \mathbf{Z}_t \overset{iid}{\sim} \mathcal{N}(\mathbf{0}, t\,I_n)
$$

**Связь с уравнением теплопроводности** (формула Фейнмана-Каца):

$$
u(x, t) = \mathbb{E}[f(W_T) | W_t = x] \implies \frac{\partial u}{\partial t} + \frac{1}{2}\frac{\partial^2 u}{\partial x^2} = 0
$$

## Ключевые свойства

- **Самоподобие**: $W_{at} \overset{d}{=} \sqrt{a}\,W_t$ (масштабная инвариантность)
- **Марковское свойство**: $\mathbb{E}[W_t | \mathcal{F}_s] = W_s$ (мартингал)
- **Нигде не дифференцируемость**: траектории непрерывны, но не дифференцируемы почти наверное
- **Плотность посещений**: любой открытый интервал посещается бесчисленное множество раз
- **Рекуррентность**: в $\mathbb{R}^1$ и $\mathbb{R}^2$ возвращается в любую точку; в $\mathbb{R}^3$ — нет

## Реализация на Python

```python
import numpy as np
import pandas as pd

def simulate_brownian_motion(T: float, n_steps: int, n_paths: int = 1,
                              seed: int = 42) -> np.ndarray:
    """Симуляция стандартного броуновского движения методом Эйлера."""
    np.random.seed(seed)
    dt = T / n_steps
    dW = np.random.normal(0, np.sqrt(dt), (n_paths, n_steps))
    W = np.concatenate([np.zeros((n_paths, 1)), np.cumsum(dW, axis=1)], axis=1)
    return W

def simulate_correlated_brownian(T: float, n_steps: int, corr_matrix: np.ndarray,
                                  seed: int = 42) -> np.ndarray:
    """Многомерное броуновское движение через разложение Холецкого."""
    np.random.seed(seed)
    n = corr_matrix.shape[0]
    dt = T / n_steps
    L = np.linalg.cholesky(corr_matrix)
    Z = np.random.normal(0, np.sqrt(dt), (n, n_steps))
    dW = L @ Z
    W = np.concatenate([np.zeros((n, 1)), np.cumsum(dW, axis=1)], axis=1)
    return W

def quadratic_variation(W: np.ndarray) -> np.ndarray:
    """Эмпирическая квадратическая вариация (должна сходиться к t)."""
    dW = np.diff(W, axis=-1)
    return np.cumsum(dW**2, axis=-1)

def first_passage_time(W: np.ndarray, level: float, dt: float) -> float:
    """Эмпирическое время первого прохождения уровня."""
    crossings = np.where(W >= level)[0]
    return crossings[0] * dt if len(crossings) > 0 else float('inf')

# Пример: 3 коррелированных процесса (акция, ставка, FX)
corr = np.array([[1.00, -0.30, 0.20],
                 [-0.30, 1.00, -0.10],
                 [0.20, -0.10, 1.00]])
paths = simulate_correlated_brownian(T=1.0, n_steps=252, corr_matrix=corr)
qv = quadratic_variation(paths)
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $W_t$ | Броуновское движение | Стандартный винеровский процесс | $(-\infty, +\infty)$ |
| $[W,W]_t$ | Квадратическая вариация | Равна $t$ для СБД | $\geq 0$ |
| $L$ | Матрица Холецкого | Нижнетреугольный квадратный корень из $\Sigma$ | Нижнетреугольная |

| $dt$ | Шаг дискретизации | Временной шаг симуляции | $1/252$ – $1/10000$ |

## Ограничения и расширения

**Ограничения:**
- Непрерывность траекторий не учитывает скачки (гэпы открытия, корпоративные события).
- Гауссово распределение приращений недооценивает вероятность экстремальных движений (тяжёлые хвосты).
- Постоянная дисперсия $dt$ не согласуется с кластеризацией волатильности.

**Расширения:**
- **[[geometric-brownian-motion]]**: $dS = \mu S\,dt + \sigma S\,dW$ — логнормальные цены.
- **[[levy-processes]]**: обобщение с скачками, тяжёлыми хвостами и асимметрией.
- **Дробное броуновское движение**: $H \neq 0.5$ для моделирования длинной памяти (реализованная волатильность).

## Практическое применение

1. **Ценообразование деривативов**: динамика цены актива в моделях BSM, Хестона, CIR.
2. **Симуляция Монте-Карло**: генерация сценариев для стресс-тестирования и расчёта VaR.
3. **Стохастические дифференциальные уравнения**: основа моделей процентных ставок (Васичек, HJM) и кредитного риска.
4. **Моделирование коррелированных активов**: многомерное БД через Холецкого для портфельных симуляций.
5. **Оптимальное исполнение**: стохастическая динамика mid-price в моделях Авелланеды-Стойкова.

# Метод Монте-Карло

**Метод Монте-Карло** — это численный метод, основанный на моделировании случайных величин для решения задач, которые могут быть детерминированными или стохастическими по своей природе. В финансах он незаменим для оценки экзотических опционов и расчета показателей риска (VaR, ES).

## Основная идея

Согласно закону больших чисел, среднее значение результатов большого количества независимых испытаний стремится к математическому ожиданию. В контексте оценки активов цена сегодня — это дисконтированное ожидаемое значение будущих выплат:

$$
V_0 = e^{-rT} E[f(S_T)] \approx e^{-rT} \frac{1}{N} \sum_{i=1}^{N} f(S_T^{(i)})
$$

Где:
- $S_T^{(i)}$ — симулированная цена актива в момент $T$ в $i$-м сценарии.
- $f(S_T)$ — функция выплат (payoff) опциона.
- $r$ — безрисковая ставка.

## Алгоритм оценки опциона

1.  **Генерация случайных путей**: Моделирование движения цены базового актива (например, через Геометрическое броуновское движение).
2.  **Расчет выплат**: Вычисление дохода по опциону для каждой траектории.
3.  **Усреднение**: Нахождение среднего арифметического всех выплат.
4.  **Дисконтирование**: Приведение среднего значения к текущему моменту времени.

## Точность и сходимость

Ошибка метода Монте-Карло убывает со скоростью $O(1/\sqrt{N})$. Чтобы уменьшить ошибку в 10 раз, нужно увеличить количество симуляций в 100 раз. Для ускорения сходимости используют методы снижения дисперсии (Antithetic Variates, Control Variates).

## Python: Оценка европейского Call-опциона

```python
import numpy as np

def monte_carlo_call(S, K, T, r, sigma, iterations=100000):
    # Генерация конечных цен по ГБД
    z = np.random.standard_normal(iterations)
    ST = S * np.exp((r - 0.5 * sigma**2) * T + sigma * np.sqrt(T) * z)
    
    # Расчет выплат
    payoffs = np.maximum(ST - K, 0)
    
    # Среднее и дисконтирование
    price = np.exp(-r * T) * np.mean(payoffs)
    return price

S, K, T, r, sigma = 100, 105, 1, 0.05, 0.2
price = monte_carlo_call(S, K, T, r, sigma)
print(f"Цена опциона (Monte Carlo): ${price:.2f}")
```

## Преимущества и недостатки

| Характеристика | Плюсы | Минусы |
| :--- | :--- | :--- |
| **Гибкость** | Работает для любых сложных функций выплат | Высокая вычислительная стоимость |
| **Многомерность** | Хорошо справляется с зависимостью от многих активов | Медленная сходимость ($1/\sqrt{N}$) |
| **Простота** | Легко распараллеливать вычисления | Трудно оценивать американские опционы |

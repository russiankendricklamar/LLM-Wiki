# Нумерайр-инвариантное квадратичное хеджирование

**Квадратичное хеджирование** (Mean-Variance Hedging) направлено на минимизацию дисперсии ошибки хеджирования на момент экспирации. Критически важным свойством в продвинутом ценообразовании является **инвариантность к нумерайру**, гарантирующая, что стратегия хеджирования остается оптимальной независимо от того, измеряем ли мы стоимость в USD, EUR или в самом активе.

## Разложение Гальтчука-Куниты-Ватанабэ (GKW)

На неполном рынке производный инструмент $H$ не может быть идеально реплицирован. Разложение GKW позволяет записать:

$$
H = H_0 + \int_0^T \xi_t dS_t + L_T
$$

Где:
- $\int_0^T \xi_t dS_t$: Часть выплаты, которая может быть реплицирована путем торговли активом $S$.
- $L_T$: Ортогональная ошибка хеджирования, такая что $E[L_T \int_0^T \xi_t dS_t] = 0$.

Оптимальный хедж $\xi_t$ — это процесс, минимизирующий $E[(H - (V_0 + \int_0^T \phi_t dS_t))^2]$.

## Смена нумерайра

Если мы меняем нумерайр с $P$ на $N$, процесс цены становится $S^N = S/N$. Для того чтобы стратегия была инвариантной, условие самофинансирования должно выполняться при новой мере $Q^N$. Это достигается за счет использования **минимальной мартингальной меры (MMM)**.

## Вариационно-оптимальная мартингальная мера (VOMM)

Стратегия хеджирования инвариантна к нумерайру, если мы выбираем оптимальную стратегию при мере VOMM. Плотность VOMM относительно физической меры $P$ имеет вид:

$$
\frac{dQ}{dP} = \frac{1 - \int_0^T \lambda_t dW_t}{E[1 - \int_0^T \lambda_t dW_t]}
$$

где $\lambda$ — рыночная цена риска.

## Python: Симуляция ошибки хеджирования

```python
import numpy as np

def delta_hedge_error(S, K, r, sigma, T, dt):
    steps = int(T/dt)
    phi = np.zeros(steps)
    # Оптимальный коэффициент хеджирования (упрощенно для случая Блэка-Шоулза)
    # На неполных рынках здесь использовалась бы логика GKW
    for t in range(steps):
        d1 = (np.log(S[t]/K) + (r + 0.5*sigma**2)*(T-t*dt)) / (sigma*np.sqrt(T-t*dt))
        phi[t] = norm.cdf(d1)
    
    # Расчет ошибки хеджирования
    error = (S[-1] - K) - sum(phi[:-1] * np.diff(S))
    return error
```

## Контекст

Квадратичное хеджирование часто является первым шагом перед переходом к более сложным методам хеджирования на основе полезности. В режимах с «тяжелыми хвостами» для калибровки параметров скачкообразной диффузии, необходимых для разложения GKW, используются методы [[mcmc]].

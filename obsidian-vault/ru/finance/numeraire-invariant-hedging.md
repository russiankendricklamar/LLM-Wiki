# Нумерайр-инвариантное квадратичное хеджирование

**Квадратичное хеджирование** (Mean-Variance Hedging) направлено на минимизацию дисперсии ошибки хеджирования на момент экспирации. Критически важным свойством в продвинутом ценообразовании является **инвариантность к нумерайру**, гарантирующая, что стратегия хеджирования остается оптимальной независимо от того, измеряем ли мы стоимость в USD, EUR или в самом активе.

## Визуализация

RMSE дельта-хеджирования убывает с ростом частоты ребалансировки, стремясь к нулю на полном GBM-рынке. Ненулевой нижний предел на высоких частотах — остаток $\mathbb{E}[L_T^2]^{1/2}$ на неполном рынке (стохастическая волатильность).

## История и мотивация

Теория квадратичного хеджирования восходит к работам Фёлльмера и Зондерманна (1986). Швейцер (1994) доказал, что разложение Гальтчука–Куниты–Ватанабэ (GKW) существует для широкого класса семимартингальных моделей. Рейнлендер и Швейцер (1997) установили нумерайр-инвариантность стратегий GKW. Это имеет практическую ценность в мультивалютных книгах деривативов: книга хеджирования, оптимальная в долларах, остаётся оптимальной при переоценке в евро.

## Разложение Гальтчука–Куниты–Ватанабэ (GKW)

На неполном рынке производный инструмент $H$ не может быть идеально реплицирован. Разложение GKW:

$$
H = H_0 + \int_0^T \xi_t^H \, dS_t + L_T^H
$$

где:
- $H_0$ — начальная стоимость хеджирующего портфеля,
- $\xi_t^H$ — оптимальная квадратичная торговая стратегия,
- $L_T^H$ — ортогональный остаток: $\langle L^H, S \rangle = 0$ (некоррелированная ошибка).

Оптимальный хедж $\xi^H$ минимизирует:

$$
\min_{\phi \in \Theta} E\!\left[\left(H - V_0 - \int_0^T \phi_t \, dS_t\right)^2\right]
$$

## Вариационно-оптимальная мартингальная мера (VOMM)

Стратегия GKW может быть записана через специальную меру — **вариационно-оптимальную мартингальную меру** (VOMM). Её плотность Радона–Никодима по отношению к физической мере $P$:

$$
\frac{dQ^*}{dP}\bigg|_{\mathcal{F}_T} = \frac{Z_T}{\mathbb{E}[Z_T]}, \quad Z_T = \exp\!\left(-\int_0^T \lambda_t \, dW_t - \frac{1}{2}\int_0^T \lambda_t^2 \, dt\right)
$$

где $\lambda_t = (\mu_t - r_t)/\sigma_t$ — рыночная цена риска. При $\lambda_t \equiv \text{const}$ VOMM совпадает с **минимальной мартингальной мерой** (MMM) Фёлльмера–Швейцера.

## Инвариантность к нумерайру

Пусть $N_t > 0$ — процесс нумерайра. Переоценённые процессы: $\tilde{S}_t = S_t / N_t$, $\tilde{H} = H / N_T$. Стратегия $\xi^H$ **инвариантна к нумерайру** в следующем смысле:

$$
\xi^{\tilde{H}} = \xi^H \cdot N_t / \tilde{S}_t
$$

то есть количество акций в портфеле не зависит от единицы измерения, а пересчёт позиции осуществляется масштабированием на $N_t$.

## Python: Симуляция ошибки хеджирования

```python
import numpy as np
from scipy.stats import norm


def simulate_hedging_error(
    S0: float, K: float, r: float, sigma: float, T: float,
    n_paths: int = 50_000, rebal_freqs: list[int] = None
) -> dict:
    """
    Симулирует ошибку дельта-хеджирования европейского колл-опциона
    при различных частотах ребалансировки.
    Возвращает среднеквадратичную ошибку (RMSE) для каждой частоты.
    """
    if rebal_freqs is None:
        rebal_freqs = [4, 12, 52, 252]

    def bs_delta(s, k, r_, sig, tau):
        if tau <= 0:
            return float(s > k)
        d1 = (np.log(s / k) + (r_ + 0.5 * sig ** 2) * tau) / (sig * np.sqrt(tau))
        return norm.cdf(d1)

    results = {}

    for steps in rebal_freqs:
        dt = T / steps
        np.random.seed(42)
        z = np.random.standard_normal((steps, n_paths))
        log_ret = (r - 0.5 * sigma ** 2) * dt + sigma * np.sqrt(dt) * z
        S = S0 * np.exp(np.cumsum(log_ret, axis=0))
        S = np.vstack([np.full(n_paths, S0), S])

        hedge_pnl = np.zeros(n_paths)
        for t in range(steps):
            tau = T - t * dt
            delta = bs_delta(S[t], K, r, sigma, tau)
            dS = S[t + 1] - S[t] * np.exp(r * dt)  # дисконтированное изменение
            hedge_pnl += delta * dS

        payoff = np.maximum(S[-1] - K, 0.0)
        bs_price = S0 * norm.cdf(
            (np.log(S0 / K) + (r + 0.5 * sigma ** 2) * T) / (sigma * np.sqrt(T))
        ) - K * np.exp(-r * T) * norm.cdf(
            (np.log(S0 / K) + (r - 0.5 * sigma ** 2) * T) / (sigma * np.sqrt(T))
        )
        hedge_error = payoff - bs_price * np.exp(r * T) - hedge_pnl
        results[steps] = float(np.sqrt(np.mean(hedge_error ** 2)))

    return results


errors = simulate_hedging_error(S0=100, K=100, r=0.05, sigma=0.2, T=1.0)
for freq, rmse in errors.items():
    print(f"Частота ребалансировки: {freq:>4}/год  |  RMSE хеджирования: ${rmse:.4f}")
```

## Таблица параметров

| Символ | Описание | Примечание |
|:---|:---|:---|
| $\xi^H$ | Оптимальная торговая стратегия GKW | Решение задачи проекции в $L^2$ |
| $L^H$ | Ортогональный остаток | Некоррелирован с $S$ |
| $\lambda_t$ | Рыночная цена риска | $(\mu - r)/\sigma$ в модели BS |
| $Q^*$ | VOMM | Минимизирует $E[(dQ/dP)^2]$ |
| MMM | Минимальная мартингальная мера | Совпадает с VOMM при постоянном $\lambda$ |

## Ограничения и расширения

- **Модели со скачками**: При введении скачков (Мертон, Ку) остаток $L^H$ не нулевой; GKW по-прежнему даёт оптимальный квадратичный хедж, но RMSE растёт.
- **Высокая размерность**: В мультиактивных книгах — корзины, спреды — требуется тензорное разложение матрицы ковариации.
- **Транзакционные издержки**: Квадратичное хеджирование без учёта комиссий приводит к чрезмерной ребалансировке; реалистичная постановка вводит штраф $c|\Delta\phi_t|$.
- **Хеджирование на основе полезности**: Квадратичное хеджирование — частный случай при экспоненциальной функции полезности с $\gamma \to 0$.
- **MCMC-калибровка**: В режимах с «тяжёлыми хвостами» параметры скачкообразной диффузии, необходимые для разложения GKW, оцениваются методами [[mcmc]].

# Теория экстремальных значений (EVT)

**Теория экстремальных значений (Extreme Value Theory, EVT)** — это раздел статистики, изучающий предельные распределения для максимумов или минимумов случайных величин. В финансах EVT используется для оценки рисков редких, но экстремальных событий («толстых хвостов»), которые не улавливаются нормальным распределением.

## Основные подходы

Существует два основных метода выделения экстремальных значений:

1.  **Block Maxima (Метод блоков)**: Выбираются максимальные значения за фиксированные интервалы времени (например, худший день в году). Предельным распределением является **GEV (Generalized Extreme Value)**.
2.  **Peaks Over Threshold (POT, Метод превышения порога)**: Выбираются все значения, превышающие достаточно высокий порог $u$. Предельным распределением является **GPD (Generalized Pareto Distribution)**.

## Обобщенное распределение Парето (GPD)

Метод POT чаще используется в финансах. Согласно теореме Пикандса-Баллемы-де Хаана, распределение превышений над высоким порогом $u$ стремится к GPD:

$$
G_{\xi, \beta}(y) = 
\begin{cases} 
1 - (1 + \xi y / \beta)^{-1/\xi} & \text{если } \xi \neq 0 \\
1 - e^{-y/\beta} & \text{если } \xi = 0
\end{cases}
$$

Где:
- $y = x - u$ — величина превышения.
- $\xi$ — параметр формы (индекс хвоста). Чем больше $\xi$, тем «тяжелее» хвост.
- $\beta$ — параметр масштаба.

## Почему EVT важна для риск-менеджмента?

Традиционный VaR на основе нормального распределения часто недооценивает риск кризисов. EVT позволяет:
- Оценивать **Expected Shortfall (ES)** в самых глубоких хвостах.
- Проводить стресс-тестирование портфелей.
- Моделировать убытки при страховании катастрофических рисков.

## Python: Аппроксимация хвоста распределением Парето

```python
import numpy as np
from scipy.stats import genpareto

# Генерация данных с тяжелыми хвостами (распределение Стьюдента)
data = np.random.standard_t(df=3, size=5000)

# Выбор порога (например, 95-й квантиль)
threshold = np.percentile(data, 95)
excesses = data[data > threshold] - threshold

# Подгонка параметров GPD
xi, loc, beta = genpareto.fit(excesses)

print(f"Параметр формы (tail index) xi: {xi:.4f}")
print(f"Параметр масштаба beta: {beta:.4f}")
```

## Сравнение методов

| Метод | Что анализируем | Предельное распределение |
| :--- | :--- | :--- |
| **Block Maxima** | Максимумы периодов | GEV (Gumbel, Frechet, Weibull) |
| **POT** | Все значения выше $u$ | GPD (Generalized Pareto) |
| **Normal Dist** | Все данные | Gaussian (Неприменимо для хвостов) |
| **Hill Estimator** | Только хвост | Степенной закон (Power Law) |

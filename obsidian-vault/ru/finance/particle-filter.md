# Частичный фильтр (Particle Filter)

**Частичный фильтр** (Particle Filter), также известный как **Последовательный метод Монте-Карло** (Sequential Monte Carlo, SMC) — это алгоритм обработки сигналов, который использует набор «частиц» (случайных выборок) для представления апостериорного распределения скрытого состояния системы.

В отличие от фильтра Калмана, частичный фильтр не требует, чтобы система была линейной, а распределения — нормальными. Это делает его идеальным для моделирования сложных финансовых процессов, таких как стохастическая волатильность или изменение режимов.

## Как это работает?

Алгоритм представляет распределение плотности вероятности набором дискретных частиц, каждой из которых присвоен определенный вес. Процесс состоит из трех повторяющихся этапов:

1.  **Предсказание (Prediction)**: Каждая частица «пробрасывается» в будущее согласно модели динамики системы (с добавлением случайного шума).
2.  **Обновление весов (Update/Weighting)**: Когда поступают новые рыночные данные, каждой частице присваивается вес, пропорциональный вероятности того, что это наблюдение могло быть получено при данном состоянии частицы.
3.  **Ресемплинг (Resampling)**: Частицы с низкими весами отсеиваются, а частицы с высокими весами дублируются. Это предотвращает «вырождение» фильтра, когда вся масса распределения концентрируется в одной частице.

## Применение в финансах

- **Оценка волатильности**: Оценка мгновенной волатильности (как в модели Хестона) на основе наблюдаемых цен.
- **Трекинг инфляционных ожиданий**: Оценка скрытых макроэкономических переменных.
- **Управление рисками**: Оценка рисков портфелей с нелинейными инструментами (опционами) при наличии «тяжелых хвостов».

## Математическая концепция

Апостериорная плотность $p(x_t | y_{1:t})$ аппроксимируется как:

$$
p(x_t | y_{1:t}) \approx \sum_{i=1}^{N} w_t^{(i)} \delta(x_t - x_t^{(i)})
$$

Где:
- $x_t^{(i)}$ — состояние $i$-й частицы.
- $w_t^{(i)}$ — нормализованный вес частицы.
- $\delta$ — дельта-функция Дирака.

## Python: Концептуальная реализация

```python
import numpy as np

def particle_filter_step(particles, weights, observation, sigma_obs):
    # 1. Предсказание (простая случайная прогулка)
    particles += np.random.normal(0, 0.1, len(particles))
    
    # 2. Обновление весов (Likelihood)
    # Насколько хорошо частица объясняет наблюдение?
    weights *= np.exp(-0.5 * ((particles - observation) / sigma_obs)**2)
    weights /= np.sum(weights) # Нормализация
    
    # 3. Ресемплинг (если нужно)
    if 1.0 / np.sum(np.square(weights)) < len(particles) / 2:
        indices = np.random.choice(len(particles), size=len(particles), p=weights)
        particles = particles[indices]
        weights.fill(1.0 / len(particles))
        
    return particles, weights
```

## Преимущества и недостатки

| Характеристика | Частичный фильтр | Фильтр Калмана |
| :--- | :--- | :--- |
| **Линейность** | Любая | Строго линейная |
| **Распределения** | Любые | Только Гауссовские |
| **Вычисления** | Высокие (зависят от N) | Очень низкие |
| **Точность** | Асимптотически точен | Оптимален только для линейных систем |

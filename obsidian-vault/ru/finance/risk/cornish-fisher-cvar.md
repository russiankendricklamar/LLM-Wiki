# CVaR Корниша-Фишера

**Разложение Корниша-Фишера (CF)**, предложенное Эдмундом Корнишем и Рональдом Фишером (1937), позволяет аппроксимировать квантили произвольного распределения через квантили нормального распределения с поправками на асимметрию и эксцесс. В финансах оно используется для расчёта модифицированных VaR и CVaR для активов с ненормальным распределением доходностей.

## Необходимые знания

[[value-at-risk]], [[extreme-value-theory]]

## Обзор и предпосылки

Классический VaR предполагает нормальность доходностей, что систематически занижает риск для активов с отрицательной асимметрией и лептокуртозисом (тяжёлыми хвостами) — типичными свойствами акций, хедж-фондов и опционных стратегий. Корниш-Фишер VaR обеспечивает более точную оценку без полного параметрического задания распределения, требуя только первых четырёх моментов.

## Математическая база

**Скорректированный z-показатель** Корниша-Фишера для уровня доверия $\alpha$:

$$
z_{CF} = z_\alpha + \frac{1}{6}(z_\alpha^2 - 1)S + \frac{1}{24}(z_\alpha^3 - 3z_\alpha)K - \frac{1}{36}(2z_\alpha^3 - 5z_\alpha)S^2
$$

где:
- $z_\alpha = \Phi^{-1}(\alpha)$ — стандартный нормальный квантиль
- $S$ — коэффициент асимметрии (skewness) распределения
- $K$ — избыточный эксцесс (excess kurtosis)

Члены разложения:
- $\frac{1}{6}(z^2-1)S$ — первичная поправка на асимметрию
- $\frac{1}{24}(z^3-3z)K$ — поправка на эксцесс
- $-\frac{1}{36}(2z^3-5z)S^2$ — квадратичная поправка на асимметрию

**Модифицированный VaR**:

$$
\text{mVaR}_\alpha = \mu + z_{CF}\,\sigma
$$

**Модифицированный CVaR (Expected Shortfall)** — ожидаемые потери в хвосте ниже mVaR:

$$
\text{mCVaR}_\alpha = -\mu + \sigma\,\frac{\phi(z_{CF})}{1-\alpha}\cdot\left[1 + \frac{S}{6}(2z_{CF}^2-1) + \frac{K}{24}(z_{CF}^3-3z_{CF}) - \frac{S^2}{36}(2z_{CF}^3-5z_{CF})\right]
$$

**Влияние моментов**:
- $S < 0$ (отрицательная асимметрия, типичная для акций): mVaR > VaR нормального — хвостовые потери больше
- $K > 0$ (лептокуртозис): дополнительное увеличение mVaR
- При $S = 0$, $K = 0$: $z_{CF} = z_\alpha$ — стандартный нормальный случай

## Ключевые свойства

- Аппроксимация второго порядка: точна при малых $|S|$ и $|K|$; при $|S| > 2$ может давать немонотонный результат
- Не требует полного задания распределения — только четыре момента
- Вычислительно эффективна по сравнению с полным историческим моделированием
- Субаддитивность mCVaR (как мера риска) сохраняется при умеренных отклонениях от нормальности

## Реализация на Python

```python
import numpy as np
from scipy.stats import norm
from scipy.integrate import quad

def cornish_fisher_quantile(alpha: float, skew: float, kurtosis: float) -> float:
    """Скорректированный квантиль Корниша-Фишера."""
    z = norm.ppf(alpha)
    cf = (z
          + (z**2 - 1) * skew / 6
          + (z**3 - 3 * z) * kurtosis / 24
          - (2 * z**3 - 5 * z) * skew**2 / 36)
    return cf

def modified_var(mu: float, sigma: float, skew: float, kurtosis: float,
                 alpha: float = 0.05) -> float:
    """Модифицированный VaR по методу Корниша-Фишера."""
    z_cf = cornish_fisher_quantile(alpha, skew, kurtosis)
    return -(mu + z_cf * sigma)  # знак: потеря положительна

def modified_cvar(mu: float, sigma: float, skew: float, kurtosis: float,
                  alpha: float = 0.05, n_points: int = 1000) -> float:
    """
    Модифицированный CVaR через численное интегрирование CF-квантилей.
    Интегрируем mVaR(p) по p от 0 до alpha.
    """
    ps = np.linspace(0.001, alpha, n_points)
    vars_ = np.array([modified_var(mu, sigma, skew, kurtosis, p) for p in ps])
    return float(np.mean(vars_))

def var_comparison_table(returns: np.ndarray,
                          alphas: list = [0.01, 0.05, 0.10]) -> dict:
    """Сравнение нормального и модифицированного VaR."""
    mu, sigma = returns.mean(), returns.std()
    skew = float(((returns - mu)**3).mean() / sigma**3)
    kurt = float(((returns - mu)**4).mean() / sigma**4 - 3)
    rows = {}
    for alpha in alphas:
        var_normal = -(mu + norm.ppf(alpha) * sigma)
        var_cf = modified_var(mu, sigma, skew, kurt, alpha)
        cvar_cf = modified_cvar(mu, sigma, skew, kurt, alpha)
        rows[alpha] = {'VaR_normal': var_normal, 'VaR_CF': var_cf, 'CVaR_CF': cvar_cf}
    return rows

# Пример: доходности хедж-фонда с тяжёлыми хвостами
np.random.seed(42)
from scipy.stats import skewnorm
returns = skewnorm.rvs(a=-5, loc=0.001, scale=0.02, size=1000)
table = var_comparison_table(returns)
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $z_\alpha$ | Нормальный квантиль | $\Phi^{-1}(\alpha)$ стандартного нормального | $-3.09$ (1%) до $-1.28$ (10%) |
| $S$ | Асимметрия | Третий стандартизованный момент | $-2$ до $2$ (акции: $-0.5$ до $-1.5$) |
| $K$ | Избыточный эксцесс | Четвёртый момент минус 3 | $1$ – $10$ (для акций) |
| $\sigma$ | Волатильность | Стандартное отклонение доходностей | $0.5\%$ – $5\%$ (ежедневно) |
| $\alpha$ | Уровень доверия | Хвостовая вероятность | $0.01$ – $0.10$ |

## Ограничения и расширения

**Ограничения:**
- Разложение CF является приближением: при сильной асимметрии ($|S| > 2$) или экстремальном эксцессе ($K > 10$) точность падает.
- Не гарантирует монотонность квантильной функции при экстремальных параметрах.
- Не захватывает форму всего хвоста — только локальную аппроксимацию квантиля.

**Расширения:**
- **Метод Грама-Шарлье**: альтернативное разложение с рядами Эрмита для более высоких порядков.
- **Историческое моделирование**: эмпирические квантили без параметрических допущений.
- **[[mcmc]]**: байесовская оценка параметров распределения для более робастных квантилей.

## Практическое применение

1. **Регуляторный VaR (Basel IV)**: банки используют CF-поправки для учёта нелинейного риска опционных портфелей.
2. **Риск хедж-фондов**: стратегии с продажей волатильности имеют отрицательную асимметрию — CF VaR критичен.
3. **Управление портфелем**: CF CVaR как мера риска в задаче оптимизации портфеля.
4. **Пенсионные фонды**: регуляторные стресс-тесты с CF-поправкой для нелинейных активов.
5. **Маржинальные требования**: биржи используют CF-метод для установки маржи по производным инструментам.

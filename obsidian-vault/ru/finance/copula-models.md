# Копула-модели

**Копула** (от лат. *copulare* — соединять) — математическая функция, объединяющая одномерные маржинальные распределения в многомерное совместное распределение. В количественных финансах копулы позволяют отделить моделирование индивидуального поведения активов от структуры зависимости между ними — принципиальное преимущество перед корреляцией Пирсона, которая улавливает только линейные связи.

## Обзор и предпосылки

Гауссова копула, применённая Дэвидом Ли (2000) для ценообразования CDO, стала ключевым инструментом структурированных финансов. Кризис 2008 года наглядно показал её главный изъян: параметр Гауссовой копулы не учитывает хвостовую зависимость — активы, слабо коррелирующие в обычное время, могут резко обрушиться одновременно. Архимедовы копулы (Clayton, Gumbel, Frank) и t-копула решают эту проблему через явное моделирование хвостовой зависимости.

## Математическая база

**Теорема Скляра** — фундамент теории копул. Для любого $n$-мерного распределения $F(x_1,\dots,x_n)$ с маржинальными функциями $F_i(x_i)$:

$$
F(x_1,\dots,x_n) = C(F_1(x_1),\dots,F_n(x_n))
$$

Если маргиналы непрерывны, копула $C$ единственна. Для совместной плотности:

$$
f(x_1,\dots,x_n) = c(F_1(x_1),\dots,F_n(x_n))\cdot\prod_{i=1}^n f_i(x_i)
$$

где $c = \partial^n C / \partial u_1\cdots\partial u_n$ — плотность копулы.

**Хвостовые коэффициенты зависимости**:

$$
\lambda_U = \lim_{u\to 1^-}\frac{1 - 2u + C(u,u)}{1-u}, \quad \lambda_L = \lim_{u\to 0^+}\frac{C(u,u)}{u}
$$

**Основные семейства копул**:

| Копула | Функция $C(u,v)$ | $\lambda_U$ | $\lambda_L$ |
|:---|:---|:---:|:---:|
| Гауссова | $\Phi_\rho(\Phi^{-1}(u),\Phi^{-1}(v))$ | $0$ | $0$ |
| Стьюдента ($t_\nu$) | $T_{\nu,\rho}(t_\nu^{-1}(u),t_\nu^{-1}(v))$ | $>0$ | $>0$ |
| Клейтон | $(u^{-\theta}+v^{-\theta}-1)^{-1/\theta}$ | $0$ | $2^{-1/\theta}$ |
| Гумбель | $\exp(-((-\ln u)^\theta+(-\ln v)^\theta)^{1/\theta})$ | $2-2^{1/\theta}$ | $0$ |
| Франк | $-\frac{1}{\theta}\ln\!\left(1+\frac{(e^{-\theta u}-1)(e^{-\theta v}-1)}{e^{-\theta}-1}\right)$ | $0$ | $0$ |

**Хвостовая зависимость t-копулы**:

$$
\lambda_U = \lambda_L = 2t_{\nu+1}\!\left(-\sqrt{\frac{(\nu+1)(1-\rho)}{1+\rho}}\right)
$$

## Ключевые свойства

- Разделение маргиналей и структуры зависимости: каждая часть калибруется независимо
- Инвариантность Скляра к монотонным преобразованиям: ранговая корреляция Спирмена и тау Кендалла определяются только копулой
- Гауссова копула: $\lambda_U = \lambda_L = 0$ — неуместна для портфелей с хвостовым риском
- Клейтон: асимметрична, $\lambda_L > 0$ — отражает «совместные обвалы» (режим кризиса)

## Реализация на Python

```python
import numpy as np
from scipy.stats import norm, t as student_t
from scipy.optimize import minimize_scalar

def generate_clayton_copula(n: int, theta: float, seed: int = 42) -> np.ndarray:
    """Симуляция двумерной копулы Клейтона методом условного обращения."""
    np.random.seed(seed)
    u = np.random.uniform(0, 1, n)
    p = np.random.uniform(0, 1, n)
    v = u * (p**(-theta / (theta + 1)) - 1 + u**theta)**(-1 / theta)
    v = np.clip(v, 1e-10, 1 - 1e-10)
    return np.column_stack([u, v])

def fit_clayton_copula(u: np.ndarray, v: np.ndarray) -> float:
    """МНК-оценка параметра theta копулы Клейтона."""
    def neg_log_lik(theta):
        if theta <= 0:
            return 1e10
        log_c = (np.log(theta + 1) + (-theta - 1) * (np.log(u) + np.log(v))
                 + (-1 / theta - 2) * np.log(u**(-theta) + v**(-theta) - 1))
        return -np.sum(log_c)
    result = minimize_scalar(neg_log_lik, bounds=(0.01, 20), method='bounded')
    return result.x

def tail_dependence_student_t(rho: float, nu: float) -> float:
    """Хвостовой коэффициент зависимости t-копулы."""
    return 2 * student_t.cdf(-np.sqrt((nu + 1) * (1 - rho) / (1 + rho)), df=nu + 1)

def gaussian_copula_sample(rho_matrix: np.ndarray, n: int, seed: int = 42) -> np.ndarray:
    """Симуляция многомерной Гауссовой копулы."""
    np.random.seed(seed)
    L = np.linalg.cholesky(rho_matrix)
    Z = np.random.randn(n, rho_matrix.shape[0])
    X = Z @ L.T
    return norm.cdf(X)

# Пример: Clayton vs Gaussian для двух активов
samples_clayton  = generate_clayton_copula(1000, theta=3.0)
theta_hat = fit_clayton_copula(samples_clayton[:, 0], samples_clayton[:, 1])
rho_mat = np.array([[1.0, 0.6], [0.6, 1.0]])
samples_gaussian = gaussian_copula_sample(rho_mat, n=1000)
lambda_t = tail_dependence_student_t(rho=0.6, nu=4)
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $\theta$ | Параметр зависимости | Сила связи в архимедовых копулах | Clayton: $>0$; Gumbel: $\geq 1$ |
| $\rho$ | Корреляция | Линейная корреляция в Гауссовой/t-копуле | $(-1, 1)$ |
| $\nu$ | Степени свободы | Тяжесть хвостов в t-копуле | $2$ – $30$ |
| $\lambda_U, \lambda_L$ | Хвостовые коэффициенты | Вероятность совместных экстремумов | $[0, 1]$ |

## Ограничения и расширения

**Ограничения:**
- Гауссова копула полностью неправильно оценивает хвостовые совместные потери — опасна для CDO/кредитных корзин.
- Параметрические копулы фиксируют структуру зависимости; реальная зависимость может быть асимметричной и нестационарной.
- Оценка в высоких размерностях ($n > 10$) требует упрощений (парные копулы, vine-копулы).

**Расширения:**
- **Vine-копулы (R-vine, C-vine)**: парное разложение для гибкого многомерного моделирования.
- **Условные копулы**: зависимость как функция рыночных состояний (режимы HMM).
- **Динамические копулы**: временно-меняющийся параметр $\rho_t$ по DCC-модели.

## Практическое применение

1. **Ценообразование кредитных корзин (CDO, CLN)**: Clayton-копула захватывает корреляцию дефолтов при системном стрессе.
2. **Портфельный VaR**: копула корректно учитывает хвостовую зависимость, недооцениваемую линейной корреляцией.
3. **Пар-трейдинг**: копула оценивает совместное распределение пары активов для определения порогов входа/выхода.
4. **Стресс-тестирование**: генерация сценариев с высокой хвостовой зависимостью (Clayton) для экстремальных потерь.
5. **Мультиактивные деривативы**: basket-опционы и spread-опционы требуют явного моделирования совместного распределения.

# Реконструкция фазового пространства

**Реконструкция фазового пространства** — это метод анализа нелинейных динамических систем, позволяющий восстановить многомерный портрет системы, используя только один наблюдаемый временной ряд. В финансах этот метод применяется для обнаружения детерминированного хаоса и построения краткосрочных прогнозов.

## Теорема Такенса

Теоретической основой метода является **теорема Такенса** (1981). Она утверждает, что если временной ряд $x(t)$ порожден некоторой динамической системой, то можно построить пространство, топологически эквивалентное оригинальному фазовому пространству системы, используя метод временных задержек.

## Метод временных задержек

Реконструированный вектор состояния $\mathbf{y}_t$ в момент времени $t$ формируется как:

$$
\mathbf{y}_t = [x(t), x(t-\tau), x(t-2\tau), \dots, x(t-(m-1)\tau)]
$$

Где:
- $x(t)$ — значение временного ряда в момент $t$.
- $\tau$ — **временная задержка** (time delay).
- $m$ — **размерность вложения** (embedding dimension).

## Выбор параметров

Для качественной реконструкции критически важен правильный выбор параметров:
1.  **Задержка $\tau$**: Обычно выбирается как первый локальный минимум функции **взаимной информации** (Mutual Information), чтобы компоненты вектора были максимально независимы.
2.  **Размерность $m$**: Определяется методом **ложных ближайших соседей** (False Nearest Neighbors). Она должна быть достаточно большой, чтобы «развернуть» аттрактор и избежать самопересечений траекторий.

## Применение в финансах

- **Обнаружение аттракторов**: Позволяет визуализировать «странные аттракторы» рынка, указывающие на наличие сложной нелинейной структуры, а не просто случайного шума.
- **Нелинейный прогноз**: Метод локальной аппроксимации (поиск похожих состояний в прошлом в фазовом пространстве) позволяет делать краткосрочные прогнозы даже для хаотических рядов.
- **Отличие хаоса от шума**: Оценка корреляционной размерности помогает понять, является ли ряд случайным или детерминированным.

## Python: Создание векторов вложения

```python
import numpy as np

def reconstruct_phase_space(series, tau, m):
    """
    Создает матрицу вложения для временного ряда.
    """
    n = len(series)
    rows = n - (m - 1) * tau
    if rows <= 0:
        return np.array([])
    
    reconstructed = np.zeros((rows, m))
    for i in range(m):
        reconstructed[:, i] = series[i * tau : i * tau + rows]
    
    return reconstructed

# Пример: Реконструкция для зашумленного цикла
t = np.linspace(0, 10, 500)
prices = np.sin(t) + np.random.normal(0, 0.1, 500)
matrix = reconstruct_phase_space(prices, tau=5, m=3)
print(f"Форма фазового пространства: {matrix.shape}")
```

## Ограничения

- **Чувствительность к шуму**: Сильный рыночный шум может полностью «размыть» структуру аттрактора.
- **Объем данных**: Для высоких размерностей вложения требуются очень длинные временные ряды.

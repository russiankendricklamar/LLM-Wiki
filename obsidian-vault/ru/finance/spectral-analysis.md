# Спектральный анализ

**Спектральный анализ** — это совокупность методов анализа данных, позволяющих разложить временной ряд на набор элементарных гармонических колебаний (синусоид и косинусоид) с разными частотами и амплитудами. В финансах этот метод используется для поиска бизнес-циклов, сезонных паттернов и фильтрации рыночного шума.

## Преобразование Фурье

Математической основой спектрального анализа является **Преобразование Фурье**. Оно переводит временной ряд $x(t)$ из временной области в частотную область $X(f)$. Для дискретных данных используется Дискретное преобразование Фурье (DFT):

$$
X(k) = \sum_{n=0}^{N-1} x(n) \cdot e^{-i \frac{2\pi}{N} kn}
$$

Где:
- $X(k)$ — комплексное число, представляющее амплитуду и фазу частоты $k$.
- $N$ — количество точек в ряду.
- $e^{-i \dots}$ — комплексная экспонента (согласно формуле Эйлера).

## Спектральная плотность мощности (PSD)

**Спектральная плотность мощности** (Power Spectral Density) показывает, какая доля дисперсии (энергии) временного ряда приходится на ту или иную частоту. Пики на графике PSD указывают на доминирующие циклы в данных.

## Применение в финансах

1.  **Обнаружение циклов**: Поиск периодичности в экономических индикаторах или ценах на сырье.
2.  **Декомпозиция рядов**: Отделение долгосрочного тренда от краткосрочных циклов и высокочастотного шума.
3.  **Индикаторы**: Некоторые технические индикаторы (например, Ehler’s Cycle Analytics) основаны на алгоритмах спектрального анализа.
4.  **Улучшение прогнозов**: Использование частотных характеристик для настройки моделей машинного обучения.

## Python: Анализ периодичности через FFT

```python
import numpy as np
import matplotlib.pyplot as plt

# Генерация сигнала с двумя циклами (10 и 50 дней) + шум
t = np.arange(256)
signal = np.sin(2 * np.pi * t / 10) + 0.5 * np.sin(2 * np.pi * t / 50) + np.random.normal(0, 0.2, 256)

# Быстрое преобразование Фурье (FFT)
fft_values = np.fft.fft(signal)
frequencies = np.fft.fftfreq(len(t))
power_spectrum = np.abs(fft_values) ** 2

# Берем только положительные частоты
mask = frequencies > 0
plt.plot(frequencies[mask], power_spectrum[mask])
plt.title("Спектр мощности (PSD)")
plt.xlabel("Частота")
plt.show()
```

## Ограничения

- **Нестационарность**: Финансовые временные ряды часто меняют свои свойства во времени, что затрудняет классический Фурье-анализ (решается через вейвлет-преобразования).
- **Длина данных**: Для обнаружения низкочастотных циклов требуется очень длинная история наблюдений.
- **Шум**: Высокий уровень рыночного шума может маскировать реальные спектральные пики.

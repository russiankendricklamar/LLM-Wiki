# Теория случайных матриц (RMT)

**Теория случайных матриц (RMT)** изучает статистические свойства собственных значений матриц, элементы которых являются случайными величинами. Первоначально разработанная в ядерной физике Вигнером в 1950-х годах для описания спектров энергий тяжёлых ядер, RMT проникла в количественные финансы через работы Лалу, Сизо, Бушо и Поттерса (1999). В портфельном менеджменте RMT — золотой стандарт фильтрации «шума» из корреляционных матриц больших портфелей активов.

## Предпосылки и мотивация

При оценке корреляционной матрицы $N$ активов по $T$ наблюдениям оценщик $\hat{\Sigma}$ имеет $O(N^2)$ параметров, но только $T \cdot N$ наблюдений. При умеренном $Q = T/N$ (например, $Q = 2$) большинство наблюдаемых корреляций являются статистическими артефактами. Подача «сырой» выборочной ковариационной матрицы в оптимизацию Марковица заставляет оптимизатор эксплуатировать погрешности оценки, что приводит к крайне сконцентрированным позициям и катастрофическому ухудшению вне выборки.

**Число обусловленности** выборочной ковариационной матрицы растёт как $O(N/T)$ для случайных данных — это означает, что малые возмущения доходностей могут вызывать огромные колебания оптимальных весов. RMT предоставляет принципиальный способ идентификации и подавления шумовых собственных значений.

## Математический аппарат

### Распределение Марченко-Пастура

Основополагающий результат RMT: для матрицы $\mathbf{W}$ с $T$ строками и $N$ столбцами, элементы которой НЗР с нулевым средним и дисперсией $\sigma^2$, эмпирическое спектральное распределение матрицы Уишарта $\mathbf{C} = \frac{1}{T}\mathbf{W}^T\mathbf{W}$ сходится при $N, T \to \infty$ (при фиксированном $Q = T/N \ge 1$) к **закону Марченко-Пастура**:

$$
\rho_{MP}(\lambda) = \frac{Q}{2\pi \sigma^2} \cdot \frac{\sqrt{(\lambda_+ - \lambda)(\lambda - \lambda_-)}}{\lambda}
$$

на носителе $[\lambda_-, \lambda_+]$, с границами шумового диапазона:

$$
\lambda_\pm = \sigma^2 \left(1 \pm \frac{1}{\sqrt{Q}}\right)^2
$$

Любое собственное значение $\lambda_i > \lambda_+$ лежит за пределами шумовой полосы и статистически отличимо от чистого случайного шума.

### Закон полукруга Вигнера

Для симметричных матриц с НЗР внедиагональными элементами (Гауссов ортогональный ансамбль) плотность собственных значений сходится к **полукругу Вигнера**:

$$
\rho_{sc}(\lambda) = \frac{1}{2\pi\sigma^2}\sqrt{4\sigma^2 - \lambda^2}, \quad |\lambda| \le 2\sigma
$$

Это описывает «балк» спектра для полностью случайных матриц без структуры.

### Процедура очистки собственных значений

Пусть $\hat{\mathbf{C}} = \mathbf{Q}\hat{\boldsymbol{\Lambda}}\mathbf{Q}^T$ — спектральное разложение выборочной корреляционной матрицы, где $\hat{\boldsymbol{\Lambda}} = \text{diag}(\hat{\lambda}_1 \ge \hat{\lambda}_2 \ge \cdots \ge \hat{\lambda}_N)$.

**Шаг 1 — подгонка MP:** Оценить $\sigma^2$ и вычислить $\lambda_+$.

**Шаг 2 — разделение сигнала и шума:**
$$
\hat{\lambda}_i^{\text{clean}} = \begin{cases} \hat{\lambda}_i & \text{если } \hat{\lambda}_i > \lambda_+ \quad (\text{сигнал}) \\ \bar{\lambda}_{noise} & \text{если } \hat{\lambda}_i \le \lambda_+ \quad (\text{шум}) \end{cases}
$$

где $\bar{\lambda}_{noise}$ — среднее значение шумовых собственных значений, выбранное для сохранения следа матрицы (суммарной дисперсии):

$$
\sum_{i: \hat{\lambda}_i \le \lambda_+} \hat{\lambda}_i^{\text{clean}} = \sum_{i: \hat{\lambda}_i \le \lambda_+} \hat{\lambda}_i
$$

**Шаг 3 — реконструкция:**
$$
\hat{\mathbf{C}}^{\text{RMT}} = \mathbf{Q}\hat{\boldsymbol{\Lambda}}^{\text{clean}}\mathbf{Q}^T
$$

## Ключевые свойства и допущения

- **Предположение о НЗР:** Закон MP справедлив точно для матриц с независимыми одинаково распределёнными элементами; реальные доходности имеют серийные корреляции, поэтому порог $\lambda_+$ приближённый.
- **Универсальность балка:** Статистика балка спектра универсальна — не зависит от конкретного распределения элементов матрицы, только от дисперсии $\sigma^2$.
- **Асимптотическая природа:** Очистка асимптотически корректна; при $N < 50$ активах необходимо применять конечновыборочные поправки.
- **Интерпретация через факторные модели:** Собственные значения выше $\lambda_+$ соответствуют рыночным и секторальным факторам риска, часто совпадая со структурой факторов Фама-Френча.

## Python-реализация

```python
import numpy as np

def marchenko_pastur_bounds(T: int, N: int, sigma: float = 1.0) -> tuple[float, float]:
    """Границы шумового диапазона Марченко-Пастура."""
    Q = T / N
    lam_plus  = sigma**2 * (1 + 1/np.sqrt(Q))**2
    lam_minus = sigma**2 * (1 - 1/np.sqrt(Q))**2
    return lam_minus, lam_plus

def rmt_denoise(corr_matrix: np.ndarray, T: int) -> np.ndarray:
    """
    Очистка корреляционной матрицы методом RMT.
    Сохраняет след матрицы (суммарную дисперсию).
    """
    N = corr_matrix.shape[0]
    eigenvalues, eigenvectors = np.linalg.eigh(corr_matrix)
    eigenvalues = eigenvalues[::-1]
    eigenvectors = eigenvectors[:, ::-1]

    _, lam_plus = marchenko_pastur_bounds(T, N)

    signal_mask = eigenvalues > lam_plus
    noise_eigenvalues = eigenvalues[~signal_mask]
    noise_mean = noise_eigenvalues.mean() if noise_eigenvalues.size > 0 else 1.0

    cleaned_eigenvalues = np.where(signal_mask, eigenvalues, noise_mean)

    cleaned_corr = eigenvectors @ np.diag(cleaned_eigenvalues) @ eigenvectors.T

    # Нормализация диагонали (корреляционная матрица)
    diag_sqrt = np.sqrt(np.diag(cleaned_corr))
    cleaned_corr = cleaned_corr / np.outer(diag_sqrt, diag_sqrt)
    return cleaned_corr

def mp_pdf(lam: np.ndarray, Q: float, sigma: float = 1.0) -> np.ndarray:
    """Плотность распределения Марченко-Пастура."""
    lam_plus  = sigma**2 * (1 + 1/np.sqrt(Q))**2
    lam_minus = sigma**2 * (1 - 1/np.sqrt(Q))**2
    pdf = np.zeros_like(lam, dtype=float)
    mask = (lam >= lam_minus) & (lam <= lam_plus)
    pdf[mask] = (Q / (2 * np.pi * sigma**2 * lam[mask]) *
                 np.sqrt((lam_plus - lam[mask]) * (lam[mask] - lam_minus)))
    return pdf

# ---- Пример ----
np.random.seed(42)
T, N = 1000, 200
returns = np.random.randn(T, N)
sample_corr = np.corrcoef(returns.T)

cleaned_corr = rmt_denoise(sample_corr, T)
lam_minus, lam_plus = marchenko_pastur_bounds(T, N)
raw_eigs = np.linalg.eigvalsh(sample_corr)

print(f"Шумовой диапазон MP: [{lam_minus:.4f}, {lam_plus:.4f}]")
print(f"Диапазон сырых собственных значений: [{raw_eigs.min():.4f}, {raw_eigs.max():.4f}]")
print(f"Доля шумовых собственных значений: {(raw_eigs <= lam_plus).mean():.1%}")
```

## Справочник параметров

| Символ | Название | Описание | Типичный диапазон |
| :--- | :--- | :--- | :--- |
| $N$ | Число активов | Количество активов в портфеле | 50 – 3000 |
| $T$ | Число наблюдений | Количество дней доходностей | 250 – 5000 |
| $Q = T/N$ | Соотношение | Контролирует ширину шумовой полосы | 1 – 20 |
| $\lambda_+$ | Верхняя граница MP | Максимальное шумовое собственное значение | 1.5 – 5.0 (при $\sigma=1$) |
| $\sigma^2$ | Дисперсия | Дисперсия индивидуальных доходностей | Калибруется по данным |
| $\bar{\lambda}_{noise}$ | Уровень шума | Значение замены для шумовых собственных значений | $\approx 1/(1 + 1/Q)$ |

## Ограничения и расширения

- **Нестационарность:** Корреляции меняются во времени; скользящая или режимно-условная очистка необходима для реальных данных.
- **Шум в собственных векторах:** RMT очищает собственные значения, но не собственные векторы; при малом $T/N$ векторы также зашумлены. Метод поворотной очистки (Bun et al. 2017) расширяет подход.
- **Негауссовы доходности:** Тяжёлые хвосты смещают порог MP; робастные альтернативы используют аппроксимации $\alpha$-стабильным распределением.
- **Известная факторная структура:** Для известных факторных моделей аналитическое сжатие Ледуа-Вольфа часто превосходит эвристическую очистку RMT.

## Практические применения

1. **Оптимизация портфеля:** Замена сырой ковариации на очищенную матрицу RMT в оптимизации Марковица даёт стабильные диверсифицированные портфели с меньшим оборотом.
2. **Обнаружение факторов риска:** Собственные портфели выше $\lambda_+$ — дата-движимые факторы риска, аналогичные главным компонентам.
3. **Мониторинг системного риска:** Скачок $\lambda_1/\lambda_+$ сигнализирует о синхронизации корреляций — опережающий индикатор рыночного стресса.
4. **Модели издержек исполнения:** Очищенные матрицы рыночного воздействия улучшают оценки транзакционных издержек в моделях Альмгрена-Криса.
5. **Стресс-тестирование:** Контролируемые возмущения очищенных матриц позволяют более реалистично генерировать сценарии по сравнению с зашумлёнными матрицами.

# Алгоритм Шора

**Алгоритм Шора (Shor's Algorithm)** — это квантовый алгоритм для факторизации целых чисел. Разработанный Питером Шором в 1994 году, он позволяет решать задачу разложения числа на множители за полиномиальное время, что делает его экспоненциально быстрее лучших классических алгоритмов.

## Теоретические основы

Значимость алгоритма заключается в его способности "взламывать" системы криптографии с открытым ключом, такие как RSA, которые базируются на вычислительной сложности факторизации больших чисел. Алгоритм использует квантовые свойства — **суперпозицию** и **интерференцию** — для нахождения периода функции, что является ключом к факторизации.

Алгоритм делится на две части:
1. **Классическое сведение**: Сведение задачи факторизации к задаче поиска порядка (периода) функции.
2. **Квантовый поиск порядка**: Использование Квантового преобразования Фурье (QFT) для нахождения периода функции $f(x) = a^x \pmod N$.

## Математическая формулировка

Чтобы факторизовать число $N$:
1. Выбрать случайное число $a < N$, такое что $\gcd(a, N) = 1$.
2. Найти период $r$ функции $f(x) = a^x \pmod N$, т.е. минимальное $r$, при котором $a^r \equiv 1 \pmod N$.
3. Если $r$ четное и $a^{r/2} \not\equiv -1 \pmod N$, то множителями $N$ будут $\gcd(a^{r/2} - 1, N)$ и $\gcd(a^{r/2} + 1, N)$.

Квантовая часть находит $r$ через создание суперпозиции состояний:
$$ \frac{1}{\sqrt{Q}} \sum_{x=0}^{Q-1} |x\rangle |a^x \pmod N\rangle $$
с последующим применением обратного QFT к первому регистру.

## Практическая значимость в финансах

Хотя полномасштабных квантовых компьютеров еще не существует, алгоритм Шора имеет критическое значение для будущего финансовых систем:
- **Угроза RSA**: Большинство банковских транзакций и цифровых подписей полагаются на RSA, который становится уязвимым.
- **Постквантовая криптография (PQC)**: Финансовые организации уже сейчас исследуют переход на квантово-устойчивые алгоритмы (на основе решеток или кодов).
- **Безопасность блокчейна**: Алгоритмы на эллиптических кривых (ECDSA), используемые в Биткоине, также уязвимы для квантовых атак на основе логики Шора.
- **Стратегическое планирование**: Оценка "квантового риска" (Quantum Risk) заставляет quants пересматривать протоколы хранения долгосрочных данных.

    a = random.randint(2, N-1)
    if gcd(a, N) > 1: return gcd(a, N)
    r = quantum_order_finding(a, N)

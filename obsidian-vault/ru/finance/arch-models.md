# Модели ARCH

**Модель авторегрессионной условной гетероскедастичности (ARCH)**, введённая Робертом Энглом в 1982 году (Нобелевская премия 2003), произвела революцию в финансовой эконометрике, предоставив строгую математическую основу для моделирования изменяющейся во времени волатильности — одного из наиболее устойчивых эмпирических свойств финансовых доходностей.

## Необходимые знания

[[ar-models]], [[arma-models]]

## Обзор и предпосылки

До работы Энгла эконометрические модели предполагали гомоскедастичность ошибок — постоянную дисперсию во времени. Однако финансовые временные ряды демонстрируют отчётливую **кластеризацию волатильности**: крупные ценовые движения группируются вместе, за ними следуют периоды спокойствия.

Ключевая идея Энгла: хотя безусловное распределение доходностей может быть стабильным, **условная** дисперсия систематически зависит от прошлых квадратов остатков. Это порождает лептокуртозные маргинальные распределения без привлечения негауссовых инноваций.

ARCH породил целое семейство моделей: GARCH (Bollerslev, 1986), EGARCH (Nelson, 1991), GJR-GARCH (Glosten et al., 1993), FIGARCH, CGARCH. Эти модели стали стандартными инструментами в ценообразовании опционов, риск-менеджменте и построении портфелей.

## Математическая база

Модель $ARCH(q)$ разлагает доходности на среднее и гетероскедастическую инновацию:

$$
r_t = \mu_t + \epsilon_t, \quad \epsilon_t = \sigma_t z_t, \quad z_t \overset{iid}{\sim} \mathcal{N}(0, 1)
$$

Условная дисперсия $\sigma_t^2 = \text{Var}(\epsilon_t | \mathcal{F}_{t-1})$:

$$
\sigma_t^2 = \alpha_0 + \sum_{i=1}^q \alpha_i \epsilon_{t-i}^2
$$

**Ограничения** для корректной условной дисперсии:
- $\alpha_0 > 0$ — гарантирует положительную базовую дисперсию
- $\alpha_i \geq 0$ для всех $i$ — гарантирует неотрицательность
- $\sum_{i=1}^q \alpha_i < 1$ — гарантирует ковариационную стационарность

**Безусловная дисперсия** при $\sum_i \alpha_i < 1$:

$$
\sigma^2 = \mathbb{E}[\sigma_t^2] = \frac{\alpha_0}{1 - \sum_{i=1}^q \alpha_i}
$$

**Эксцесс**: даже при $z_t \sim \mathcal{N}(0,1)$ ARCH-процесс генерирует избыточный эксцесс. Для $ARCH(1)$:

$$
\text{Kurt}(\epsilon_t) = 3 \cdot \frac{1 - \alpha_1^2}{1 - 3\alpha_1^2} \quad \text{(требует } 3\alpha_1^2 < 1 \text{ для конечного эксцесса)}
$$

**Тест ARCH-LM** (Engle, 1982): регрессия $\hat{\epsilon}_t^2$ на собственные лаги; статистика $LM = T \cdot R^2 \sim \chi^2(q)$ при нулевой гипотезе об отсутствии ARCH-эффектов.

## Ключевые свойства

- **Кластеризация волатильности**: $\sigma_t^2$ высока, когда $\epsilon_{t-1}^2$ было большим.
- **Тяжёлые хвосты**: маргинальное распределение $r_t$ имеет избыточный эксцесс даже при гауссовых $z_t$.
- **Возврат к среднему**: при $\sum_i \alpha_i < 1$ дисперсия возвращается к безусловному уровню $\sigma^2$.
- **Отсутствие плеча**: ARCH одинаково реагирует на положительные и отрицательные шоки. EGARCH и GJR-GARCH улавливают асимметрию.

## Реализация на Python

```python
import numpy as np
import pandas as pd
from arch import arch_model
from scipy import stats

def fit_arch_model(returns: pd.Series, q: int = 1) -> dict:
    """Оценить ARCH(q) модель для ряда доходностей."""
    model = arch_model(returns * 100, vol='ARCH', p=q, mean='Constant', dist='Normal')
    result = model.fit(disp='off')
    return {
        'params': result.params,
        'aic': result.aic,
        'bic': result.bic,
        'conditional_vol': result.conditional_volatility / 100,
    }

def arch_lm_test(residuals: np.ndarray, lags: int = 5) -> dict:
    """Тест ARCH-LM Энгла на условную гетероскедастичность."""
    n = len(residuals)
    resid_sq = residuals ** 2
    X = np.column_stack([resid_sq[i:n-lags+i] for i in range(lags)])
    y = resid_sq[lags:]
    XtX_inv = np.linalg.pinv(X.T @ X)
    beta = XtX_inv @ X.T @ y
    y_hat = X @ beta
    r_squared = 1 - np.sum((y - y_hat)**2) / np.sum((y - y.mean())**2)
    lm_stat = n * r_squared
    p_value = 1 - stats.chi2.cdf(lm_stat, df=lags)
    return {'lm_stat': lm_stat, 'p_value': p_value, 'lags': lags}

def simulate_arch(alpha0: float, alphas: list, n: int, seed: int = 42) -> dict:
    """Симуляция ARCH(q) процесса."""
    np.random.seed(seed)
    q = len(alphas)
    eps = np.zeros(n)
    sigma2 = np.zeros(n)
    sigma2[:q] = alpha0 / (1 - sum(alphas))
    eps[:q] = np.random.normal(0, np.sqrt(sigma2[:q]))
    for t in range(q, n):
        sigma2[t] = alpha0 + sum(a * eps[t-i-1]**2 for i, a in enumerate(alphas))
        eps[t] = np.sqrt(sigma2[t]) * np.random.normal()
    return {'returns': eps, 'sigma': np.sqrt(sigma2)}

result = simulate_arch(alpha0=0.0001, alphas=[0.8], n=1000)
returns_series = pd.Series(result['returns'])
arch_test = arch_lm_test(returns_series.values, lags=5)
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $\alpha_0$ | Базовая дисперсия | Долгосрочная компонента дисперсии | > 0 |
| $\alpha_i$ | Коэффициенты ARCH | Вес на прошлые квадраты шоков | $[0, 1)$ |
| $q$ | Порядок ARCH | Число запаздывающих квадратов остатков | 1 – 10 |
| $\sigma^2$ | Безусловная дисперсия | $\alpha_0 / (1 - \sum \alpha_i)$ | > 0 |
| $z_t$ | Стандартизированная инновация | Нормальная, t-Стьюдента или GED | — |

## Ограничения и расширения

**Ограничения:**
- **Требует большого $q$**: для длинной памяти волатильности нужно много лагов. GARCH(1,1) решает это двумя параметрами.
- **Нет эффекта плеча**: одинаковая реакция на положительные и отрицательные шоки. В акциях негативные шоки усиливают волатильность сильнее.
- **Нестандартные распределения**: $z_t$ часто имеет более тяжёлые хвосты, чем нормальное распределение; использование t-Стьюдента или GED улучшает качество подгонки.

**Расширения:**
- **GARCH(p,q)** ([[garch-models]]): добавляет запаздывающие $\sigma_{t-j}^2$, парсимонично захватывая персистентность волатильности.
- **EGARCH**: асимметричная реакция волатильности через $\ln \sigma_t^2$.
- **GJR-GARCH**: индикаторный член для эффекта плеча.

## Практическое применение

1. **Ценообразование опционов**: прогнозы волатильности на основе ARCH улучшают подгонку поверхности подразумеваемой волатильности.
2. **Расчёт VaR**: динамический VaR с условной дисперсией ARCH превосходит исторический.
3. **Оптимизация портфеля**: меняющиеся во времени ковариационные матрицы из многомерных ARCH-моделей улучшают среднедисперсионную оптимизацию.
4. **Бэктестинг риск-моделей**: тест ARCH-LM применяется к остаткам модели для проверки отсутствия предсказуемой дисперсии.
5. **Алгоритмическая торговля**: прогнозы волатильности из ARCH используются для динамического масштабирования позиций и корректировки стоп-лоссов.

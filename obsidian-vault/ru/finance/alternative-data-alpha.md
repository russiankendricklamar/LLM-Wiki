# Альтернативная альфа (Alternative Data Alpha)

**Альтернативная альфа** — это инвестиционные сигналы, полученные из нетрадиционных источников. В эпоху, когда традиционные финансовые данные (цены, объемы, отчетность) анализируются миллионами участников рынка, преимущество все чаще находят в нестандартных наборах данных: спутниковых снимках, транзакциях по картам, геолокации и данных о цепочках поставок.

## Практическое руководство по внедрению

1.  **Поиск источников (Sourcing)**: Работа с поставщиками (например, Orbital Insight для спутников или Eagle Alpha для общих наборов данных).
2.  **Маппинг сущностей (Entity Mapping)**: Самая сложная часть. Связывание «парковки торгового центра» на снимке с конкретным публичным тикером ритейлера.
3.  **Нормализация и очистка**: Альтернативные данные часто неструктурированы и «зашумлены». Например, данные о посещаемости магазинов должны корректироваться с учетом погоды и праздников.
4.  **Интеграция сигналов**: Объединение альт-сигналов с традиционными факторами через [[alpha-stacking]] или ML-модели.

## Математическая формулировка

Пусть $X_{alt}$ — сигнал альтернативных данных (например, количество машин на парковках Walmart). Прогнозная модель:
$$
R_{t+1} = \beta_0 + \beta_{alt} X_{alt, t} + \sum \beta_i F_{i, t} + \epsilon
$$
где $F_i$ — традиционные факторы (Size, Value). Чтобы убедиться, что сигнал не является шумом, можно использовать [[mcmc]] для оценки вероятности того, что связь $\beta_{alt} \neq 0$ статистически значима. Если данных мало, [[gan]] можно применять для генерации синтетических, но реалистичных паттернов альтернативных данных.

## Ключевые метрики
- **Эксклюзивность**: У скольких участников рынка есть доступ к этим данным?
- **Разреженность (Sparsity)**: Какая часть времени или вселенной активов покрыта данными?
- **Опережение (Lead Time)**: Насколько раньше официальных отчетов данные дают сигнал?
- **Затухание информации**: Скорость, с которой альфа из данных арбитражируется рынком.

## Python: Симуляция сигнала посещаемости

```python
import numpy as np
import pandas as pd

def simulate_alt_signal(actual_revenue):
    """
    Симуляция сигнала посещаемости, который опережает выручку.
    """
    # Сигнал = реальная выручка + шум
    noise = np.random.normal(0, 0.1, len(actual_revenue))
    signal = actual_revenue + noise
    return pd.Series(signal).rolling(3).mean()

# Тестовые данные: Рост выручки
revenue = np.cumsum(np.random.normal(0.01, 0.05, 100))
alt_signal = simulate_alt_signal(revenue)

correlation = np.corrcoef(alt_signal[3:], revenue[3:])[0, 1]
print(f"Корреляция Alt-Data и Выручки: {correlation:.4f}")
```

## Финансовый контекст

Альтернативные данные — главный драйвер **квантаментального (Quantamental)** инвестирования, объединяющего количественные модели и фундаментальный анализ. Например, подсчет танкеров в океане дает сигнал о мировых поставках нефти раньше официальных данных. Риск-менеджеры используют [[value-at-risk]], чтобы моделировать влияние «исчезновения» сигнала (например, из-за изменения законов о конфиденциальности).

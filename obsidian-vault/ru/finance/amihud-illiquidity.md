# Коэффициент неликвидности Амихуда

**Коэффициент неликвидности Амихуда**, предложенный Яковом Амихудом в 2002 году, — широко используемая мера для оценки влияния объёма торгов на цену. Он отражает взаимосвязь между абсолютной доходностью и долларовым объёмом торгов и является основой обширной литературы о премии за риск ликвидности в ценообразовании активов.

## Обзор и предпосылки

Измерение ликвидности остаётся центральной проблемой в ценообразовании активов с новаторской работы Amihud & Mendelson (1986), показавших, что спреды bid-ask объясняют различия в поперечных доходностях. Сложность состоит в том, что высокочастотные меры микроструктуры (спреды, глубина, лямбда Кайла) требуют тиковых данных, недоступных для длительных исторических исследований или развивающихся рынков. Амихуд (2002) предложил низкочастотный прокси, вычислимый из дневных данных CRSP: отношение абсолютной доходности к долларовому объёму.

Коэффициент Амихуда оказался чрезвычайно полезным: он высоко коррелирует с внутридневными мерами ценового влияния, предсказывает будущие ожидаемые доходности и резко возрастает в периоды рыночного стресса (2008 год, крах COVID-19 в 2020 году).

## Математическая база

Коэффициент неликвидности Амихуда $ILLIQ_{i,y}$ для акции $i$ в году $y$ определяется как среднедневное отношение абсолютной доходности к долларовому объёму:

$$
ILLIQ_{i,y} = \frac{1}{D_{i,y}} \sum_{t=1}^{D_{i,y}} \frac{|R_{i,t}|}{DVOL_{i,t}}
$$

где:
- $R_{i,t}$ — доходность акции $i$ в день $t$
- $DVOL_{i,t} = \text{Volume}_{i,t} \times P_{i,t}$ — долларовый объём
- $D_{i,y}$ — число торговых дней в году $y$

**Интерпретация единиц**: $ILLIQ$ измеряет процентное изменение цены на доллар объёма торгов. Акция с $ILLIQ = 0.001$ движется на 0.1% при $100\,000$ долларов объёма — она значительно более чувствительна к ценовому давлению, чем акция с $ILLIQ = 0.0001$.

**Общерыночная неликвидность** — равновзвешенное среднее:

$$
ILLIQ_{m,y} = \frac{1}{N_y} \sum_{i=1}^{N_y} ILLIQ_{i,y}
$$

**Ценовое влияние сделки объёмом $Q$ долларов:**

$$
\Delta P \approx \lambda \cdot Q, \quad \lambda = ILLIQ
$$

Это линейное приближение связывает $ILLIQ$ с $\lambda$ Кайла (1985) из литературы по микроструктуре рынка.

### Фактор риска ликвидности

**Инновация ликвидности** (неожиданное изменение рыночной ликвидности) используется как ценовой фактор риска:

$$
\Delta ILLIQ_{m,t} = ILLIQ_{m,t} - \mathbb{E}[ILLIQ_{m,t}]
$$

Активы, доходность которых положительно коррелирует с $\Delta ILLIQ_m$ (плохо себя ведут при росте рыночной неликвидности), зарабатывают **премию за риск ликвидности**.

## Реализация на Python

```python
import pandas as pd
import numpy as np

def amihud_ratio(returns: pd.Series, dollar_volume: pd.Series) -> float:
    """
    Рассчитать коэффициент неликвидности Амихуда.
    returns      : Series ежедневных доходностей (дроби, не проценты)
    dollar_volume: Series ежедневного долларового объёма
    """
    valid = dollar_volume > 0
    return (returns.abs()[valid] / dollar_volume[valid]).mean()

def rolling_amihud(
    returns: pd.Series, dollar_volume: pd.Series, window: int = 63
) -> pd.Series:
    """
    Скользящий коэффициент Амихуда для отслеживания динамики ликвидности.
    window: торговых дней (63 ≈ 1 квартал)
    """
    ratio = returns.abs() / dollar_volume.replace(0, np.nan)
    return ratio.rolling(window).mean()

def illiquidity_premium_factor(
    amihud_series: pd.Series, market_returns: pd.Series, n_quantiles: int = 5
) -> pd.DataFrame:
    """
    Средние доходности по квинтилям Амихуда для оценки премии за ликвидность.
    """
    quantile_labels = pd.qcut(amihud_series, q=n_quantiles, labels=False)
    result = market_returns.groupby(quantile_labels).agg(['mean', 'std', 'count'])
    result['sharpe'] = result['mean'] / (result['std'] + 1e-10) * np.sqrt(252)
    return result

# Пример
np.random.seed(42)
n_days = 252
prices = 100 * np.exp(np.cumsum(np.random.normal(0.0003, 0.015, n_days)))
volume = np.random.lognormal(14, 0.5, n_days)
dollar_vol = pd.Series(prices * volume)
returns = pd.Series(np.diff(np.log(prices), prepend=np.log(prices[0])))

illiq = amihud_ratio(returns, dollar_vol)
rolling_illiq = rolling_amihud(returns, dollar_vol, window=21)
```

## Таблица параметров

|:---|:---:|:---|:---|
| $ILLIQ_{i,y}$ | Коэффициент Амихуда | Среднее $|R_t|/DVOL_t$ за год | $10^{-8}$ до $10^{-5}$ |

## Ограничения и расширения

**Ограничения:**
- Амихуд использует $|R_t|$ и не различает покупки и продажи.
- Абсолютный долларовый объём делает сравнение акций корректным только после нормировки по капитализации.
- Долгосрочные тренды в объёмах торгов (рост алгоритмической торговли после 2005 года) создают структурные разрывы в временных рядах ILLIQ.
- В дни с очень низким объёмом ILLIQ может быть экстремальным; необходима фильтрация дней ниже порогового объёма.

**Расширения:**
- **Фактор ликвидности Pastor-Stambaugh (2003)**: использует знаковую версию меры Амихуда для построения торгуемого фактора ликвидности.
- **Декомпозиция Sadka (2006)**: разделяет ILLIQ на постоянную (информационную) и временную (инвентарную) компоненты.
- **Внутридневная агрегация**: сочетание дневных оценок Амихуда с реализованным спредом даёт более полную картину ликвидности.

## Практическое применение

1. **Конструирование портфеля**: систематические стратегии используют скрининг по Амихуду для избегания позиций, слишком крупных для выхода за целевое число дней.
2. **Оценка транзакционных издержек**: $\text{impact} \approx ILLIQ \times \text{размер сделки в долларах}$ используется для оценки проскальзывания в бэктестах.
3. **Риск-менеджмент**: VaR-модели, дополненные горизонтом ликвидности (время ликвидации без существенного влияния), используют ILLIQ для определения горизонта на актив.
4. **Факторное исследование**: документально подтверждённая Амихудом (2002) премия за неликвидность обосновывает длинно-короткий фактор.
5. **Развивающиеся рынки**: ILLIQ особенно полезен на рынках, где данные по спредам bid-ask часто недоступны.

# Коэффициент неликвидности Амихуда

**Коэффициент неликвидности Амихуда** — это широко используемая мера для оценки влияния объема торгов на цену. Он отражает взаимосвязь между абсолютной доходностью и долларовым объемом торгов.

## Формула

Коэффициент $I_t$ для заданного периода определяется как:

$$
\text{Illiquidity}_{i,T} = \frac{1}{D_{i,T}} \sum_{t=1}^{D_{i,T}} \frac{|R_{i,t}|}{\text{Volume}_{i,t} \times P_{i,t}}
$$

Где:
- $R_{i,t}$ — доходность актива $i$ в день $t$
- $\text{Volume}_{i,t} \times P_{i,t}$ — долларовый объем (стоимость торгов)
- $D_{i,T}$ — количество дней в периоде

Более высокое значение указывает на то, что актив менее ликвиден, так как для изменения цены на определенный процент требуется меньший объем торгов.

## Премия за риск ликвидности
Исследования показывают, что активы с более высоким коэффициентом Амихуда, как правило, имеют более высокую ожидаемую доходность, что отражает премию за несение риска ликвидности.

## Реализация на Python

```python
import pandas as pd

def amihud_ratio(returns, dollar_volume):
    """
    Рассчитать коэффициент неликвидности Амихуда.
    returns: Series ежедневных доходностей
    dollar_volume: Series ежедневного долларового объема
    """
    illiquidity = (returns.abs() / dollar_volume).mean()
    return illiquidity
```

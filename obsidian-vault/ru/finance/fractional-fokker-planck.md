# Дробное уравнение Фоккера-Планка (FFPE)

**Дробное уравнение Фоккера-Планка (FFPE)** — обобщение классического уравнения Фоккера-Планка (уравнения Колмогорова) для описания временной эволюции плотности вероятности физической или финансовой системы. Введение производных дробного порядка даёт инструментарий для моделирования **аномальной диффузии** — процессов, в которых дисперсия растёт нелинейно со временем, что эмпирически наблюдается на многих финансовых рынках.

## История и контекст

Классическое уравнение Фоккера-Планка разработано независимо Фоккером (1914) и Планком (1917) для описания броуновского движения. Его решение — распространяющаяся во времени гауссова плотность, что соответствует лог-нормальной модели Блэка-Шоулза. Дробные обобщения изучены Метцлером и Клафтером (2000): они показали, что модели случайных блужданий непрерывного времени (CTRW) с тяжёлыми хвостами времён ожидания и/или прыжков сходятся в скейлинговом пределе к FFPE.

## Аномальная диффузия в финансах

В стандартной модели Блэка-Шоулза дисперсия доходностей растёт линейно: $\text{Var}(x) \sim t$. В реальных данных часто наблюдается иное:

- **Субдиффузия** ($\text{Var} \sim t^\alpha,\ \alpha < 1$): процесс распространяется медленнее броуновского. Характерна для низколиквидных рынков, сильного возврата к среднему, долгих времён ожидания между сделками.
- **Супердиффузия** ($\text{Var} \sim t^\alpha,\ \alpha > 1$): процесс распространяется быстрее. Характерна для трендовых рынков с «долгой памятью», полётами Леви, устойчивыми автокорреляциями.
- **Нормальная диффузия** ($\alpha = 1$): стандартный случай броуновского движения, согласующийся с гипотезой эффективного рынка.

## Математическая постановка

Классическое уравнение Фоккера-Планка для плотности вероятности $p(x,t)$ с дрейфом $A(x)$ и диффузией $B(x)$:

$$
\frac{\partial p(x,t)}{\partial t} = -\frac{\partial}{\partial x}[A(x)\,p(x,t)] + \frac{\partial^2}{\partial x^2}[B(x)\,p(x,t)]
$$

**Дробное обобщение** с дробной производной Римана–Лиувилля порядка $\alpha \in (0,1]$ по времени:

$$
\frac{\partial p(x, t)}{\partial t} = {}_0D_t^{1-\alpha} \left[ -\frac{\partial}{\partial x} (A(x) p(x, t)) + \frac{\partial^2}{\partial x^2} (B(x) p(x, t)) \right]
$$

Оператор Римана–Лиувилля:

$$
{}_0D_t^{1-\alpha} f(t) = \frac{1}{\Gamma(\alpha)} \frac{d}{dt} \int_0^t (t-s)^{\alpha-1} f(s)\, ds
$$

Интеграл свёртки встраивает **всю предысторию** процесса в текущую скорость изменения — именно это порождает эффекты памяти и не-марковскую динамику.

## Пространственно-дробное FFPE (полёты Леви)

Отдельный класс FFPE заменяет пространственную вторую производную дробным лапласианом порядка $\mu \in (0,2]$:

$$
\frac{\partial p(x,t)}{\partial t} = -A \frac{\partial p}{\partial x} + D_\mu (-\Delta)^{\mu/2} p(x,t)
$$

Это моделирует **полёты Леви** — процессы, в которых размеры прыжков подчиняются степенному закону. Решения имеют алгебраические хвосты $p(x,t) \sim |x|^{-(1+\mu)}$, непосредственно воспроизводя тяжёлые хвосты в распределениях доходностей.

## Связь с показателем Херста

Порядок дробности $\alpha$ напрямую связан с показателем Херста $H$. Для субдиффузивных процессов CTRW:

$$
\langle x^2(t) \rangle \sim t^\alpha \quad \Rightarrow \quad H = \alpha/2
$$

Для дробного броуновского движения (fBm) с показателем Херста $H$ соответствующее FFPE имеет $\alpha = 2H$. Таким образом, наблюдаемый $H$ (оцениваемый по ценовым данным) напрямую связан с дробным порядком уравнения эволюции.

## Ключевые свойства

- Решения FFPE **негауссовы** и имеют степенные хвосты при $\alpha < 1$ (временное) или $\mu < 2$ (пространственное).
- Процесс **не-марковский**: будущее состояние зависит от полной предыстории через свёртку в дробной производной.
- **Стационарность** не гарантирована: субдиффузивные процессы могут демонстрировать старение (зависимость от момента начала наблюдения).

## Python: Численная концепция (симуляция CTRW)

```python
import numpy as np

def simulate_ctrw(n_steps: int, alpha: float, seed: int = 42) -> tuple:
    """
    Симуляция CTRW, сходящегося к решению FFPE.

    alpha < 1: субдиффузия (тяжёлые хвосты времён ожидания)
    alpha > 1: супердиффузия (тяжёлые хвосты прыжков)
    """
    rng = np.random.default_rng(seed)
    waiting_times = rng.pareto(alpha, n_steps)   # тяжёлый хвост
    jumps = rng.normal(0, 1, n_steps)            # гауссовы прыжки

    operational_time = np.cumsum(waiting_times)
    position = np.cumsum(jumps)
    return operational_time, position

def estimate_diffusion_exponent(times: np.ndarray, positions: np.ndarray, n_bins: int = 20) -> float:
    """Оценка alpha по наклону log-log графика MSD vs time."""
    max_t = times[-1]
    t_bins = np.logspace(0, np.log10(max_t), n_bins)
    msd = []
    for t_bin in t_bins:
        idx = np.searchsorted(times, t_bin)
        if idx > 0:
            msd.append(np.mean(positions[:idx] ** 2))
    log_t = np.log(t_bins[:len(msd)])
    log_msd = np.log(msd)
    slope, _ = np.polyfit(log_t, log_msd, 1)
    return slope

# Субдиффузия (alpha=0.6) и почти нормальная (alpha=0.9)
t_sub, x_sub = simulate_ctrw(5000, alpha=0.6)
t_norm, x_norm = simulate_ctrw(5000, alpha=0.9)

print(f"Субдиффузия, оценка alpha:  {estimate_diffusion_exponent(t_sub, x_sub):.3f}")
print(f"Норм. диффузия, оценка alpha: {estimate_diffusion_exponent(t_norm, x_norm):.3f}")
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|--------|----------|----------|-------------------|
| $\alpha$ | Дробный порядок | Сила аномалии | $(0, 1]$ для субдиффузии |
| $H$ | Показатель Херста | Мера памяти; $H = \alpha/2$ | $(0, 1)$ |
| $\mu$ | Индекс Леви | Хвост пространственных прыжков | $(0, 2]$ |
| $A(x)$ | Дрейф | Детерминированная сила | Зависит от модели |
| $B(x)$ | Коэффициент диффузии | Локальная волатильность | $B > 0$ |

## Ограничения и расширения

- **Вычислительная стоимость**: численные решения FFPE требуют $O(N^2)$ памяти для $N$ временных шагов из-за нелокальности оператора.
- **Идентификация параметров**: различить субдиффузию и возврат к среднему по данным нетривиально; нужны R/S анализ и DFA.
- **Нестационарность**: FFPE предполагает фиксированный $\alpha$; реальные рынки показывают изменяющиеся во времени свойства памяти.
- **Затухающие процессы Леви**: современное расширение заменяет чистые степенные хвосты экспоненциально усечёнными, сохраняя тяжёлые хвосты на промежуточных масштабах при конечных моментах.
- **Мультифрактальность**: некоторые финансовые ряды демонстрируют разные $H$ на разных масштабах; мультифрактальные модели (MMAR) обобщают однопараметрическое FFPE.

## Практическое применение

1. **Оценка опционов при аномальной диффузии**: замена стандартного оператора диффузии дробным даёт PDE ценообразования, чьи решения воспроизводят улыбку подразумеваемой волатильности.
2. **VaR с памятью**: включение дробной динамики в риск-модели лучше захватывает персистентность крупных убытков.
3. **Моделирование ликвидности**: на тонких рынках время между сделками подчиняется тяжёлому хвосту — субдиффузивный режим FFPE.
4. **Крипторынки и EM**: эмпирические показатели Херста значительно выше 0.5 на этих рынках мотивируют дробные модели вместо GBM.

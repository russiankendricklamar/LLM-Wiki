# Дробное уравнение Фоккера-Планка (FFPE)

**Дробное уравнение Фоккера-Планка (FFPE)** — это обобщение классического уравнения Фоккера-Планка (уравнения Колмогорова), которое описывает эволюцию плотности вероятности системы во времени. В отличие от классического случая, FFPE использует производные дробного порядка, что позволяет моделировать **аномальную диффузию**.

## Аномальная диффузия в финансах

В стандартной модели Блэка-Шоулза (основанной на броуновском движении) дисперсия растет линейно со временем: $\text{Var}(x) \sim t$. Однако на реальных рынках часто наблюдается иное поведение:
- **Субдиффузия** ($\text{Var} \sim t^\alpha, \alpha < 1$): Процесс замедляется, сильный возврат к среднему.
- **Супердиффузия** ($\text{Var} \sim t^\alpha, \alpha > 1$): Процесс ускоряется, сильные тренды и «память».

## Математическая форма

Типичное дробное уравнение Фоккера-Планка с дробной производной по времени Римана-Лиувилля порядка $\alpha$ имеет вид:

$$
\frac{\partial p(x, t)}{\partial t} = {}_0D_t^{1-\alpha} \left[ -\frac{\partial}{\partial x} (A(x) p(x, t)) + \frac{\partial^2}{\partial x^2} (B(x) p(x, t)) \right]
$$

Где:
- $p(x, t)$ — плотность вероятности.
- ${}_0D_t^{1-\alpha}$ — оператор дробного дифференцирования.
- $A(x)$ и $B(x)$ — коэффициенты дрейфа и диффузии.

## Зачем это нужно квинтам?

1.  **Моделирование «толстых хвостов»**: FFPE естественным образом порождает распределения с тяжелыми хвостами, которые лучше соответствуют рыночным данным.
2.  **Эффекты памяти**: Дробные производные учитывают всю предысторию процесса, в отличие от локальных марковских моделей.
3.  **Оценка опционов**: Позволяет строить модели оценки, учитывающие долгосрочную зависимость (Long-range dependence).

## Связь с показателем Херста

Параметр дробности $\alpha$ напрямую связан с показателем Херста $H$. Для многих процессов аномальной диффузии выполняется соотношение:
$$
H = \alpha / 2
$$

## Python: Численное решение (схематично)

Решение FFPE обычно требует сложных сеточных методов или симуляции соответствующих случайных блужданий (Continuous Time Random Walks, CTRW).

```python
import numpy as np

def simulate_ctrw(n_steps, alpha):
    """
    Симуляция случайного блуждания с непрерывным временем (CTRW),
    которое в пределе сходится к решению FFPE.
    """
    # Время ожидания между прыжками (распределение Парето для аномалии)
    waiting_times = np.random.pareto(alpha, n_steps)
    # Прыжки (нормальное распределение)
    jumps = np.random.normal(0, 1, n_steps)
    
    time = np.cumsum(waiting_times)
    path = np.cumsum(jumps)
    return time, path

# Пример супердиффузии
t, x = simulate_ctrw(1000, 0.8)
```

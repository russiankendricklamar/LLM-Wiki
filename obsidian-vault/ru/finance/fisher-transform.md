# Преобразование Фишера (Fisher Transform)

**Преобразование Фишера** — это математический метод, используемый в техническом анализе для преобразования временного ряда цен (которые обычно не распределены нормально) в данные, имеющие распределение, близкое к нормальному (Гауссовскому). Это делает пики и впадины на графике более выраженными, облегчая поиск точек разворота.

## Зачем это нужно?

Цены на активы часто «залипают» около своих экстремумов, создавая широкие плато на обычных индикаторах (например, RSI или Stochastics). Преобразование Фишера резко переводит эти значения в область бесконечности, превращая плавные закругления в острые пики.

## Математическая формула

Для данных $x$, нормированных в диапазоне $[-1, 1]$, преобразование Фишера $y$ рассчитывается как:

$$
y = \frac{1}{2} \ln \left( \frac{1 + x}{1 - x} \right) = \text{arctanh}(x)
$$

Где:
- $x$ — нормированная цена (или значение другого индикатора).
- $y$ — трансформированное значение.

## Алгоритм применения в трейдинге

1.  Выбирается окно (например, 10 дней).
2.  Цена нормируется так, чтобы максимум окна стал $+1$, а минимум — $-1$.
3.  Применяется формула преобразования Фишера.
4.  Сигналом к покупке или продаже служит пересечение линии индикатора со своей сглаженной копией (триггерной линией).

## Python: Реализация индикатора Фишера

```python
import numpy as np

def fisher_transform(price, period=10):
    # Псевдокод для концепции
    high = price.rolling(period).max()
    low = price.rolling(period).min()
    
    # Нормировка в [-1, 1]
    value = 0.66 * ((price - low) / (high - low) - 0.5) + 0.67 * 0
    value = np.clip(value, -0.999, 0.999) # Избегаем деления на 0
    
    # Преобразование Фишера
    fisher = 0.5 * np.log((1 + value) / (1 - value))
    return fisher

# Пример данных
prices = np.random.normal(100, 2, 100)
f_line = fisher_transform(prices)
```

## Преимущества

- **Скорость реакции**: Позволяет обнаруживать развороты раньше, чем традиционные осцилляторы.
- **Четкость сигналов**: Убирает неопределенность «зависания» индикатора в зонах перекупленности/перепроданности.
- **Универсальность**: Может применяться к ценам, объемам или другим индикаторам.

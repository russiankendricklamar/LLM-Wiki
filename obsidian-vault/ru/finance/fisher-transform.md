# Преобразование Фишера (Fisher Transform)

**Преобразование Фишера** — математический метод, применяемый в техническом анализе для преобразования ценовых данных в распределение, близкое к нормальному (Гауссовскому). Разработан Джоном Эхлерсом в книге *Cybernetic Analysis for Stocks and Futures* (2002). Преобразование использует обратную гиперболическую тангенциальную функцию для резкого обострения сигналов разворота, делая экстремумы статистически выраженными и легко идентифицируемыми.

## Предпосылки и мотивация

Цены на активы не следуют нормальному распределению: они демонстрируют тяжёлые хвосты и склонны «задерживаться» у своих экстремумов, создавая широкие плато на обычных осцилляторах (RSI, Stochastics). Стандартные индикаторы «залипают» в зонах перекупленности/перепроданности и не дают чётких сигналов. Преобразование Фишера устраняет этот недостаток: оно нелинейно растягивает значения вблизи границ диапазона к $\pm\infty$, превращая плавные закругления в острые пики, которые легко интерпретировать статистически.

## Математическая формула

Для данных $x$, нормированных в диапазоне $(-1, 1)$, преобразование Фишера $y$ — это обратный гиперболический тангенс:

$$
y = \frac{1}{2} \ln \left( \frac{1 + x}{1 - x} \right) = \text{arctanh}(x)
$$

Ключевые свойства:
- При $x \to \pm 1$ значение $y \to \pm\infty$: экстремальные позиции цены дают крайние показания индикатора.
- При $x = 0$ значение $y = 0$: середина диапазона отображается в нуль.
- Производная $dy/dx = 1/(1-x^2)$ расходится на границах, усиливая малые изменения у экстремумов.

Обратное преобразование:
$$
x = \tanh(y) = \frac{e^{2y} - 1}{e^{2y} + 1}
$$

## Статистическая интерпретация

Если нормированная цена $x$ равномерно распределена на $(-1,1)$, то $y = \text{arctanh}(x)$ следует логистическому распределению. На практике это приближает распределение индикатора к нормальному: значение $|y| > 2$ приблизительно соответствует «2-сигмовому» событию, что даёт статистически обоснованный порог для генерации сигналов — чего не могут обеспечить ограниченные осцилляторы.

## Алгоритм применения в трейдинге

1. **Нормировка**: цена отображается на диапазон $[-1, 1]$ через максимум и минимум за период $n$ (обычно 9–14 баров):
   $$x_t = 0.66 \cdot \frac{P_t - \min_{t-n:t}}{\max_{t-n:t} - \min_{t-n:t}} - 0.5 \cdot 0.66$$
2. **Трансформация**: применяется формула $y_t = \text{arctanh}(x_t)$.
3. **Триггерная линия**: $y_{t-1}$ — копия индикатора со сдвигом на 1 бар.
4. **Генерация сигналов**: покупка при пересечении $y_t$ триггерной линии снизу вверх; продажа — при пересечении сверху вниз. Экстремальные значения $|y_t| > 2$ указывают на вероятный разворот.

## Python: Реализация преобразования Фишера

```python
import numpy as np
import pandas as pd

def calculate_fisher(prices: pd.Series, period: int = 10) -> pd.DataFrame:
    """
    Расчёт индикатора Фишера и триггерной линии.

    Возвращает DataFrame с колонками 'fisher' и 'trigger'.
    """
    roll_max = prices.rolling(window=period).max()
    roll_min = prices.rolling(window=period).min()

    # Нормировка в (-1, 1); обрезка во избежание сингулярности arctanh
    denom = (roll_max - roll_min).replace(0, np.nan)
    raw = 2 * (prices - roll_min) / denom - 1
    clamped = raw.clip(-0.999, 0.999)

    # Преобразование Фишера: arctanh = 0.5 * ln((1+x)/(1-x))
    fisher = 0.5 * np.log((1 + clamped) / (1 - clamped))

    return pd.DataFrame({
        "fisher":  fisher,
        "trigger": fisher.shift(1),
    })

# Пример: симуляция цены
np.random.seed(42)
raw_returns = np.random.normal(0.002, 0.015, 200)
prices = pd.Series(100 * np.exp(np.cumsum(raw_returns)), name="price")

indicator = calculate_fisher(prices, period=10)

# Простые сигналы по пересечению
long_signal  = (indicator["fisher"] > indicator["trigger"]) & \
               (indicator["fisher"].shift(1) <= indicator["trigger"].shift(1))
short_signal = (indicator["fisher"] < indicator["trigger"]) & \
               (indicator["fisher"].shift(1) >= indicator["trigger"].shift(1))

print(f"Сигналов к покупке:  {long_signal.sum()}")
print(f"Сигналов к продаже: {short_signal.sum()}")
```

## Таблица параметров

| Символ | Название | Описание | Типичное значение |
|--------|----------|----------|-------------------|
| $n$ | Период нормировки | Окно для максимума/минимума | 9–14 баров |
| $x$ | Нормированная цена | Позиция цены в $(-1, 1)$ | $(-0.999, 0.999)$ |
| $y$ | Значение Фишера | Трансформированный индикатор | $(-\infty, +\infty)$, значимо: $\pm 3$ |
| Порог | Экстремальный сигнал | Уровень вероятного разворота | $\pm 1.5$ до $\pm 2.5$ |

## Ограничения и расширения

- **Запаздывание**: окно нормировки вводит задержку, равную периоду. Короткие периоды снижают запаздывание, но увеличивают шум.
- **Зависимость от диапазона**: прорыв к новым максимумам может подавить сигнал, даже если импульс экстремален в абсолютном выражении.
- **Ложные пересечения**: как все осцилляторы, преобразование Фишера может давать множество сигналов в трендовом рынке; комбинирование с фильтром тренда (например, EMA-200) снижает долю ложных сигналов.
- **Адаптивный Фишер**: некоторые реализации используют переменный период нормировки, основанный на доминирующей длине цикла (из HHT), что улучшает качество сигнала на нестационарных рынках.
- **Кросс-секционный рэнкинг**: гауссизированный выход делает ранжирование активов по значению Фишера статистически корректным — полезно в систематических стратегиях.

## Практическое применение

1. **Сигналы возврата к среднему**: значения Фишера выше $\pm 2$ часто совпадают с локальными экстремумами цены.
2. **Нормализация сигналов**: предварительная обработка технических индикаторов через преобразование Фишера делает их объединение в составной сигнал статистически обоснованным.
3. **Подтверждение импульса**: растущее значение Фишера подтверждает ценовой импульс и позволяет удерживать или наращивать позицию.
4. **Опционные стратегии**: экстремальные показания Фишера помогают выбирать момент для шорт-гаммовых или лонг-гаммовых позиций.

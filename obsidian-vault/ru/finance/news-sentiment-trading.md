# Торговая система на анализе новостей (News Sentiment Trading)

**Система торговли на анализе новостей (Sentiment Trading)** использует обработку естественного языка (NLP) для количественной оценки эмоционального тона финансовых новостей, социальных сетей или отчетов компаний. В современных финансах это часть «Альтернативной альфы» или событийного трейдинга (Event-Driven Trading).

## Практическое руководство по внедрению

1.  **Сбор данных**: Получение потоковых новостей от Bloomberg, Reuters или парсинг Twitter (X).
2.  **Предобработка**: Токенизация, удаление стоп-слов и распознавание именованных сущностей (определения того, к какому тикеру относится новость).
3.  **Оценка сентимента**:
    *   **На базе словарей**: Использование специализированных словарей (Loughran-McDonald).
    *   **BERT/FinBERT**: Использование моделей-трансформеров, предобученных на финансовых текстах.
4.  **Генерация сигналов**: Агрегация оценок за временное окно (например, 1 час) и использование Z-score для выявления резких изменений настроений.

## Математическая формулировка

Оценка сентимента $S_t$ для актива является средневзвешенным значением $N$ новостей:
$$
S_t = \frac{\sum_{i=1}^N w_i \cdot s_i}{\sum w_i}
$$
где $s_i$ — сентимент отдельной новости, $w_i$ — её важность или охват. Для фильтрации шума можно использовать [[mcmc]] для моделирования скрытого «истинного» настроения рынка. Сценарии с «фейковыми новостями» можно генерировать с помощью [[gan]] для проверки устойчивости системы.

## Ключевые метрики
- **Точность сентимента**: Корреляция между оценкой системы и немедленной реакцией цены.
- **Затухание сигнала (Signal Decay)**: Как долго длится эффект новости (обычно от минут до нескольких часов).
- **Разброс мнений (Dispersion)**: Высокое разногласие между источниками часто предвещает высокую волатильность.
- **Охват (Coverage)**: Количество активов, которые система может отслеживать.

## Python: Инференс FinBERT для сентимента

```python
# Требуются библиотеки transformers и torch
from transformers import BertTokenizer, BertForSequenceClassification
import torch

def get_sentiment(text):
    tokenizer = BertTokenizer.from_pretrained('ProsusAI/finbert')
    model = BertForSequenceClassification.from_pretrained('ProsusAI/finbert')
    
    inputs = tokenizer(text, return_tensors="pt", padding=True)
    outputs = model(**inputs)
    
    # Вероятности для [positive, negative, neutral]
    probs = torch.nn.functional.softmax(outputs.logits, dim=-1)
    return probs.detach().numpy()

text = "Federal Reserve signals potential rate cuts by summer, boosting market confidence."
scores = get_sentiment(text)
print(f"Сентимент (Pos/Neg/Neu): {scores}")
```

## Финансовый контекст

Анализ настроений жизненно важен для **событийных хедж-фондов**. Например, изменение тональности CEO во время телеконференции может предсказать падение акций до того, как оно станет очевидным. Однако в HFT сентимент часто отстает от цены. Риск-менеджеры используют [[value-at-risk]], чтобы подготовиться к «черным лебедям», вызванным вирусными негативными новостями.

# Временной vs Кросс-секционный факторный анализ

В эмпирическом ценообразовании активов существует два основных способа оценки премий за факторный риск и факторных нагрузок: **регрессия временных рядов (TS)** и **кросс-секционная регрессия (CS)**.

## Регрессия временных рядов (TS)

В подходе TS (например, Блэка-Йенсена-Шоулза) мы регрессируем избыточную доходность актива $i$ на доходность известного фактора $F$ во времени:

$$
R_{i,t} - R_{f,t} = \alpha_i + \beta_i (R_{m,t} - R_{f,t}) + \epsilon_{i,t}
$$

- **Цель**: Оценить нагрузку ($\beta_i$) актива $i$ на фактор $F$.
- **Интерпретация**: $\alpha_i$ представляет собой неверную оценку (mispricing) относительно факторной модели.

## Кросс-секционная регрессия (CS)

В подходе CS (например, Фамы-Макбета) мы регрессируем доходность всех активов на их расчетные нагрузки на каждом временном шаге $t$:

$$
R_{i,t} = \gamma_{0,t} + \gamma_{1,t} \hat{\beta}_i + \eta_{i,t}
$$

- **Цель**: Оценить премию за риск ($\gamma_1$), связанную с фактором.
- **Процедура Фамы-Макбета**:
    1. Оценка $\beta_i$ с помощью TS-регрессии.
    2. Запуск CS-регрессий в каждый момент $t$ для получения временного ряда $\gamma_t$.
    3. Вычисление среднего значения $\gamma_t$ для определения средней премии за риск.

## Ключевые различия

| Характеристика | Временные ряды (TS) | Кросс-секции (CS) |
| :--- | :--- | :--- |
| **Фокус** | Риск конкретного актива | Премия за факторный риск |
| **Входные данные** | Доходности факторов | Факторные нагрузки (беты) |
| **Результаты** | Беты и Альфы | Премии за риск ($\lambda$) |

## Валидация

Надежный факторный анализ должен учитывать «ошибку предвзятости относительно будущего» (look-ahead bias) и «проблему множественного тестирования». Практики используют методы [[probability-of-backtest-overfitting]], чтобы убедиться, что обнаруженные факторы не являются результатом простого поиска по данным.

# Непрерывное вейвлет-преобразование (CWT)

**Непрерывное вейвлет-преобразование (CWT)** — это метод частотно-временного анализа, который позволяет исследовать сигнал одновременно во временной и частотной областях. В отличие от преобразования Фурье, которое использует бесконечные синусоиды, вейвлет-преобразование использует локализованные во времени функции — «маленькие волны» (вейвлеты).

## Почему вейвлеты важны для финансов?

Финансовые рынки нестационарны: их циклы и волатильность меняются со временем. Фурье-анализ «размазывает» эти изменения по всему ряду. Вейвлеты же позволяют увидеть:
- **Когда** произошел всплеск волатильности на определенной частоте.
- **Как** меняется сила краткосрочных и долгосрочных циклов со временем.

## Математическое определение

Для сигнала $x(t)$ непрерывное вейвлет-преобразование $W(a, b)$ определяется как свертка сигнала с масштабированным и смещенным материнским вейвлетом $\psi(t)$:

$$
W(a, b) = \frac{1}{\sqrt{|a|}} \int_{-\infty}^{\infty} x(t) \psi^* \left( \frac{t - b}{a} \right) dt
$$

Где:
- $a$ — параметр масштаба (отвечает за частоту).
- $b$ — параметр смещения (отвечает за положение во времени).
- $\psi(t)$ — материнский вейвлет (например, вейвлет Морле).

## Python: Генерация скалограммы

Использование библиотеки `PyWavelets` для анализа доходностей:

```python
import pywt
import numpy as np
import matplotlib.pyplot as plt

def plot_wavelet(data):
    # Используем вейвлет Морле
    coef, freqs = pywt.cwt(data, np.arange(1, 128), 'cmor1.5-1.0')
    
    plt.imshow(np.abs(coef), extent=[0, len(data), 1, 128], 
               cmap='jet', aspect='auto', vmax=abs(coef).max(), vmin=0)
    plt.title("Вейвлет-скалограмма (CWT)")
    plt.ylabel("Масштаб (Частота)")
    plt.xlabel("Время")
    plt.show()

# Пример: синусоида с меняющейся частотой
t = np.linspace(0, 1, 1000)
signal = np.sin(2 * np.pi * (10 + 20 * t) * t) 
plot_wavelet(signal)
```

## Применение

1.  **Детектирование разрывов**: Обнаружение резких изменений тренда или структурных сдвигов.
2.  **Мультимасштабный анализ корреляции**: Изучение того, как два актива коррелируют на разных горизонтах (например, слабая корреляция на днях, но сильная на месяцах).
3.  **Денойзинг**: Удаление шума при сохранении резких скачков цены (которые Фурье-фильтры часто сглаживают).

# Скрытые марковские модели (HMM)

**Скрытая марковская модель (Hidden Markov Model, HMM)** — это статистическая модель, в которой система предполагается марковским процессом с неизвестными (скрытыми) параметрами. Задача состоит в том, чтобы определить скрытые параметры на основе наблюдаемых параметров.

В финансах HMM идеально подходят для моделирования **рыночных режимов** (Market Regimes). Например, волатильность и доходность акций могут зависеть от скрытого состояния экономики (рост, рецессия, стагнация), которое мы не видим напрямую, но можем оценить по рыночным котировкам.

## Основные компоненты

HMM определяется тремя наборами параметров ($\lambda = (A, B, \pi)$):
1.  **Матрица переходов (A)**: Вероятности перехода из одного скрытого состояния в другое $P(z_t | z_{t-1})$.
2.  **Матрица эмиссий (B)**: Вероятность наблюдения конкретного значения при условии нахождения в скрытом состоянии $P(x_t | z_t)$.
3.  **Начальное распределение ($\pi$)**: Вероятность того, с какого состояния начнется процесс.

## Свойства Марковской цепи

Процесс обладает свойством отсутствия памяти: будущее состояние зависит только от текущего состояния и не зависит от всей предыстории:

$$
P(z_{t+1} | z_t, z_{t-1}, \dots, z_1) = P(z_{t+1} | z_t)
$$

## Три основные задачи HMM

1.  **Оценка (Задча 1)**: Какова вероятность того, что наблюдаемая последовательность была порождена данной моделью? (Алгоритм Forward-Backward).
2.  **Декодирование (Задача 2)**: Какова наиболее вероятная последовательность скрытых состояний для данной последовательности наблюдений? (Алгоритм Витерби).
3.  **Обучение (Задача 3)**: Как настроить параметры модели $\lambda$, чтобы максимизировать вероятность наблюдений? (Алгоритм Баума-Уэлша).

## Python: Обнаружение рыночных режимов

Использование библиотеки `hmmlearn` для поиска двух режимов (низкая и высокая волатильность) в доходностях акций:

```python
from hmmlearn import hmm
import numpy as np

# Пример доходностей (наблюдения)
returns = np.random.normal(0.001, 0.02, 1000).reshape(-1, 1)

# Создание и обучение модели с 2 скрытыми состояниями
model = hmm.GaussianHMM(n_components=2, covariance_type="full", n_iter=100)
model.fit(returns)

# Предсказание скрытых состояний
hidden_states = model.predict(returns)

print("Матрица переходов:")
print(model.transmat_)
# Состояния: 0 - низкая вол-ть, 1 - высокая вол-ть (или наоборот)
```

## Применение в трейдинге

- **Адаптивные стратегии**: Использование разных параметров (например, длин скользящих средних) в зависимости от текущего режима волатильности.
- **Риск-менеджмент**: Снижение позиций при переходе модели в режим высокой волатильности или системного риска.
- **Прогнозирование**: Оценка вероятности того, что текущий тренд сохранится в следующем периоде.

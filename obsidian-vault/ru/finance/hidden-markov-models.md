# Скрытые марковские модели (HMM)

**Скрытая марковская модель (Hidden Markov Model, HMM)** — статистическая модель, в которой система предполагается марковским процессом с ненаблюдаемыми (скрытыми) состояниями. Наблюдаемые данные порождаются скрытыми состояниями через распределение эмиссий. HMM — стандартная модель для **обнаружения рыночных режимов** в количественных финансах.

## Предпосылки

HMM изначально разработаны для распознавания речи в 1960–70-х годах (Баум, Уэлш, Витерби). В финансах они решают фундаментальную проблему: статистические свойства доходностей (среднее, волатильность, корреляция) меняются со временем ненаблюдаемым образом. Скрытое состояние («бычий рынок», «медвежий рынок», «кризис высокой волатильности») управляет наблюдаемыми доходностями, а модель выводит активное состояние из самой последовательности доходностей.

## Основные компоненты

HMM полностью задаётся тройкой $\lambda = (A, B, \pi)$:

1. **Матрица переходов $A$**: $A_{ij} = P(z_t = j \mid z_{t-1} = i)$ — вероятность перехода из состояния $i$ в $j$.
2. **Распределение эмиссий $B$**: $B_i(x) = P(x_t = x \mid z_t = i)$ — правдоподобие наблюдения $x$ при скрытом состоянии $i$. Для непрерывных доходностей обычно гауссово: $x_t \mid z_t = i \sim \mathcal{N}(\mu_i, \sigma_i^2)$.
3. **Начальное распределение $\pi$**: $\pi_i = P(z_1 = i)$ — вероятность старта в состоянии $i$.

## Марковское свойство

Последовательность скрытых состояний — цепь Маркова первого порядка (без памяти):

$$
P(z_{t+1} \mid z_t, z_{t-1}, \dots, z_1) = P(z_{t+1} \mid z_t)
$$

При заданных скрытых состояниях наблюдения условно независимы:

$$
P(x_1, \dots, x_T \mid z_1, \dots, z_T) = \prod_{t=1}^T P(x_t \mid z_t)
$$

## Три основные задачи HMM

### 1. Оценка правдоподобия (алгоритм Forward-Backward)

Вычислить $P(X \mid \lambda)$ для наблюдений $X = (x_1, \dots, x_T)$.

Прямые переменные $\alpha_t(i) = P(x_1, \dots, x_t, z_t = i \mid \lambda)$:

$$
\alpha_1(i) = \pi_i B_i(x_1), \qquad \alpha_{t+1}(j) = B_j(x_{t+1}) \sum_i \alpha_t(i) A_{ij}
$$

Тогда $P(X \mid \lambda) = \sum_i \alpha_T(i)$.

### 2. Декодирование (алгоритм Витерби)

Найти наиболее вероятную последовательность скрытых состояний $z^* = \arg\max_{z} P(z \mid X, \lambda)$.

$$
\delta_t(j) = \max_{z_1,\dots,z_{t-1}} P(z_1,\dots,z_{t-1},z_t=j, x_1,\dots,x_t \mid \lambda)
$$
$$
\delta_{t+1}(j) = B_j(x_{t+1}) \max_i [\delta_t(i) A_{ij}]
$$

Обратный проход даёт оптимальную последовательность состояний.

### 3. Обучение (алгоритм Баума-Уэлша / EM)

Найти $\lambda^* = \arg\max_\lambda P(X \mid \lambda)$ — оценка параметров методом максимального правдоподобия.

**E-шаг** вычисляет апостериорные вероятности состояний через алгоритм Forward-Backward. **M-шаг** обновляет параметры:

$$
\hat{\mu}_i = \frac{\sum_t \gamma_t(i) x_t}{\sum_t \gamma_t(i)}, \qquad \hat{\sigma}_i^2 = \frac{\sum_t \gamma_t(i)(x_t - \hat{\mu}_i)^2}{\sum_t \gamma_t(i)}
$$
$$
\hat{A}_{ij} = \frac{\sum_t \xi_t(i,j)}{\sum_t \gamma_t(i)}
$$

где $\gamma_t(i) = P(z_t = i \mid X, \lambda)$ и $\xi_t(i,j) = P(z_t = i, z_{t+1} = j \mid X, \lambda)$.

## Python: Обнаружение рыночных режимов

```python
import numpy as np
import pandas as pd
from hmmlearn import hmm

def fit_market_regime_hmm(
    returns: np.ndarray,
    n_regimes: int = 2,
    n_iter: int = 200,
    seed: int = 42,
) -> dict:
    """
    Подгонка гауссовой HMM для выявления режимов рыночной волатильности.
    """
    X = returns.reshape(-1, 1)
    model = hmm.GaussianHMM(
        n_components=n_regimes,
        covariance_type="full",
        n_iter=n_iter,
        random_state=seed,
    )
    model.fit(X)
    states = model.predict(X)
    probs  = model.predict_proba(X)

    vols = [np.sqrt(model.covars_[i][0, 0]) for i in range(n_regimes)]
    regime_labels = {i: ("Низкая вол." if v == min(vols) else "Высокая вол.") for i, v in enumerate(vols)}

    return {
        "model":            model,
        "states":           states,
        "state_probs":      probs,
        "regime_means":     model.means_.flatten(),
        "regime_vols":      np.array(vols),
        "transition_matrix":model.transmat_,
        "regime_labels":    regime_labels,
    }

# --- Симуляция двухрежимных доходностей ---
np.random.seed(0)
calm   = np.random.normal(0.0005,  0.008, 600)
crisis = np.random.normal(-0.001,  0.025, 400)
returns_sim = np.concatenate([calm, crisis])

result = fit_market_regime_hmm(returns_sim, n_regimes=2)
print("Средние режимов (дневные):", result["regime_means"].round(5))
print("Волатильности режимов:    ", result["regime_vols"].round(5))
print("Матрица переходов:\n",        result["transition_matrix"].round(3))
```

## Таблица параметров

| Символ | Название | Описание | Типичное значение |
|--------|----------|----------|-------------------|
| $K$ | Число состояний | Количество рыночных режимов | 2–4 |
| $A_{ij}$ | Вероятность перехода | Персистентность режима на диагонали | 0.95–0.99 |
| $\mu_i$ | Средняя доходность режима | Ожидаемая доходность в режиме $i$ | Варьируется |
| $\sigma_i$ | Волатильность режима | Стандартное отклонение доходностей | Низкая: 0.5–1%, высокая: 2–4% |
| $\pi_i$ | Начальные вероятности | Распределение стартового режима | Часто равномерное |

## Ограничения и расширения

- **Фиксированное число состояний**: байесовские непараметрические расширения (бесконечная HMM, HDP-HMM) позволяют обучать число режимов из данных.
- **Геометрическое распределение длительностей**: скрытые полумарковские модели (HSMM) допускают явные распределения длительностей, лучше описывая персистентность медвежьих рынков.
- **Гауссовы эмиссии**: тяжёлохвостые эмиссии (Student-$t$ или смесь гауссиан) лучше подходят для данных доходностей.
- **Многомерные HMM**: совместное моделирование нескольких активов с многомерным гауссовым распределением эмиссий на каждое состояние; захватывает зависящую от режима корреляционную структуру.

## Применение в трейдинге

- **Адаптивные стратегии**: переключение между параметрами (длины скользящих средних) в зависимости от текущего режима.
- **Риск-менеджмент**: снижение позиций при высокой апостериорной вероятности кризисного режима.
- **Динамическое распределение активов**: корректировка весов акции/облигации по вектору вероятностей текущего режима.
- **Прогнозирование волатильности**: взвешенное по режимам среднее волатильностей состояний — краткосрочный прогноз волатильности.

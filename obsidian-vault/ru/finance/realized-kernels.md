# Прогнозирование волатильности через реализованные ядра (Realized Kernels)

**Реализованные ядра (Realized Kernels)** — это класс статистических оценок, предназначенных для измерения интегральной волатильности цены актива, устойчивых к **шуму микроструктуры рынка (MMN)**. Обычная реализованная дисперсия (RV) дает смещенные результаты на высоких частотах (например, 1-секундные интервалы) из-за влияния спреда и ошибок округления.

## Математическое определение

Оценка через реализованное ядро $RK$ определяется как:
$$
RK = \sum_{h=-H}^H k\left(\frac{h}{H+1}\right) \gamma_h
$$
Где:
- $\gamma_h = \sum_{j=|h|+1}^n x_j x_{j-|h|}$ — автоковариация доходностей $x_j$ с лагом $h$.
- $k(x)$ — весовая функция ядра (например, ядро Парзена или Тьюки-Ханнинга).
- $H$ — параметр ширины окна (bandwidth), определяющий количество лагов для подавления шума.

## Устойчивость к шуму

При наличии микроструктурного шума $\epsilon_t$ наблюдаемая цена $Y_t = X_t + \epsilon_t$ отклоняется от истинной. Реализованное ядро является состоятельной оценкой, то есть оно сходится к истинной интегральной волатильности при увеличении частоты наблюдений:
$$
RK \xrightarrow{p} \int_0^T \sigma_s^2 ds
$$

## Применение в финансах

1. **Ценообразование опционов**: Точные внутридневные оценки волатильности критически важны для расчета греков и стоимости краткосрочных опционов.
2. **Управление рисками**: Улучшение моделей Value-at-Risk (VaR) за счет использования более точных данных о текущей волатильности.
3. **Алгоритмы исполнения**: Корректировка «срочности» исполнения по модели Алмгрена-Крисса в зависимости от текущего уровня шума и волатильности.

## Python: Расчет реализованного ядра

```python
import numpy as np

def realized_kernel(returns, H):
    n = len(returns)
    rk = 0
    # Веса ядра Парзена
    def parzen(x):
        x = abs(x)
        if 0 <= x <= 0.5: return 1 - 6*x**2 + 6*x**3
        if 0.5 < x <= 1: return 2 * (1 - x)**3
        return 0

    for h in range(-H, H + 1):
        weight = parzen(h / (H + 1))
        if h >= 0:
            gamma_h = np.sum(returns[h:] * returns[:n-h])
        else:
            gamma_h = np.sum(returns[:n+h] * returns[-h:])
        rk += weight * gamma_h
    return rk

# Пример: 1000 высокочастотных доходностей
rets = np.random.normal(0, 0.01, 1000)
vol_est = np.sqrt(realized_kernel(rets, 5))
print(f"Оценка волатильности: {vol_est:.6f}")
```

# Байесовская оптимизация портфеля Блэка-Литтермана

**Модель Блэка-Литтермана (BL)** — это сложный метод построения портфеля, который преодолевает чувствительность оптимизации среднего-дисперсии Марковица к входным оценкам. Объединяя **равновесное рыночное априорное распределение** с **субъективными взглядами инвестора**, она позволяет создавать стабильные и диверсифицированные портфели.

## Смешанное оценивание Тейла-Голдбергера

В рамках байесовского подхода взгляды инвестора рассматриваются как дополнительные точки данных. Равновесное априорное распределение предполагает, что рыночный портфель является оптимальным. Мы предполагаем, что ожидаемая доходность $\Pi$ следует распределению, центрированному вокруг рыночного равновесия:

$$
\Pi = \delta \Sigma w_{mkt}
$$

где $\delta$ — коэффициент неприятия риска, $\Sigma$ — ковариационная матрица, а $w_{mkt}$ — веса рыночной капитализации.

## Постериорное распределение

Формула Блэка-Литтермана для комбинированной ожидаемой доходности $E[R]$ выводится из свойств многомерного нормального распределения:

$$
E[R] = [(\tau\Sigma)^{-1} + P^T \Omega^{-1} P]^{-1} [(\tau\Sigma)^{-1} \Pi + P^T \Omega^{-1} Q]
$$

Где:
- $\tau$: Скаляр, указывающий на неопределенность априорного распределения.
- $P$: Матрица выбора (picking matrix), которая сопоставляет взгляды с активами.
- $Q$: Вектор выраженных взглядов.
- $\Omega$: Диагональная ковариационная матрица неопределенности взглядов.

Постериорная ковариационная матрица $\Sigma_{post}$ имеет вид:
$$
\Sigma_{post} = \Sigma + [(\tau\Sigma)^{-1} + P^T \Omega^{-1} P]^{-1}
$$

## Python: Реализация с использованием PyPortfolioOpt

```python
from pypfopt import black_litterman, risk_models
from pypfopt.black_litterman import BlackLittermanModel
import pandas as pd
import numpy as np

# Рыночное априорное распределение (равновесные доходности)
cov_matrix = risk_models.sample_cov(prices)
delta = 2.5 # Неприятие риска
prior = black_litterman.market_implied_prior_returns(mcaps, delta, cov_matrix)

# Взгляды инвестора
viewdict = {"AAPL": 0.10, "GOOGL": -0.05}
confidences = [0.6, 0.4]

bl = BlackLittermanModel(cov_matrix, pi=prior, absolute_views=viewdict, omega="idzorev")
ret_bl = bl.bl_returns()
cov_bl = bl.bl_covariance()

# Получение оптимизированных весов
from pypfopt.efficient_frontier import EfficientFrontier
ef = EfficientFrontier(ret_bl, cov_bl)
weights = ef.max_sharpe()
```

## Продвинутые аспекты

Модель BL часто интегрируется с [[hierarchical-risk-parity]] для обработки ошибок оценки в ковариационной матрице $\Sigma$. Кроме того, когда взгляды нелинейны или не подчиняются гауссовскому распределению, для сэмплирования из постериорного распределения вместо аналитической формулы можно использовать методы [[mcmc]].

# Байесовская оптимизация портфеля Блэка-Литтермана

**Модель Блэка-Литтермана (BL)**, предложенная Фишером Блэком и Робертом Литтерманом из Goldman Sachs (1990, 1992), преодолевает главный недостаток оптимизации Марковица — экстремальную чувствительность весов к входным оценкам ожидаемой доходности. BL объединяет **равновесное рыночное априорное распределение** с субъективными взглядами инвестора в байесовской структуре.

## Необходимые знания

[[hierarchical-risk-parity]], [[mcmc]]

## Обзор и предпосылки

Проблема Марковица: ошибки в ожидаемой доходности приводят к концентрированным, нестабильным портфелям. Блэк и Литтерман предложили начинать с рыночного равновесия (CAPM) как нейтральной стартовой точки, а не с исторических средних. Если у инвестора нет взглядов — портфель BL воспроизводит рыночный. Взгляды отклоняют его в сторону от рынка пропорционально уверенности.

Математическая основа — смешанное оценивание Тейла-Голдбергера, аналог Ridge-регрессии с информативным априорным распределением.

## Математическая база

**Рыночное равновесие (априорное распределение)**. Обратная оптимизация даёт равновесные доходности:

$$
\Pi = \delta\Sigma w_{mkt}
$$

где $\delta$ — подразумеваемый коэффициент неприятия риска рынка, $\Sigma$ — ковариационная матрица активов, $w_{mkt}$ — веса рыночной капитализации.

**Взгляды инвестора**: матрица выбора $P \in \mathbb{R}^{k \times n}$ задаёт $k$ взглядов относительно $n$ активов. Вектор $Q$ — ожидаемые значения взглядов с матрицей неопределённости $\Omega$:

$$
P\mu = Q + \eta, \quad \eta \sim \mathcal{N}(0, \Omega)
$$

**Апостериорная ожидаемая доходность** — формула BL:

$$
\mu_{BL} = \left[(\tau\Sigma)^{-1} + P^\top\Omega^{-1}P\right]^{-1} \left[(\tau\Sigma)^{-1}\Pi + P^\top\Omega^{-1}Q\right]
$$

**Апостериорная ковариация**:

$$
\Sigma_{BL} = \Sigma + \left[(\tau\Sigma)^{-1} + P^\top\Omega^{-1}P\right]^{-1}
$$

**Метод Идзорека** для калибровки $\Omega$ из доверия $c_k \in [0,1]$:

$$
\omega_k = \frac{1 - c_k}{c_k} \cdot (P_k \Sigma P_k^\top)
$$

**Оптимальные веса** методом максимизации функции полезности:

$$
w^* = (\delta\Sigma_{BL})^{-1}\mu_{BL}
$$

## Ключевые свойства

- При нулевой уверенности в взглядах ($\Omega \to \infty$): $\mu_{BL} \to \Pi$ — портфель сходится к рыночному
- При полной уверенности ($\Omega \to 0$): $\mu_{BL} \to$ взгляды инвестора
- Параметр $\tau$ регулирует относительное доверие к рынку vs истории; обычно $\tau \in [0.01, 0.05]$
- BL даёт устойчивые недиверсифицированные веса только при ненулевых взглядах

## Реализация на Python

```python
import numpy as np
import pandas as pd
from scipy.linalg import solve

def market_implied_returns(w_mkt: np.ndarray, sigma: np.ndarray,
                           delta: float = 2.5) -> np.ndarray:
    """Равновесные ожидаемые доходности (обратная оптимизация)."""
    return delta * sigma @ w_mkt

def idzorek_omega(P: np.ndarray, sigma: np.ndarray,
                  confidences: np.ndarray) -> np.ndarray:
    """Матрица Omega из уровней доверия по методу Идзорека."""
    k = P.shape[0]
    omega = np.zeros((k, k))
    for i in range(k):
        p_i = P[i:i+1, :]
        variance = float(p_i @ sigma @ p_i.T)
        omega[i, i] = (1 - confidences[i]) / confidences[i] * variance
    return omega

def black_litterman(pi: np.ndarray, sigma: np.ndarray, P: np.ndarray,
                    Q: np.ndarray, omega: np.ndarray, tau: float = 0.025) -> dict:
    """
    Апостериорные доходности и ковариация по формуле Блэка-Литтермана.
    """
    tau_sigma = tau * sigma
    tau_sigma_inv = np.linalg.inv(tau_sigma)
    omega_inv = np.linalg.inv(omega)
    # Апостериорная ковариация
    m_inv = tau_sigma_inv + P.T @ omega_inv @ P
    sigma_bl_adj = np.linalg.inv(m_inv)
    # Апостериорная доходность
    mu_bl = sigma_bl_adj @ (tau_sigma_inv @ pi + P.T @ omega_inv @ Q)
    sigma_bl = sigma + sigma_bl_adj
    return {'mu': mu_bl, 'sigma': sigma_bl, 'sigma_adj': sigma_bl_adj}

def optimal_weights_bl(mu_bl: np.ndarray, sigma_bl: np.ndarray,
                       delta: float = 2.5) -> np.ndarray:
    """Оптимальные веса через обращение матрицы апостериорной ковариации."""
    raw = np.linalg.solve(delta * sigma_bl, mu_bl)
    return raw / raw.sum()

# Пример: 4 актива, 1 взгляд (актив 0 превзойдёт актив 1 на 2%)
n = 4
np.random.seed(42)
sigma = np.array([[0.04, 0.02, 0.01, 0.00],
                  [0.02, 0.09, 0.02, 0.01],
                  [0.01, 0.02, 0.06, 0.01],
                  [0.00, 0.01, 0.01, 0.03]])
w_mkt = np.array([0.40, 0.30, 0.20, 0.10])
pi = market_implied_returns(w_mkt, sigma, delta=2.5)
P = np.array([[1, -1, 0, 0]])
Q = np.array([0.02])
omega = idzorek_omega(P, sigma, np.array([0.65]))
bl = black_litterman(pi, sigma, P, Q, omega)
w_opt = optimal_weights_bl(bl['mu'], bl['sigma'])
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $\delta$ | Неприятие риска рынка | Коэффициент обратной оптимизации | $2.0$ – $3.5$ |
| $\tau$ | Масштаб неопределённости | Уверенность в априорном распределении | $0.01$ – $0.05$ |
| $P$ | Матрица выбора | Отображает взгляды на активы | $k \times n$ бинарная |
| $Q$ | Вектор взглядов | Ожидаемая доходность по взглядам | Любые вещественные |
| $\Omega$ | Матрица неопределённости | Ковариация ошибок взглядов | Диагональная, > 0 |
| $c_k$ | Уверенность во взгляде | Доверие Идзорека к $k$-му взгляду | $(0, 1)$ |

## Ограничения и расширения

**Ограничения:**
- Результат чувствителен к $\tau$ и $\Omega$ — параметрам, которые трудно откалибровать объективно.
- Гауссово априорное распределение неадекватно при тяжёлых хвостах и рыночных режимах.
- Матрица $\Sigma$ оценивается с ошибкой; неточная $\Sigma$ искажает равновесие $\Pi$.

**Расширения:**
- **[[hierarchical-risk-parity]]**: комбинирование BL-доходностей с HRP-ковариационной матрицей для повышения устойчивости.
- **[[mcmc]]**: байесовское сэмплирование апостериорного распределения весов без гауссового приближения.
- **Нечёткие взгляды**: взгляды выражаются как интервалы, а не точечные оценки.

## Практическое применение

1. **Долгосрочное распределение активов**: суверенные фонды используют BL для включения макро-взглядов при сохранении диверсификации.
2. **Систематические факторные стратегии**: факторные прогнозы альфа преобразуются в BL-взгляды для построения взвешенного портфеля.
3. **Риск-паритет с взглядами**: BL позволяет добавить направленные взгляды к нейтральному риск-паритетному портфелю.
4. **Мульти-менеджерные фонды**: каждый менеджер выражает взгляды, которые объединяются в общую аллокацию через BL.
5. **ESG-скрининг**: исключение активов через матрицу $P$ без полного пересмотра модели.

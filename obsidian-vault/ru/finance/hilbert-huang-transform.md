# Преобразование Гильберта-Хуанга (HHT)

**Преобразование Гильберта-Хуанга (HHT)** — это мощный метод анализа временных рядов, специально разработанный для работы с нелинейными и нестационарными данными. В отличие от преобразования Фурье, HHT позволяет определять мгновенную частоту и амплитуду сигнала, что делает его крайне эффективным для поиска скрытых циклов в рыночных ценах.

## Структура метода

HHT состоит из двух основных этапов:

1.  **Эмпирическая модовая декомпозиция (EMD)**: Разложение сложного сигнала на набор простых функций, называемых **Intrinsic Mode Functions (IMF)**. Каждая IMF представляет собой локальный цикл данных на определенном временном масштабе.
2.  **Спектральный анализ Гильберта**: Применение преобразования Гильберта к каждой извлеченной IMF для получения мгновенной частоты и амплитуды.

## В чем отличие от Фурье?

| Метод | Базисные функции | Применимость | Разрешение |
| :--- | :--- | :--- | :--- |
| **Фурье** | Синусы и косинусы | Линейные, стационарные | Глобальное |
| **Вейвлеты** | Материнский вейвлет | Нестационарные | Фиксированное (шкала) |
| **HHT** | Адаптивные (IMF) | **Нелинейные, нестационарные** | **Мгновенное (локальное)** |

## Применение в финансах

- **Обнаружение рыночных циклов**: Извлечение циклов разной длины (от внутридневных до многолетних) без шума.
- **Денойзинг (Фильтрация)**: Удаление высокочастотных IMF, которые часто соответствуют рыночному шуму, для получения чистого тренда.
- **Анализ волатильности**: Изучение того, как амплитуда циклов меняется со временем.

## Математическая концепция (Преобразование Гильберта)

Для функции $x(t)$ преобразование Гильберта $\hat{x}(t)$ определяется как:

$$
\hat{x}(t) = \frac{1}{\pi} P \int_{-\infty}^{\infty} \frac{x(\tau)}{t - \tau} d\tau
$$

Это позволяет построить аналитический сигнал $z(t) = x(t) + i\hat{x}(t) = a(t)e^{i\theta(t)}$, где $a(t)$ — мгновенная амплитуда, а $\theta(t)$ — мгновенная фаза.

## Python: Концептуальный пример EMD

```python
# Требуется библиотека PyEMD
from PyEMD import EMD
import numpy as np

def decompose_series(prices):
    emd = EMD()
    # Извлечение IMF (Intrinsic Mode Functions)
    IMFs = emd(prices)
    return IMFs

# Пример: извлечение циклов из цен
prices = np.random.normal(0, 1, 100).cumsum()
imfs = decompose_series(prices)
print(f"Извлечено циклов (IMF): {len(imfs)}")
```

## Ограничения

- **Эффект перемешивания мод (Mode Mixing)**: Иногда разные циклы могут попадать в одну IMF.
- **Вычислительная сложность**: Алгоритм итеративен и требует больше ресурсов, чем FFT.

# Модель Блэка (1976)

**Модель Блэка** (Fischer Black, 1976) адаптирует формулу [[black-scholes]] для ценообразования опционов на **фьючерсы и форварды**. Замена спот-цены $S$ на форвардную цену $F$ устраняет необходимость моделировать дивиденды, стоимость хранения и convenience yield — всё это уже заложено в $F$. Модель является рыночным стандартом для каплетов, свопционов и товарных опционов.

## Формула Блэка

Для европейского колла на форвард со страйком $K$ и экспирацией $T$:

$$
C = e^{-rT}\bigl[F\,N(d_1) - K\,N(d_2)\bigr]
$$

$$
P = e^{-rT}\bigl[K\,N(-d_2) - F\,N(-d_1)\bigr]
$$

где:

$$
d_1 = \frac{\ln(F/K) + \frac{1}{2}\sigma^2 T}{\sigma\sqrt{T}}, \qquad d_2 = d_1 - \sigma\sqrt{T}
$$

По сравнению с [[black-scholes]]: спот $S$ заменён на $F$, а член дрифта $(r + \sigma^2/2)$ сводится к $\sigma^2/2$, поскольку форвардная цена — мартингал под $T$-форвардной мерой.

## Вывод под форвардной мерой

Под $T$-форвардной мерой $\mathbb{Q}^T$ (нумерер — нулькупонная облигация $P(t,T)$) форвардная цена $F(t,T) = S_t / P(t,T)$ является мартингалом (см. [[change-of-numeraire]]):

$$
\frac{dF}{F} = \sigma\,dW_t^T
$$

Нет дрифта — ключевое упрощение. При экспирации $F(T,T) = S_T$, форвард логнормален:

$$
\ln F_T \sim \mathcal{N}\!\left(\ln F_0 - \tfrac{1}{2}\sigma^2 T,\;\sigma^2 T\right)
$$

Цена колла: $P(0,T)\,\mathbb{E}^{\mathbb{Q}^T}[\max(F_T - K, 0)]$, что даёт формулу Блэка напрямую.

## Каплеты и кэпы

Процентный **каплет** выплачивает $\delta\,\max(L(T_i, T_{i+1}) - K, 0)$ в момент $T_{i+1}$, где $L$ — форвардная ставка LIBOR/SOFR. Под моделью Блэка форвардная ставка логнормальна:

$$
\text{Caplet} = \delta\,P(0, T_{i+1})\bigl[F_L\,N(d_1) - K\,N(d_2)\bigr]
$$

**Кэп** — это набор каплетов; цена равна сумме каплетных цен. Подробнее: [[caps-floors-swaptions]].

## Свопционы

**Свопцион плательщика** даёт право войти в своп с фиксированной ставкой $K$. Форвардная своп-ставка $S_0$ логнормальна:

$$
\text{Swaption} = A(0)\bigl[S_0\,N(d_1) - K\,N(d_2)\bigr]
$$

где $A(0) = \sum_{i=1}^n \delta_i P(0, T_i)$ — аннуитетный фактор.

## Нормальная (Башелье) модификация

При отрицательных ставках (реальность после 2014 года) логнормальное предположение нарушается. **Нормальная** модель: $dF = \sigma_n\,dW_t^T$:

$$
C = e^{-rT}\bigl[(F - K)\,N(d) + \sigma_n\sqrt{T}\,\varphi(d)\bigr], \quad d = \frac{F - K}{\sigma_n\sqrt{T}}
$$

Нормальная волатильность $\sigma_n$ котируется в базисных пунктах в год. См. [[bachelier-formula]].

## Сдвинутая логнормальная модель

Практический гибрид: формула Блэка применяется к сдвинутому форварду $(F + s)$ и сдвинутому страйку $(K + s)$, $s > 0$:

$$
C = e^{-rT}\bigl[(F+s)\,N(d_1) - (K+s)\,N(d_2)\bigr]
$$

Это позволяет работать с умеренно отрицательными ставками, сохраняя логнормальную машину. Сдвиг $s$ — рыночная конвенция (обычно $1\%$ или $3\%$).

## Конвенции котирования волатильности

| Продукт | Стандарт котирования | Модель |
|:--------|:---:|:---|
| Опционы на акции | Логнормальная (Black) | % волатильность |
| Кэпы/флоры | Нормальная или логнормальная | б.п./год или % |
| Свопционы | Нормальная (после 2014) | б.п./год |
| Товарные опционы | Логнормальная (Black) | % волатильность |

Приближение для пересчёта при ATM: $\sigma_N \approx \sigma_{\text{LN}} \cdot F$.

## Реализация на Python

```python
import numpy as np
from scipy.stats import norm

def black76(F: float, K: float, T: float, r: float,
            sigma: float, option_type: str = 'call') -> float:
    """Цена опциона по модели Блэка (1976)."""
    d1 = (np.log(F / K) + 0.5 * sigma**2 * T) / (sigma * np.sqrt(T))
    d2 = d1 - sigma * np.sqrt(T)
    df = np.exp(-r * T)
    if option_type == 'call':
        return df * (F * norm.cdf(d1) - K * norm.cdf(d2))
    return df * (K * norm.cdf(-d2) - F * norm.cdf(-d1))

def bachelier(F: float, K: float, T: float, r: float,
              sigma_n: float, option_type: str = 'call') -> float:
    """Нормальная (Башелье) модель для опционов на форварды."""
    d = (F - K) / (sigma_n * np.sqrt(T))
    df = np.exp(-r * T)
    sign = 1 if option_type == 'call' else -1
    return df * (sign * (F - K) * norm.cdf(sign * d)
                 + sigma_n * np.sqrt(T) * norm.pdf(d))

# Пример: каплет
caplet = 0.25 * 0.96 * (0.035 * norm.cdf(1.2) - 0.030 * norm.cdf(0.95))
```

## Применения

1. **Процентные деривативы**: каплеты, флорлеты, свопционы — [[caps-floors-swaptions]], [[swaps]].
2. **Товарные опционы**: опционы на нефть, газ, металлы — Блэк 76 без моделирования стоимости хранения.
3. **Энергетические рынки**: спарк-спреды и крэк-спреды оцениваются через Блэка.
4. **Котирование волатильности**: подразумеваемая волатильность Блэка — lingua franca рынка ставок.
5. **Калибровка**: Блэк-волы — входные данные для продвинутых моделей ([[heston-model]], [[sabr-model]], [[volatility-surface]]).

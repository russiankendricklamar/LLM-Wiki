# Поиск и майнинг альфа-факторов

**Поиск альфа-факторов (Alpha Factor Discovery)** — это систематический процесс нахождения новых предсказательных сигналов (альф), способных генерировать избыточную доходность относительно бенчмарка. В современном количественном анализе это часто называют «майнингом альф» — использованием генетических алгоритмов, обучения с подкреплением или символьной регрессии для исследования пространства математических операторов.

## Практическое руководство по внедрению

Процесс поиска обычно включает следующие этапы:
1.  **Проектирование признаков (Feature Engineering)**: Применение операторов типа `Moving Average`, `Standard Deviation`, `Rank` и `Z-Score` к данным о ценах, объемах и фундаментальным показателям.
2.  **Генетическое программирование (GP)**: Использование GP для эволюции математических выражений, представляющих альфа-факторы. Выражения вида `rank(delta(close, 5)) / stddev(volume, 20)` тестируются на предсказательную силу.
3.  **Ортогонализация**: Проверка того, что новый фактор не дублирует уже существующие (Value, Momentum, Size).
4.  **Бэктестинг и Walk-Forward**: Валидация фактора на данных вне выборки (out-of-sample) для предотвращения переобучения.

## Математическая формулировка

Альфа-фактор $f_t$ является функцией исторических данных $D_{t-n:t}$:
$$
f_t = \Phi(D_{t-n:t})
$$
Цель — максимизировать Информационный коэффициент (Information Coefficient, IC), определяемый как корреляция между фактором и будущей доходностью $R_{t+1}$:
$$
IC = \rho(f_t, R_{t+1})
$$
Для обеспечения робастности можно использовать методы [[mcmc]] для сэмплирования из распределения параметров потенциальных факторов.

## Ключевые метрики
- **Information Coefficient (IC)**: Ранговая корреляция между значениями фактора и будущей доходностью.
- **IC IR (Information Ratio)**: Среднее значение IC, деленное на его стандартное отклонение (мера стабильности).
- **Оборачиваемость (Turnover)**: Скорость изменения рангов фактора, влияющая на транзакционные издержки.
- **Затухание (Decay)**: Скорость снижения предсказательной силы фактора со временем.

## Python: Генетическое выражение для альфы

```python
import pandas as pd
import numpy as np
from scipy.stats import spearmanr

def alpha_factor_mining(prices):
    # Пример найденного выражения: rank(delta(log(close), 5))
    log_return = np.log(prices).diff(5)
    factor = log_return.rank(pct=True)
    return factor

# Генерация тестовых данных
np.random.seed(42)
dates = pd.date_range("2020-01-01", periods=100)
prices = pd.Series(np.cumsum(np.random.normal(0, 1, 100)) + 100, index=dates)
returns = prices.pct_change().shift(-1) # Будущая доходность

factor = alpha_factor_mining(prices)
ic, _ = spearmanr(factor.dropna(), returns.dropna())
print(f"IC Фактора: {ic:.4f}")
```

## Финансовый контекст

Поиск альф эволюционировал от ручной проверки гипотез к высокопроизводительному автоматизированному майнингу. Это привело к проблеме «Зоопарка факторов», где многие «открытые» факторы являются просто шумом. Проверки на устойчивость часто включают симуляции [[monte-carlo-method]] для оценки вероятности нахождения такого IC случайным образом.

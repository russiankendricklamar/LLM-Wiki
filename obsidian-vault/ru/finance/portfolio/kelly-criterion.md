# Критерий Келли

**Критерий Келли** — формула определения оптимальной доли капитала для ставки или инвестиции с целью максимизации долгосрочного геометрического темпа роста благосостояния. Выведен Джоном Л. Келли-младшим в Bell Labs в 1956 году, он укоренён в теории информации и устанавливает математически доказуемую верхнюю границу устойчивого кредитного плеча. В количественных финансах — ключевой инструмент размещения капитала, определения размера позиций и управления рисками.

## Предпосылки

Келли вывел формулу, анализируя связь между скоростью передачи информации и оптимальным размером ставки. Ключевое открытие: максимизация ожидаемого логарифма благосостояния эквивалентна максимизации геометрического темпа роста — а эта цель, в отличие от максимизации арифметического ожидаемого дохода, естественным образом штрафует за риск разорения. Ставка выше доли Келли даёт ту же ожидаемую логарифмическую скорость роста, что и меньшая ставка, но с более высокой дисперсией и положительной вероятностью итогового разорения.

## Максимизация логарифма капитала

Основная цель — максимизировать ожидаемый логарифм итогового капитала:

$$
\max_f \; \mathbb{E}[\ln W_n] = \mathbb{E}\!\left[\sum_{i=1}^n \ln(1 + f X_i)\right]
$$

где $f$ — доля вложенного капитала, $X_i$ — доходность $i$-й ставки. Это эквивалентно максимизации геометрического среднего, определяющего долгосрочное накопление капитала.

## Дискретный случай

Для ставки с двумя исходами:
- Выигрыш с вероятностью $p$, чистая прибыль $b$ на единицу ставки.
- Проигрыш с вероятностью $q = 1 - p$, потеря всей ставки.

Максимизируем $\mathbb{E}[\ln W] = p\ln(1+fb) + q\ln(1-f)$ по $f$:

$$
\frac{d}{df}\mathbb{E}[\ln W] = \frac{pb}{1+fb} - \frac{q}{1-f} = 0
$$

Решение:

$$
f^* = \frac{bp - q}{b} = \frac{p(b+1) - 1}{b}
$$

Формула даёт положительное значение (ставка оправдана) только когда $\mathbb{E}[X] = pb - q > 0$ — то есть ставка имеет положительное математическое ожидание.

## Непрерывный случай

Для рискового актива, следующего геометрическому броуновскому движению с дрейфом $\mu$, волатильностью $\sigma$ и безрисковой ставкой $r$:

$$
f^* = \frac{\mu - r}{\sigma^2}
$$

Это **коэффициент Шарпа, делённый на $\sigma$**, или избыточная доходность на единицу дисперсии. Совпадает с оптимальным весом портфеля в непрерывной задаче Мертона с логарифмической функцией полезности.

Для портфеля из $n$ рискованных активов с вектором доходностей $\mu$ и матрицей ковариаций $\Sigma$:

$$
f^* = \Sigma^{-1}(\mu - r\mathbf{1})
$$

Это вектор весов касательного портфеля на эффективной границе — Келли и Марковиц совпадают при логарифмической функции полезности.

## Теоретико-полезностная основа

Критерий Келли соответствует **логарифмической функции полезности** $U(W) = \ln W$, что подразумевает постоянное относительное неприятие риска (CRRA) равное 1:

$$
\mathbb{E}[U(W)] = p\ln(1 + fb) + q\ln(1 - f)
$$

Инвесторы с CRRA $> 1$ (более склонные к избеганию риска) оптимально ставят меньше Келли; с CRRA $< 1$ — оптимально ставят больше, ценой более высокой вероятности разорения.

## Дробный Келли

На практике полный размер по Келли даёт высокую краткосрочную волатильность и серьёзные просадки при ошибках оценки $(\mu, \sigma)$. **Дробная стратегия Келли** ставит $c \times f^*$ при $c \in (0,1)$:

- **Половина Келли** ($c = 0.5$): ~75% от асимптотического темпа роста Келли при ~50% дисперсии.
- **Четверть Келли** ($c = 0.25$): ~44% темпа роста при ~25% дисперсии.

Темп роста масштабируется как $c(2-c)$ от полного темпа Келли, а дисперсия — как $c^2$, поэтому дробный Келли всегда более эффективен с точки зрения риска скорректированного роста.

## Анализ просадок

При стратегии Келли вероятность просадки величины $d$ (падение до доли $(1-d)$ от пика капитала) составляет:

$$
P(\text{просадка} \geq d) = (1-d)^{b/(b+1)} \quad \text{(бинарный случай)}
$$

Например, при $b=1$ (равные шансы) вероятность 50%-й просадки равна 50%. Именно поэтому практики предпочитают половину Келли, несмотря на меньший асимптотический темп роста.

## Python: Симуляция роста капитала

```python
import numpy as np
from typing import Optional

def kelly_fraction_binary(p: float, b: float) -> float:
    """Оптимальная доля Келли для бинарной ставки: выигрыш b с вер. p, проигрыш 1 с вер. 1-p."""
    return (b * p - (1 - p)) / b

def kelly_fraction_continuous(mu: float, r: float, sigma: float) -> float:
    """Доля Келли для актива GBM: (mu - r) / sigma^2."""
    return (mu - r) / sigma**2

def simulate_kelly_wealth(
    p: float,
    b: float,
    n_steps: int,
    fraction: float = 1.0,
    initial_wealth: float = 1.0,
    seed: Optional[int] = None,
) -> np.ndarray:
    """Симуляция траектории капитала при дробном критерии Келли."""
    rng = np.random.default_rng(seed)
    f_star = kelly_fraction_binary(p, b)
    bet = f_star * fraction

    wealth = np.empty(n_steps + 1)
    wealth[0] = initial_wealth

    for i in range(n_steps):
        w = wealth[i]
        outcome = b if rng.random() < p else -1.0
        wealth[i+1] = max(w * (1 + bet * outcome), 1e-12)  # поглощение у нуля

    return wealth

# --- Сравнение стратегий ---
p, b = 0.55, 1.0    # 55% вероятность выигрыша, шансы 1:1
f_star = kelly_fraction_binary(p, b)
print(f"Доля Келли f* = {f_star:.4f}  ({f_star*100:.1f}% капитала на ставку)")

n_steps = 500
seeds = range(10)

results = {}
for label, frac in [("Полный Келли", 1.0), ("Половина Келли", 0.5), ("Превышение 1.5x", 1.5)]:
    terminal = [simulate_kelly_wealth(p, b, n_steps, frac, seed=s)[-1] for s in seeds]
    results[label] = np.mean(terminal)
    print(f"{label:22s}: средний итоговый капитал = {results[label]:.2f}")
```

## Таблица параметров

|--------|----------|----------|-------------------|
| $f^*$ | Доля Келли | Оптимальная ставка как доля капитала | 0.01–0.30 |
| $p$ | Вер. выигрыша | Вероятность благоприятного исхода | 0.50–0.65 |
| $b$ | Коэффициент выплаты | Чистая прибыль на единицу ставки | 0.5–3 |
| $\mu - r$ | Избыточная доходность | Ожидаемая доходность сверх безрисковой | Годовая |
| $\sigma^2$ | Дисперсия | Дисперсия доходности (годовая) | 0.01–0.25 |
| $c$ | Доля Келли | Масштабирующий коэффициент дробного Келли | 0.25–0.75 |

## Ограничения и расширения

- **Риск ошибки оценки**: критерий Келли требует точных $\mu$ и $\sigma$; малые ошибки в $\mu$ дают большие ошибки в $f^*$. Доля Келли очень чувствительна к переоценке преимущества.
- **Нестационарность**: если распределение ставок меняется во времени, фиксированная доля Келли субоптимальна; адаптивный Келли обновляет $f^*$ по мере уточнения оценок.
- **Корреляции**: в многоактивных портфелях непрерывная формула Келли требует полной матрицы ковариаций; игнорирование корреляций может привести к совокупной избыточной ставке.
- **Дискретное vs. непрерывное время**: непрерывная формула предполагает GBM; для дискретных доходностей с тяжёлыми хвостами рекомендуется более консервативная доля.
- **Практические ограничения**: транзакционные издержки, минимальный размер позиции, регуляторные требования к капиталу и ограничения ликвидности снижают достижимую долю ниже $f^*$.
- **Пропорциональный Келли с контролем просадки**: сочетание размера по Келли с правилом стоп-лосса — снижение коэффициента $c$ пропорционально по мере приближения портфеля к максимально допустимому уровню просадки.

## Практическое применение

1. **Определение размера позиции в алготрейдинге**: размер позиции каждой стратегии как доля оптимальной аллокации по Келли от преимущества стратегии с поправкой на неопределённость оценки.
2. **Спортивные ставки и предсказательные рынки**: исходное применение; Келли — доказуемо оптимальная стратегия долгосрочного роста для любого игрока с преимуществом.
3. **Аллокация в венчурном капитале**: распределение между портфельными компаниями по весам Келли, рассчитанным из оценок вероятности успеха и мультипликаторов выплат.
4. **Управление кредитным плечом**: непрерывная доля Келли задаёт максимально устойчивое кредитное плечо для рискового актива; превышение его ведёт к неизбежному разорению с вероятностью 1.

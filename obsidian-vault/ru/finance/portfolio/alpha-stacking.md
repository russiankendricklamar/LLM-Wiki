# Ортогональное стекирование альф и резидуализация

**Ортогональное стекирование альф (Orthogonal Alpha Stacking)** — метод комбинирования нескольких предсказательных сигналов (альф) таким образом, чтобы каждый новый сигнал приносил уникальную информацию. Если два сигнала сильно коррелируют, их простое сложение не улучшает качество прогноза; резидуализация гарантирует уникальность каждого сигнала.

## Обзор и предпосылки

По мере открытия новых факторов их наивная комбинация — с равными весами или взвешенная по корреляции — приводит к двойному учёту: портфель непропорционально концентрируется в доминирующем факторе риска, встроенном во все коррелирующие сигналы. Классический пример: сигналы импульса прибыли, ценового импульса и пересмотров аналитиков все частично отражают один и тот же эффект (устойчивость сюрпризов прибыли). Без резидуализации их объединение — это, по сути, трёхкратная ставка на импульс.

Математическая база опирается на процедуру ортогонализации Грама-Шмидта из линейной алгебры и работы по мультиколлинеарности в регрессии. В контексте факторного инвестирования модели рисков Barra (MSCI Barra) формализовали это в «чистые факторные портфели», каждый из которых ортогонален остальным.

## Математическая база

### Резидуализация через регрессию

Чтобы сделать сигнал $Y$ (Альфа 2) ортогональным сигналу $X$ (Альфа 1):

$$
Y = \beta X + \epsilon
$$

Оценка МНК $\hat{\beta} = \frac{\text{Cov}(X, Y)}{\text{Var}(X)}$ даёт проекцию $Y$ на $X$. Остаток:

$$
\epsilon = Y - \hat{\beta} X
$$

— часть Альфы 2, некоррелированная с Альфой 1. Скомбинированная альфа:

$$
\alpha_{stacked} = w_1 A_1 + w_2 \epsilon
$$

### Многосигнальный процесс Грама-Шмидта

Для $n$ сигналов процедура Грама-Шмидта строит ортонормальный базис $\{e_1, \dots, e_n\}$:

$$
e_k = \frac{A_k - \sum_{j=1}^{k-1} \langle A_k, e_j \rangle e_j}{\left\| A_k - \sum_{j=1}^{k-1} \langle A_k, e_j \rangle e_j \right\|}
$$

где $\langle \cdot, \cdot \rangle$ — поперечное произведение по вселенной активов в момент $t$.

### Нейтрализация по факторам

Резидуализация часто применяется для устранения влияния системных факторов риска (рынок, сектора). Если $F \in \mathbb{R}^{n \times k}$ — матрица из $k$ факторных экспозиций для $n$ активов:

$$
\alpha_{clean} = \alpha - F(F^T F)^{-1} F^T \alpha = (I - P_F) \alpha
$$

где $P_F = F(F^T F)^{-1} F^T$ — матрица проектора на столбцовое пространство $F$. Результат $(I - P_F)\alpha$ лежит в нуль-пространстве $F^T$ — он одновременно ортогонален всем $k$ факторам.

### Взвешенная комбинация

При $n$ ортогонализированных сигналах оптимальная взвешенная комбинация:

$$
\alpha_{combined} = \sum_{i=1}^n \frac{IC_i}{\sigma_{IC_i}^2} \tilde{A}_i
$$

Это **взвешивание по ICIR**: каждый сигнал получает вес, пропорциональный его информационному отношению. Сигналы с высоким IC, но нестабильным ICIR получают меньший вес.

## Ключевые свойства и допущения

- **Зависимость от порядка в Грама-Шмидте**: результат ортогонализации зависит от порядка обработки сигналов. Рекомендуется упорядочивать по убыванию ICIR.
- **Поперечное сечение vs временной ряд**: резидуализация должна применяться поперечно (по активам в каждый момент времени), а не по временной оси.
- **Нестационарность**: если корреляции факторов меняются со временем, статическая резидуализация устаревает. Скользящее окно МНК или фильтр Калмана повышают устойчивость.
- **Регуляризация**: при многих факторах $(F^T F)$ может быть почти вырожденной. Гребневая регрессия с штрафом $\lambda I$ решает эту проблему.

## Реализация на Python

```python
import numpy as np
import pandas as pd
from sklearn.linear_model import LinearRegression, Ridge

def residualize_signal(
    target_alpha: np.ndarray,
    basis_alphas: np.ndarray,
    method: str = 'ols',
    ridge_alpha: float = 1e-4
) -> np.ndarray:
    """
    Возвращает компонент target_alpha, ортогональный basis_alphas.
    """
    if method == 'ridge':
        model = Ridge(alpha=ridge_alpha, fit_intercept=True)
    else:
        model = LinearRegression(fit_intercept=True)
    model.fit(basis_alphas, target_alpha)
    return target_alpha - model.predict(basis_alphas)

def gram_schmidt_stack(alphas: list) -> list:
    """Ортогонализация списка альфа-сигналов методом Грама-Шмидта."""
    orthogonal = [alphas[0] / (np.std(alphas[0]) + 1e-10)]
    for alpha in alphas[1:]:
        for prev in orthogonal:
            alpha = alpha - np.dot(alpha, prev) / np.dot(prev, prev) * prev
        orthogonal.append(alpha / (np.std(alpha) + 1e-10))
    return orthogonal

# Пример: два коррелирующих сигнала импульса
np.random.seed(42)
n = 200
alpha1 = np.random.normal(0, 1, n)
alpha2 = 0.8 * alpha1 + 0.2 * np.random.normal(0, 1, n)  # 80% общего

unique_alpha2 = residualize_signal(alpha2, alpha1.reshape(-1, 1))
corr_before = np.corrcoef(alpha1, alpha2)[0, 1]
corr_after = np.corrcoef(alpha1, unique_alpha2)[0, 1]
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $P_F$ | Матрица проектора | Проектор на пространство факторных экспозиций | — |
| $k$ | Число базисных факторов | Факторы риска для нейтрализации | 5 – 100 |
| $IC_i$ | Информационный коэффициент | Ранговая корреляция сигнала $i$ с доходностью | –0.10 до 0.10 |
| $ICIR_i$ | IC Information Ratio | Мера стабильности сигнала $i$ | > 0.5 для включения |
| $\lambda$ | Гребневой штраф | Регуляризация при вырожденной $(F^TF)$ | $10^{-4}$ до $10^{-2}$ |

## Ограничения и расширения

**Ограничения:**
- Резидуализация удаляет коррелированную дисперсию, которая может содержать истинный сигнал: если два фактора предсказывают доходность через разные механизмы, резидуализация одного к другому уничтожает часть предсказательного содержания.
- Линейная резидуализация не устраняет нелинейные зависимости.
- Нейтрализация по факторам предполагает правильно специфицированную модель рисков; устаревшие ковариационные матрицы приводят к неполной нейтрализации.

**Расширения:**
- **[[random-matrix-theory]]**: фильтрация ковариационной матрицы сигналов методами теории случайных матриц.
- **Динамическая резидуализация**: фильтр Калмана для отслеживания меняющихся во времени корреляций сигналов.
- **Нелинейная нейтрализация**: нейросетевая факторная нейтрализация.

## Практическое применение

1. **Многофакторные модели**: объединение импульса, стоимости и качества без двойного учёта.
2. **Проп-трейдинг**: проверка того, что новая стратегия добавляет альфу, а не дублирует существующий портфель.
3. **Управление рисками**: разграничение навыков (альфа) и скрытой бета-экспозиции.
4. **Регуляторная отчётность**: чистые факторные портфели обеспечивают точную атрибуцию по каждому фактору риска.

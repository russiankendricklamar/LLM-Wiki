# Модель Хестона

**Модель Хестона** (Стивен Хестон, 1993) — наиболее широко используемая модель стохастической волатильности в количественных финансах. Она расширяет Блэка-Шоулза, позволяя мгновенной дисперсии $v_t$ следовать возвращающемуся к среднему процессу CIR, коррелированному с ценой актива. Это воспроизводит **улыбку и скос подразумеваемой волатильности**, наблюдаемые на рынках опционов.

## Необходимые знания

[[black-scholes]], [[geometric-brownian-motion]]

## Предпосылки

Блэк-Шоулз предполагает постоянную волатильность, что соответствует плоской поверхности подразумеваемой волатильности. Эмпирически опционы на один и тот же базовый актив с одним сроком, но разными страйками торгуются при разных подразумеваемых волатильностях — «улыбка» (симметрична) или «скос» (асимметрична, характерен для индексов акций). Хестон (1993) решил эту проблему, сделав волатильность стохастической и выведя полуаналитическую формулу ценообразования через характеристическую функцию.

## Динамика (СДУ)

Под риск-нейтральной мерой $\mathbb{Q}$ цена актива $S_t$ и дисперсия $v_t$ удовлетворяют:

$$
\begin{aligned}
dS_t &= r S_t\, dt + \sqrt{v_t}\, S_t\, dW_t^S \\
dv_t &= \kappa(\theta - v_t)\, dt + \xi\sqrt{v_t}\, dW_t^v
\end{aligned}
$$

С корреляцией $d[W^S, W^v]_t = \rho\, dt$.

### Параметры

| Символ | Название | Роль | Типичный диапазон |
|--------|----------|------|-------------------|
| $r$ | Безрисковая ставка | Дрейф под $\mathbb{Q}$ | 0–0.06 |
| $\kappa$ | Скорость возврата | Скорость возврата $v_t$ к $\theta$ | 0.5–5 |
| $\theta$ | Долгосрочная дисперсия | Уровень, к которому возвращается $v_t$ | 0.01–0.09 |
| $\xi$ | Вол-оф-вол | Волатильность процесса дисперсии | 0.1–1.0 |
| $\rho$ | Корреляция | Корреляция броуновских движений $S$ и $v$ | $-0.9$ до $0$ (акции) |
| $v_0$ | Начальная дисперсия | Стартовая дисперсия | Близко к ATM $\sigma^2$ |

## Условие Феллера

Чтобы $v_t > 0$ п.н. (дисперсия не достигает нуля):

$$
2\kappa\theta > \xi^2
$$

Если условие нарушено, процесс CIR может достигать нуля, оставаясь неотрицательным. На практике калибровки рынка акций часто немного нарушают условие Феллера; для симуляции используется схема Эйлера с полным усечением.

## Полуаналитическое ценообразование через характеристическую функцию

Ключевая идея Хестона — **характеристическая функция** $\ln S_T$ под $\mathbb{Q}$ имеет замкнутую форму. Для европейского колла $C(S_0, K, T)$:

$$
C = S_0 e^{-qT} P_1 - K e^{-rT} P_2
$$

где $q$ — дивидендная доходность, а вероятности $P_{1,2}$ восстанавливаются через инверсионную формулу Гила-Пелаеза:

$$
P_j = \frac{1}{2} + \frac{1}{\pi} \int_0^\infty \text{Re}\!\left[\frac{e^{-i\phi \ln K} f_j(\phi)}{i\phi}\right] d\phi
$$

Характеристические функции $f_1, f_2$:

$$
f_j(\phi) = \exp\!\left[C_j(\phi, T) + D_j(\phi, T) v_0 + i\phi \ln S_0\right]
$$

## Метод Карра-Мадана (FFT-ценообразование)

**Метод Карра-Мадана** (1999) оценивает стрип европейских коллов по всем страйкам одновременно с помощью быстрого преобразования Фурье, снижая сложность с $O(N_K)$ инверсий до $O(N \log N)$. Модифицированная цена колла $z_T(k) = e^{\alpha k} C(k)$ имеет преобразование Фурье, выражаемое через характеристическую функцию:

$$
\hat{z}_T(\phi) = \frac{e^{-rT} f(\phi - (\alpha+1)i)}{\alpha^2 + \alpha - \phi^2 + i(2\alpha+1)\phi}
$$

где $\alpha > 0$ — параметр затухания, $k = \ln(K/S_0)$ — логарифм монейнесса.

## Python: Симуляция траекторий и ценообразование

```python
import numpy as np
from scipy.integrate import quad

def simulate_heston(
    S0: float, v0: float, T: float, r: float,
    kappa: float, theta: float, xi: float, rho: float,
    steps: int, n_paths: int = 1, seed: int = 42
) -> tuple:
    """Дискретизация Эйлера-Маруямы с полным усечением дисперсии."""
    rng = np.random.default_rng(seed)
    dt = T / steps
    S = np.full(n_paths, S0, dtype=float)
    v = np.full(n_paths, v0, dtype=float)

    for _ in range(steps):
        z1 = rng.standard_normal(n_paths)
        z2 = rho * z1 + np.sqrt(1 - rho**2) * rng.standard_normal(n_paths)
        v_pos = np.maximum(v, 0)
        v = (v + kappa * (theta - v_pos) * dt
             + xi * np.sqrt(v_pos * dt) * z2)
        v = np.maximum(v, 0)
        S = S * np.exp((r - 0.5 * v_pos) * dt + np.sqrt(v_pos * dt) * z1)

    return S, v


def heston_call_cf(
    S0: float, K: float, T: float, r: float,
    kappa: float, theta: float, xi: float, rho: float, v0: float
) -> float:
    """Полуаналитический европейский колл (Хестон 1993)."""
    def cf(phi: complex, j: int) -> complex:
        u = 0.5 if j == 1 else -0.5
        b = kappa - rho * xi if j == 1 else kappa
        d = np.sqrt((rho * xi * phi * 1j - b) ** 2
                    - xi**2 * (2 * u * phi * 1j - phi**2))
        g = (b - rho * xi * phi * 1j + d) / (b - rho * xi * phi * 1j - d)
        C = (r * phi * 1j * T
             + kappa * theta / xi**2
             * ((b - rho * xi * phi * 1j + d) * T
                - 2 * np.log((1 - g * np.exp(d * T)) / (1 - g))))
        D = ((b - rho * xi * phi * 1j + d) / xi**2
             * (1 - np.exp(d * T)) / (1 - g * np.exp(d * T)))
        return np.exp(C + D * v0 + 1j * phi * np.log(S0))

    def integrand(phi: float, j: int) -> float:
        val = cf(phi, j) * np.exp(-1j * phi * np.log(K))
        return np.real(val / (1j * phi))

    P1 = 0.5 + (1 / np.pi) * quad(integrand, 1e-8, 200, args=(1,))[0]
    P2 = 0.5 + (1 / np.pi) * quad(integrand, 1e-8, 200, args=(2,))[0]
    return S0 * P1 - K * np.exp(-r * T) * P2


price = heston_call_cf(
    S0=100, K=100, T=1.0, r=0.05,
    kappa=2.0, theta=0.04, xi=0.3, rho=-0.7, v0=0.04
)
print(f"Цена колла по Хестону: {price:.4f}")
```

## Процедура калибровки

Калибровка подбирает $(\kappa, \theta, \xi, \rho, v_0)$ по рыночной поверхности подразумеваемой волатильности:

1. Собираем рыночные цены коллов/путов по страйкам $K_i$ и срокам $T_j$.
2. Конвертируем в рыночные подразумеваемые волатильности $\sigma^{\text{mkt}}_{ij}$.
3. Минимизируем взвешенный RMSE:
$$\min_{\kappa,\theta,\xi,\rho,v_0} \sum_{i,j} w_{ij} \left(\sigma^{\text{Heston}}_{ij} - \sigma^{\text{mkt}}_{ij}\right)^2$$
4. Глобальный поиск: `scipy.optimize.differential_evolution`; локальная доводка: `L-BFGS-B`.
5. Проверки: условие Феллера, $v_0 > 0$, $\xi > 0$, $|\rho| < 1$.

## Сравнение с Блэком-Шоулзом

| Характеристика | Блэк-Шоулз | Модель Хестона |
|:----------------|:-----------|:---------------|
| **Волатильность** | Постоянная | Стохастическая (CIR с возвратом к среднему) |
| **Распределение актива** | Логнормальное | Нелогнормальное (тяжёлые хвосты, скос) |
| **Улыбка волатильности** | Плоская | Воспроизводится через $\rho$ и $\xi$ |
| **Ценообразование** | Замкнутая форма | Полуаналитическое (характ. функция) |
| **Калибровка** | 1 параметр ($\sigma$) | 5 параметров |

## Ограничения и расширения

- **Грубая волатильность**: эмпирически показатель Херста волатильности $H \approx 0.1$ (Gatheral et al. 2018). Модель «грубого Хестона» заменяет CIR процессом дробного броуновского движения.
- **Постоянная корреляция**: $\rho$ фиксирована; в реальности корреляция стохастична.
- **Нестабильность калибровки**: 5-параметрическая поверхность сложно калибруется; необходимы регуляризация и глобальные оптимизаторы.
- **Расширение с прыжками**: модель Бейтса (1996) добавляет компоненту прыжков к Хестону для лучшей подгонки краткосрочного скоса.

## Практическое применение

1. **Экзотические опционы**: барьерные, азиатские и клике-опционы требуют симуляции траекторий; траектории Хестона воспроизводят более реалистичную динамику волатильности, чем GBM.
2. **Интерполяция поверхности волатильности**: Хестон даёт параметрическую арбитражно-свободную интерполяцию по страйкам и срокам.
3. **Риск-менеджмент**: греки (дельта, вега, волга, ванна) отражают экспозицию к стохастической волатильности, обеспечивая более точное хеджирование.

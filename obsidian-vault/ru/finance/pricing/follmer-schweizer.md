# Квадратичное хеджирование и подход Фёллмера-Швейцера

**Квадратичное хеджирование** — семейство методов ценообразования и хеджирования в неполных рынках, минимизирующих среднеквадратичный риск. Подход Фёллмера-Швейцера (Föllmer-Schweizer, 1989–1991) определяет **локально минимизирующий риск** портфель и связанную с ним **минимальную мартингальную меру** — алтернативу нейтральной мере Блэка-Шоулза в мирах с несовершенным хеджированием.

## Мотивация: неполные рынки

На рынке Блэка-Шоулза любой дериватив воспроизводится точно. В реальности:
- **Стохастическая волатильность** (модель Хестона) — нет ликвидного рынка волатильности
- **Прыжки** — невозможно мгновенно захеджировать скачок
- **Дискретная торговля** — непрерывное дельта-хеджирование неосуществимо

В неполных рынках **выбор меры не единственен**: существует континуум эквивалентных мартингальных мер (EMM). Разные меры дают разные «справедливые» цены.

## Локальная минимизация риска (Schweizer, 1991)

Стратегия $(\theta_t, \eta_t)$: $\theta_t$ — доля в рисковом активе, $\eta_t$ — в облигации.

**Стоимость** портфеля:

$$
V_t = \theta_t S_t + \eta_t B_t
$$

**Стоимость хеджирования (cost process)**:

$$
C_t = V_t - \int_0^t \theta_s \, dS_s
$$

Стратегия **локально минимизирует риск**, если $C_t$ — мартингал и $\theta_t$ минимизирует мгновенный остаточный риск:

$$
\min_{\theta} E\!\left[\left(dC_t\right)^2 \mid \mathcal{F}_t\right]
$$

## Фёллмер-Швейцерово разложение

При слабых условиях на $S_t$ любой квадратично-интегрируемый выплата $H$ разлагается:

$$
H = H_0 + \int_0^T \xi_t^H \, dS_t + L_T^H
$$

где:
- $H_0 = E^{\tilde{P}}[H]$ — начальная стоимость под минимальной мартингальной мерой $\tilde{P}$
- $\xi_t^H$ — оптимальная стратегия хеджирования (прогнозируемая)
- $L_T^H$ — **хеджируемый остаток** (orthogonal to $S$: $\langle L^H, S \rangle = 0$)

## Минимальная мартингальная мера (MMM)

**Мотивация**: среди всех EMM $Q$ выбираем ту, которая минимально «искажает» вероятностную структуру, связанную с $S$.

В модели $dS_t = \mu_t S_t \, dt + \sigma_t S_t \, dW_t$:

MMM $\tilde{P}$ определяется **плотностью Радона-Никодима**:

$$
\frac{d\tilde{P}}{dP}\bigg|_{\mathcal{F}_T} = \exp\!\left(-\int_0^T \lambda_t \, dW_t - \frac{1}{2}\int_0^T \lambda_t^2 \, dt\right)
$$

где $\lambda_t = \mu_t / \sigma_t$ — **цена рыночного риска (market price of risk)**.

**Ключевое свойство**: под $\tilde{P}$ процесс $S_t / B_t$ — мартингал, а ортогональные компоненты $L^H$ остаются мартингалами. Никакой дополнительной «искусственной» меры не вносится.

## Среднеквадратичное хеджирование (mean-variance hedging)

Альтернативная постановка: найти начальную стоимость $V_0$ и стратегию $\theta$, минимизирующие:

$$
\min_{V_0, \theta} E\!\left[\left(H - V_0 - \int_0^T \theta_t \, dS_t\right)^2\right]
$$

Решение связано с **меняющейся мерой Варианс-Оптимального Хеджирования** (variance-optimal hedging measure) $\hat{P}$ — другим кандидатом среди EMM.

### Дискретное приближение

```python
import numpy as np
from scipy.linalg import solve

def local_risk_minimizing_hedge(S: np.ndarray,
                                 payoff: np.ndarray,
                                 sigma_path: np.ndarray,
                                 mu_path: np.ndarray,
                                 rf: float = 0.0) -> np.ndarray:
    """
    Дискретное приближение к локально минимизирующей стратегии.

    S: цены актива (T+1,)
    payoff: выплата в каждый момент T
    sigma_path: волатильность (T,)
    mu_path: дрейф (T,)
    rf: безрисковая ставка

    Возвращает: theta (T,) — оптимальная доля
    """
    T = len(S) - 1
    dt = 1.0 / T

    # Дельта-хедж под мерой Фёллмера-Швейцера
    # theta_t = Cov(H, dS_t) / Var(dS_t)
    theta = np.zeros(T)

    for t in range(T-1, -1, -1):
        dS = S[t+1] - S[t] * (1 + rf * dt)

        # Мгновенный риск: Var(dS) ~ sigma^2 * S^2 * dt
        var_dS = sigma_path[t]**2 * S[t]**2 * dt

        # В непрерывном пределе theta = d(payoff_value) / d(S)
        # Численно: конечная разность опциона по S
        eps = S[t] * 0.01
        payoff_up = max(S[t] + eps - payoff.mean(), 0)   # упрощение
        payoff_down = max(S[t] - eps - payoff.mean(), 0)
        d_payoff_dS = (payoff_up - payoff_down) / (2 * eps)

        theta[t] = d_payoff_dS

    return theta


def mean_variance_hedge_discrete(
    S_paths: np.ndarray,
    payoffs: np.ndarray
) -> tuple[float, np.ndarray]:
    """
    Среднеквадратичное хеджирование методом наименьших квадратов.

    S_paths: (n_sims, T+1) матрица сценариев цен
    payoffs: (n_sims,) выплаты

    Возвращает: (V0_optimal, theta: (n_sims, T))
    """
    n_sims, T_plus_1 = S_paths.shape
    T = T_plus_1 - 1

    # dS для каждого сценария
    dS = np.diff(S_paths, axis=1)   # (n_sims, T)

    # Задача: min_{V0, theta} E[(H - V0 - sum theta_t * dS_t)^2]
    # Упрощение: theta предполагается константой для каждого шага
    thetas = np.zeros(T)
    V0 = 0.0

    for t in range(T):
        X = np.column_stack([np.ones(n_sims), dS[:, t]])
        y = payoffs if t == T-1 else dS[:, t]
        coeffs, _, _, _ = np.linalg.lstsq(X, y, rcond=None)
        thetas[t] = coeffs[1]

    V0 = np.mean(payoffs) - np.sum(thetas * dS.mean(axis=0))
    return V0, thetas
```

## Сравнение подходов к неполным рынкам

| Подход | Критерий | Мера | Применение |
|--------|---------|------|-----------|
| Нейтральная мера (BS) | Нет риска при точном воспроизведении | $Q^*$ (уникальна в BS) | Полные рынки |
| Локальная min-variance (Föllmer-Schweizer) | Min мгновенный остаточный риск | MMM $\tilde{P}$ | SV, прыжки |
| Mean-variance hedge | Min глобальный $E[(H - \hat{H})^2]$ | Variance-optimal $\hat{P}$ | Дискретная торговля |
| Superhedging | Гарантированное покрытие | Нейтральная (sup) | Консервативный |
| Utility indifference | Макс EU | Зависит от U | Неполные рынки + риск |

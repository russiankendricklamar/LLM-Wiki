# Процесс Кокса (Cox Process)

**Процесс Кокса**, или **дважды стохастический пуассоновский процесс**, введённый Дэвидом Коксом (1955) — обобщение пуассоновского процесса, в котором сама интенсивность событий является стохастическим процессом. Это позволяет моделировать кластеризацию событий и сверхдисперсию, наблюдаемые в потоках дефолтов, торговых ордеров и страховых случаев.

## Обзор и предпосылки

В стандартном процессе Пуассона интенсивность $\lambda$ постоянна или детерминирована. Реальные финансовые события — дефолты, высокочастотные сделки, наступление страховых случаев — демонстрируют **сверхдисперсию**: дисперсия счётчика событий превышает его среднее. Процесс Кокса объясняет это через случайность самой интенсивности.

Модели интенсивности (Jarrow-Turnbull, Duffie-Singleton) стали стандартом кредитного риска после работ 1990-х. CIR-интенсивность (Cox-Ingersoll-Ross) особенно популярна: она гарантирует положительность и обладает аффинной структурой, дающей аналитические формулы для выживаемости.

## Математическая база

**Определение**. Процесс $N(t)$ — процесс Кокса с интенсивностью $\lambda(t)$, если условно при фиксированной траектории $(\lambda_s)_{s \geq 0}$ он является неоднородным пуассоновским:

$$
P(N_t - N_s = k \mid (\lambda_u)_{s \leq u \leq t}) = \frac{(\Lambda_{s,t})^k}{k!}e^{-\Lambda_{s,t}}, \quad \Lambda_{s,t} = \int_s^t \lambda_u\,du
$$

**Безусловные моменты** — ключевое свойство сверхдисперсии:

$$
\mathbb{E}[N(t)] = \mathbb{E}[\Lambda_{0,t}], \quad \mathrm{Var}[N(t)] = \mathbb{E}[\Lambda_{0,t}] + \mathrm{Var}[\Lambda_{0,t}] > \mathbb{E}[N(t)]
$$

**Динамика интенсивности CIR** (гарантирует $\lambda_t \geq 0$ при условии Феллера $2\kappa\theta \geq \sigma^2$):

$$
d\lambda_t = \kappa(\theta - \lambda_t)\,dt + \sigma\sqrt{\lambda_t}\,dW_t
$$

**Аффинная вероятность выживания** (Cox-процесс с CIR-интенсивностью):

$$
\mathbb{E}\!\left[e^{-\int_0^T \lambda_t\,dt}\right] = A(T)\,e^{-B(T)\lambda_0}
$$

$$
B(T) = \frac{2(e^{\gamma T}-1)}{(\gamma+\kappa)(e^{\gamma T}-1)+2\gamma}, \quad A(T) = \left(\frac{2\gamma e^{(\kappa+\gamma)T/2}}{(\gamma+\kappa)(e^{\gamma T}-1)+2\gamma}\right)^{2\kappa\theta/\sigma^2}
$$

где $\gamma = \sqrt{\kappa^2 + 2\sigma^2}$.

**RPV01 кредитно-дефолтного свопа** (рисковая аннуитетная стоимость):

$$
\text{RPV01} = \sum_{i=1}^n \Delta_i\cdot B(0,T_i)\cdot S(T_i)
$$

где $S(T) = \mathbb{E}[e^{-\int_0^T \lambda_t\,dt}]$ — вероятность выживания, $B(0,T)$ — дисконт-фактор.

## Ключевые свойства

- **Сверхдисперсия**: $\mathrm{Var}[N(t)] > \mathbb{E}[N(t)]$ — отличительная черта от пуассоновского процесса
- **Условная пуассоновская структура**: позволяет аналитически работать с каждой фиксированной траекторией
- **Кластеризация событий**: высокая интенсивность в одном периоде увеличивает вероятность последующих событий
- **Аффинная структура CIR**: даёт замкнутые формулы для цен дефолтных инструментов

## Реализация на Python

```python
import numpy as np
import pandas as pd

def simulate_cir(kappa: float, theta: float, sigma: float, lambda0: float,
                 T: float, steps: int, seed: int = 42) -> np.ndarray:
    """Симуляция процесса CIR методом Эйлера-Маруямы."""
    np.random.seed(seed)
    dt = T / steps
    lam = np.zeros(steps + 1)
    lam[0] = lambda0
    for t in range(steps):
        dW = np.random.normal(0, np.sqrt(dt))
        dlam = kappa * (theta - lam[t]) * dt + sigma * np.sqrt(max(lam[t], 0)) * dW
        lam[t+1] = max(lam[t] + dlam, 0)
    return lam

def cir_survival_probability(lambda0: float, kappa: float, theta: float,
                              sigma: float, T: float) -> float:
    """Аналитическая вероятность выживания для CIR-интенсивности."""
    gamma = np.sqrt(kappa**2 + 2 * sigma**2)
    exp_gT = np.exp(gamma * T)
    denom = (gamma + kappa) * (exp_gT - 1) + 2 * gamma
    B = 2 * (exp_gT - 1) / denom
    log_A = (2 * kappa * theta / sigma**2 *
             np.log(2 * gamma * np.exp((kappa + gamma) * T / 2) / denom))
    return float(np.exp(log_A - B * lambda0))

def cds_risky_pv01(lambda0: float, kappa: float, theta: float, sigma: float,
                   r: float, maturities: np.ndarray) -> float:
    """RPV01 кредитно-дефолтного свопа (рисковая аннуитетная стоимость)."""
    rpv01 = 0.0
    prev_t = 0.0
    for T in maturities:
        dt = T - prev_t
        disc = np.exp(-r * T)
        surv = cir_survival_probability(lambda0, kappa, theta, sigma, T)
        rpv01 += dt * disc * surv
        prev_t = T
    return rpv01

# Симуляция Cox-процесса
lam = simulate_cir(kappa=0.5, theta=0.05, sigma=0.1, lambda0=0.03, T=5.0, steps=1000)
events = np.random.poisson(lam[:-1] * (5.0 / 1000))
surv_1y = cir_survival_probability(0.03, 0.5, 0.05, 0.1, 1.0)
mats = np.array([0.25, 0.5, 1.0, 2.0, 3.0, 5.0])
rpv01 = cds_risky_pv01(0.03, 0.5, 0.05, 0.1, 0.04, mats)
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---:|:---|:---|
| $\lambda_0$ | Начальная интенсивность | Мгновенная вероятность дефолта | $0.001$ – $0.10$ |
| $\kappa$ | Скорость возврата | Скорость возврата к долгосрочному среднему | $0.1$ – $5.0$ |
| $\theta$ | Долгосрочное среднее | Безусловная средняя интенсивность | $0.01$ – $0.15$ |
| $\sigma$ | Волатильность интенсивности | Коэффициент диффузии CIR | $0.05$ – $0.50$ |
| $\gamma$ | Вспомогательный параметр | $\sqrt{\kappa^2 + 2\sigma^2}$ в аффинных формулах | — |

## Ограничения и расширения

**Ограничения:**
- CIR-интенсивность может нарушать условие Феллера ($2\kappa\theta < \sigma^2$) — риск обнуления интенсивности.
- Линейная диффузия $\sigma\sqrt{\lambda}$ ограничивает гибкость моделирования всплесков (jumps) интенсивности.
- Оценка параметров требует либо панельных данных по дефолтам, либо цен CDS для калибровки.

**Расширения:**
- **Процесс Хоука**: самовозбуждающаяся интенсивность — каждый дефолт увеличивает вероятность следующего.
- **Скачкообразный Cox**: добавление пуассоновских скачков в динамику $\lambda_t$ для моделирования кредитных кризисов.
- **Многомерный Cox**: корреляция интенсивностей для корзин дефолтов через копулу или общий фактор.

## Практическое применение

1. **Ценообразование CDS**: RPV01 и защитная нога CDS через вероятности выживания CIR.
2. **Бутстрап кривой дефолта**: калибровка $(\kappa, \theta, \sigma, \lambda_0)$ по котировкам CDS на разных горизонтах.
3. **CDO**: корреляция дефолтов через совместный CIR-фактор для всего портфеля.
4. **HFT-моделирование потока ордеров**: динамическая интенсивность заявок в модели Авелланеды-Стойкова.
5. **Страхование**: катастрофические риски с нестационарной интенсивностью (климатические события).

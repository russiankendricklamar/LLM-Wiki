# Равновесие Нэша

**Равновесие Нэша (Nash Equilibrium)** — это концепция в теории игр, при которой ни один участник не может увеличить свой выигрыш, изменив свою стратегию в одностороннем порядке, если остальные участники сохраняют свои стратегии неизменными. Это состояние устойчивости в некооперативной игре.

## Теоретические основы

Концепция, названная в честь Джона Нэша, применима к ситуациям, где взаимодействуют несколько лиц, принимающих решения (игроков). Набор стратегий является равновесием Нэша, если стратегия каждого игрока является наилучшим ответом на стратегии всех остальных участников.

Основные определения:
- **Чистая стратегия**: Конкретный план действий игрока.
- **Смешанная стратегия**: Распределение вероятностей по набору чистых стратегий.
- **Игра с нулевой суммой**: Игра, в которой выигрыш одного игрока в точности равен проигрышу другого.

## Математическая формулировка

В игре с $n$ игроками, пусть $S_i$ — множество стратегий игрока $i$, а $u_i(s_1, \dots, s_n)$ — функция выигрыша игрока $i$. Профиль стратегий $s^* = (s_1^*, \dots, s_n^*)$ является равновесием Нэша, если:

$$ \forall i, \forall s_i \in S_i : u_i(s_i^*, s_{-i}^*) \geq u_i(s_i, s_{-i}^*) $$

Где $s_{-i}^*$ обозначает стратегии всех игроков, кроме $i$. Нэш доказал, что любая конечная игра имеет как минимум одно равновесие (возможно, в смешанных стратегиях).

## Практическая значимость в финансах

Равновесие Нэша критически важно для понимания рыночной динамики:
- **Микроструктура рынка**: Моделирование взаимодействия между поставщиками ликвидности и агрессивными трейдерами.
- **Алгоритмическая торговля**: Анализ "гонки вооружений" в HFT, где фирмы конкурируют за скорость доступа.
- **Теория аукционов**: Оптимизация ставок в государственных облигациях или при выкупе частных компаний.
- **Системный риск**: Понимание банковских паник как перехода между различными состояниями равновесия.
- **Модели олигополии**: Прогнозирование ценовых войн и объемов производства на рынках с ограниченным числом крупных игроков (модели Курно, Бертрана).

## Python: Поиск равновесия в игре 2x2

```python
import numpy as np

def find_pure_nash(payoff_a, payoff_b):
    # payoff_a: матрица выигрышей игрока A
    # payoff_b: матрица выигрышей игрока B
    
    rows, cols = payoff_a.shape
    equilibria = []
    
    for i in range(rows):
        for j in range(cols):
            # Проверка: является ли выбор A лучшим ответом на j игрока B
            is_a_best = payoff_a[i, j] == np.max(payoff_a[:, j])
            # Проверка: является ли выбор B лучшим ответом на i игрока A
            is_b_best = payoff_b[i, j] == np.max(payoff_b[i, :])
            
            if is_a_best and is_b_best:
                equilibria.append((i, j))
                
    return equilibria

# Пример: Дилемма заключенного
A = np.array([[-1, -3], [0, -2]])
B = np.array([[-1, 0], [-3, -2]])

print(f"Равновесия (индексы): {find_pure_nash(A, B)}")
```

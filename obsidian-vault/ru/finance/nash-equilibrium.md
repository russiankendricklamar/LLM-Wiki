# Равновесие Нэша

**Равновесие Нэша (Nash Equilibrium)** — это концепция в теории игр, при которой ни один участник не может увеличить свой выигрыш, изменив свою стратегию в одностороннем порядке. Это состояние устойчивости в некооперативной игре.

## История и мотивация

Джон Нэш доказал в своей докторской диссертации 1950 года, что каждая конечная игра имеет как минимум одно равновесие в смешанных стратегиях — результат, удостоенный Нобелевской премии по экономике в 1994 году. Более ранние работы фон Неймана и Моргенштерна (1944) установили теорему минимакса для игр с нулевой суммой, но вклад Нэша распространил анализ равновесия на несравненно более богатый класс игр с ненулевой суммой.

На финансовых рынках равновесие Нэша обеспечивает теоретическую основу для понимания стратегических взаимодействий между участниками: конкурирующими маркет-мейкерами, HFT-фирмами в гонке задержек и институциональными инвесторами во время кризиса.

## Теоретические основы

Основные определения:
- **Чистая стратегия**: детерминированный план действий игрока.
- **Смешанная стратегия**: вероятностное распределение по чистым стратегиям.
- **Игра с нулевой суммой**: выигрыш одного игрока в точности равен проигрышу другого.
- **Наилучший ответ**: стратегия, максимизирующая выигрыш игрока при заданных стратегиях остальных.

## Математическая формулировка

В игре с $n$ игроками пусть $S_i$ — множество стратегий игрока $i$, $u_i$ — функция выигрыша. Профиль $s^* = (s_1^*, \dots, s_n^*)$ является равновесием Нэша, если:

$$\forall i, \forall s_i \in S_i : u_i(s_i^*, s_{-i}^*) \geq u_i(s_i, s_{-i}^*)$$

Для смешанных стратегий каждая чистая стратегия, разыгрываемая с положительной вероятностью, должна быть наилучшим ответом:

$$\sigma_i(s_i) > 0 \implies s_i \in \arg\max_{s_i'} u_i(s_i', \sigma_{-i}^*)$$

## Вычисление равновесий в смешанных стратегиях

Для двухигровой игры с нулевой суммой с матрицей выплат $A$ равновесие является решением линейной программы:

$$\max_{p \in \Delta^m} \min_{q \in \Delta^n} p^\top A q$$

Это эквивалентно теореме минимакса: $\max_p \min_q p^\top A q = \min_q \max_p p^\top A q$.

## Практическая значимость в финансах

Равновесие Нэша критически важно для понимания рыночной динамики:
- **Микроструктура рынка**: моделирование взаимодействия поставщиков ликвидности и агрессивных трейдеров.
- **Алгоритмическая торговля**: анализ гонки вооружений в HFT.
- **Теория аукционов**: оптимизация ставок в государственных облигациях или при buyout.
- **Системный риск**: банковские паники как переход между равновесиями.
- **Модели олигополии**: прогнозирование ценовых войн (Курно, Бертран).

## Маркет-мейкинг как игра

В модели Авелланеды-Стойкова бид-аск спред маркет-мейкера анализируется в игровой рамке. При $N$ одинаковых маркет-мейкерах в равновесии каждый выставляет спред, балансирующий риск неблагоприятного отбора и риск потери потока ордеров конкурентам. При $N \to \infty$ равновесный спред стремится к минимальному жизнеспособному спреду, объясняя, почему фрагментированные рынки с многими маркет-мейкерами имеют узкие спреды.

## Python: Поиск равновесий

```python
import numpy as np
from scipy.optimize import linprog


def find_pure_nash(payoff_a, payoff_b):
    """Найти все равновесия Нэша в чистых стратегиях двуматричной игры."""
    rows, cols = payoff_a.shape
    equilibria = []

    for i in range(rows):
        for j in range(cols):
            is_a_best = payoff_a[i, j] == np.max(payoff_a[:, j])
            is_b_best = payoff_b[i, j] == np.max(payoff_b[i, :])
            if is_a_best and is_b_best:
                equilibria.append((i, j))

    return equilibria


def find_mixed_nash_zerosum(A):
    """
    Найти равновесие в смешанных стратегиях для игры с нулевой суммой.
    A: матрица выплат игрока-строки.
    Возвращает (p_opt, q_opt, значение_игры).
    """
    m, n = A.shape

    c = np.zeros(m + 1)
    c[-1] = -1.0  # максимизируем v

    A_ub = np.hstack([-A.T, np.ones((n, 1))])
    b_ub = np.zeros(n)

    A_eq = np.ones((1, m + 1))
    A_eq[0, -1] = 0
    b_eq = np.array([1.0])

    bounds = [(0, None)] * m + [(None, None)]
    res = linprog(c, A_ub=A_ub, b_ub=b_ub, A_eq=A_eq, b_eq=b_eq,
                  bounds=bounds, method='highs')

    p_opt = res.x[:m]
    value = res.x[-1]

    # Оптимальная стратегия игрока-столбца (двойственная задача)
    c2 = np.zeros(n + 1)
    c2[-1] = 1.0
    A_ub2 = np.hstack([A, -np.ones((m, 1))])
    b_ub2 = np.zeros(m)
    A_eq2 = np.ones((1, n + 1))
    A_eq2[0, -1] = 0
    b_eq2 = np.array([1.0])
    bounds2 = [(0, None)] * n + [(None, None)]
    res2 = linprog(c2, A_ub=A_ub2, b_ub=b_ub2, A_eq=A_eq2, b_eq=b_eq2,
                   bounds=bounds2, method='highs')
    q_opt = res2.x[:n]

    return p_opt, q_opt, value


# Дилемма заключённого (не нулевая сумма)
A = np.array([[-1.0, -3.0], [0.0, -2.0]])
B = np.array([[-1.0,  0.0], [-3.0, -2.0]])
print("Равновесия Нэша (чистые):", find_pure_nash(A, B))

# Орёл или решка (нулевая сумма)
M = np.array([[1.0, -1.0], [-1.0, 1.0]])
p, q, v = find_mixed_nash_zerosum(M)
print(f"\nРавновесие в смешанных стратегиях: p={p}, q={q}, значение={v:.4f}")
# Ожидаемый результат: p=[0.5, 0.5], q=[0.5, 0.5], значение=0
```

| Фокальная точка | Равновесие, выбранное по соглашению | Кластеризация цен на круглых числах |

## Ограничения и расширения

- **Множественные равновесия**: в большинстве игр много равновесий Нэша; для предсказания конкретного нужны критерии отбора (риск-доминирование, выплата-доминирование).
- **Предположение о рациональности**: равновесие Нэша предполагает общее знание рациональности — нарушается когнитивными искажениями и стадным поведением.
- **Повторяющиеся игры**: «народная теорема» показывает, что бесконечное повторение допускает Парето-оптимальные исходы, объясняя долгосрочные отношения маркет-мейкеров.
- **Эволюционная теория игр**: заменяет рациональность эволюционным отбором; применима для моделирования эволюции алгоритмических стратегий на конкурентных рынках.
- **Игры среднего поля**: при $N \to \infty$ одинаковых игроках задача равновесия сводится к системе связанных УЧП, применимых к моделям торговли с большим числом участников.

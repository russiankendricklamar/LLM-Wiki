# Марковские цепи Монте-Карло (MCMC)

**Марковские цепи Монте-Карло (Markov Chain Monte Carlo, MCMC)** — это класс алгоритмов для генерации выборок из сложных вероятностных распределений. Суть метода заключается в построении цепи Маркова, для которой целевое распределение является стационарным (равновесным).

## Теоретические основы

Эффективность MCMC проявляется при работе с многомерными распределениями, из которых невозможно получить выборку напрямую. 
- **Цепь Маркова**: Процесс, в котором следующее состояние зависит только от текущего.
- **Эргодичность**: Свойство цепи, гарантирующее, что она посетит все области пространства состояний и сойдется к целевому распределению.
- **Условие детального равновесия**: Достаточное условие стационарности распределения $\pi(x)$: $\pi(x)P(x \to y) = \pi(y)P(y \to x)$.

## Математическая формулировка

Наиболее известным алгоритмом MCMC является **алгоритм Метрополиса — Гастингса**. Для генерации выборки из $\pi(x)$:
1. Предложить новое состояние $y$ из распределения $q(y|x)$.
2. Вычислить вероятность принятия $\alpha$:
   $$ \alpha = \min\left(1, \frac{\pi(y)q(x|y)}{\pi(x)q(y|x)}\right) $$
3. С вероятностью $\alpha$ перейти в $y$, иначе остаться в $x$.

В байесовском выводе, где $\pi(\theta|D) \propto P(D|\theta)P(\theta)$, MCMC позволяет оценивать апостериорное распределение без вычисления сложного нормировочного коэффициента (свидетельства).

## Практическая значимость в финансах

MCMC является фундаментом современного количественного анализа:
- **Байесовская оценка параметров**: Оценка параметров сложных моделей, таких как [[heston-model]] или [[sabr-model]].
- **Управление рисками**: Расчет [[value-at-risk]] (VaR) и ожидаемых потерь для портфелей с нелинейными зависимостями.
- **Стохастическая волатильность**: Калибровка моделей, где волатильность является скрытой переменной.
- **Кредитный скоринг**: Использование иерархических байесовских моделей для оценки вероятности дефолта.

## Python: Алгоритм Метрополиса — Гастингса

```python
import numpy as np

def metropolis_hastings(target_pdf, iterations, initial_state, proposal_std):
    x = initial_state
    samples = []
    
    for _ in range(iterations):
        # Предложение нового состояния
        x_prop = x + np.random.normal(0, proposal_std)
        
        # Коэффициент принятия
        acceptance_ratio = target_pdf(x_prop) / target_pdf(x)
        
        # Принятие или отклонение
        if np.random.rand() < acceptance_ratio:
            x = x_prop
            
        samples.append(x)
        
    return np.array(samples)

# Пример: выборка из нормального распределения
target = lambda x: np.exp(-0.5 * x**2)
samples = metropolis_hastings(target, 10000, 0, 0.5)
```

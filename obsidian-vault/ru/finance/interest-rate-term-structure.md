# Временная структура процентных ставок

**Временная структура процентных ставок** — кривая доходности $y(T)$ как функция срока до погашения $T$ — фундаментальнейший объект в мире фиксированного дохода. Она определяет ценообразование каждой облигации, свопа и процентного деривативa. Понимание формы кривой, извлечение информации из неё и моделирование динамики лежат в основе анализа монетарной политики, относительной стоимости облигаций и управления рисками.

## Ставки и их связи

### Спот-ставки

Непрерывно начисляемая **бескупонная (спот) ставка** $r(T)$ определяется ценой бескупонной облигации:

$$
P(0, T) = e^{-r(T) \cdot T}
$$

Спот-ставки извлекаются из купонных облигаций методом **бутстрэппинга** — последовательным вычислением нулевых ставок на всё более длинных сроках.

### Форвардные ставки

**Мгновенная форвардная ставка** $f(0, T)$ — ставка бесконечно малого заимствования в момент $T$, зафиксированная сегодня:

$$
f(0, T) = -\frac{\partial \ln P(0,T)}{\partial T} = r(T) + T \cdot r'(T)
$$

Дискретная форвардная ставка между $T_1$ и $T_2$:

$$
f(T_1, T_2) = \frac{r(T_2)\, T_2 - r(T_1)\, T_1}{T_2 - T_1}
$$

### Паритетные ставки

**Паритетная ставка** $c(T)$ — купонная ставка, при которой облигация торгуется по номиналу. Она равна фиксированной ставке процентного свопа той же дюрации:

$$
c(T) = \frac{1 - P(0,T)}{\sum_{i=1}^{n} P(0, t_i) \cdot \Delta t_i}
$$

Паритетная кривая — наиболее непосредственно наблюдаемая кривая доходности (из своповых ставок).

### Связи между ставками

По любой из $\{P(0,T),\; r(T),\; f(0,T),\; c(T)\}$ определяются остальные:

$$
P(0,T) = e^{-r(T)T} = \exp\!\left(-\int_0^T f(0,u)\,du\right)
$$

## Теории временной структуры

### Гипотеза ожиданий

**Чистая гипотеза ожиданий** (PEH) утверждает, что форвардные ставки равны ожидаемым будущим коротким ставкам:

$$
f(0, T) = \mathbb{E}[r(T)]
$$

Эквивалентно, длинная ставка есть среднее ожидаемых будущих коротких ставок:

$$
r(T) = \frac{1}{T}\int_0^T \mathbb{E}[r(u)]\,du
$$

По PEH, плоская кривая сигнализирует неизменность ставок, восходящая — ожидаемое повышение. Эмпирически PEH отвергается: форвардные ставки систематически завышают будущие спот-ставки.

### Предпочтение ликвидности (Хикс, 1939)

Инвесторы предпочитают краткосрочные облигации (меньший процентный риск). Для побуждения к покупке длинных облигаций необходима **срочная премия** $\text{TP}(T)$:

$$
f(0, T) = \mathbb{E}[r(T)] + \text{TP}(T), \qquad \text{TP}(T) > 0
$$

Срочная премия растёт со сроком, объясняя типичный восходящий наклон кривой даже при неизменных ожиданиях по ставкам.

### Предпочтительная среда обитания (Модильяни-Сатч, 1966)

Инвесторы и эмитенты имеют **предпочтительные сроки**, определяемые структурой обязательств. Дисбалансы спроса и предложения на каждом сроке создают локальные премии, не обязательно монотонно возрастающие. Это объясняет горбы и перегибы кривой.

### Сегментация рынка

Крайняя форма предпочтительной среды обитания: инвесторы на разных сроках действуют в полностью изолированных рынках. Доходность на каждом сроке определяется исключительно локальным спросом и предложением. Эмпирически слишком радикально — межсрочное замещение очевидно существует.

## Аффинные модели временной структуры

Доходность **аффинна** по вектору переменных состояния $\mathbf{X}_t$:

$$
r(t, T) = A(T-t) + \mathbf{B}(T-t)^{\top} \mathbf{X}_t
$$

где $A(\cdot)$ и $\mathbf{B}(\cdot)$ удовлетворяют ОДУ Риккати, выведенным из динамики состояния под риск-нейтральной мерой. Основные примеры:

| Модель | Динамика | Факторы |
|:-------|:---------|:--------|
| [[short-rate-models\|Vasicek]] | $dr = a(b-r)\,dt + \sigma\,dW$ | 1 (гауссовский с возвратом к среднему) |
| [[short-rate-models\|CIR]] | $dr = a(b-r)\,dt + \sigma\sqrt{r}\,dW$ | 1 (неотрицательный) |
| Многофакторный | $d\mathbf{X} = \mathbf{K}(\boldsymbol{\theta} - \mathbf{X})\,dt + \boldsymbol{\Sigma}\,d\mathbf{W}$ | 2-3 (уровень, наклон, кривизна) |

Цена облигации в аффинных моделях:

$$
P(t, T) = \exp\!\left(-A(T-t) - \mathbf{B}(T-t)^{\top} \mathbf{X}_t\right)
$$

## Оценка срочной премии

### Модель Адриана-Крампа-Мёнча (ACM, 2013)

ACM-модель разделяет доходности на ожидания и срочную премию через VAR форвардных ставок под физической и риск-нейтральной мерами:

$$
y(T) = \frac{1}{T}\sum_{h=0}^{T-1} \mathbb{E}^{\mathbb{P}}[r_{t+h}] + \text{TP}(T)
$$

Оценки срочной премии существенно варьируются во времени: близки к нулю в 2012-2013, достигали 100-200 б.п. в 1980-х. Они дают более чистый сигнал о позиции монетарной политики, чем сырые доходности.

## Эмпирическая модель Нельсона-Зигеля

Параметризация [[nelson-siegel-svensson]] подгоняет кривую доходности интерпретируемыми факторами:

$$
y(\tau) = \beta_0 + \beta_1 \frac{1 - e^{-\lambda\tau}}{\lambda\tau} + \beta_2 \left(\frac{1 - e^{-\lambda\tau}}{\lambda\tau} - e^{-\lambda\tau}\right)
$$

где $\beta_0$ = долгосрочный уровень, $\beta_1$ = наклон, $\beta_2$ = кривизна, $\lambda$ = скорость затухания. Используется центробанками по всему миру.

## Инвертированная кривая доходности

Когда $r(T_{\text{short}}) > r(T_{\text{long}})$, кривая **инвертирована**. По гипотезе ожиданий инверсия сигнализирует ожидаемое снижение ставок — обычно предшествуя рецессиям. Спред 10Y-2Y инвертировался перед каждой рецессией США с 1955 года с единственным ложным сигналом.

Предсказательная сила обусловлена динамикой срочной премии: при ослаблении экономики рынок закладывает агрессивное смягчение, опуская длинные ставки ниже коротких.

## Реализация на Python

```python
import numpy as np

def bootstrap_zero_rates(
    par_rates: np.ndarray, maturities: np.ndarray
) -> np.ndarray:
    """Бутстрэппинг нулевых ставок из паритетных своповых ставок."""
    n = len(maturities)
    zero_rates = np.zeros(n)
    discount = np.zeros(n)

    for i in range(n):
        if i == 0:
            zero_rates[i] = par_rates[i]
            discount[i] = 1 / (1 + zero_rates[i]) ** maturities[i]
        else:
            coupon_pv = par_rates[i] * np.sum(discount[:i])
            discount[i] = (1 - coupon_pv) / (1 + par_rates[i])
            zero_rates[i] = (1 / discount[i]) ** (1 / maturities[i]) - 1
    return zero_rates

def forward_rate(zero_rates, maturities, t1_idx, t2_idx):
    """Дискретная форвардная ставка между двумя сроками."""
    r1, T1 = zero_rates[t1_idx], maturities[t1_idx]
    r2, T2 = zero_rates[t2_idx], maturities[t2_idx]
    return (r2 * T2 - r1 * T1) / (T2 - T1)

def nelson_siegel(tau, beta0, beta1, beta2, lam):
    """Модель кривой доходности Нельсона-Зигеля."""
    factor1 = (1 - np.exp(-lam * tau)) / (lam * tau)
    factor2 = factor1 - np.exp(-lam * tau)
    return beta0 + beta1 * factor1 + beta2 * factor2

# Пример: типичная восходящая кривая
maturities = np.array([0.25, 0.5, 1, 2, 3, 5, 7, 10, 20, 30])
yields = nelson_siegel(maturities, beta0=0.045, beta1=-0.02,
                        beta2=-0.01, lam=0.6)
```

## Применения

1. **Относительная стоимость облигаций**: выявление дорогих/дешёвых бумаг сравнением рыночных доходностей с подогнанной кривой
2. **Торговля кривой**: стипенеры, флэттенеры и баттерфляи выражают взгляды на изменение формы кривой
3. **Анализ монетарной политики**: разложение доходностей на ожидания ставок и срочную премию для оценки позиции политики
4. **Ценообразование деривативов**: кривая дисконтирования лежит в основе всех свопов, свопционов и кэпов/флоров
5. **Управление рисками**: PCA кривой (уровень, наклон, кривизна) объясняет >99% вариации доходностей для хеджирования ключевых дюраций

# Стратегия парного трейдинга (Pairs Trading)

**Парный трейдинг** — это рыночно-нейтральная стратегия, основанная на открытии длинной позиции по одному активу и короткой позиции по другому, тесно коррелирующему с ним. Стратегия использует временные отклонения цен между двумя активами, предполагая, что они в конечном итоге вернутся к своей исторической взаимосвязи (**Коинтеграция**).

## Практическое руководство по внедрению

1.  **Выбор пары**: Поиск активов в одном секторе (например, PEP и KO), демонстрирующих стабильную долгосрочную зависимость.
2.  **Тест на коинтеграцию**: Использование теста **Энгла-Грейнджера** или **Йохансена** для проверки стационарности спреда.
3.  **Расчет спреда**:
    *   $\text{Spread} = \ln(P_A) - \beta \ln(P_B)$
    *   $\beta$ — коэффициент хеджирования, обычно определяемый через МНК или фильтр Калмана.
4.  **Правила входа/выхода**: Вход при выходе Z-score спреда за пороговые значения (например, $\pm 2.0$) и выход при возврате к среднему ($0.0$).

## Математическая формулировка

Спред $S_t$ часто моделируется как процесс Орнштейна-Уленбека (OU):
$$
dS_t = \theta (\mu - S_t) dt + \sigma dW_t
$$
где $\theta$ — скорость возврата к среднему. Параметры $\theta, \mu, \sigma$ могут быть оценены с помощью [[mcmc]] для учета динамики во времени. Если спред не возвращается слишком долго, вступают в силу лимиты [[value-at-risk]] (VaR).

## Ключевые метрики
- **Коэффициент хеджирования ($\beta$)**: Относительный размер позиций.
- **Период полураспада (Half-Life)**: $\ln(2) / \theta$, показывающий, как долго сделка будет открыта в среднем.
- **Z-Score**: Количество стандартных отклонений текущего спреда от его среднего значения.
- **P-Value коинтеграции**: Статистическая уверенность в том, что связь не случайна.

## Python: Тест на коинтеграцию и расчет Z-score

```python
import numpy as np
import pandas as pd
from statsmodels.tsa.stattools import coint
from sklearn.linear_model import LinearRegression

# Генерация коинтегрированных данных
np.random.seed(42)
x = np.cumsum(np.random.normal(0, 1, 100))
y = x + np.random.normal(0, 0.5, 100) # x и y связаны

# Тест Энгла-Грейнджера
score, p_value, _ = coint(x, y)
print(f"P-Value коинтеграции: {p_value:.4f}")

# Расчет Z-score спреда
model = LinearRegression().fit(x.reshape(-1, 1), y)
beta = model.coef_[0]
spread = y - beta * x
z_score = (spread - np.mean(spread)) / np.std(spread)
print(f"Текущий Z-Score: {z_score[-1]:.2f}")
```

## Финансовый контекст

Парный трейдинг — основа **статистического арбитража**. Риск заключается в «распаде связи», когда активы перестают коррелировать из-за фундаментальных изменений (например, слияния или банкротства). Устойчивость проверяется через симуляции [[monte-carlo-method]], имитирующие тысячи сценариев разрыва связи между активами.

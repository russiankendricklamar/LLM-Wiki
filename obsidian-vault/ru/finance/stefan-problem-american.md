# Задача Стефана и американские опционы

В непрерывном времени [[optimal-stopping-american|оптимальная остановка]] для американского опциона превращается в задачу с **препятствием** для PDE Блэка-Шоулса — это классическая **задача Стефана** (Stefan problem). Свободная граница, разделяющая «область удержания» и «область исполнения», сама становится неизвестной функцией, и её положение определяется одновременно с ценой.

Ширяев ([[shiryaev-stochastic-finance|т. 2, гл. VIII, § 3b-3c]]) посвящает этому подробный раздел. Задача Стефана — один из самых старых сюжетов прикладной математики (восходит к работам Йозефа Стефана 1889 г. о таянии льда), и финансовые опционы — её современное применение, ничуть не уступающее классическим тепловым задачам.

## Постановка для американского пут-опциона

Рассмотрим американский пут на акцию в модели Блэка-Шоулса: $S_t$ — [[geometric-brownian-motion|GBM]] с волатильностью $\sigma$ и сносом $r$ под мартингальной мерой. Выплата при исполнении в момент $\tau$:

$$G(\tau, S_\tau) = (K - S_\tau)^+.$$

**Цена опциона** $V(t, S)$ — функция времени $t \in [0, T]$ и текущей цены $S$. Свобода исполнения означает, что в каждой точке $(t, S)$ держатель решает: «удержать ещё мгновение» или «исполнить сейчас». Этот выбор разбивает плоскость $(t, S)$ на две области:

- **Область удержания** $\mathcal{C} = \{(t, S) \,:\, V(t, S) > G(t, S)\}$ — здесь оптимально подождать.
- **Область исполнения** $\mathcal{E} = \{(t, S) \,:\, V(t, S) = G(t, S)\}$ — здесь оптимально исполнить.

Граница между ними — **свободная граница** $S^*(t)$.

## Уравнения

В области удержания цена удовлетворяет PDE Блэка-Шоулса:

$$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + r S \frac{\partial V}{\partial S} - r V = 0, \quad (t, S) \in \mathcal{C}.$$

В области исполнения цена тождественно равна выплате:

$$V(t, S) = K - S, \quad (t, S) \in \mathcal{E}.$$

На свободной границе $S = S^*(t)$ выполняются два условия согласования (smooth pasting):

$$V(t, S^*(t)) = K - S^*(t) \quad \text{(непрерывность)},$$

$$\frac{\partial V}{\partial S}(t, S^*(t)) = -1 \quad \text{(непрерывность дельты)}.$$

Второе условие — **smooth pasting condition** (Самуэльсон-МакКин, 1965) — обеспечивает оптимальность; без него граница не была бы оптимальной точкой исполнения.

## Структура задачи Стефана

В классической задаче Стефана о таянии льда «свободная граница» — это интерфейс между льдом и водой; её положение зависит от температуры, которая сама подчиняется уравнению теплопроводности. В американском пут-опционе:

- «Лёд» $\equiv$ область исполнения,
- «Вода» $\equiv$ область удержания,
- «Температура» $\equiv$ цена опциона $V(t, S)$,
- «Уравнение теплопроводности» $\equiv$ PDE Блэка-Шоулса,
- «Скрытая теплота» (условие Стефана на границе) $\equiv$ smooth pasting.

Эта аналогия не поверхностная: после стандартной замены переменных PDE Блэка-Шоулса прямо превращается в [[black-scholes-derivations|уравнение теплопроводности]], а граничные условия — в условия Стефана.

## Аналитика свободной границы

**Утверждение.** Для американского пут-опциона на бездивидендной акции свободная граница $S^*(t)$ — монотонно возрастающая функция $t$, со значением $S^*(T) = K$ в момент истечения и

$$\lim_{t \to -\infty} S^*(t) = \frac{2r}{2r + \sigma^2} \cdot K$$

в пределе бесконечно длинного срока (это значение — известное «бесконечно-горизонтная» цена исполнения, найденная МакКином в 1965).

**Замкнутого выражения для $S^*(t)$ не существует.** Это редкий случай в финансовой математике: задача формулируется элементарно, но решение — нет. Численные методы (конечные разности, биномиальное дерево, интегральные уравнения) обязательны.

## Численные методы

1. **[[crr-model|Биномиальное дерево CRR]] с проверкой раннего исполнения.** На каждом узле дерева берётся максимум из «удержания» и «исполнения». Это дискретный аналог огибающей Снелла. Сходится к точному решению со скоростью $O(1/N)$.
2. **Конечные разности (PSOR).** Решение PDE с препятствием на сетке $(t, S)$ с явной проверкой $V \geq K - S$ на каждом шаге. Используется метод проектируемой последовательной сверх-релаксации (Projected SOR).
3. **Интегральные уравнения для $S^*(t)$.** Карр-Якка-Майерс (1992) выводят интегральное уравнение Вольтерры для свободной границы и решают его численно. Самый быстрый метод для одномерных опционов.
4. **Регрессия по Монте-Карло (Лонгстафф-Шварц, 2001).** Ключевая идея: «удержание» оценивается через регрессию ожидаемой выплаты по базисным функциям от текущего состояния. Стандартный метод для многомерных американских опционов.

## Эквивалентность с задачей оптимальной остановки

Задача Стефана — **PDE-форма** того же объекта, что в дискретном времени даёт огибающая Снелла. Связь делается строгой через теорему Майера-Шуттенберга:

$$V(t, S) = \sup_{\tau \in \mathcal{T}_{[t,T]}} \mathbb{E}^\mathbb{Q}_{(t,S)}\!\left[ e^{-r(\tau-t)} (K - S_\tau)^+ \right],$$

где супремум берётся по моментам остановки. Эта вариационная формулировка эквивалентна PDE с препятствием — это и есть «опциональная остановка $\Leftrightarrow$ свободная граница».

## Американский колл: тривиальный случай

На бездивидендной акции американский **колл** имеет ту же цену, что и европейский — задача Стефана для него **вырождена**: область исполнения пуста, $S^*(t) = +\infty$. Причина: $(S - K)^+$ растёт быстрее, чем $r$ может «съесть» через дисконтирование, и дольжать всегда выгоднее.

При наличии дивидендов ситуация меняется: дивиденды могут сделать раннее исполнение колла оптимальным, и появляется нетривиальная свободная граница.

# Фильтр Калмана

**Фильтр Калмана** (Рудольф Калман, 1960) — оптимальный рекурсивный байесовский оценщик для линейных динамических систем с гауссовским шумом. Он даёт оценку скрытого состояния с минимальной среднеквадратической ошибкой по всем наблюдениям до текущего момента, обновляя оценку с поступлением каждого нового наблюдения — что делает его идеальным для финансовых приложений реального времени.

## Предпосылки

В финансах «истинное» состояние системы — справедливая стоимость, коэффициент хеджирования, скрытая факторная нагрузка — никогда не наблюдается напрямую. Мы наблюдаем лишь зашумлённые прокси. Фильтр Калмана объединяет динамическую модель эволюции состояния (**уравнение состояния**) и модель измерений (**уравнение наблюдения**), взвешивая каждую по соответствующей неопределённости. Оптимальность при гауссовских допущениях делает его золотым стандартом линейной фильтрации.

## Представление в пространстве состояний

Любая линейная динамическая система записывается в **форме пространства состояний**:

**Уравнение состояния (переходов):**
$$
x_t = F_t x_{t-1} + B_t u_t + w_t, \qquad w_t \sim \mathcal{N}(0, Q_t)
$$

**Уравнение наблюдения:**
$$
z_t = H_t x_t + v_t, \qquad v_t \sim \mathcal{N}(0, R_t)
$$

Где:
- $x_t \in \mathbb{R}^n$ — вектор скрытого состояния
- $z_t \in \mathbb{R}^m$ — вектор наблюдений
- $F_t$ — матрица переходов состояний ($n \times n$)
- $H_t$ — матрица наблюдений ($m \times n$)
- $B_t, u_t$ — опциональный управляющий вход
- $Q_t$ — ковариация шума процесса ($n \times n$)
- $R_t$ — ковариация шума измерений ($m \times m$)

## Двухшаговый алгоритм

### Шаг 1: Прогноз

Проецируем оценку состояния и её неопределённость вперёд по динамической модели:

$$
\hat{x}_{t \mid t-1} = F_t \hat{x}_{t-1 \mid t-1} + B_t u_t
$$

$$
P_{t \mid t-1} = F_t P_{t-1 \mid t-1} F_t^T + Q_t
$$

$P_{t \mid t-1}$ — **прогнозная ковариация** — неопределённость состояния до получения $z_t$.

### Шаг 2: Обновление

Включаем новое наблюдение $z_t$ для коррекции прогноза:

$$
\nu_t = z_t - H_t \hat{x}_{t \mid t-1} \qquad \text{(инновация)}
$$

$$
S_t = H_t P_{t \mid t-1} H_t^T + R_t \qquad \text{(ковариация инновации)}
$$

$$
K_t = P_{t \mid t-1} H_t^T S_t^{-1} \qquad \text{(усиление Калмана)}
$$

$$
\hat{x}_{t \mid t} = \hat{x}_{t \mid t-1} + K_t \nu_t
$$

$$
P_{t \mid t} = (I - K_t H_t) P_{t \mid t-1}
$$

**Усиление Калмана** $K_t$ балансирует доверие к модели и к новому наблюдению:
- $K_t \to 0$: игнорируем наблюдение (высокий шум измерений $R_t$ или малый шум процесса $Q_t$).
- $K_t \to H_t^{-1}$: полностью доверяем наблюдению (малый $R_t$, большой $Q_t$).

## Логарифм правдоподобия для оценки параметров

Инновации $\nu_t \sim \mathcal{N}(0, S_t)$ независимы при верной модели. Логарифм правдоподобия:

$$
\ell = -\frac{1}{2} \sum_{t=1}^T \left[\ln|S_t| + \nu_t^T S_t^{-1} \nu_t + m\ln(2\pi)\right]
$$

Это позволяет **оценивать неизвестные параметры** $F, H, Q, R$ методом максимального правдоподобия через численную оптимизацию — ключевой шаг при применении фильтра к реальным финансовым данным.

## Python: Полная реализация

```python
import numpy as np
from dataclasses import dataclass
from typing import Optional

@dataclass
class KalmanState:
    x: np.ndarray        # среднее состояния
    P: np.ndarray        # ковариация состояния

def kalman_filter(
    observations: np.ndarray,  # форма (T, m)
    F: np.ndarray,             # (n, n) матрица переходов
    H: np.ndarray,             # (m, n) матрица наблюдений
    Q: np.ndarray,             # (n, n) ковариация шума процесса
    R: np.ndarray,             # (m, m) ковариация шума измерений
    x0: Optional[np.ndarray] = None,
    P0: Optional[np.ndarray] = None,
) -> tuple:
    """
    Линейный фильтр Калмана.

    Возвращает:
        filtered_means:  (T, n) оценки состояний
        filtered_covs:   (T, n, n) ковариации состояний
        log_likelihood:  полное логарифмическое правдоподобие
    """
    T, m = observations.shape
    n = F.shape[0]

    x = x0 if x0 is not None else np.zeros(n)
    P = P0 if P0 is not None else np.eye(n) * 1e6   # диффузный априор

    filtered_means = np.zeros((T, n))
    filtered_covs  = np.zeros((T, n, n))
    log_lik = 0.0

    for t in range(T):
        # --- Прогноз ---
        x_pred = F @ x
        P_pred = F @ P @ F.T + Q

        # --- Обновление ---
        innovation = observations[t] - H @ x_pred
        S = H @ P_pred @ H.T + R
        K = P_pred @ H.T @ np.linalg.inv(S)
        x = x_pred + K @ innovation
        P = (np.eye(n) - K @ H) @ P_pred

        # Вклад в логарифм правдоподобия
        sign, logdet = np.linalg.slogdet(S)
        log_lik -= 0.5 * (logdet + innovation @ np.linalg.inv(S) @ innovation + m * np.log(2 * np.pi))

        filtered_means[t] = x
        filtered_covs[t]  = P

    return filtered_means, filtered_covs, log_lik


# --- Пример: оценка коэффициента хеджирования в парном трейдинге ---
np.random.seed(42)
T = 500
# Истинный нестационарный коэффициент хеджирования (случайное блуждание)
true_beta = 1.0 + np.cumsum(np.random.normal(0, 0.01, T))
spread_noise = np.random.normal(0, 0.5, T)
y1 = 100 + np.cumsum(np.random.normal(0, 1, T))
y2 = true_beta * y1 + spread_noise

observations = (y2 - y1).reshape(-1, 1)  # наблюдаемый спред

F = np.array([[1.0]])       # коэффициент хеджирования — случайное блуждание
H = np.array([[1.0]])
Q = np.array([[0.0001]])    # медленно меняющийся коэффициент
R = np.array([[0.25]])      # дисперсия шума наблюдений

means, covs, ll = kalman_filter(observations, F, H, Q, R)
print(f"Логарифм правдоподобия:       {ll:.2f}")
print(f"Итоговая оценка коэф. хеджир.: {means[-1, 0]:.4f}")
print(f"Истинный итоговый коэф.:       {true_beta[-1]:.4f}")
```

## Расширения

### Расширенный фильтр Калмана (EKF)
Для нелинейных переходов $x_t = f(x_{t-1}) + w_t$ или нелинейных наблюдений $z_t = h(x_t) + v_t$ производится линеаризация через разложение Тейлора первого порядка: матрица $F_t$ заменяется якобианом $\partial f / \partial x$ в точке $\hat{x}_{t-1}$.

### Сигма-точечный фильтр Калмана (UKF)
Вместо линеаризации через нелинейные функции пропускается специально подобранный набор **сигма-точек**. Точнее EKF при сильной нелинейности; предпочтителен для моделей стохастической волатильности.

### Сглаживатель Калмана (RTS-сглаживатель)
Имея полную последовательность наблюдений, вычисляет оптимальные оценки состояний с использованием как прошлых, так и будущих наблюдений. **Обратный проход Рауха-Тунга-Штрибеля (RTS)** уточняет отфильтрованные оценки, снижая неопределённость по всей последовательности — полезно при оценке параметров и бэктестировании.

## Таблица параметров

| Символ | Название | Роль | Типичная настройка |
|--------|----------|------|--------------------|
| $F_t$ | Матрица переходов | Динамика состояния | Часто AR(1) или единичная |
| $H_t$ | Матрица наблюдений | Связь состояния с наблюдениями | Единичная или регрессионная |
| $Q_t$ | Шум процесса | Неопределённость состояния | Малая (гладкие состояния) |
| $R_t$ | Шум измерений | Неопределённость наблюдений | Оценивается из данных |
| $K_t$ | Усиление Калмана | Коэффициент смешивания | Вычисляется на каждом шаге |
| $P_t$ | Ковариация ошибки | Неопределённость оценки | Отслеживает сходимость |

## Практическое применение в финансах

1. **Парный трейдинг**: оценка нестационарного коэффициента хеджирования коинтегрированных активов; торговля на возврат к среднему по сглаженному Калманом спреду.
2. **Оценка факторной модели**: отслеживание нестационарных факторных нагрузок (бета) в динамической факторной модели с фильтром Калмана в качестве движка вывода.
3. **Подгонка кривой доходности**: модель Нельсона-Зигеля может быть сформулирована как модель пространства состояний; фильтрация Калманом извлекает скрытые факторы уровня, наклона и кривизны из доходностей облигаций в реальном времени.
4. **Дисбаланс потока ордеров**: фильтрация зашумлённых данных микроструктуры (середина спреда) для извлечения сигнала эффективной цены.
5. **Оценка волатильности**: модель стохастической волатильности Харви et al. оценивается фильтром Калмана на логарифмах квадратов доходностей.

# Процессы Леви

**Процесс Леви** — это стохастический процесс с независимыми и стационарными приращениями. Он представляет собой непрерывный аналог случайного блуждания. Процессы Леви фундаментальны в количественных финансах для моделирования цен активов, которые демонстрируют скачки и «тяжелые хвосты».

## История и мотивация

Вездесущее предположение о геометрическом броуновском движении (ГБД), лежащее в основе модели Блэка-Шоулза, приводит к логнормальному распределению доходностей — свойству, эмпирически нарушаемому практически каждым финансовым активом. Реальные распределения доходностей обладают избыточным эксцессом (тяжёлые хвосты) и асимметрией, а цены активов иногда скачкообразно меняются в ответ на публикацию отчётности, геополитические события или шоки ликвидности.

Поль Леви разработал общую теорию процессов с независимыми и стационарными приращениями в 1930-х годах. Теорема Леви-Хинчина (1937) даёт полную характеризацию всех таких процессов через тройку: дрейф, диффузия и мера скачков.

## Определение и свойства

Стохастический процесс $X = \{X_t : t \ge 0\}$ является процессом Леви, если:
1. $X_0 = 0$ (почти наверное).
2. **Независимые приращения**: Для любых $s < t$ приращение $X_t - X_s$ не зависит от траектории до момента $s$.
3. **Стационарные приращения**: Распределение $X_t - X_s$ зависит только от длины интервала $t-s$.
4. **Непрерывность по вероятности**: Для любого $\epsilon > 0$, $\lim_{h \to 0} P(|X_{t+h} - X_t| > \epsilon) = 0$.

## Представление Леви-Хинчина

Каждый процесс Леви однозначно определяется своим характеристическим показателем $\psi(\theta)$:

$$
\psi(\theta) = i\gamma\theta - \frac{1}{2}\sigma^2\theta^2 + \int_{\mathbb{R}\setminus\{0\}} (e^{i\theta x} - 1 - i\theta x \mathbf{1}_{|x|<1}) \nu(dx)
$$

Где:
- $\gamma$ — параметр дрейфа.
- $\sigma^2$ — гауссовская (диффузионная) дисперсия.
- $\nu$ — мера Леви, описывающая интенсивность и размер скачков.

Тройка $(\gamma, \sigma^2, \nu)$ — **тройка Леви** — однозначно идентифицирует процесс.

## Декомпозиция Леви-Ито

Каждый процесс Леви разлагается на:

$$
X_t = \gamma t + \sigma W_t + \int_{|x|<1} x \tilde{N}(dt, dx) + \int_{|x|\ge 1} x N(dt, dx)
$$

где $N(dt, dx)$ — пуассоновская случайная мера с интенсивностью $dt \cdot \nu(dx)$, $\tilde{N} = N - dt\,\nu(dx)$ — компенсированная мера. Декомпозиция разделяет:
1. Детерминированный дрейф $\gamma t$
2. Непрерывную диффузионную компоненту $\sigma W_t$
3. Малые скачки (компенсированные) — мартингальный вклад
4. Большие скачки (составной пуассоновский процесс)

## Основные примеры

- **Винеровский процесс**: $\nu = 0$. Единственный непрерывный процесс Леви.
- **Пуассоновский процесс**: $\sigma = 0$, $\nu = \lambda \delta_1$.
- **Составной пуассоновский процесс**: скачки в пуассоновские моменты со случайным размером.
- **Процесс вариационной гаммы (VG)**: броуновское движение, субординированное гамма-процессом. Конечная вариация, аналитически управляемый.
- **Нормальный обратный гауссов процесс (NIG)**: улавливает одновременно асимметрию и избыточный эксцесс доходностей акций.
- **CGMY/KoBoL**: обобщает VG; раздельно контролирует «тяжесть» хвостов скачков вверх и вниз через параметры $C, G, M, Y$.

## Ключевые свойства

- **Бесконечная делимость**: распределение $X_t$ при любом фиксированном $t$ бесконечно делимо.
- **Мартингальное условие**: $X_t - X_0 - \mathbb{E}[X_t - X_0]$ — мартингал.
- **Асимметрия и эксцесс**: мера Леви $\nu$ напрямую управляет моментами высшего порядка.

## Python: Симуляция процессов Леви

```python
import numpy as np

def simulate_jump_diffusion(S0, T, r, sigma, lamb, mu_j, sigma_j, steps):
    """Модель скачкообразной диффузии Мертона."""
    dt = T / steps
    W = np.random.standard_normal(steps)
    N = np.random.poisson(lamb * dt, steps)
    J = np.where(N > 0, np.random.normal(mu_j, sigma_j, steps), 0)

    log_S = np.zeros(steps)
    log_S[0] = np.log(S0)

    kappa = np.exp(mu_j + 0.5 * sigma_j**2) - 1
    for i in range(1, steps):
        drift = (r - 0.5 * sigma**2 - lamb * kappa) * dt
        diffusion = sigma * np.sqrt(dt) * W[i]
        jumps = N[i] * J[i]
        log_S[i] = log_S[i-1] + drift + diffusion + jumps

    return np.exp(log_S)


def simulate_variance_gamma(S0, T, r, sigma, nu, theta, steps):
    """
    Процесс вариационной гаммы:
    броуновское движение с дрейфом, субординированное гамма-процессом.
    nu: дисперсия субординатора на единицу времени
    theta: дрейф субординированного броуновского движения
    """
    dt = T / steps
    dg = np.random.gamma(dt / nu, nu, steps)
    dX = theta * dg + sigma * np.sqrt(dg) * np.random.standard_normal(steps)

    omega = (1 / nu) * np.log(1 - theta * nu - 0.5 * sigma**2 * nu)
    log_S = np.zeros(steps + 1)
    log_S[0] = np.log(S0)
    for i in range(steps):
        log_S[i + 1] = log_S[i] + (r + omega) * dt + dX[i]

    return np.exp(log_S)


np.random.seed(42)
S_merton = simulate_jump_diffusion(100, 1.0, 0.05, 0.2, 3.0, -0.05, 0.1, 252)
S_vg = simulate_variance_gamma(100, 1.0, 0.05, 0.15, 0.1, -0.1, 252)

print(f"Итоговая цена (Merton):        {S_merton[-1]:.2f}")
print(f"Итоговая цена (Variance Gamma): {S_vg[-1]:.2f}")
```

## Таблица параметров

| Символ | Название | Описание | Типичный диапазон |
|:---|:---|:---|:---|
| $\gamma$ | Дрейф | Детерминированная трендовая составляющая | $[-0.5, 0.5]$ |
| $\sigma^2$ | Дисперсия диффузии | Непрерывная волатильность | $[0, 1]$ |
| $\lambda$ | Интенсивность скачков | Ожидаемое число скачков в год | $[0.5, 10]$ |
| $\mu_j$ | Средний размер скачка | Среднее лог-скачка | $[-0.2, 0.1]$ |
| $\sigma_j$ | Волатильность скачков | СКО лог-скачка | $[0.05, 0.3]$ |
| $\nu$ (VG) | Параметр гаммы | Управляет эксцессом VG-процесса | $[0.01, 0.5]$ |

## Параметры модели

| Компонента | Описание | Финансовое влияние |
| :--- | :--- | :--- |
| **Диффузия** | Непрерывное движение цены | Обычная рыночная волатильность |
| **Скачки** | Внезапные разрывы в цене | Рыночные шоки / новости |
| **Дрейф** | Средняя скорость доходности | Ожидаемый рост |

## Ограничения и расширения

- **Сложность калибровки**: тройка Леви обладает бесконечномерной свободой. Параметрические семейства (Merton, VG, NIG, CGMY) ограничивают её 3–5 параметрами, но калибровка по поверхности волатильности нетривиальна.
- **Временная неоднородность**: стандартные процессы Леви имеют стационарные приращения — GARCH-эффекты не захватываются. Подчинённые Леви-процессы или Леви-управляемые модели стохастической волатильности решают эту проблему.
- **Корреляционная структура**: построение многомерных Леви-моделей с реалистичной взаимозависимостью активов требует копул Леви.

## Практические применения

Процессы Леви применяются в деск-маркет-мейкинге для построения более точных поверхностей волатильности, чем Блэк-Шоулз. Модели NIG и VG стандартны в ценообразовании производных на энергию и валюту. В кредитном риске Леви-управляемые модели интенсивности (расширяющие Cox-процесс) захватывают риск мгновенного дефолта, который диффузионные модели упускают.

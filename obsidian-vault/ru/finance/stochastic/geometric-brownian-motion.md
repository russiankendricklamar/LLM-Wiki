# Геометрическое броуновское движение

**Геометрическое броуновское движение (GBM)** — стохастический процесс с непрерывным временем, в котором логарифм случайно меняющейся величины следует броуновскому движению с дрейфом. Это фундаментальная модель для цен акций в рамках структуры Блэка-Шоулза — контрольная точка, с которой сравниваются все более сложные модели.

## История

GBM введено в финансы Полем Самуэльсоном (1965) и формализовано Блэком, Шоулзом и Мертоном (1973). Его ключевое достоинство — сочетание математической tractability (существуют замкнутые формулы для европейских опционов) с экономически естественным свойством строгой положительности цен. Модель лежит в основе всего здания непрерывного ценообразования производных инструментов.

## Стохастическое дифференциальное уравнение

Динамика цены актива $S_t$, следующего GBM:

$$
dS_t = \mu S_t\, dt + \sigma S_t\, dW_t
$$

Где:
- $\mu$ — коэффициент дрейфа (мгновенная ожидаемая доходность, $\mu \in \mathbb{R}$)
- $\sigma > 0$ — волатильность (мгновенное стандартное отклонение доходностей)
- $W_t$ — стандартное броуновское движение при реальной мере $\mathbb{P}$

СДУ линейно по $S_t$, что обеспечивает пропорциональные приращения: движение на 1% одинаково вероятно независимо от уровня цены.

## Лемма Ито и замкнутая форма решения

Применяя лемму Ито к $f(S_t) = \ln S_t$:

$$
d\ln S_t = \left(\mu - \frac{1}{2}\sigma^2\right)dt + \sigma\, dW_t
$$

Член $-\frac{1}{2}\sigma^2$ — **поправка Ито**, возникающая из выпуклости (неравенство Йенсена). Интегрируя от 0 до $T$:

$$
S_T = S_0 \exp\!\left[\left(\mu - \frac{1}{2}\sigma^2\right)T + \sigma W_T\right]
$$

Поскольку $W_T \sim \mathcal{N}(0, T)$, логарифмическая доходность нормально распределена:

$$
\ln\frac{S_T}{S_0} \sim \mathcal{N}\!\left(\left(\mu - \frac{1}{2}\sigma^2\right)T,\ \sigma^2 T\right)
$$

Это означает, что $S_T$ имеет **логнормальное распределение** с:
- $\mathbb{E}[S_T] = S_0 e^{\mu T}$
- $\text{Var}(S_T) = S_0^2 e^{2\mu T}(e^{\sigma^2 T} - 1)$

## Риск-нейтральное ценообразование

Под риск-нейтральной мерой $\mathbb{Q}$ (теорема Гирсанова) дрейф $\mu$ заменяется безрисковой ставкой $r$:

$$
dS_t = r S_t\, dt + \sigma S_t\, d\tilde{W}_t
$$

Цена европейского колл-опциона — дисконтированное риск-нейтральное ожидание:

$$
C = e^{-rT}\,\tilde{\mathbb{E}}\!\left[\max(S_T - K, 0)\right] = S_0 N(d_1) - K e^{-rT} N(d_2)
$$

с $d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}$, $d_2 = d_1 - \sigma\sqrt{T}$ — формула Блэка-Шоулза.

## Ключевые статистические свойства

| Свойство | Значение для GBM |
|----------|-----------------|
| $\mathbb{E}[\ln S_T/S_0]$ | $(\mu - \sigma^2/2)T$ |
| $\text{Var}[\ln S_T/S_0]$ | $\sigma^2 T$ |
| Асимметрия логарифмических доходностей | 0 (гауссово) |
| Избыточный эксцесс логарифмических доходностей | 0 (гауссово) |
| Автокорреляция доходностей | 0 (марковский) |
| Положительность цены | Гарантирована ($S_t > 0$ п.н.) |

## Дискретизация Эйлера-Маруямы

Для симуляции СДУ дискретизируется с шагом $\Delta t = T/N$:

$$
S_{t+\Delta t} = S_t \exp\!\left[\left(\mu - \frac{1}{2}\sigma^2\right)\Delta t + \sigma\sqrt{\Delta t}\,Z_t\right], \quad Z_t \sim \mathcal{N}(0,1)
$$

Эта **точная дискретизация** (использующая замкнутую форму) не имеет ошибки дискретизации — только ошибку выборки Монте-Карло.

## Реализация на Python

```python
import numpy as np
import pandas as pd

def simulate_gbm(
    S0: float,
    mu: float,
    sigma: float,
    T: float,
    n_steps: int,
    n_paths: int = 1,
    seed: int = 42,
) -> np.ndarray:
    """
    Симуляция траекторий GBM через точную дискретизацию.
    Возвращает массив формы (n_paths, n_steps + 1).
    """
    rng = np.random.default_rng(seed)
    dt = T / n_steps
    drift = (mu - 0.5 * sigma ** 2) * dt
    diffusion = sigma * np.sqrt(dt)

    Z = rng.standard_normal((n_paths, n_steps))
    log_returns = drift + diffusion * Z
    log_price = np.concatenate(
        [np.zeros((n_paths, 1)), np.cumsum(log_returns, axis=1)], axis=1
    )
    return S0 * np.exp(log_price)

def gbm_statistics(paths: np.ndarray) -> dict:
    """Средний путь и 95%-й доверительный интервал."""
    mean_path = paths.mean(axis=0)
    std_path  = paths.std(axis=0)
    return {
        "mean":     mean_path,
        "upper_95": mean_path + 1.96 * std_path,
        "lower_95": mean_path - 1.96 * std_path,
    }

# --- Пример: 1 год, дневные шаги, 1000 траекторий ---
paths = simulate_gbm(S0=100, mu=0.08, sigma=0.20, T=1.0, n_steps=252, n_paths=1000)
stats = gbm_statistics(paths)
print(f"Средняя конечная цена: {stats['mean'][-1]:.2f}")
print(f"95% интервал при T=1:  [{stats['lower_95'][-1]:.2f}, {stats['upper_95'][-1]:.2f}]")
```

## Ограничения и расширения

- **Постоянная волатильность**: GBM предполагает неизменный $\sigma$. Эмпирически волатильность стохастична (улыбка/скос в опционах). [[heston-model]] и модель SABR решают эту проблему.
- **Гауссовы логарифмические доходности**: реальные дневные доходности имеют избыточный эксцесс и отрицательную асимметрию. [[levy-processes]] (Variance Gamma, NIG) обобщают GBM с прыжками и более тяжёлыми хвостами.
- **Нет возврата к среднему**: GBM не является возвращающимся к среднему, что делает его непригодным для ставок, спредов или волатильности. Для них используются процессы Орнштейна-Уленбека или CIR.
- **Непрерывные траектории**: в GBM нет прыжков; модели с диффузией и прыжками (Merton 1976, Kou 2002) добавляют составной процесс Пуассона для захвата внезапных обвалов.

## Практическое применение

1. **Оценка опционов**: формула Блэка-Шоулза предполагает GBM; остаётся отраслевым эталоном.
2. **Симуляция Монте-Карло**: траектории GBM — стандарт для оценки путезависимых деривативов (азиатские, барьерные, lookback-опционы).
3. **Симуляция портфеля**: долгосрочные проекции благосостояния и пенсионные модели используют GBM для генерации сценариев доходностей.
4. **Реальные опционы**: решения о капиталовложениях (инвестировать сейчас vs. ждать) моделируются как опционы на стоимость проекта, следующую GBM.

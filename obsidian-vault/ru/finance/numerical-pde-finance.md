# Численные PDE-методы в финансах

Уравнение [[black-scholes]] в частных производных (PDE) и его обобщения редко допускают аналитические решения за пределами ванильных европейских опционов. **Конечно-разностные методы** (FDM) дискретизируют PDE на сетке и решают его в обратном времени от терминального условия, обеспечивая быстрое и точное ценообразование американских опционов, барьерных опционов и многофакторных задач.

## Уравнение Блэка-Шоулза

Для дериватива $V(S,t)$ на актив с GBM под мерой $\mathbb{Q}$:

$$
\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} + rS\frac{\partial V}{\partial S} - rV = 0
$$

с терминальным условием $V(S,T) = H(S)$. Решается **назад во времени** от $t = T$ к $t = 0$.

## Сведение к уравнению теплопроводности

Подстановка $x = \ln(S/K)$, $\tau = \frac{1}{2}\sigma^2(T-t)$, $V = K\,e^{-\alpha x - \beta\tau}\,u(x,\tau)$ при подходящих $\alpha, \beta$ сводит PDE Блэка-Шоулза к стандартному уравнению теплопроводности:

$$
\frac{\partial u}{\partial \tau} = \frac{\partial^2 u}{\partial x^2}
$$

Полезно для теоретического анализа и применения результатов математической физики. См. [[partial-differential-equations]].

## Конечно-разностные схемы

Дискретизация: $S_0, S_1, \ldots, S_M$ по пространству, $t_0, t_1, \ldots, t_N$ по времени. Обозначение $V_j^n \approx V(S_j, t_n)$.

### Явная схема (прямой Эйлер)

$$
\frac{V_j^{n+1} - V_j^n}{\Delta t} = \frac{1}{2}\sigma^2 S_j^2 \frac{V_{j+1}^n - 2V_j^n + V_{j-1}^n}{\Delta S^2} + rS_j\frac{V_{j+1}^n - V_{j-1}^n}{2\Delta S} - rV_j^n
$$

Каждое $V_j^{n+1}$ вычисляется явно — СЛАУ не решается. Но **условие устойчивости CFL** ограничивает шаг:

$$
\Delta t \leq \frac{\Delta S^2}{\sigma^2 S_{\max}^2}
$$

В логарифмической координате $x = \ln S$ с равномерным шагом $\Delta x$: $\Delta t \leq \Delta x^2 / \sigma^2$.

### Неявная схема (обратный Эйлер)

Требует решения **трёхдиагональной** СЛАУ на каждом шаге (алгоритм прогонки, $O(M)$). **Безусловно устойчива**. Первый порядок по времени, второй по пространству.

### Кранка-Николсон ($\theta = 1/2$)

Усредняет явную и неявную части:

$$
V_j^{n+1} - V_j^n = \frac{\Delta t}{2}\bigl[\mathcal{L}V^{n+1} + \mathcal{L}V^n\bigr]
$$

**Второй порядок по времени и пространству**, безусловно устойчива, но может порождать **паразитные осцилляции** вблизи разрывов (например, излом при $S = K$ для цифровых опционов). Исправление: несколько начальных неявных шагов перед переключением на КН (приём Раннахера).

### Общая $\Theta$-схема

$$
V^{n+1} = V^n + \Delta t\bigl[\theta\,\mathcal{L}V^{n+1} + (1-\theta)\,\mathcal{L}V^n\bigr]
$$

| $\theta$ | Схема | Точность | Устойчивость |
|:---------:|:------|:---------|:-------------|
| 0 | Явная | $O(\Delta t, \Delta S^2)$ | Условная (CFL) |
| 1 | Полностью неявная | $O(\Delta t, \Delta S^2)$ | Безусловная |
| 1/2 | Кранка-Николсон | $O(\Delta t^2, \Delta S^2)$ | Безусловная |

## Построение сетки

**Равномерная в логарифме**: $x_j = \ln S_{\min} + j\,\Delta x$ концентрирует узлы вблизи страйка.

**Неравномерная (растянутая)**: более мелкая сетка около $S = K$. Преобразования sinh или tanh:

$$
S(x) = K + \alpha\sinh\!\bigl(\beta(x - x_0)\bigr)
$$

Снижает число узлов для заданной точности в 3-5 раз.

## Граничные условия

При $S = 0$: для колла $V(0,t) = 0$; для пута $V(0,t) = Ke^{-r(T-t)}$.

При $S = S_{\max}$: для колла $V(S_{\max},t) \approx S_{\max} - Ke^{-r(T-t)}$; для пута $V = 0$. Граница должна быть достаточно далеко ($S_{\max} \geq 4K$).

## Американские опционы: задача со свободной границей

Досрочное исполнение вводит **условие дополнительности**:

$$
V(S,t) \geq H(S), \qquad (V - H)(\mathcal{L}V) = 0
$$

На каждом шаге: $V_j^{n+1} = \max(V_j^{n+1}, H(S_j))$. Методы:

- **Проектированная SOR**: итеративный решатель с проекцией на ограничение.
- **Метод штрафа**: добавление большого штрафного члена $\rho\,\max(H - V, 0)$ к PDE. См. [[optimal-stopping-american]].

## Многомерный случай: ADI-методы

Для двухфакторного опциона $V(S_1, S_2, t)$ PDE содержит перекрёстные производные. **Метод переменных направлений** (ADI) разделяет каждый шаг:

1. Неявно по $S_1$, явно по $S_2$ (полушаг).
2. Неявно по $S_2$, явно по $S_1$ (полушаг).

Схемы Дугласа-Рэкфорда и Хундсдорфера-Вервера расширяют ADI до 3+ измерений. Свыше 3 активов FDM непрактичен — предпочтителен [[monte-carlo-method]].

## Сравнение: PDE vs деревья vs Монте-Карло

| Критерий | FDM/PDE | Биномиальное дерево | Монте-Карло |
|:---------|:--------|:-------------------|:------------|
| **Размерность** | 1-3 | 1 | Любая |
| **Американские** | Естественно | Естественно | Регрессия (LSM) |
| **Сходимость** | $O(\Delta t^p + \Delta S^q)$ | $O(1/N)$ | $O(1/\sqrt{N})$ |
| **Греки** | Прямые | Bump-and-reprice | Pathwise / LR |

## Реализация на Python

```python
import numpy as np
from scipy.linalg import solve_banded

def cn_put(S0: float, K: float, T: float, r: float, sigma: float,
           M: int = 200, N: int = 200) -> float:
    """Кранк-Николсон для европейского пута."""
    x_min, x_max = np.log(S0 / 5), np.log(S0 * 5)
    dx = (x_max - x_min) / M
    dt = T / N
    x = np.linspace(x_min, x_max, M + 1)

    S = np.exp(x)
    V = np.maximum(K - S, 0.0)

    alpha = 0.5 * dt * (sigma**2 / dx**2 - (r - 0.5 * sigma**2) / dx)
    beta  = -dt * (sigma**2 / dx**2 + r)
    gamma = 0.5 * dt * (sigma**2 / dx**2 + (r - 0.5 * sigma**2) / dx)

    for n in range(N):
        rhs = np.copy(V)
        rhs[1:-1] += 0.5 * (alpha * V[:-2] + beta * V[1:-1] + gamma * V[2:])
        rhs[0]  = K * np.exp(-r * (N - n) * dt)
        rhs[-1] = 0.0

        lower = np.zeros(M + 1); diag = np.ones(M + 1); upper = np.zeros(M + 1)
        lower[2:]   = -0.5 * alpha
        diag[1:-1]  = 1 - 0.5 * beta
        upper[:-2]  = -0.5 * gamma

        ab = np.array([np.append(0, upper[:-1]), diag, np.append(lower[1:], 0)])
        V = solve_banded((1, 1), ab, rhs)

    idx = np.searchsorted(x, np.log(S0))
    w = (np.log(S0) - x[idx - 1]) / dx
    return V[idx - 1] * (1 - w) + V[idx] * w

put_price = cn_put(S0=100, K=100, T=1.0, r=0.05, sigma=0.20)
```

## Применения

1. **Американские опционы**: FDM — метод выбора при отсутствии аналитических решений.
2. **Барьерные опционы**: естественная обработка knock-in/knock-out границ на сетке.
3. **[[convertible-bonds]]**: PDE с кредитно-скорректированным дисконтированием и свободными границами.
4. **Локальная волатильность**: $\sigma(S,t)$ Дюпира входит в коэффициенты PDE напрямую.
5. **Риск в реальном времени**: предвычисленные PDE-сетки дают греки за субмиллисекунды через интерполяцию.

# Дискретное преобразование Фурье (ДПФ)

**Дискретное преобразование Фурье (ДПФ)** — конечномерный аналог [[fourier-transform]] для последовательностей из $N$ чисел. Переводит $N$ временны́х отсчётов в $N$ комплексных частотных коэффициентов. Именно ДПФ (а не непрерывное ПФ) используется на практике: реальные данные всегда дискретны и конечны. Быстрый алгоритм его вычисления — [[fft]].

## Определение

Для последовательности $x[0], x[1], \dots, x[N-1]$:

$$
X[k] = \sum_{n=0}^{N-1} x[n]\, e^{-i\frac{2\pi}{N}kn}, \quad k = 0, 1, \dots, N-1
$$

Обратное ДПФ:

$$
x[n] = \frac{1}{N}\sum_{k=0}^{N-1} X[k]\, e^{i\frac{2\pi}{N}kn}, \quad n = 0, 1, \dots, N-1
$$

**Матричная форма:** $\mathbf{X} = W_N \mathbf{x}$, где $W_N$ — матрица ДПФ со элементами $(W_N)_{kn} = e^{-2\pi i kn/N}$. Обратная: $W_N^{-1} = \tfrac{1}{N}W_N^*$ (сопряжённая-транспонированная / $N$).

## Интерпретация коэффициентов

$k$-й коэффициент $X[k]$ соответствует частоте:

$$
f_k = \frac{k \cdot f_s}{N}
$$

где $f_s$ — **частота дискретизации** (отсчётов в секунду). Период соответствующей гармоники: $T_k = N / (k \cdot f_s)$.

**Однозначность:** ДПФ периодично с периодом $N$, т.е. $X[k] = X[k + N]$. Для вещественного $x[n]$ спектр **симметричен**: $X[N-k] = X^*[k]$. Это значит, что независимой информации содержится лишь в $k = 0, 1, \dots, \lfloor N/2 \rfloor$.

| Коэф. | Частота | Смысл |
| :--- | :--- | :--- |
| $X[0]$ | $0$ | DC-компонента (среднее значение) |
| $X[1]$ | $f_s/N$ | Наинизшая ненулевая частота |
| $X[N/2]$ | $f_s/2$ | Частота Найквиста (верхний предел) |
| $X[N-k]$ | $-f_k$ | Зеркало $X[k]$ для вещественного сигнала |

## Теорема Найквиста–Шеннона

При дискретизации непрерывного сигнала с частотой $f_s$ корректно восстанавливаются только компоненты с частотой $f < f_s/2$ (**частота Найквиста** $f_{Nyq}$).

$$
f_{Nyq} = \frac{f_s}{2}
$$

Компоненты выше $f_{Nyq}$ **накладываются** (aliasing) на низкочастотные — неотличимы от них. Для предотвращения перед дискретизацией применяют **антиалиасинговый фильтр** нижних частот.

## Спектральная утечка и оконные функции

**Утечка спектра** возникает, когда частота сигнала не кратна $f_s/N$: энергия «вытекает» из одного бина в соседние. Причина — периодическое продолжение конечного отрезка порождает разрывы на его краях.

Решение — **оконная функция** $w[n]$, умножённая на сигнал перед ДПФ:

$$
X_w[k] = \sum_{n=0}^{N-1} x[n]\,w[n]\, e^{-i\frac{2\pi}{N}kn}
$$

| Окно | Уровень боковых лепестков | Ширина главного лепестка | Применение |
| :--- | :--- | :--- | :--- |
| **Прямоугольное** | −13 дБ | Минимальная | Когда нет утечки |
| **Хэнн** | −31 дБ | Умеренная | Общего назначения |
| **Хэмминг** | −41 дБ | Умеренная | Речевые сигналы |
| **Блэкман** | −57 дБ | Широкая | Высокая точность |
| **Блэкман-Харрис** | −92 дБ | Широкая | Спектральный анализ |
| **Флэттоп** | −93 дБ | Очень широкая | Измерение амплитуды |

Компромисс: чем лучше подавление боковых лепестков, тем шире главный лепесток (хуже разрешение по частоте).

## Спектральная плотность мощности (PSD)

Односторонняя PSD из ДПФ:

$$
S[k] = \frac{2}{N \cdot f_s}\,|X[k]|^2, \quad k = 1, \dots, \lfloor N/2 \rfloor
$$

Теорема Парсеваля для ДПФ:

$$
\sum_{n=0}^{N-1} |x[n]|^2 = \frac{1}{N}\sum_{k=0}^{N-1}|X[k]|^2
$$

Полная мощность сигнала одинакова в обоих доменах.

## Zero-padding

Добавление $M$ нулей в конец сигнала перед ДПФ даёт $N + M$ коэффициентов — **интерполяцию** спектра, но не новую информацию:

- Разрешение по частоте $\Delta f = f_s/N$ **не улучшается** (определяется только длиной данных)
- Пики становятся более «гладкими» — удобнее для визуализации и интерполяции истинной частоты пика
- Стандартная практика: дополнять до ближайшей степени 2 для [[fft]]

## Python-реализация

```python
import numpy as np


def dft_matrix(N: int) -> np.ndarray:
    """
    Матрица ДПФ размера N×N.

    (W_N)_{kn} = exp(-2πi·kn/N)
    Прямое ДПФ: X = W @ x
    Обратное:   x = (1/N) * W.conj().T @ X
    """
    n = np.arange(N)
    k = n[:, None]
    return np.exp(-2j * np.pi * k * n / N)


def dft(x: np.ndarray) -> np.ndarray:
    """Наивное ДПФ через матричное умножение — O(N²)."""
    return dft_matrix(len(x)) @ x


def idft(X: np.ndarray) -> np.ndarray:
    """Наивное обратное ДПФ — O(N²)."""
    N = len(X)
    return (dft_matrix(N).conj().T @ X) / N


def power_spectrum(
    x: np.ndarray,
    fs: float = 1.0,
    window: str = "hann",
) -> dict:
    """
    Односторонняя спектральная плотность мощности.

    Args:
        x:      временной ряд длины N
        fs:     частота дискретизации (отсч./с)
        window: 'hann' | 'blackman' | 'hamming' | 'none'

    Returns:
        dict с 'freqs' (Гц), 'psd', 'amplitude', 'phase_deg'
    """
    N = len(x)

    win = {
        "hann":     np.hanning(N),
        "blackman": np.blackman(N),
        "hamming":  np.hamming(N),
        "none":     np.ones(N),
    }.get(window, np.hanning(N))

    # Нормировка по coherent gain сохраняет амплитуду
    x_win = x * win / win.mean()

    X = np.fft.rfft(x_win)
    freqs = np.fft.rfftfreq(N, d=1.0 / fs)

    # Одностороннее нормирование: ×2 для всех бинов, кроме DC и Найквиста
    amp = np.abs(X) * 2 / N
    amp[0] /= 2
    if N % 2 == 0:
        amp[N // 2] /= 2

    psd = amp ** 2 / fs          # В единицах [сигнал²/Гц]

    return {
        "freqs":     freqs,
        "psd":       psd,
        "amplitude": amp,
        "phase_deg": np.angle(X, deg=True),
    }


def spectral_resolution(N: int, fs: float) -> dict:
    """Параметры спектра: разрешение, Найквист, число бинов."""
    return {
        "delta_f":  fs / N,
        "nyquist":  fs / 2,
        "n_bins":   N // 2 + 1,
        "max_period": N / fs,
    }


# ---- Пример: утечка спектра с оконной функцией и без ----
np.random.seed(0)
N, fs = 256, 1.0                 # 256 ежедневных отсчётов

# Частота НЕ кратна f_s/N → сильная утечка без окна
f_signal = 0.123                 # «некратная» частота
t = np.arange(N)
x = np.sin(2 * np.pi * f_signal * t)

spec_rect = power_spectrum(x, fs, window="none")
spec_hann = power_spectrum(x, fs, window="hann")

# Пик без окна — «размазан» по многим бинам
peak_rect = spec_rect["freqs"][np.argmax(spec_rect["psd"])]
peak_hann = spec_hann["freqs"][np.argmax(spec_hann["psd"])]
print(f"Истинная частота:    {f_signal:.4f}")
print(f"Пик (прямоугольное): {peak_rect:.4f}")
print(f"Пик (Хэнн):         {peak_hann:.4f}")

res = spectral_resolution(N, fs)
print(f"Разрешение Δf = {res['delta_f']:.4f}, Найквист = {res['nyquist']:.3f}")
```

## Ограничения

- **Стационарность:** ДПФ предполагает неизменность частотных характеристик на всём отрезке. Для нестационарных сигналов — [[wavelet-transform]] или STFT (краткосрочное ПФ).
- **Периодическое продолжение:** ДПФ неявно считает сигнал периодическим. Если конец не совпадает с началом — возникают краевые артефакты (устраняются окнами).
- **Разрешение ограничено:** $\Delta f = f_s / N$ фиксировано данными. Zero-padding лишь интерполирует спектр.
- **Чувствительность к выбросам:** Одиночный выброс в $x[n]$ влияет на все $N$ коэффициентов.

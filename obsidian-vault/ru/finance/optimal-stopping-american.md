# Оптимальная остановка и американские опционы

Американский опцион отличается от европейского одной деталью — его можно исполнить **в любой момент** до экспирации, а не только в финал. Эта свобода превращает задачу ценообразования из вычисления ожидания в **задачу оптимальной остановки**: держатель опциона выбирает случайный момент времени $\tau$ так, чтобы максимизировать дисконтированную ожидаемую выплату.

Ширяев ([[shiryaev-stochastic-finance|т. 2, гл. VI, § 2]]) даёт исчерпывающее изложение через **супермартингальную характеризацию** цены — конструкцию Снелла. Это один из самых элегантных кусков всего тома.

## Платёжное обязательство

Пусть $G_n \geq 0$ — выплата, доступная держателю при исполнении в момент $n$. Например, для американского пут-опциона с страйком $K$:

$$G_n = (K - S_n)^+.$$

Цель держателя — найти **момент остановки** $\tau \leq N$ (с произвольным распределением, согласованным с фильтрацией $\mathcal{F}_n$), максимизирующий дисконтированное ожидание выплаты:

$$\sup_{\tau} \mathbb{E}^\mathbb{Q}\!\left[ \frac{G_\tau}{B_\tau} \right],$$

где $\mathbb{Q}$ — эквивалентная мартингальная мера. На полном рынке $\mathbb{Q}$ единственна и эта величина — корректная цена американского опциона; на неполном — берётся [[upper-lower-hedging-prices|верхняя цена]].

## Огибающая Снелла

Ключевой объект — **огибающая Снелла** (Snell envelope) последовательности дисконтированных выплат $\widetilde{G}_n = G_n / B_n$.

**Определение.** Огибающая Снелла $V = (V_n)_{n=0}^N$ задаётся обратной индукцией:

$$V_N = \widetilde{G}_N, \qquad V_n = \max\!\left( \widetilde{G}_n, \; \mathbb{E}^\mathbb{Q}[V_{n+1} \mid \mathcal{F}_n] \right), \quad n < N.$$

Интерпретация: на каждом шаге держатель сравнивает «исполнить сейчас» ($\widetilde{G}_n$) с «удержать и продолжать» ($\mathbb{E}^\mathbb{Q}[V_{n+1} \mid \mathcal{F}_n]$) и выбирает большее.

**Утверждение 1.** $V_n$ — наименьший $\mathbb{Q}$-супермартингал, мажорирующий $\widetilde{G}$:

$$V_n \geq \widetilde{G}_n, \qquad \mathbb{E}^\mathbb{Q}[V_{n+1} \mid \mathcal{F}_n] \leq V_n.$$

**Утверждение 2 (Доб-Мейер).** $V$ единственным образом раскладывается как $V_n = M_n - A_n$, где $M$ — $\mathbb{Q}$-мартингал, а $A$ — предсказуемый возрастающий процесс с $A_0 = 0$. Этот процесс $A$ показывает, насколько «рано» опцион стоит исполнить.

## Оптимальный момент остановки

**Теорема.** Цена американского опциона в момент 0 есть $V_0$, а оптимальный момент остановки

$$\tau^* = \inf\{ n \geq 0 \,:\, V_n = \widetilde{G}_n \}.$$

То есть исполнять надо в первый момент, когда «удержание» перестаёт давать преимущество над немедленной выплатой.

**Доказательство (эскиз).** Сверху: для любого $\tau$ в силу свойства супермартингальности $V_0 \geq \mathbb{E}^\mathbb{Q}[V_\tau] \geq \mathbb{E}^\mathbb{Q}[\widetilde{G}_\tau]$, значит $V_0$ — верхняя оценка цены. Снизу: при $\tau = \tau^*$ процесс $V$ остановленный в момент $\tau^*$ становится мартингалом (поскольку $A$ ещё не возросло), и $V_0 = \mathbb{E}^\mathbb{Q}[V_{\tau^*}] = \mathbb{E}^\mathbb{Q}[\widetilde{G}_{\tau^*}]$.

## Хеджирование американского опциона

Как воспроизвести продавцу обязательство, которое может быть исполнено в любой момент? Он должен иметь портфель, **в каждый момент** покрывающий текущую выплату $\widetilde{G}_n$, а не только в финал.

Это и есть смысл [[optional-decomposition|опционального разложения]]: $V_n = M_n - A_n$. Мартингальная часть $M$ — это в точности воспроизводящий портфель (его капитал = $M$); возрастающая часть $A$ — это «отвод средств», который продавец может позволить себе изымать, пока опцион не исполнен. Подробности — в отдельной статье.

## Связь с PDE: задача Стефана

В **непрерывном** времени оптимальная остановка превращается в задачу с препятствием — **задачу Стефана**: цена $V(t, S)$ удовлетворяет PDE Блэка-Шоулса в области, где «удерживать» оптимально, и совпадает с выплатой $G(S)$ в области, где «исполнить» оптимально. Граница между этими областями — **свободная граница** (free boundary), и её положение само определяется решением PDE.

Для американского пут-опциона на бездивидендной акции в модели Блэка-Шоулса свободная граница $S^*(t)$ — кривая, ниже которой пут стоит немедленно исполнить. Аналитического выражения для $S^*$ не существует (несмотря на десятилетия попыток); численно — конечно-разностный метод или биномиальное дерево.

Подробное обсуждение — в статье [[stefan-problem-american|задача Стефана и американские опционы]].

## Отличие от европейского

На бездивидендной акции американский **колл** имеет ту же цену, что и европейский (это знаменитый результат Мертона: оптимальный момент остановки для колла — всегда $\tau = N$, т. е. ждать до конца). Для **пута** же досрочное исполнение может быть оптимальным, и американский пут стоит **строго дороже** европейского. Именно для пута и задача Стефана нетривиальна.

# Ортогональное стекирование альф и резидуализация

**Ортогональное стекирование альф (Orthogonal Alpha Stacking)** — это метод комбинирования нескольких предсказательных сигналов (альф) таким образом, чтобы каждый новый сигнал приносил уникальную информацию. Если два сигнала сильно коррелируют, их простое сложение не улучшает качество прогноза; резидуализация гарантирует, что каждый сигнал в стеке «уникален».

## Математическая формулировка

Пусть имеется набор сигналов $A_1, A_2, \dots, A_n$. Мы хотим добавить новый сигнал $A_{new}$ так, чтобы он был ортогонален имеющимся. Это достигается через **резидуализацию** (выделение остатков).

### Резидуализация через регрессию

Чтобы сделать сигнал $Y$ (Альфа 2) ортогональным сигналу $X$ (Альфа 1), мы строим линейную регрессию:
$$
Y = \beta X + \epsilon
$$
Остаток $\epsilon = Y - \hat{\beta} X$ — это часть Альфы 2, которая не коррелирует с Альфой 1. Итоговая комбинированная альфа:
$$
\alpha_{stacked} = w_1 A_1 + w_2 \epsilon
$$

## Очистка от факторов (Factor Neutralization)

Резидуализация часто используется для удаления влияния общих факторов риска (рынок, сектора). Если $f$ — вектор экспозиций на факторы:
$$
\alpha_{clean} = \alpha - f(f^T f)^{-1} f^T \alpha
$$
Это гарантирует, что альфа-сигнал является **рыночно-нейтральным** или **секторно-нейтральным**.

## Применение в финансах

1. **Мультифакторные модели**: Объединение сигналов Momentum, Value и Quality без «двойного учета» одного и того же эффекта.
2. **Проп-трейдинг**: Проверка того, приносит ли новая стратегия реальную прибыль или она просто копирует существующий портфель стратегий.
3. **Управление рисками**: Определение того, вызвана ли доходность портфеля мастерством (альфой) или скрытым риском (бетой).

## Python: Резидуализация сигналов

```python
import numpy as np
from sklearn.linear_model import LinearRegression

def get_unique_alpha(target, reference):
    """
    Возвращает часть целевого сигнала, ортогональную базовому.
    """
    model = LinearRegression()
    model.fit(reference.reshape(-1, 1), target)
    prediction = model.predict(reference.reshape(-1, 1))
    return target - prediction

# Пример: Два коррелирующих сигнала
alpha1 = np.random.normal(0, 1, 100)
alpha2 = alpha1 * 0.9 + np.random.normal(0, 0.1, 100)

clean_alpha2 = get_unique_alpha(alpha2, alpha1)
corr = np.corrcoef(alpha1, clean_alpha2)[0, 1]

print(f"Корреляция после очистки: {corr:.10f}")
```

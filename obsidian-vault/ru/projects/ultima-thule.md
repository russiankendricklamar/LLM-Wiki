# Ultima Thule

**Ultima Thule** — нативное iOS-приложение для планирования путешествий. Объединяет AI-генератор маршрутов, трекинг рейсов в реальном времени, умный чемодан, офлайн-карты, AR-навигацию и дневник в одном месте. Все внешние API проходят через Supabase Edge Functions — работает из любой страны без VPN.

## Мотивация

Планирование поездки разбросано по 6+ приложениям: рейсы, отели, маршруты, расходы, погода, документы — всё в разных местах, нет персонализации, офлайн не работает. Ultima Thule убирает этот хаос.

## Возможности

**AI Trip Generator** — 4-шаговый визард: направление (с подсказками AI на основе сезона и дешёвых рейсов), даты, бюджет (слайдер + чипы Эконом/Средний/Без ограничений), стиль поездки (Активный, Культурный, Гастро и др.). AI строит маршрут по дням с реальными ценами через Aviasales/Travelpayouts.

**Трекинг рейсов** — статус, задержки, гейты в реальном времени через AirLabs. Посадочные талоны добавляются через OCR-сканирование или парсинг Gmail/Яндекс.Почты.

**Умная паковка** — AI генерирует список под конкретную поездку с учётом климата, длительности и профиля путешественника.

**Дневник** — drag-and-drop фотокнига с геотегами, заметками и экспортом. Отдельный вид Memory Book.

**GPS-трекинг маршрута** — запись пути в реальном времени, восстановление трекинга после перезапуска приложения. Геозоны с уведомлениями ("Отметить посещённым").

**Карта посещённых стран** — визуализация всей географии поездок.

**AR-навигация** — ARKit-оверлей с указателями к точкам маршрута через RealityKit.

**Офлайн-доступ** — MKMapSnapshotter кэширует тайлы карт, погода и данные мест сохраняются локально через OfflineCacheManager.

**Widget Extension** — виджет на домашнем экране с информацией о текущей поездке.

## Архитектура

- **UI**: SwiftUI + SwiftData (локальное хранилище с миграциями)
- **Бэкенд**: Supabase — авторизация (Apple/Google/Email + OAuth callback), синхронизация через SyncManager, хранилище фото, Edge Functions как единый API-прокси
- **AI-слой**: Gemini (основной), Claude, GPT-4 — через облачный прокси, без ключей в приложении
- **Карты**: MapKit + CoreLocation + GeofenceManager
- **AR**: ARKit + RealityKit
- **Валюты**: CurrencyService с авто-обновлением и паузой в фоне
- **Безопасность**: биометрическая блокировка при выходе из фона, Secrets через xcconfig (не в коде)

## Дизайн

Несколько цветовых палитр на выбор (включая Sakura), полная поддержка тёмной и светлой темы. Языки: русский и английский через Localizable.xcstrings.

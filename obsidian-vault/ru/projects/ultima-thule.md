# Ultima Thule

**Ultima Thule** — нативное iOS-приложение для планирования путешествий. Объединяет AI-генератор маршрутов, трекинг рейсов в реальном времени, умный чемодан, офлайн-карты и AR-навигацию в одном месте. Работает из любой страны через облачный API-прокси на Supabase.

## Мотивация

Планирование поездки разбросано по 6+ приложениям: рейсы, отели, маршруты, расходы, погода, документы — всё в разных местах, нет персонализации, офлайн не работает. Ultima Thule убирает этот хаос.

## Возможности

- **AI Trip Generator**: 4-шаговый визард — AI строит маршрут по интересам, стилю и бюджету; интеграция с Aviasales для реальных цен на билеты
- **Трекинг рейсов**: статус, задержки, гейты в реальном времени через AirLabs; OCR-сканирование бронирований и парсинг Gmail/Яндекс.Почты
- **Умная паковка**: AI-генерация списка под конкретную поездку с учётом климата и профиля путешественника
- **Дневник путешествий**: drag-and-drop фотокнига, геотеги, заметки, экспорт
- **AR-навигация**: ARKit-оверлей с указателями к точкам маршрута
- **Офлайн-доступ**: MKMapSnapshotter кэширует карты, погода и данные мест сохраняются локально

## Архитектура

- **UI**: SwiftUI + SwiftData (локальное хранилище)
- **Бэкенд**: Supabase — авторизация (Apple/Google/Email), синхронизация, хранилище фото, Edge Functions как единый API-прокси для всех внешних сервисов
- **AI-слой**: Gemini (основной), Claude, GPT-4 — через облачный прокси, без ключей в приложении
- **Карты**: MapKit + CoreLocation + GeofenceManager
- **Дополненная реальность**: ARKit + RealityKit

## Статус

В активной разработке, 26 сессий. Все основные модули реализованы; идёт полировка UX и интеграция Live Activities.

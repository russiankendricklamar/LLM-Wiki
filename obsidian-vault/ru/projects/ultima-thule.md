# Ultima Thule

**Ultima Thule** — приложение для путешествий и исследований, созданное для тех, кто хочет выйти за пределы проторённых маршрутов. Название отсылает к древнегреческой концепции далёкой таинственной земли на краю известного мира.

## Концепция

Большинство туристических приложений оптимизированы под эффективность — самый быстрый маршрут, самый популярный ресторан, самый дешёвый отель. Ultima Thule создан для исследования: обнаружения малоизвестных исторических мест, поиска правильного контекста для локации и планирования путешествий вокруг любопытства, а не удобства.

## Возможности

- **Умный конструктор маршрутов**: генерирует планы поездок на основе интересов, а не только близости
- **Офлайн-карты**: полные данные маршрута кешируются для районов с плохим соединением
- **Контекстуальное открытие**: историческая, культурная и географическая информация о каждой локации
- **Дневник путешествий**: документируйте исследования с заметками, фото и геотегами

## Стек

- **Мобильный**: React Native с Expo
- **Бэкенд**: Supabase (PostgreSQL + Edge Functions)
- **Карты**: Mapbox GL для карт с кастомными стилями
- **AI-слой**: Claude для генерации контекста и предложений маршрутов

## Статус

MVP в разработке. Основные функции карт и маршрутов работают; AI-генерация контекста находится в процессе интеграции.

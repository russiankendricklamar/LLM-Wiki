# Zeta Terminal

**Zeta Terminal** — AI-native рабочее пространство для специалистов в области количественных финансов и AI-исследований. Объединяет данные рынка в реальном времени, высокопроизводительные численные вычисления и помощь LLM в единую терминальную среду.

## Мотивация

Современные рабочие процессы квантов разбросаны по Bloomberg-терминалам, Jupyter-ноутбукам, риск-системам и чат-интерфейсам. Zeta Terminal стремится свернуть этот стек — чтобы можно было запустить калибровку стохастической волатильности, запросить рыночные данные и попросить LLM интерпретировать результаты, не переключая контекст.

## Архитектура

- **Среда выполнения**: Python + FastAPI бэкенд с WebSocket-стримингом
- **Численный движок**: QuantLib для оценки деривативов, scipy для оптимизации
- **LLM-слой**: Claude API для генерации кода, интерпретации моделей и запросов к финансовым данным на естественном языке
- **Фронтенд**: TypeScript + React терминальный UI с обновлением графиков в реальном времени

## Ключевые возможности

- Диалоговый интерфейс для количественного анализа ("Какова поверхность implied vol для SPX?")
- Живая калибровка моделей Хестона, SABR и jump-diffusion
- Интегрированный бэктестинг с портфельной аналитикой
- LLM-помощь в генерации отчётов из сырых выходных данных моделей

## Статус

В активной разработке. Основной слой приёма данных и маршрутизация LLM функциональны; движок оценки деривативов находится в процессе интеграции.

# NLP в финансовом анализе

## Обзор

Финансовые рынки генерируют огромные объемы неструктурированного текста: расшифровки звонков по итогам квартала, отчеты SEC (10-K, 8-K), аналитические обзоры, протоколы центробанков, новостные ленты и социальные сети. Извлечение структурированных сигналов из этого корпуса — ключевая область исследований с 2000-х годов. Путь пролегал от словарей сентимента (Loughran-McDonald, 2011) до дообученных трансформеров (FinBERT, 2019) и современных [[llm|LLM]], способных к глубоким рассуждениям.

Сигналы NLP относятся к **альтернативным данным**, так как они ортогональны информации о ценах и объемах, обеспечивая инкрементальную альфу. Событийные стратегии (trading material events) полностью полагаются на быстрый и точный парсинг текстов.

## Математический аппарат

### Сентимент как латентная переменная

Пусть $D_t$ — финансовый документ в момент $t$. Сентимент менеджмента $\psi_t \in \{-1, 0, +1\}$ (медвежий, нейтральный, бычий) — латентная переменная. Классификатор $p_\theta(\psi \mid D_t)$ аппроксимирует апостериорную вероятность. Торговый сигнал:

$$
z_t = \mathbb{E}_{p_\theta}[\psi \mid D_t] = p_\theta(\psi=+1 \mid D_t) - p_\theta(\psi=-1 \mid D_t)
$$

Для предсказания доходности используется модель:

$$
r_{t+\Delta} = \beta_0 + \beta_1 z_t + \beta_2 z_t^{\text{surprise}} + \varepsilon_t
$$

где $z_t^{\text{surprise}} = z_t - \mathbb{E}[z_t \mid \text{prior calls}]$ отражает изменение сентимента относительно исторической базы.

### Дообучение FinBERT

FinBERT (Yang et al., 2020) — это BERT, предобученный на Reuters, отчетах 10-K и звонках аналитиков. Голова классификации вычисляет:

$$
p(\psi \mid D) = \text{softmax}(W_c \cdot \text{CLS}(D) + b_c)
$$

где CLS$(D)$ — эмбеддинг токена [CLS]. FinBERT достигает ~88% точности на 3-классовом финансовом сентименте против ~75% у общего BERT.

### Извлечение информации через [[llm|LLM]]

Для структурированного извлечения из отчетов 8-K LLM вычисляет:

$$
P(\text{fields} \mid D, \text{schema}) \propto \prod_{j} P(\text{field}_j \mid D, \text{schema}, \text{fields}_{<j})
$$

Константное декодирование обеспечивает соблюдение JSON-схемы на уровне токенов. См. [[llm-financial-analysis|LLM в финансовом анализе]] для описания полного [[rag|RAG]]-пайплайна.

## Основные подходы

### Словарь Loughran-McDonald
Список слов (позитивные/негативные/рисковые), специфичный для финансов. Быстр и интерпретируем, но не учитывает контекст и иронию. Является стандартным академическим бейзлайном.

### Финансовые трансформеры (FinBERT)
Модели, учитывающие специфику жаргона и отрицаний («not expecting growth»). BloombergGPT расширяет этот подход на задачи генерации и сложных рассуждений.

### Майнинг отчетов 10-K / 8-K
Анализ рисков (Item 1A) и тона обсуждения менеджмента (MD&A). Позволяет классифицировать события (M&A, смена CEO) за секунды после публикации.

### Анализ звонков инвесторов (Earnings Calls)
Исследования показывают, что секция вопросов-ответов (Q&A) содержит больше инкрементальной информации, чем подготовленная речь. Анализируется сдвиг тона, частота слов неуверенности и разрыв в сентименте между звонками.

## Пример на Python

```python
from transformers import pipeline

class FinancialSentimentAnalyser:
    """Анализ сентимента на базе FinBERT."""
    
    def __init__(self, model_name: str = "ProsusAI/finbert") -> None:
        self.pipe = pipeline(
            "text-classification",
            model=model_name,
            return_all_scores=True
        )

    def analyse(self, text: str):
        # Разбиение на куски по 512 токенов и усреднение
        scores = self.pipe(text, truncation=True)[0]
        agg = {s["label"]: s["score"] for s in scores}
        # Сигнал: positive - negative
        return agg["positive"] - agg["negative"]
```

## Ключевые результаты

**Loughran & McDonald (2011)**: Финансовые словари значительно лучше предсказывают доходность акций, чем общие словари. Частота позитивных слов в 10-K коррелирует с аномальной доходностью в следующем месяце.

**NLP в звонках инвесторов (Li et al., 2021)**: Лонг-шорт стратегия на базе сентимента звонков достигает 3–5% годовой аномальной доходности при Шарпе ~0.7.

## Ограничения

**Рыночное воздействие**: Как только сигнал становится стандартом, его доходность падает. Преимущество смещается в сторону скорости парсинга и охвата редких источников.

**Стилистика менеджмента**: Компании знают об инструментах NLP и активно управляют тоном. Исследования отмечают «стратегическое запутывание» — использование более сложных предложений при сообщении плохих новостей.

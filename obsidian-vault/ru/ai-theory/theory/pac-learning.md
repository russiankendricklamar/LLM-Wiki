# PAC-обучение и VC-размерность

## Обзор

PAC-обучение (Probably Approximately Correct — вероятно приблизительно корректное), введённое Лесли Валиантом в 1984 году, — фундаментальная теория, объясняющая, когда и почему алгоритмы машинного обучения обобщаются с конечных обучающих данных на новые примеры. PAC-теория отвечает на вопрос: _сколько обучающих примеров достаточно, чтобы гарантировать вероятную приблизительную корректность выученной гипотезы на новых данных?_

Фреймворк формализует статистическое обучение в безраспределительном сеттинге и лежит в основе теории SVMs, деревьев решений и глубокого обучения. Его ключевой вклад — перевод неформального понятия «обучение» в точные математические гарантии через параметры ошибки $\epsilon$ и доверия $1-\delta$.

## Математическая основа

**Определение PAC-обучения.** Пусть $\mathcal{X}$ — пространство входов, $\mathcal{Y} = \{0,1\}$ — метки, $\mathcal{H}$ — класс гипотез, $\mathcal{D}$ — неизвестное распределение на $\mathcal{X} \times \mathcal{Y}$.

Алгоритм $\mathcal{A}$ PAC-обучает $\mathcal{H}$, если для всех $\epsilon, \delta \in (0,1)$ и всех распределений $\mathcal{D}$ при объёме выборки:

$$m \geq \frac{1}{\epsilon}\left(\ln\frac{|\mathcal{H}|}{\delta}\right)$$

$\mathcal{A}$ возвращает $h \in \mathcal{H}$ такое, что с вероятностью не менее $1-\delta$:

$$R(h) = \Pr_{(x,y)\sim\mathcal{D}}[h(x) \neq y] \leq \epsilon$$

Для бесконечных классов гипотез вместо $|\mathcal{H}|$ используется VC-размерность.

**VC-размерность.** VC-размерность (Вапника-Червоненкиса) класса $\mathcal{H}$:

$$\text{VC}(\mathcal{H}) = \max\{|S| : \Pi_{\mathcal{H}}(S) = 2^{|S|}\}$$

где $\Pi_{\mathcal{H}}(S) = |\{h|_S : h \in \mathcal{H}\}|$ — число различных разметок, которые $\mathcal{H}$ порождает на множестве $S$. Множество $S$ **разбивается** $\mathcal{H}$, если $\Pi_{\mathcal{H}}(S) = 2^{|S|}$ (достижима любая разметка).

**Лемма Сауэра.** Функция роста ограничена:

$$\Pi_{\mathcal{H}}(m) \leq \sum_{i=0}^{d} \binom{m}{i} \leq \left(\frac{em}{d}\right)^d$$

где $d = \text{VC}(\mathcal{H})$.

**Фундаментальная теорема статистического обучения.** Следующие утверждения равносильны:
1. $\mathcal{H}$ PAC-обучаем (конечная сложность выборки)
2. $\text{VC}(\mathcal{H}) < \infty$
3. $\mathcal{H}$ удовлетворяет свойству равномерной сходимости

**Сложность выборки через VC-размерность:**

$$m \geq \frac{C}{\epsilon^2}\left(d \ln\frac{1}{\epsilon} + \ln\frac{1}{\delta}\right)$$

где $d = \text{VC}(\mathcal{H})$, $C$ — абсолютная константа.

## Принцип работы

**Границы обобщения.** Через неравенство объединения и неравенство Хёффдинга:

$$R(h_S) \leq \hat{R}(h_S) + \sqrt{\frac{\ln|\mathcal{H}| + \ln(1/\delta)}{2m}}$$

где $\hat{R}$ — эмпирический риск, $h_S$ — гипотеза ERM.

**Сложность Радемахера.** Более точная, зависящая от данных мера:

$$\mathcal{R}_m(\mathcal{H}) = \mathbb{E}_{S, \sigma}\left[\sup_{h \in \mathcal{H}} \frac{1}{m}\sum_{i=1}^m \sigma_i h(x_i)\right]$$

где $\sigma_i$ — радемахеровские переменные ($\pm 1$ с равными вероятностями).

**Декомпозиция смещение-дисперсия.** В языке PAC: сложный $\mathcal{H}$ (высокая VC-размерность) имеет малое смещение, но высокую дисперсию; простой $\mathcal{H}$ — наоборот. Оптимальный $\mathcal{H}$ минимизирует их сумму.

**MDL и бритва Оккама.** PAC-теория формализует бритву Оккама: предпочтительна гипотеза с малой обучающей ошибкой и коротким описанием. Это связывает PAC-теорию с принципом минимальной длины описания (MDL) и [[kolmogorov-complexity]].

## Реализация на Python

```python
import numpy as np
from itertools import combinations

def can_shatter_with_intervals(points, n_grid=50):
    """Проверяет, разбивается ли набор точек интервальными классификаторами."""
    labellings_achieved = set()
    xs = np.linspace(-0.1, 1.1, n_grid)
    for a in xs:
        for b in xs[xs > a]:
            labelling = tuple((a < points[i] < b) for i in range(len(points)))
            labellings_achieved.add(labelling)
    return len(labellings_achieved) == 2 ** len(points)


def find_vc_dim(n_candidates=10):
    """Находит VC-размерность интервальных классификаторов эмпирически."""
    points = np.linspace(0.1, 0.9, n_candidates)

    print("=== Интервальные классификаторы h_{a,b}(x) = 1[a < x < b] ===")
    for k in range(1, 5):
        shattered = False
        for subset in combinations(range(n_candidates), k):
            pts = np.array([points[i] for i in subset])
            if can_shatter_with_intervals(pts):
                shattered = True
                break
        print(f"  Разбиваются {k} точек: {shattered}")
    print("  => VC-размерность = 2")


def pac_sample_complexity(vc_dim, epsilon, delta):
    """Верхняя оценка сложности PAC-выборки."""
    return (8 / epsilon**2) * (vc_dim * np.log(16 / epsilon) + np.log(2 / delta))


find_vc_dim()

print("\n=== Границы сложности выборки (eps=0.05, delta=0.05) ===")
for d in [1, 10, 100, 1000]:
    m = pac_sample_complexity(d, epsilon=0.05, delta=0.05)
    print(f"  VC-размерность={d:4d}: нужно ~{int(m):,} примеров")


def empirical_rademacher(X, n_trials=1000):
    """Эмпирическая сложность Радемахера для линейных классификаторов."""
    m = X.shape[0]
    rads = []
    for _ in range(n_trials):
        sigma = np.random.choice([-1, 1], size=m)
        rad = np.linalg.norm(sigma @ X) / m
        rads.append(rad)
    return float(np.mean(rads))


X = np.random.randn(100, 10)
X_norm = X / np.linalg.norm(X, axis=1, keepdims=True)
print(f"\nЭмпирическая сложность Радемахера: {empirical_rademacher(X_norm):.4f}")
```

## Ключевые результаты и свойства

- **Фундаментальная теорема:** $\mathcal{H}$ PAC-обучаем $\iff$ $\text{VC}(\mathcal{H}) < \infty$. Один из самых чётких результатов в теории обучения.
- **Точные оценки для SVM:** VC-размерность линейных классификаторов в $\mathbb{R}^d$ равна точно $d+1$; оценки с отступом для SVM точнее через размерность жирного разбиения.
- **Парадокс глубокого обучения:** Современные нейронные сети имеют VC-размерность, многократно превышающую объём выборки, и при этом хорошо обобщаются — стандартные PAC-оценки слишком слабы. Это мотивирует изучение алгоритмически-зависимых оценок, PAC-Bayes и [[neural-scaling-laws]].
- **Нижние оценки:** Информационно-теоретические нижние оценки показывают $\Omega(d/\epsilon)$, так что VC-размерность оптимальна с точностью до логарифмических факторов.

## Ограничения

**Сеттинг худшего случая.** PAC-оценки безраспределительные и наихудшего случая, что делает их слабыми для структурированных распределений. Байесовская теория или PAC-Bayes дают более точные практические предсказания.

**Реализуемое предположение.** Классическое PAC-обучение предполагает, что цель лежит в $\mathcal{H}$. Агностическое PAC более реалистично, но требует больше примеров.

**Разрыв с глубоким обучением.** PAC-оценки для переопараметризованных нейронных сетей тривиально велики. [[neural-scaling-laws]] эмпирически характеризуют обобщение за пределами классической VC-теории.

**Вычислительная сложность.** PAC-обучаемость ничего не говорит об эффективности. Многие PAC-обучаемые классы требуют экспоненциального времени, если $\text{P} \neq \text{NP}$.

## Применения

- **Теоретическая основа SVM:** Структурная минимизация риска явно минимизирует VC-оценку.
- **Бустинг:** AdaBoost и PAC-анализ — объединение слабых алгоритмов с точностью чуть выше $1/2$.
- **Активное обучение:** PAC-теория направляет, когда запросы меток снижают сложность выборки.
- **Дифференциальная приватность:** DP-PAC обучение сочетает ограничения приватности с гарантиями обобщения.

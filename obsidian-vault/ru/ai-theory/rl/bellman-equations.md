Уравнения Беллмана разбивают функцию ценности на немедленную награду и дисконтированную ценность следующего состояния.

### Уравнение ожидания Беллмана
Для заданной политики $\pi$:
$V^\pi(s) = \sum_a \pi(a|s) \sum_{s', r} p(s', r | s, a) [r + \gamma V^\pi(s')]$
Это система линейных уравнений, которая имеет аналитическое решение для малых MDP.

### Уравнение оптимальности Беллмана
Для оптимальной политики $\pi^*$:
$V^*(s) = \max_a \sum_{s', r} p(s', r | s, a) [r + \gamma V^*(s')]$
Это нелинейное уравнение лежит в основе Value Iteration и Q-learning, опираясь на свойства сжимающего отображения для нахождения единственной неподвижной точки.

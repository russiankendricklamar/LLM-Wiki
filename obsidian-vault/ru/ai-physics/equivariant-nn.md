# Эквивариантные нейронные сети

## Необходимые знания

[[pinns]]

## Обзор

Эквивариантные нейронные сети встраивают ограничения симметрии непосредственно в архитектуру. Когда входные данные подвергаются преобразованию симметрии (вращению, отражению, перестановке), выход сети преобразуется соответствующим, предсказуемым образом — вместо того чтобы заново обучать симметрию на данных.

Это принципиально отличается от аугментации данных, которая учит сеть быть приблизительно инвариантной, демонстрируя много повёрнутых копий тренировочных примеров. Эквивариантность обеспечивает точную встроенную гарантию. Практический результат впечатляет: эквивариантные сети достигают наилучших результатов в задачах предсказания молекулярных свойств, структуры белков и физики элементарных частиц при значительно меньшем числе параметров и обучающих примеров.

## Математическая основа

Пусть $G$ — группа, $\rho_{\text{вх}}: G \to GL(V_{\text{вх}})$, $\rho_{\text{вых}}: G \to GL(V_{\text{вых}})$ — линейные представления. Отображение $f: V_{\text{вх}} \to V_{\text{вых}}$ называется **$G$-эквивариантным**, если:

$$f(\rho_{\text{вх}}(g) \cdot x) = \rho_{\text{вых}}(g) \cdot f(x), \quad \forall g \in G, x \in V_{\text{вх}}$$

Частный случай: когда $\rho_{\text{вых}}$ — тривиальное представление, $f$ является **$G$-инвариантным**:

$$f(\rho_{\text{вх}}(g) \cdot x) = f(x)$$

Для трёхмерного евклидова пространства релевантные группы:
- $O(3)$: ортогональная группа (вращения + отражения)
- $SO(3)$: специальная ортогональная группа (только собственные вращения)
- $E(3)$: евклидова группа $= O(3) \ltimes \mathbb{R}^3$ (вращения, отражения, переносы)
- $SE(3)$: специальная евклидова группа (вращения + переносы)

Неприводимые представления (НП) $SO(3)$ — **матрицы Вигнера** $D^{(\ell)}(R)$ для каждого угла момента $\ell = 0, 1, 2, \ldots$. Признаки, преобразующиеся как НП степени $\ell$, называются **сферическими тензорами**:
- $\ell = 0$: скаляры (инвариантны при вращении)
- $\ell = 1$: векторы (вращаются со стандартной матрицей вращения 3D)
- $\ell = 2$: симметричные бесследовые тензоры ранга 2
- $\ell \geq 3$: тензоры высших порядков

## Принцип работы

**Слои тензорного произведения.** Фундаментальный строительный блок $E(3)$-эквивариантных сетей — тензорное произведение Клебша-Гордана. Для двух признаков типов $\ell_1$ и $\ell_2$ эквивариантное произведение даёт признаки типов $|\ell_1 - \ell_2|, \ldots, \ell_1 + \ell_2$:

$$(\mathbf{u}^{(\ell_1)} \otimes_{\text{КГ}} \mathbf{v}^{(\ell_2)})_m^{(\ell)} = \sum_{m_1, m_2} C_{\ell_1 m_1 \ell_2 m_2}^{\ell m} \, u_{m_1}^{(\ell_1)} v_{m_2}^{(\ell_2)}$$

где $C_{\ell_1 m_1 \ell_2 m_2}^{\ell m}$ — коэффициенты Клебша-Гордана.

**Передача сообщений по облаку точек.** В молекулярных графах атомы — узлы с позициями $\mathbf{r}_i$. Сообщения между атомами $i$ и $j$ используют единичный вектор $\hat{\mathbf{r}}_{ij} = (\mathbf{r}_j - \mathbf{r}_i)/|\mathbf{r}_j - \mathbf{r}_i|$, проецированный на сферические гармоники $Y_m^{(\ell)}(\hat{\mathbf{r}}_{ij})$. Обновление узла:

$$\mathbf{h}_i^{\text{новый}} = \mathbf{h}_i + \sum_{j \in \mathcal{N}(i)} W(|\mathbf{r}_{ij}|) \left( \mathbf{h}_j \otimes_{\text{КГ}} Y^{(\ell)}(\hat{\mathbf{r}}_{ij}) \right)$$

**Ключевые архитектуры:**
- **SchNet (2017):** Инвариантная; использует радиальные базисные функции, без тензорных произведений.
- **SE(3)-[[transformer-architecture|Transformer]] (2020):** Эквивариантное внимание над облаком точек.
- **SEGNN (2021):** Управляемая $E(3)$-эквивариантная граф-нейронная сеть.
- **NequIP (2022):** $E(3)$-эквивариантный межатомный потенциал; точность DFT при малом числе данных.
- **MACE (2022):** Передача сообщений высшего порядка; современный уровень для молекулярной динамики.

## Реализация на Python

```python
# Минимальный E(3)-эквивариантный слой с использованием библиотеки e3nn
# pip install e3nn

import torch
from e3nn import o3
from e3nn.nn import FullyConnectedNet

# Определяем НП: 1 скаляр (l=0), 1 вектор (l=1), 1 тензор ранга 2 (l=2)
irreps_in = o3.Irreps("1x0e + 1x1o + 1x2e")   # e=чётная чётность, o=нечётная
irreps_sh = o3.Irreps.spherical_harmonics(lmax=2)
irreps_out = o3.Irreps("2x0e + 2x1o")

# Тензорное произведение: комбинирует признаки узла со сферическими гармониками рёбер
tp = o3.FullyConnectedTensorProduct(
    irreps_in, irreps_sh, irreps_out, shared_weights=False,
)

# Радиальная сеть: производит веса для тензорного произведения
radial_net = FullyConnectedNet(
    [1, 64, 64, tp.weight_numel], act=torch.nn.functional.silu,
)

def equivariant_message(h_j, r_vec):
    """h_j: признаки исходного узла, r_vec: вектор смещения (3,)"""
    r_norm = r_vec.norm(dim=-1, keepdim=True)
    r_hat = r_vec / r_norm.clamp(min=1e-8)
    sh = o3.spherical_harmonics(irreps_sh, r_hat, normalize=True)
    weights = radial_net(r_norm)
    return tp(h_j, sh, weights)


# Проверка эквивариантности
x = torch.randn(irreps_in.dim)
r = torch.randn(3)
R = o3.rand_matrix()
rot_in = irreps_in.D_from_matrix(R)
rot_out = irreps_out.D_from_matrix(R)

msg_original = equivariant_message(x, r)
msg_rotated = equivariant_message(rot_in @ x, R @ r)

error = (msg_rotated - rot_out @ msg_original).norm()
print(f"Ошибка эквивариантности: {error.item():.2e}")  # Должно быть ~1e-6
```

## Ключевые результаты и свойства

- **Эффективность обучения:** NequIP достигает точности силовых полей DFT при в 1000 раз меньшем числе данных, чем неэквивариантные аналоги (Batzner et al. 2022).
- **Точная симметрия:** Ошибка вращения равна нулю по конструкции.
- **Физическая согласованность:** Сохраняющие энергию силовые поля требуют $\mathbf{F}_i = -\nabla_{\mathbf{r}_i} E$; эквивариантные сети обеспечивают это автоматически.
- **Инсайт AlphaFold:** «Инвариантное точечное внимание» в AlphaFold2 — ключевой $SE(3)$-эквивариантный компонент.
- **Физика частиц:** Лоренц-эквивариантные сети для маркировки джетов превосходят неэквивариантные аналоги при значительно меньшем числе параметров.

## Ограничения

**Вычислительная сложность.** Тензорные произведения Клебша-Гордана стоят $O(L^6)$ при максимальном угловом моменте $L$. Высокие $\ell$ дают большую выразительность, но замедляют обучение.

**Тонкости чётности.** Различие псевдоскаляров ($\ell=0$, нечётная чётность) от скаляров требует отслеживания чётности во всей сети. Хиральность важна в фармакологии.

**Узкое место передачи сообщений.** Как и все GNN, эквивариантные сети могут страдать от чрезмерного сжатия информации.

**Некомпактные группы.** Группа Лоренца $SO(1,3)$ имеет только бесконечномерные унитарные представления; построение практических Лоренц-эквивариантных сетей требует усечения.

## Применения

- **Молекулярная динамика:** Эквивариантные силовые поля (NequIP, MACE) для симуляции биомолекул с точностью ab initio.
- **Предсказание структуры белков:** $SE(3)$-эквивариантное внимание в AlphaFold2 и RoseTTAFold.
- **Разработка лекарств:** Предсказание свойств молекул из 3D-структур.
- **Физика частиц:** Классификация джетов и реконструкция событий на детекторах LHC.
- **Предсказание свойств кристаллов:** Ширина запрещённой зоны, энергия образования.
- **Робототехника:** $SE(3)$-эквивариантные политики для манипуляций.

## Две стороны

Эквивариантные сети — пересечение двух больших тем:

- **Физика / математика:** [[group-theory]] — группы Ли, их представления, неприводимые представления, коэффициенты Клебша-Гордана. Вся тензорная арифметика, используемая в эквивариантных сетях, заимствована из теории групп симметрии, разработанной для квантовой механики и физики элементарных частиц.
- **AI:** эта статья — архитектура нейросети, в которой ограничения симметрии встроены по конструкции. Практически — тензорные произведения Клебша-Гордана как эквивариантные «ядра», сферические гармоники как направленные фильтры.

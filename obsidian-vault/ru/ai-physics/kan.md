# Сети Колмогорова-Арнольда (KAN)

## Необходимые знания

[[pinns]]

## Обзор

Сети Колмогорова-Арнольда (Kolmogorov-Arnold Networks, KAN), предложенные Лю и соавторами (2024), — архитектура нейронной сети, вдохновлённая теоремой Колмогорова-Арнольда о представлении. Вместо фиксированных нелинейных функций активации в узлах (как в MLP) KAN размещают **обучаемые функции активации на рёбрах**. Это незначительное на вид изменение порождает сети, более интерпретируемые, способные к символьной регрессии и зачастую достигающие сравнимой точности при меньшем числе параметров, чем MLP, на научных задачах.

KAN привлекли широкое внимание, поскольку предложили путь к _нейронным сетям, открывающим математические формулы_ — давней цели научного машинного обучения. Архитектура тесно связана с теорией аппроксимации функций.

## Математическая основа

**Теорема Колмогорова-Арнольда (1957).** Любую непрерывную многомерную функцию $f: [0,1]^n \to \mathbb{R}$ можно записать как:

$$f(x_1, \ldots, x_n) = \sum_{q=0}^{2n} \Phi_q\!\left(\sum_{p=1}^{n} \varphi_{q,p}(x_p)\right)$$

где $\Phi_q: \mathbb{R} \to \mathbb{R}$ и $\varphi_{q,p}: [0,1] \to \mathbb{R}$ — непрерывные одномерные функции. Это двухслойное представление с $2n+1$ внутренними узлами.

**Архитектура KAN.** Слой KAN преобразует $n$-мерный вход в $m$-мерный выход:

$$(\text{KAN-слой})_j(x_1, \ldots, x_n) = \sum_{i=1}^{n} \phi_{i,j}(x_i), \quad j = 1, \ldots, m$$

Каждая $\phi_{i,j}$ — **обучаемая одномерная функция**, параметризованная B-сплайном:

$$\phi(x) = w_b \cdot b(x) + w_s \cdot \sum_{k} c_k B_k(x)$$

где $b(x) = x / (1 + e^{-x})$ (базис SiLU), $B_k$ — базисные функции B-сплайна, $c_k, w_b, w_s$ — обучаемые параметры. Сетку сплайна можно **адаптивно уточнять** в процессе обучения.

KAN глубины $L$ компонирует несколько KAN-слоёв:

$$\text{KAN}(\mathbf{x}) = (\Phi_{L-1} \circ \cdots \circ \Phi_1 \circ \Phi_0)(\mathbf{x})$$

## Принцип работы

**Активации B-сплайна.** B-сплайн порядка $k$ с $G$ узлами сетки имеет $G+k$ базисных функций. Больший $G$ даёт более мелкое разрешение. Это аналог $h$-уточнения в методе конечных элементов.

**Расширение сетки.** Обучение начинается на грубой сетке для быстрой сходимости и постепенно уточняется («расширение сетки») для захвата мелких деталей.

**Прореживание и обрезка.** После обучения рёбра с малыми $L_1$-нормами функций активации удаляются. Остаточная разреженная сеть интерпретируема: можно прочитать, какие входы соединены с какими выходами через какую нелинейность.

**Интерфейс символьной регрессии.** После обрезки отдельные $\phi_{i,j}$ могут быть заменены символьными кандидатами ($\sin$, $\exp$, $x^2$, …) визуальной подгонкой или автоматическим поиском формул.

**Сравнение с MLP:**
| Характеристика | MLP | KAN |
|---|---|---|
| Активации | Фиксированы в узлах | Обучаемы на рёбрах |
| Интерпретируемость | Низкая | Высокая (символьная регрессия) |
| Скорость обучения | Быстрая | Медленнее |
| Индуктивный смещение | Универсальный аппроксиматор | Декомпозиция КАТ |

## Реализация на Python

```python
# pip install pykan
import torch
from kan import KAN

# KAN для символьной регрессии f(x,y) = exp(sin(pi*x) + y^2)
# Архитектура: [2, 5, 1] означает 2 входа -> 5 скрытых узлов -> 1 выход

model = KAN(width=[2, 5, 1], grid=5, k=3, seed=0)

# Генерация синтетических обучающих данных
torch.manual_seed(42)
x_train = torch.rand(1000, 2) * 2 - 1   # Равномерно на [-1, 1]^2
y_train = torch.exp(
    torch.sin(torch.pi * x_train[:, 0]) + x_train[:, 1] ** 2
).unsqueeze(1)

dataset = {
    "train_input": x_train,
    "train_label": y_train,
    "test_input": x_train[:100],
    "test_label": y_train[:100],
}

# Обучение
results = model.train(dataset, opt="LBFGS", steps=200, lamb=0.001)

# Предложение символьных кандидатов
model.suggest_symbolic(0, 0, 0)   # слой 0, из узла 0, в узел 0
model.suggest_symbolic(0, 1, 0)

# Автоматическое извлечение формулы
model.auto_symbolic()
formula = model.symbolic_formula()
print("Обнаруженная формула:", formula)

# Расширение сетки для большей точности
model = model.refine(grid=10)
results2 = model.train(dataset, opt="LBFGS", steps=100)
```

## Ключевые результаты и свойства

- **Бенчмарк символьной регрессии.** На задачах Фейнмановской символьной регрессии KAN восстанавливают известные физические формулы в нескольких случаях, где MLP выходят на плато.
- **Эффективность представления.** Для гладких научных функций KAN с $O(n)$ параметрами на слой могут соответствовать или превосходить точность MLP, требующих $O(n^2)$.
- **Композиционная структура.** KAN естественно представляют функции, разлагающиеся как композиции более простых функций — распространённый паттерн в физических законах.
- **Теорема:** KAN вида $[n, 2n+1, 1]$ может представить любую непрерывную функцию на $[0,1]^n$ (прямая реализация КАТ).
- **Экстраполяция:** Активации B-сплайна допускают продолжение за пределы обучающей области.

## Ограничения

**Скорость обучения.** KAN сейчас в 10–100 раз медленнее обучаются, чем MLP аналогичной ёмкости. Вычисление B-сплайнов не так аппаратно оптимизировано, как матричные умножения.

**Масштабируемость.** Число рёбер растёт как $\prod_\ell n_\ell$. Для высокоразмерных задач (изображения) декомпозиция КАТ непрактична.

**Выбор разрешения сетки.** Начальный размер сетки и схема расширения — дополнительные гиперпараметры.

**Хрупкость символьной регрессии.** Автоматическое присвоение символов зависит от графиков активаций, которые могут быть зашумлены для сложных функций.

**Зрелость экосистемы.** Архитектура опубликована в 2024 году; программная экосистема менее зрелая, чем MLP-стеки PyTorch.

## Применения

- **Открытие физических уравнений:** Восстановление известных физических законов из данных симуляций.
- **Символьная регрессия в финансах:** Открытие замкнутых формул ценообразования для экзотических деривативов.
- **Научная интерполяция:** Компактные интерпретируемые суррогатные модели для дорогостоящих симуляций.
- **Дифференциальные уравнения:** KAN-PINNs — интерпретируемые решения УЧП.
- **Спектральный анализ:** Открытие спектральных соотношений в физических системах.

## Две стороны KAN

KAN — точка соединения двух больших тем, каждой из которых посвящена отдельная статья:

- **Математика:** [[kolmogorov-arnold-theorem]] — теорема 1957 года о представлении многомерных функций через конечную суперпозицию одномерных, решившая 13-ю проблему Гильберта.
- **AI:** [[learnable-activations]] — обучаемые B-сплайн-активации как общий приём, выходящий за рамки KAN; связь с формирующими потоками и интерпретируемостью.

# Нейронные сети, информированные физикой (PINNs)

## Обзор

Нейронные сети, информированные физикой (Physics-Informed Neural Networks, PINNs), встраивают физические законы непосредственно в функцию потерь нейронной сети. Вместо обучения исключительно на данных PINN одновременно удовлетворяет управляющему дифференциальному уравнению в частных производных (УЧП), граничным условиям и доступным наблюдениям. Результат — безсеточный дифференцируемый решатель УЧП, хорошо обобщающийся даже при малом объёме данных.

Предложенные Райси, Пердикарисом и Карниадакисом (2019), PINNs стали одним из наиболее влиятельных пересечений глубокого обучения и научных вычислений. Особенно привлекательны там, где экспериментальные данные дороги, геометрия области сложна или требуется дифференцируемая прямая модель внутри оптимизационного цикла.

## Математическая основа

Пусть $u(\mathbf{x}, t)$ — искомое поле, управляемое оператором УЧП $\mathcal{N}$:

$$\mathcal{N}[u](\mathbf{x}, t) = 0, \quad (\mathbf{x}, t) \in \Omega \times [0, T]$$

с граничными условиями $\mathcal{B}[u] = 0$ на $\partial\Omega$ и начальным условием $u(\mathbf{x}, 0) = u_0(\mathbf{x})$.

PINN аппроксимирует $u$ нейронной сетью $u_\theta(\mathbf{x}, t)$. Полная функция потерь:

$$\mathcal{L}(\theta) = \mathcal{L}_{\text{УЧП}} + \mathcal{L}_{\text{ГУ}} + \mathcal{L}_{\text{данные}}$$

где

$$\mathcal{L}_{\text{УЧП}} = \frac{1}{N_r}\sum_{i=1}^{N_r} \left|\mathcal{N}[u_\theta](\mathbf{x}_i^r, t_i^r)\right|^2$$

$$\mathcal{L}_{\text{ГУ}} = \frac{1}{N_b}\sum_{j=1}^{N_b} \left|\mathcal{B}[u_\theta](\mathbf{x}_j^b)\right|^2$$

$$\mathcal{L}_{\text{данные}} = \frac{1}{N_d}\sum_{k=1}^{N_d} \left|u_\theta(\mathbf{x}_k^d, t_k^d) - u_k^{\text{набл}}\right|^2$$

Все невязки УЧП вычисляются через автоматическое дифференцирование по $u_\theta$ — никаких конечно-разностных шаблонов не требуется.

## Принцип работы

**Выборка коллокационных точек.** Точки невязки $\{(\mathbf{x}_i^r, t_i^r)\}$ сэмплируются случайно (или квазислучайно — латинский гиперкуб, последовательности Соболя) внутри области. Плотная выборка вблизи границ улучшает точность.

**Автоматическое дифференцирование.** Производные $u_\theta$ по пространству и времени получаются точно через обратное распространение. Для уравнения теплопроводности $u_t = \alpha u_{xx}$ вычисляются $\partial u_\theta / \partial t$ и $\partial^2 u_\theta / \partial x^2$ как операции на вычислительном графе.

**Выбор архитектуры.** Стандартный выбор — полносвязные сети с активациями tanh или sin (гладкие активации важны для производных высшего порядка). Расширенные архитектуры включают:
- Встраивания Фурье для снижения спектрального смещения
- Магистральные сети для улучшения потока градиентов
- Декомпозицию области (XPINNs) для больших областей

**Адаптивные веса потерь.** Составляющие потерь часто имеют разные масштабы. Для стабилизации используются самоадаптивные веса, балансировка через NTK или взвешивание по неопределённости.

## Реализация на Python

```python
import torch
import torch.nn as nn

# PINN для 1D уравнения теплопроводности: u_t = alpha * u_xx
# Область: x in [0,1], t in [0,1], alpha=0.01
# ГУ: u(0,t) = u(1,t) = 0;  НУ: u(x,0) = sin(pi*x)

ALPHA = 0.01

class PINN(nn.Module):
    def __init__(self, hidden=64, layers=4):
        super().__init__()
        dims = [2] + [hidden] * layers + [1]
        self.net = nn.Sequential(
            *[
                nn.Sequential(nn.Linear(dims[i], dims[i + 1]), nn.Tanh())
                for i in range(len(dims) - 2)
            ],
            nn.Linear(dims[-2], dims[-1]),
        )

    def forward(self, x, t):
        inputs = torch.cat([x, t], dim=1)
        return self.net(inputs)


def pde_residual(model, x, t):
    x = x.requires_grad_(True)
    t = t.requires_grad_(True)
    u = model(x, t)
    u_t = torch.autograd.grad(u.sum(), t, create_graph=True)[0]
    u_x = torch.autograd.grad(u.sum(), x, create_graph=True)[0]
    u_xx = torch.autograd.grad(u_x.sum(), x, create_graph=True)[0]
    return u_t - ALPHA * u_xx


model = PINN()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3)

for step in range(5000):
    optimizer.zero_grad()

    # Коллокационные точки (внутренняя область)
    xr = torch.rand(1000, 1)
    tr = torch.rand(1000, 1)
    res = pde_residual(model, xr, tr)
    loss_pde = (res ** 2).mean()

    # Граничные условия
    tb = torch.rand(200, 1)
    x0 = torch.zeros(200, 1)
    x1 = torch.ones(200, 1)
    loss_bc = (model(x0, tb) ** 2 + model(x1, tb) ** 2).mean()

    # Начальное условие
    xi = torch.rand(200, 1)
    ti = torch.zeros(200, 1)
    u_ic = model(xi, ti)
    u_true = torch.sin(torch.pi * xi)
    loss_ic = ((u_ic - u_true) ** 2).mean()

    loss = loss_pde + 10 * loss_bc + 10 * loss_ic
    loss.backward()
    optimizer.step()

    if step % 1000 == 0:
        print(f"Шаг {step}: loss={loss.item():.4e}")
```

## Ключевые результаты и свойства

- **Отсутствие сетки:** работает на нерегулярных областях без построения сетки; сложность масштабируется по числу коллокационных точек.
- **Ассимиляция данных:** слагаемое потерь по данным допускает обратные задачи — оценку неизвестных параметров УЧП по наблюдениям.
- **Точность производных:** производные вычисляются с машинной точностью через AD, в отличие от конечно-разностных схем.
- **Теория сходимости:** при достаточной ёмкости сети и плотности коллокационных точек PINNs сходятся к истинному решению (Mishra & Molinaro, 2022).
- **Тестовые задачи:** Навье-Стокс, уравнение Бюргерса, Гельмгольца, [[schrodinger-equation]], Блэк-Шоулз — все успешно решены.

## Ограничения

**Спектральное смещение.** Нейронные сети предпочтительно обучают низкочастотные компоненты. Высокочастотные решения (турбулентные течения) требуют встраиваний Фурье или многомасштабных архитектур.

**Жёсткие УЧП.** При широко разделённых временных масштабах (реакционно-диффузионные уравнения с быстрыми реакциями) стандартные PINNs не сходятся. Каузальное обучение (Wang et al. 2022) устраняет эту проблему.

**Масштабируемость.** Для больших 3D-задач PINNs существенно медленнее оптимизированных конечно-элементных решателей.

**Сложные ландшафты потерь.** Плохая обусловленность потерь PINN характерна; NTK-анализ показывает, что спектры собственных значений компонент потерь могут различаться на несколько порядков.

**Требуется известное УЧП.** Если управляющее уравнение неизвестно, нельзя построить $\mathcal{L}_{\text{УЧП}}$. Neural ODE или SINDy — альтернативы в режиме, управляемом данными.

## Применения

- **Гидродинамика:** решение несжимаемых уравнений Навье-Стокса в сложных геометриях; восстановление поля скоростей из разреженных датчиков давления.
- **Квантовая механика:** решение стационарного [[schrodinger-equation]] для потенциальных ям; задачи электронной структуры.
- **Финансы:** оценка опционов через уравнение [[black-scholes]] без метода Монте-Карло.
- **Теплопередача:** управление тепловым режимом электроники; обратные задачи определения теплопроводности.
- **Аномальная диффузия:** решение дробных УЧП типа [[fractional-fokker-planck]], где целочисленные методы затруднены.
- **Геофизика:** распространение сейсмических волн; моделирование потоков в недрах.
